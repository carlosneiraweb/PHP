DOMElement::getAttributeNS

(PHP 5, PHP 7)

DOMElement::getAttributeNS — Devuelve el valor de un atributo

Descripción

public string DOMElement::getAttributeNS ( string $namespaceURI , string $localName )

Obtiene el valor del atributo en el espacio de nombres namespaceURI con el nombre local localName para el nodo actual.

Parámetros

namespaceURI

La URI del espacio de nombres.

localName

El nombre local.

Valores devueltos

El valor del atributo, o una cadena vacía si no se ecuentra el atributo con los localName y namespaceURI dados.
