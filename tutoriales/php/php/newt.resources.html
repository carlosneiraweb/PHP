Tipos de recursos

Esta extensión utiliza dos tipos de recursos: "newt component" y "newt grid".

El tipo de recurso "newt component" es devuelto por funciones, las cuales crean widgets comunes de newt widgets (por ejemplo: newt_button())

El tipo de recurso "newt grid" es un identificador de enlace especial para componentes, devuelto por las funciones de fabricación de cuadríagulas de newt (por ejemplo: newt_create_grid())
