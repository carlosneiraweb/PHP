GearmanWorker::getErrno

(PECL gearman >= 0.5.0)

GearmanWorker::getErrno — Obtiene el código de error

Descripción

public int GearmanWorker::getErrno ( void )

Retorna el valor del código de error (errno) en caso de que se obtenga un GEARMAN_ERRNO.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Un errno válido.
