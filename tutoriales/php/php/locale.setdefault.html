Locale::setDefault

locale_set_default

(PHP 5 >= 5.3.0, PHP 7, PECL intl >= 1.0.0)

Locale::setDefault -- locale_set_default — Establecer la configuración regional predeterminada en tiempo de ejecución

Descripción

Estilo orientado a objetos

public static bool Locale::setDefault ( string $locale )

Estilo por procedimientos

bool locale_set_default ( string $locale )

Establece la configuración regional predeterminada en tiempo de ejecución a $locale. Esto cambia el valor del identificador regional global 'default_locale' de INTL. Se aceptan extensiones UAX #35.

Parámetros

locale

Es una etiqueta de lenguaje que satisface el estándar BCP 47.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.

Ejemplos

Ejemplo #1 Ejemplo de locale_set_default()

<?php
locale_set_default('de-DE');
echo locale_get_default();
?>

Ejemplo #2 Ejemplo orientado a objetos

<?php
Locale::setDefault('de-DE');
echo Locale::getDefault();
?>

El resultado del ejemplo sería:

de-DE
