fbsql_read_blob

(PHP 4 >= 4.2.0, PHP 5, PHP 7)

fbsql_read_blob — Lee un BLOB de la base de datos

Descripción

string fbsql_read_blob ( string $blob_handle [, resource $link_identifier ] )

Lee datos BLOB de la base de datos.

Si la consulta SELECT contiene columnas BLOB y/o CLOB, Frontbase los devolverá directamente cuando se recuperen los datos. Este funcionamiento por defecto puede cambiarse con fbsql_set_lob_mode() para que las funciones de recuperación devuelvan handles a los datos BLOB y CLOB. Si se recupera un handle, se deberá llamar a fbsql_read_blob() para obtener los datos BLOB de la base de datos.

Parámetros

blob_handle

Un handle BLOB, devuelto por fbsql_create_blob().

link_identifier

Un identificador de enlace FrontBase devuelto por fbsql_connect() o fbsql_pconnect().

Es opcional y si no se especifica, la función intentará encontrar un enlace abierto al servidor FrontBase y de no encontrarse ningún enlace abierto intentará crear uno como si fbsql_connect() fuese invocada sin parámetros.

Valores devueltos

Devuelve una cadena conteniendo los datos BLOB especificados.

Ejemplos

Ejemplo #1 Ejemplo de fbsql_read_blob()

<?php
$link = fbsql_pconnect("localhost", "_SYSTEM", "secret")
    or die("No se pudo conectar");
$sql = "SELECT BLOB_COLUMN FROM BLOB_TABLE;";
$rs = fbsql_query($sql, $link);
$row_data = fbsql_fetch_row($rs);
// $row_data[0] contendrá ahora los datos BLOB de la primera fila
fbsql_free_result($rs);

$rs = fbsql_query($sql, $link);
fbsql_set_lob_mode($rs, FBSQL_LOB_HANDLE);
$row_data = fbsql_fetch_row($rs);
// $row_data[0] contiene ahora un handle a los datos BLOB de la primera fila
$blob_data = fbsql_read_blob($row_data[0]);
fbsql_free_result($rs);

?>
