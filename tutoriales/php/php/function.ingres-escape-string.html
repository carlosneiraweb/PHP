ingres_escape_string

(PECL ingres >= 2.1.0)

ingres_escape_string — Escapa caracteres especiales para usuarlos en una consulta

Descripción

string ingres_escape_string ( resource $link , string $source_string )

ingres_escape_string() se usa para escapar ciertos caracteres de un string antes de eviarlo al sevidor de la base de datos.

Parámetros

link

El identificador de enlace de la conexión

source_string

El string origen a analizar

Valores devueltos

Deveulve un string que contiene los datos escapados

Ejemplos

Ejemplo #1 Escape special characters for use in a query

<?php
$enlace = ingres_connect($base_datos, $usuario, $contraseña);

$apellido = "O'Connor";

$sql = sprintf("select * from user_profile where up_last = '%s'", ingres_escape_string( $enlace, $apellido));

$resultado = ingres_query($enlace, $sql);

while ($usuario = ingres_fetch_object($resultado))
{
  echo $usuario->up_first . '<BR/>';
}

ingres_commit($enlace);

ingres_close($enlace);

?>
