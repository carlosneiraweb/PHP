setrawcookie

(PHP 5, PHP 7)

setrawcookie — Enviar una cookie sin codificar su valor

Descripción

bool setrawcookie ( string $name [, string $value [, int $expire = 0 [, string $path [, string $domain [, bool $secure = false [, bool $httponly = false ]]]]]] )

setrawcookie() es exactamente el mismo que setcookie() excepto que el valor de la cookie no será automáticamente codificado cuando se envía al navegador.

Parámetros

Para información sobre los parámetros, consulte la función setcookie().

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.

Historial de cambios

Versión Descripción
5.5.0 Ahora se incluye un atribugo Max-Age en la cabecera Set-Cookie enviada al cliente.
5.2.0 Añadido el parámetro httponly.
