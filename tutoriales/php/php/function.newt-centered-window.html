newt_centered_window

(PECL newt >= 0.1)

newt_centered_window — Abrir una ventana centrada de un tamaño específico

Descripción

int newt_centered_window ( int $width , int $height [, string $title ] )

Abre una ventana centrada de un tamaño específico.

Parámetros

width

Ancho de la ventana

height

Alto de la vetana

title

Título de la ventana

Valores devueltos

Valor no definido.

Ver también

  • newt_pop_window() - Elimina la ventana superior de la exposición
  • newt_open_window() - Abre una ventana con el tamaño y posición especificados
