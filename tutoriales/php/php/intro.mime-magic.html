Advertencia

Esta extensión está obsoleta debido a que la extensión de PECL Fileinfo proporciona la misma funcionalidad (y más) de una manera mucho más limpia.

Las funciones de este módulo intentan adivinar el tipo de contenido y la codificación de un fichero buscando ciertas secuencias de bytes mágicas en posiciones específicas dentro del mismo. Aunque no es una aproximación a prueba de balas, la heurística usada realiza un buen trabajo.

Esta extensión está derivada del módulo mod_mime_magic de Apache, que está basado a su vez en el comando file mantenido por Ian F. Darwin. Vea el código fuente para más información sobre el histórico y el copyright.
