hebrevc

(PHP 4, PHP 5, PHP 7)

hebrevc — Convertir texto de hebreo lógico a texto visual con conversión de línea nueva

Descripción

string hebrevc ( string $hebrew_text [, int $max_chars_per_line = 0 ] )

Esta función es parecida a hebrev() con la diferencia que esta convierte líneas nuevas (\n) a "<br>\n".

Esta función intenta evitar la separación de palabras.

Parámetros

hebrew_text

Una cadena de entrada en hebreo.

max_chars_per_line

Este parámetro es opcional, e indica el número máximo de caracteres por línea que serán devueltos.

Valores devueltos

Devuelve la cadena visual.
