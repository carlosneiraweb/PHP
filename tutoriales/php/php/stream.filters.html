Filtros de Flujos

Un filtro es una pieza final de código que puede realizar operaciones sobre información que está siendo leída o escrita en un flujo. Se puede apilar cualquier número de filtros en un flujo. Los filtros personalizados se pueden definir en un script de PHP usando stream_filter_register() o en una extensión usando la Referencia de la API en Trabajar con flujos. Para acceder a la lista de los filtros actualmente registrados, use stream_get_filters().
