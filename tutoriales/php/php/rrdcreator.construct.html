RRDCreator::__construct

(PECL rrd >= 0.9.0)

RRDCreator::__construct — Crea una nueva instancia RRDCreator

Descripción

public RRDCreator::__construct ( string $path [, string $startTime [, int $step = 0 ]] )

Crea una nueva instancia RRDCreator.

Parámetros

path

Ruta de acceso para el archivo de base de datos recién creada RRD.

startTime

Tiempo para el primer valor en la base de datos RRD.Parámetro soporta todos los formatos que son soportados por la llamada a rrd create.

intstep

Base de intervalo en segundos con que los datos se introducen en la base de datos RRD.

Valores devueltos

No devuelve ningún valor.
