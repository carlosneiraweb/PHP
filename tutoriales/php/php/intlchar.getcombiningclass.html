IntlChar::getCombiningClass

(PHP 7)

IntlChar::getCombiningClass — Obtener la clase de combinación de un punto de cógido

Descripción

public static int IntlChar::getCombiningClass ( mixed $codepoint )

Devuelve la clase de combinación de un punto de cógido.

Parámetros

codepoint

El valor de tipo integer del punto de código (p.ej. 0x2603 para U+2603 SNOWMAN), o el carácter codificado como un string UTF-8 (p.ej. "\u{2603}")

Valores devueltos

Devuelve la clase de combinación del carácter.

Ejemplos

Ejemplo #1 Probar diferentes puntos de código

<?php
var_dump(IntlChar::getCombiningClass("A"));
var_dump(IntlChar::getCombiningClass("\u{0334}"));
var_dump(IntlChar::getCombiningClass("\u{0358}"));
?>

El resultado del ejemplo sería:

int(0)
int(1)
int(232)
