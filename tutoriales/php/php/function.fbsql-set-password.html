fbsql_set_password

(PHP 5, PHP 7)

fbsql_set_password — Cambia la contraseña para el usuario especificado

Descripción

bool fbsql_set_password ( resource $link_identifier , string $user , string $password , string $old_password )

Cambia la contraseña para el user especificado.

Parámetros

link_identifier

Un identificador de enlace FrontBase devuelto por fbsql_connect() o fbsql_pconnect().

Es opcional y si no se especifica, la función intentará encontrar un enlace abierto al servidor FrontBase y de no encontrarse ningún enlace abierto intentará crear uno como si fbsql_connect() fuese invocada sin parámetros.

user

El nombre de usuario.

password

La nueva contraseña a asignar.

old_password

La antigua contraseña a sustituir.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.
