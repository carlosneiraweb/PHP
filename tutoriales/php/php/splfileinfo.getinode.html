SplFileInfo::getInode

(PHP 5 >= 5.1.2, PHP 7)

SplFileInfo::getInode — Obtiene el i-nodo de el fichero

Descripción

public int SplFileInfo::getInode ( void )

Obtiene el número de i-nodo de el objeto filesystem.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve el número de i-nodo de el objeto filesystem.

Errores/Excepciones

Lanza una RuntimeException en caso de error.
