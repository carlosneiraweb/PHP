fbsql_fetch_assoc

(PHP 4 >= 4.0.6, PHP 5, PHP 7)

fbsql_fetch_assoc — Obtiene una fila de resultados como array asociativo

Descripción

array fbsql_fetch_assoc ( resource $result )

Llamar a fbsql_fetch_assoc() es equivalente a hacerlo a fbsql_fetch_array() con FBSQL_ASSOC como segundo parámetro. Sólo devuelve un array asociativo.

fbsql_fetch_array() funcionaba originalmente así. Si se necesitan índices numéricos además de los asociativos, , se puede usar fbsql_fetch_array().

Es importante saber que usar fbsql_fetch_assoc() NO es significativamente más lento que usar fbsql_fetch_row(), y además proporciona un reseñable valor añadido.

Parámetros

result

Un identificador de resultado devuelto por fbsql_query() o fbsql_db_query().

Valores devueltos

Devuelve un array que corresponde a la fila obtenida, o FALSE si no quedan más filas.

Si hay dos o más columnas del resultado con los mismos nombres de campo, la última columna tendrá prioridad. Para acceder a la otra(s) columna(s) con el mismo nombre será necesario utilizar fbsql_fetch_array() y hacer que devuelva también los índices numéricos.

Ejemplos

Ejemplo #1 fbsql_fetch_assoc() example

<?php
fbsql_connect($host, $user, $password);
$result = fbsql_db_query("database", "select * from table");
while ($row = fbsql_fetch_assoc($result)) {
    echo $row["user_id"];
    echo $row["fullname"];
}
fbsql_free_result($result);
?>
