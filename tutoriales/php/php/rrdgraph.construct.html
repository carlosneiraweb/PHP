RRDGraph::__construct

(PECL rrd >= 0.9.0)

RRDGraph::__construct — Crea una nueva instancia RRDGraph

Descripción

public RRDGraph::__construct ( string $path )

Crea una nueva instancia RRDGraph. Esta instancia es la encargada de mostrar el resultado de la consulta de base de datos RRD en la imagen.

Parámetros

path

Ruta completa de la imagen recién creada.

Valores devueltos

No devuelve ningún valor.
