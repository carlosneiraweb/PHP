ifx_num_rows

(PHP 4, PHP <=5.2.0)

ifx_num_rows — Contar las filas que ya han sido devueltas por una consulta

Descripción

int ifx_num_rows ( resource $result_id )

Devuelve el número de filas extraidas por una consulta a través del identificador result_id después de haber llamado a ifx_query() o ifx_do().

Parámetros

result_id

result_id es el identificador válido devuelto por ifx_query() o ifx_prepare() (sólo para consultas "SELECT").

Valores devueltos

Devuelve el número de filas extraidas o FALSE en caso de error.
