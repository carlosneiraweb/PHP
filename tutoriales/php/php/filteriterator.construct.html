FilterIterator::__construct

(PHP 5 >= 5.1.0, PHP 7)

FilterIterator::__construct — Construye un filterIterator

Descripción

public FilterIterator::__construct ( Iterator $iterator )

Construir un nuevo FilterIterator, que consiste en pasar un iterator y aplicar varios filtros a este.

Advertencia

Esta función no está documentada actualmente, solamente se encuentra disponible la lista de parámetros.

Parámetros

iterator

El iterador que va a ser filtrado.

Valores devueltos

The FilterIterator.
