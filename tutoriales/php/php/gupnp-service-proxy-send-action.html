gupnp_service_proxy_send_action

(PECL gupnp >= 0.2.0)

gupnp_service_proxy_send_action — Envía la acción con múltiples parámetros de forma sincronizada

Descripción

array gupnp_service_proxy_send_action ( resource $proxy , string $action , array $in_params , array $out_params )

Envía la acción con los parámetros in_params a través de un servicio proxy de forma sincronizada y devuelve out_params con valores o FALSE en caso de error.

Parámetros

proxy

Un identificador de servicio proxy.

action

Una acción.

in_params

Un array de parámetros de entrada. Cada entrada en in_params se supone que contenga un array con nombre, tipo y valor de los parámetros.

out_params

Un array de parámetros de salida. Cada entrada en out_params se supone que contenga un array con nombre, tipo y valor de los parámetros.

Valores devueltos

Devuelve out_params un array con los valores o FALSE en caso de error.

Errores/Excepciones

Lanza un E_WARNING si in_params y out_params no están correctamente definidos o no es posible enviar la acción.

  • gupnp_service_proxy_action_set() - Envía la acción al servicio y establece el valor
