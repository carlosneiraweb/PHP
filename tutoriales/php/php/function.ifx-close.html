ifx_close

(PHP 4, PHP <=5.2.0)

ifx_close — Cerrar conexión Informix

Descripción

bool ifx_close ([ resource $link_identifier ] )

ifx_close() cierra el enlace a una base de datos Informix que está asociado al identificador de enlace especificado.

Tenga en cuenta que esto no suele ser necesario, los enlaces abiertos que no son permanentes son cerrados automáticamente al finalizar el script.

ifx_close() no cerrará enlaces permanentes generados por ifx_pconnect().

Parámetros

link_identifier

El identificador de enlace. Si no se especifica, el último enlace abierto es asumido.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.

Ejemplos

Ejemplo #1 Closing a Informix connection

<?php
$conn_id = ifx_connect ("mydb@ol_srv", "itsme", "mypassword");
/* ... Algunas consultas y cosas ... */
ifx_close($conn_id);
?>
