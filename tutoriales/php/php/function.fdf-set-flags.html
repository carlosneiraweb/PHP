fdf_set_flags

(PHP 4 >= 4.0.2, PHP 5, PHP 7)

fdf_set_flags — Establecer banderas de un campo

Descripción

bool fdf_set_flags ( resource $fdf_document , string $fieldname , int $whichFlags , int $newFlags )

Establece determinadas banderas al campo proporcionado.

Parámetros

fdf_document

Manejador de documentos FDF, devuelto por fdf_create(), fdf_open() o por fdf_open_string().

fieldname

Nombre del campo FDF, en forma de string.

whichFlags

newFlags

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.
