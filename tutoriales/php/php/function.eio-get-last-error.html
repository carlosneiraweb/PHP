eio_get_last_error

(PECL eio >= 1.0.0)

eio_get_last_error — Devuelve un string que describe el último error asociado con un recurso de petición

Descripción

string eio_get_last_error ( resource $req )

eio_get_last_error() devuelve un string que describe el último error asociado con req.

Parámetros

req

El recurso de petición

Valores devueltos

eio_get_last_error() devuelve un string que describe el último error asociado con el recurso de petición dado por req.

Advertencia

Esta función ha sido declarada EXPERIMENTAL. Su funcionamiento, nombre y la documentación que le acompaña puede cambiar sin previo aviso en futuras versiones de PHP. Utilícela bajo su propia responsabilidad.
