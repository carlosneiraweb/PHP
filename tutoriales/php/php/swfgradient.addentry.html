SWFGradient->addEntry

(PHP 5 <= 5.3.0, PECL ming SVN)

SWFGradient->addEntry — Añade una entrada a la lista de gradientes

Descripción

void SWFGradient::addEntry ( float $ratio , int $red , int $green , int $blue [, int $alpha = 255 ] )
Advertencia

Esta función ha sido declarada EXPERIMENTAL. Su funcionamiento, nombre y la documentación que le acompaña puede cambiar sin previo aviso en futuras versiones de PHP. Utilícela bajo su propia responsabilidad.

swfgradient::addentry() añade una entrada a la lista de gradientes. ratio es un número entre 0 y 1 indicando dónde aparece este color en el gradiente. Añadirás entradas en orden de ratio creciente.

red, green, blue es un color (modo RGB).

Valores devueltos

No devuelve ningún valor.
