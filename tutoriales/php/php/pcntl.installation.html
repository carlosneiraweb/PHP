Instalación

Soporte de Control de procesos en PHP no está habilitado de forma predeterminada. Tiene que compilar la versión CGI o CLI de PHP con la opción de configuración --enable-pcntl al compilar PHP para habilitar el soporte de Control de procesos.

Nota:

Actualmente, este módulo no funciona en las plataformas no Unix (Windows).
