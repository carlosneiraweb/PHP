sqlite_seek

SQLiteResult::seek

(PHP 5 < 5.4.0, PECL sqlite >= 1.0.0)

sqlite_seek -- SQLiteResult::seek — Busca un número de fila en particular en un conjunto de resultados almacenado en buffer

Descripción

bool sqlite_seek ( resource $result , int $rownum )

Estilo orientado a objetos (method):

bool SQLiteResult::seek ( int $rownum )

sqlite_seek() busca la fila dada por el parámetro rownum.

Parámetros

result

El recurso de resultados de SQLite. Este parámetro no es necesario cuando se usa el método orientado a objetos.

Nota:

Esta función no puede ser utilizada con manejos de resultados no pasados por buffer.

rownum

El número ordinal de fila a buscar. El número de fila está basado en cero (0 es la primera fila).

Nota:

Esta función no puede ser utilizada con manejos de resultados no pasados por buffer.

Valores devueltos

Devuelve FALSE si la fila no existe, TRUE en caso contrario.

Ver también

  • sqlite_next() - Busca el número de fila siguiente
  • sqlite_current() - Obtiene la fila actual como un array de un conjunto de resultado
  • sqlite_rewind() - Busca el primer número de fila
