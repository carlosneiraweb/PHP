La clase SolrDocumentField

(PECL solr >= 0.9.2)

Introducción

Representa un campo de un documento Solr. Todas sus propiedades son de sólo lectura.

public __construct ( void )

}

Propiedades

name

El nombre del campo.

boost

El valor de boost del campo

values

Una matriz de valores para este campo
