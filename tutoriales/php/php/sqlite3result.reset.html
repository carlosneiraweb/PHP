SQLite3Result::reset

(PHP 5 >= 5.3.0, PHP 7)

SQLite3Result::reset — Reiniciar el conjunto de resultados a la primera fila

Descripción

public bool SQLite3Result::reset ( void )

Reinicia el conjunto de resultados a la primera fila.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve TRUE si el conjunto de resultados es reiniciado con éxito a la primera fila, FALSE en caso de error.
