SWFText::getWidth

(PHP 5 <= 5.3.0, PECL ming SVN)

SWFText::getWidth — Computa el ancho de una cadena

Descripción

float SWFText::getWidth ( string $string )
Advertencia

Esta función ha sido declarada EXPERIMENTAL. Su funcionamiento, nombre y la documentación que le acompaña puede cambiar sin previo aviso en futuras versiones de PHP. Utilícela bajo su propia responsabilidad.

Devuelve el ancho renderizado de la cadena string en la configuración de fuente, escala y espaciado actuales del objeto de texto.
