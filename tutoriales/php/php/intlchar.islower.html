IntlChar::islower

(PHP 7)

IntlChar::islower — Comprobar si un punto de código es una letra minúscula

Descripción

public static bool IntlChar::islower ( mixed $codepoint )

Determina si el punto de código especificado tiene la categoría general "Ll" (letra minúscula).

Nota:

Este método erra con algunos caracteres que tambíen son minúscula pero tienen un valor diferente de categoría general. Para incluirlos, use IntlChar::isULowercase().

Parámetros

codepoint

El valor de tipo integer del punto de código (p.ej. 0x2603 para U+2603 SNOWMAN), o el carácter codificado como un string UTF-8 (p.ej. "\u{2603}")

Valores devueltos

Devuelve TRUE si codepoint es una letra minúscula de Ll, FALSE si no.

Ejemplos

Ejemplo #1 Probar diferentes puntos de código

<?php
var_dump(IntlChar::islower("A"));
var_dump(IntlChar::islower("a"));
var_dump(IntlChar::islower("Φ"));
var_dump(IntlChar::islower("φ"));
var_dump(IntlChar::islower("1"));
?>

El resultado del ejemplo sería:

bool(false)
bool(true)
bool(false)
bool(true)
bool(false)
