fann_get_connection_rate

(PECL fann >= 1.0.0)

fann_get_connection_rate — Obtener el índice de conexión empleado al crear la red

Descripción

float fann_get_connection_rate ( resource $ann )

Obtener el índice de conexión empleado al crear la red.

Parámetros

ann

Un resource de red neuronal.

Valores devueltos

El índice de conexión empleado al crear la red, o FALSE en caso de error.
