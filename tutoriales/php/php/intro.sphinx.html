Introducción

Esta extensión proporciona enlaces para la biblioteca cliente de Sphinx search. Sphinx es un motor de búsqueda independiente de la intención de proporcionar rápidamente un tamaño eficiente y adecuado para las funciones de búsqueda de texto completo a otras aplicaciones.
