Introducción

El módulo dbx es una capa de abstracción de base de datos (db 'X', donde 'X' es una base de datos soportada). Las funciones de dbx permiten acceder a todas las bases de datos soportadas usando una simple convención de llamada. La funciones de dbx por sí mismas no son la interfaz directa de las bases de datos, sino que son la interfaz de los módulos que se usan para soportar estas bases de datos.

Nota:

Esta extensión se ha movido al repositorio de » PECL y ya no se distribuye con PHP a partir de PHP 5.1.0.
