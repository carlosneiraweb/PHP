fbsql_create_db

(PHP 4 >= 4.0.6, PHP 5, PHP 7)

fbsql_create_db — Crea una base de datos FrontBase

Descripción

bool fbsql_create_db ( string $database_name [, resource $link_identifier [, string $database_options ]] )

Intenta crear una nueva base de datos en el servidor especificado.

Parámetros

database_name

El nombre de la base de datos.

link_identifier

Un identificador de enlace FrontBase devuelto por fbsql_connect() o fbsql_pconnect().

Es opcional y si no se especifica, la función intentará encontrar un enlace abierto al servidor FrontBase y de no encontrarse ningún enlace abierto intentará crear uno como si fbsql_connect() fuese invocada sin parámetros.

database_options

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.

Ejemplos

Ejemplo #1 Ejemplo de fbsql_create_db().

<?php
$link = fbsql_pconnect("localhost", "_SYSTEM", "secret")
    or die("No se puede conectar");
if (fbsql_create_db("my_db")) {
    echo "Base de datos creada correctamente\n";
} else {
    printf("Error creando base de datos: %s\n", fbsql_error());
}
?>
