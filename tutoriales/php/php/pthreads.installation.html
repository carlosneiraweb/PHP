Instalación

Las versiones de pthreads están alojadas por PECL, y el código fuente por » github; la ruta más sencilla para instalarla es la ruta normal de PECL: » http://pecl.php.net/package/pthreads.

Los usuarios de Windows pueden descargar los binarios de una versión preconstruida desde el sitio web de » PECL.

Precaución

Los usuarios de Windows necesitan realizar un paso más para añadir pthreadVC2.dll (distribuido con las versiones de Windows) a su PATH.
