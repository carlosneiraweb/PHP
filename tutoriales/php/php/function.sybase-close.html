sybase_close

(PHP 4, PHP 5, PHP 7)

sybase_close — Cierra una conexión Sybase

Descripción

bool sybase_close ([ resource $link_identifier ] )

sybase_close() cierra la conexión a una base de datos Sybase que se ha asociado al identificador de conexión link_identifier.

Notar que esto no es normalmente necesario, puesto que las conexiones no persistentes se cierran automáticamente al final de la ejecución del script.

sybase_close() no cerrará las conexiones persistentes generadas por sybase_pconnect().

Parámetros

link_identifier

Si no se especifica el identificador de conexión, se asumirá que es el correspondiente a la última conexión abierta.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.

Ver también

  • sybase_connect() - Establece una conexión a un servidor Sybase
  • sybase_pconnect() - Abre una conexión persistente a un servidor Sybase
