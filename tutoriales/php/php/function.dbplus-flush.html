dbplus_flush

(PHP 4 <= 4.1.0, PECL dbplus >= 0.9)

dbplus_flush — Vuelca todos los cambios hechos en una relación

Descripción

int dbplus_flush ( resource $relation )

Escribe todos los cambios aplicados a relation desde el último vuelco a disco.

Parámetros

relation

Una relación abierta con dbplus_open().

Valores devueltos

Devuelve DBPLUS_ERR_NOERR si se tuvo éxito o un código de error de db++ si falló.

Notas

Advertencia

Esta función ha sido declarada EXPERIMENTAL. Su funcionamiento, nombre y la documentación que le acompaña puede cambiar sin previo aviso en futuras versiones de PHP. Utilícela bajo su propia responsabilidad.
