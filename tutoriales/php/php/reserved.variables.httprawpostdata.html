$HTTP_RAW_POST_DATA

(PHP 4, PHP 5)

$HTTP_RAW_POST_DATA — Datos POST sin tratar

Descripción

Advertencia

Esta característica fue declarada OBSOLETA en PHP 5.6.0 y ELILMINADA a partir de PHP 7.0.0.

$HTTP_RAW_POST_DATA contiene los datos POST sin tratar. Véase always_populate_raw_post_data.

En general, se debería utilizar php://input en lugar de $HTTP_RAW_POST_DATA.
