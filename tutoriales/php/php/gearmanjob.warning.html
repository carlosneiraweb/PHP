GearmanJob::warning

(PECL gearman <= 0.5.0)

GearmanJob::warning — Envía un aviso (obsoleto)

Descripción

public bool GearmanJob::warning ( string $warning )

Envía un aviso para el trabajo mientras está en ejecución.

Nota:

Este método ha sido reemplazado por GearmanJob::sendWarning() en la versión 0.6.0 de la extensión Gearman.

Parámetros

warning

Un mensaje de aviso.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.

Ver también

  • GearmanJob::sendComplete() - Envía el resultado y el estado completado
  • GearmanJob::sendException() - Envía una excepción para un trabajo en ejecución
  • GearmanJob::sendFail() - Envía el estado de fallo
