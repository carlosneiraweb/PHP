GmagickPixel::setcolor

(PECL gmagick >= Unknown)

GmagickPixel::setcolor — Establece el color

Descripción

public GmagickPixel GmagickPixel::setcolor ( string $color )

Establece el color descrito por el objeto GmagickPixel, con una cadena (p.ej. "blue", "#0000ff", "rgb(0,0,255)", "cmyk(100,100,100,10)", etc.).

Parámetros

color

La definición de color a usar para inicializar el objeto GmagickPixel.

Valores devueltos

El objeto GmagickPixel si se tuvo éxito.
