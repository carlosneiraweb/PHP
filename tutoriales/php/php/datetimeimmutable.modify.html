DateTimeImmutable::modify

(PHP 5 >= 5.5.0, PHP 7)

DateTimeImmutable::modify — Crea un nuevo objeto con la marca de tiempo modificada

Descripción

public DateTimeImmutable DateTimeImmutable::modify ( string $modify )

Crea un nuevo objeto DateTimeImmutable con la marca de tiempo modificada. El objeto original no se modifica.

Parámetros

object

Solamente para el estilo por procedimientos: Un objeto DateTime devuelto por date_create(). La función modifica este objeto.

modify

Una cadena de fecha/hora. Los formatos válidos se explican en Formatos de fecha y hora.

Valores devueltos

Devuelve el objeto recién creado o FALSE en caso de error.
