ImagickDraw::getStrokeLineJoin

(PECL imagick 2.0.0)

ImagickDraw::getStrokeLineJoin — Devuelve la forma que se va a usar en la esquinas de los trazados cuando están contorneados

Descripción

int ImagickDraw::getStrokeLineJoin ( void )
Advertencia

Esta función no está documentada actualmente, solamente se encuentra disponible la lista de parámetros.

Devuelve la forma que se va a usar en las esquinas de los trazados (u otras formas vectoriales) cuando están contorneados.

Valores devueltos

Devuelve una de las constantes LINEJOIN_ o 0 si la junta de líneas de contorno no está establecida.
