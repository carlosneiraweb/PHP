ifx_get_char

(PHP 4, PHP <=5.2.0)

ifx_get_char — Devolver el contenido del objeto char

Descripción

string ifx_get_char ( int $bid )

Devuelve el contenido del objeto char.

Parámetros

bid

El id del objeto char.

Valores devueltos

Devuelve el contenido como un string, o FALSE en caso de error.
