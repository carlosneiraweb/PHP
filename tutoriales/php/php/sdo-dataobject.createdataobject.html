SDO_DataObject::createDataObject

(^)

SDO_DataObject::createDataObject — Crear un SDO_DataObject hijo

Descripción

SDO_DataObject SDO_DataObject::createDataObject ( mixed $identifier )
Advertencia

Esta función ha sido declarada EXPERIMENTAL. Su funcionamiento, nombre y la documentación que le acompaña puede cambiar sin previo aviso en futuras versiones de PHP. Utilícela bajo su propia responsabilidad.

Crea un SDO_DataObject hijo del tipo predeterminado para la propiedad identificada. El objeto de datos es insertado automáticamente en el árbol y se devuelve una referencia a él.

Parámetros

identifier

Identifica la propiedad para el tipo de objeto de datos a ser creado. Puede ser un nombre de propiedad (string), un índice de propiedad (int), o un SDO_Model_Property.

Valores devueltos

Devuelve el recién creado SDO_DataObject.

Errores/Excepciones

SDO_PropertyNotFoundException

Es lanzada si el identificador no se corresponde con una propiedad del objeto de datos.
