imagecolorstotal

(PHP 4, PHP 5, PHP 7)

imagecolorstotal — Averiguar el número de colores de la paleta de una imagen

Descripción

int imagecolorstotal ( resource $image )

Devuelve el número de colores de la paleta de una imagen.

Parámetros

image

Un recurso de imagen, devuelto por una de las funciones de creación de imágenes, como imagecreatefromgif().

Valores devueltos

Devuelve el número de colores de la paleta de la imagen especificada o 0 para imágenes de color verdadero.

Ejemplos

Ejemplo #1 Obtener el número total de colores de una imagen usando imagecolorstotal()

<?php
// Crear una instancia de una imagen
$im = imagecreatefromgif('php.gif');

echo 'Total de colores en la imagen: ' . imagecolorstotal($im);

// Liberar memoria
imagedestroy($im);
?>

El resultado del ejemplo sería algo similar a:

Total colors in image: 128
