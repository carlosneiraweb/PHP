ifxus_close_slob

(PHP 4, PHP <=5.2.0)

ifxus_close_slob — Cerrar un objeto SLOB

Descripción

bool ifxus_close_slob ( int $bid )

Cierra el objecto SLOB asociado con el identificador SLOB bid.

Parámetros

bid

Un identificador SLOB valido.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.
