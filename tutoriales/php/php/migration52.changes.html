Qué ha cambiado en PHP 5.2.x

La mayoría de las mejoras en PHP 5.2.x no tienen impacto en códigos existentes. Existen algunas incompatibilidades y nuevos mensajes de error que deberían ser considerados, y el código debería ser probado antes de hacer el cambio de versión de PHP en ambientes de producción.

Si el sistema está siendo actualizado desde PHP 5.0.x, también debería leer la sección del manual Migración de PHP 5.0.x a PHP 5.1.x.

De la misma forma, si el sistema está siendo actualizado desde PHP 4, debería leer la sección del manual Migración desde PHP 4 a PHP 5.0.x.
