SDO_Model_Type::getProperties

(^)

SDO_Model_Type::getProperties — Obtener los objetos SDO_Model_Property definidos en el tipo

Descripción

array SDO_Model_Type::getProperties ( void )
Advertencia

Esta función ha sido declarada EXPERIMENTAL. Su funcionamiento, nombre y la documentación que le acompaña puede cambiar sin previo aviso en futuras versiones de PHP. Utilícela bajo su propia responsabilidad.

Obtiene un array de objetos SDO_Model_Property que describe las propiedades definidas en el SDO_Model_Type. Cada SDO_Model_Property contiene información tal como el nombre de propiedad, el valor predeterminado, etc.

Parámetros

Ninguno.

Valores devueltos

Devuelve un array de objetos SDO_Model_Property.
