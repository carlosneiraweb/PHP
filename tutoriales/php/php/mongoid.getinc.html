MongoId::getInc

(PECL mongo >= 1.0.11)

MongoId::getInc — Obtiene el valor incremental usado para crear este id

Esta extenisón que define este método está obsoleta. En su lugar debe usarse la extensión MongoDB. No hay equivalente para este método en la nueva extensión.

Descripción

public int MongoId::getInc ( void )

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve el valor incremental usado para crear este MongoId.
