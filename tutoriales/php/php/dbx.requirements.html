Requerimientos

Para ser capaz de utilizar una base de datos con el módulo dbx, el módulo debe ser enlazado o cargado dentro de PHP, y el módulo de la base de datos debe ser soportado por el módulo dbx. Actualmente están soportadas las siguientes bases de datos, aunque otras le seguirán:

  • FrontBase (disponible desde PHP 4.1.0).
  • Microsoft SQL Server
  • MySQL
  • ODBC
  • PostgreSQL
  • Sybase-CT (disponible desde PHP 4.2.0).
  • Oracle (oci8) (disponible desde PHP 4.3.0).
  • SQLite (PHP 5).

La documentación para añadir soporte de una base de datos adicional a dbx se puede encontrar en » http://www.guidance.nl/php/dbx/doc/.
