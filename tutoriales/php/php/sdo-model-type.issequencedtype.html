SDO_Model_Type::isSequencedType — Comprobar si este tipo es un tipo secuenciado

Descripción

bool SDO_Model_Type::isSequencedType ( void )
Advertencia

Esta función ha sido declarada EXPERIMENTAL. Su funcionamiento, nombre y la documentación que le acompaña puede cambiar sin previo aviso en futuras versiones de PHP. Utilícela bajo su propia responsabilidad.

Comprueba si este tipo es un tipo secuenciado. Devuelve TRUE si este tipo e secuenciado, si no, devuelve FALSE. Los tipos secuenciado pueden tener el orden de las propiedades preservado y pueden contener texto no estructurado. Para más información sobre tipos secuenciados véase la sección sobre Trabajar con Objetos de Datos Secuenciados.

Parámetros

Ninguno.

Valores devueltos

Devuelve TRUE si este tipo es secuenciado, si no, devuelve FALSE.
