fdf_header

(PHP 4 >= 4.3.0, PHP 5, PHP 7)

fdf_header — Establecer cabeceras de salida específicas de FDF

Descripción

void fdf_header ( void )

Es una función apropiada para establecer cabeceras HTTP para salidas de FDF. Establece el Content-type: a application/vnd.fdf.

Valores devueltos

No devuelve ningún valor.
