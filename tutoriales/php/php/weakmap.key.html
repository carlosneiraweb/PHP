WeakMap::key

(PECL weakref >= 0.2.0)

WeakMap::key — Devuelve la clave actual de la iteración

Descripción

public object WeakMap::key ( void )

Devuelve el objeot que sirve como clave del mapa, en la posición actual de la iteración.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

La clave del objeto que está siendo iterado.
