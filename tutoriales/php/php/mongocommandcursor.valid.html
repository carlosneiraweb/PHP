MongoCommandCursor::valid

(PECL mongo >=1.5.0)

MongoCommandCursor::valid — Comprueba si el cursor está leyendo un resultado válido

Descripción

public bool MongoCommandCursor::valid ( void )

Parámetros

Esta función no tiene parámetros.

Valores devueltos

TRUE si el resultado actual no es nulo, y FALSE en caso contrario.
