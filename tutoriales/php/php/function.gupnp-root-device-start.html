gupnp_root_device_start

(PECL gupnp >= 0.1.0)

gupnp_root_device_start — Inicio el bucle principal

Descripción

bool gupnp_root_device_start ( resource $root_device )

Inicia el bucle principal en el servidor raíz.

Parámetros

root_device

Un identificadorde servicio raíz, devuelto por gupnp_root_device_new().

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.
