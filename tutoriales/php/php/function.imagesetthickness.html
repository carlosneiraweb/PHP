imagesetthickness

(PHP 4 >= 4.0.6, PHP 5, PHP 7)

imagesetthickness — Establecer el grosor para el dibujo de líneas

Descripción

bool imagesetthickness ( resource $image , int $thickness )

imagesetthickness() establece el grosor de las líneas cuando se dibujan rectágulos, polígonos, elipses, etc..., a thickness píxeles.

Parámetros

image

Un recurso image, es devuelto por una de las funciones de creación de imágenes, como imagecreatetruecolor().

thickness

El grosor, en píxeles.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.

Ejemplos

Ejemplo #1 Ejemplo de imagesetthickness()

<?php
// Crear una imagen de 200x100
$im = imagecreatetruecolor(200, 100);
$blanco = imagecolorallocate($im, 0xFF, 0xFF, 0xFF);
$negro = imagecolorallocate($im, 0x00, 0x00, 0x00);

// Esteblecer el fondo a blanco
imagefilledrectangle($im, 0, 0, 299, 99, $blanco);

// Establecer el grosor de línea a 5
imagesetthickness($im, 5);

// Dibujar un rectángulo
imagerectangle($im, 14, 14, 185, 85, $negro);

// Imprimir la imagen al navegador
header('Content-Type: image/png');

imagepng($im);
imagedestroy($im);
?>
