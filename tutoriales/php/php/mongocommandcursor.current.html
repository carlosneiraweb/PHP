MongoCommandCursor::current

(PECL mongo >=1.5.0)

MongoCommandCursor::current — Devuelve el elemento actual

Descripción

public array MongoCommandCursor::current ( void )

Devuelve NULL hasta que se invoque a MongoCommandCursor::rewind().

Parámetros

Esta función no tiene parámetros.

Valores devueltos

El documento de resultados actual como un array asociativo. Se devolverá NULL si no hay más resultados.
