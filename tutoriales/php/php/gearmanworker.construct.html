GearmanWorker::__construct

(PECL gearman >= 0.5.0)

GearmanWorker::__construct — Crea una instancia GearmanWorker

Descripción

public GearmanWorker::__construct ( void )

Crea una instancia GearmanWorker que representa a un trabajador que se conecta a un servidor de trabajo y acepta tareas a ejecutar.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Un objeto GearmanWorker
