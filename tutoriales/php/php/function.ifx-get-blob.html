ifx_get_blob

(PHP 4, PHP <=5.2.0)

ifx_get_blob — Devolver el contenido de un objeto blob

Descripción

string ifx_get_blob ( int $bid )

Devuelve el contenido del objeto blob.

Parámetros

bid

El id del objeto BLOB.

Valores devueltos

El contenido de el BLOB como un string, o FALSE en caso de error.
