CairoContext::copyPage

cairo_copy_page

(PECL cairo >= 0.1.0)

CairoContext::copyPage -- cairo_copy_page — Emite la página actual

Descripción

Estilo orientado a objetos (método):

public void CairoContext::copyPage ( void )

Estilo por procedimientos:

void cairo_copy_page ( CairoContext $context )

Emite la página actual para entornos que soportan múltiples páginas, pero no la borra, así, el contenido de la página actual se conservará también para la siguiente página. Use CairoContext::showPage() si quiere obtener una página vacía después de la emisión.

Esta es una función conveniente que simplemente llama a CairoSurface::copyPage() sobre los objetivos de CairoContext.

Parámetros

context

Un objeto CairoContext válido creado con CairoContext::__construct() o cairo_create()

Valores devueltos

No devuelve ningún valor.

Ejemplos

Ejemplo #1 Estilo orientado a objetos

<?php

$superficie = new CairoImageSurface(CairoFormat::ARGB32, 50, 50);

$contexto = new CairoContext($superficie);

$contexto->copyPage();

?>

Ejemplo #2 Estilo por procedimientos

<?php

$superficie = cairo_image_surface_create(CAIRO_FORMAT_ARGB32, 50, 50);

$contexto = cairo_create($superficie);

cairo_copy_page($contexto);

?>
