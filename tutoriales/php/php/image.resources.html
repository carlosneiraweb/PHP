Tipos de recursos

Esta extensió define los siguientes tipos de recursos:

Lista de tipos de recursos en GD
Nombre Descripción Notas
gd Recurso de imagen, empleado por funciones como imagecreatefrompng()  
gd font Recurso de fuente devuelto por imageloadfont()  
gd PS font Recurso de fuente de PostScript Tipo 1, devuelto por imagepsloadfont() Eliminado a partir de PHP 7.0.0.
gd PS encoding Recurso de codificación de PostScript Tipo 1, devuelto por imagepsencodefont() Eliminado a partir de PHP 7.0.0.
