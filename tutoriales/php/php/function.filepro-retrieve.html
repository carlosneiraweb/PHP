filepro_retrieve

(PHP 4, PHP 5 <= 5.1.6, PHP 7)

filepro_retrieve — Obtiene datos de una base de datos filePro

Descripción

string filepro_retrieve ( int $row_number , int $field_number )

Devuelve los datos de una localización específica en la base de datos.

Nota: Cuando el modo seguro está habilitado, PHP comprueba si los ficheros o directorios sobre los que se opera tienen el mismo UID (propietario) que el script que está siendo ejecutado.

Parámetros

row_number

El número de la fila. Debe estar entre cero y el número total de filas menos uno (0..filepro_rowcount() - 1)

field_number

El número de la columna. Acepta valores entre cero y el número total de columnas menos uno (0..filepro_fieldcount() - 1)

Valores devueltos

Devuelve los datos especificados, o FALSE en caso de errores.
