MongoCommandCursor::dead

(PECL mongo >=1.5.0)

MongoCommandCursor::dead — Comprueba si hay resultados que aún no han sido enviados desde la base de datos

Descripción

public bool MongoCommandCursor::dead ( void )

Este método comprueba si el cursor MongoCommandCursor ha expirado y la base de datos no tiene más resultados que enviar al cliente. Un cursor "muerto" no significa necesariamente que ya no haya más resultados disponibles para la iteración.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve TRUE si no hay más resultados que aún no han sido enviados al cliente, y FALSE de lo contrario.
