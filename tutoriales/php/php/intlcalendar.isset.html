IntlCalendar::isSet

(PHP 5.5.0, PHP 7, PECL >= 3.0.0a1)

IntlCalendar::isSet — Determina si un campo está definido

Descripción

Estilo orientado a objetos

public bool IntlCalendar::isSet ( int $field )

Estilo por procedimientos

bool intlcal_is_set ( IntlCalendar $cal , int $field )

Indica si un campo está definido (lo opuesto a clear). Los campos definidos tienen prioridad sobre los indefinidos y sus valores por defecto cuando la fecha/hora está siendo calculada. Los campos definidos con posterioridad tienen prioridad sobre los definidos con anterioridad.

Parámetros

cal

El recurso IntlCalendar.

field

Una de las constantes del campo fecha/hora de IntlCalendar. Son valores de tipo integer entre 0 y IntlCalendar::FIELD_COUNT.

Valores devueltos

Asumiendo que no hay errores, devuelve TRUE si el campo está definido.

Ejemplos

Vea el ejemplo en IntlCalendar::clear().

Ver también

  • IntlCalendar::clear() - Limpiar uno o todos los campos
  • IntlCalendar::set() - Establecer un campo de instante o varios campos en común de una vez
