MongoClient::dropDB

(PECL mongo >=1.3.0)

MongoClient::dropDB — Elimina una base de datos [obsoleto]

Esta extenisón que define este método está obsoleta. En su lugar debe usarse la extensión MongoDB. No hay equivalente para este método en la nueva extensión, aunque existe una alterenativa en la biblioteca de PHP:

  • » MongoDB\Database\drop

Descripción

public array MongoClient::dropDB ( mixed $db )
Advertencia

Obsoleto

Use MongoDB::drop() en su lugar.

Parámetros

db

La base de datos a eliminar. Puede ser un objeto MongoDB o el nombre de la base de datos.

Valores devueltos

Devuelve la respuesta de la base de datos.
