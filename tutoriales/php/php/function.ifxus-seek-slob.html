ifxus_seek_slob

(PHP 4, PHP <=5.2.0)

ifxus_seek_slob — Establecer la posición del archivo para la siguiente operación de lectura o escritura

Descripción

int ifxus_seek_slob ( int $bid , int $mode , int $offset )

Establece la posición de búsqueda del archivo para la siguiente operación de lectura o escritura en un objeto SLOB.

Parámetros

bid

Un identificador SLOB válido.

mode

0 = LO_SEEK_SET (Inicio del objeto SLOB), 1 = LO_SEEK_CUR (Posición de búsqueda actual del objeto SLOB), 2 = LO_SEEK_END (Final del objeto SLOB).

offset

Offset a partir de la posición de búsqueda.

Valores devueltos

Devuelve la posición de búsqueda como entero, o FALSE en caso de error.
