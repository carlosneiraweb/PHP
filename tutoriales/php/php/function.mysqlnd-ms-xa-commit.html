mysqlnd_ms_xa_commit

(PECL mysqlnd_ms < 1.6.0)

mysqlnd_ms_xa_commit — Consigna una transacción distribuida/XA entre servidores de MySQL

Descripción

int mysqlnd_ms_xa_commit ( mixed $connection , string $gtrid )

Consigna una transacción global entre servidores de MySQL iniciada por mysqlnd_ms_xa_begin().

Si cualquier participante de la transacción global falla en la consignación, se realizará una reversión implícita. Podría ocurrir que no todos los casos puedan ser manejados durante la reversión. Por ejemplo, no se intentará realizar una conexión a un participante después de que la conexión al participante se haya perdido. La resolución de casos que no pueden ser revertidos con facilidad se deja a la recolección de basura.

Nota: Experimental

Esta característica está actualmente en desarrollo. Podrían existir problemas y/o limitaciones. No la use en entornos de producción.

Parámetros

connection

Un gestor de conexión de MySQL obtenido de cualesquiera de las extensiones connect functions of the mysqli, mysql o PDO_MYSQL.

gtrid

El identificador de transacción global (gtrid).

Valores devueltos

Devuelve TRUE si la transacción global ha sido consignada. De lo contrario, devuelve FALSE
