fam_next_event

(PHP 5 <= 5.0.5)

fam_next_event — Obtiene el próximo evento FAM en espera

Descripción

array fam_next_event ( resource $fam )

Obtiene el próximo evento FAM en espera.

Esta función se bloqueará hasta que un evento esté disponible, lo cual puede verificarse usando la función fam_pending().

Parámetros

fam

Un recurso que representa una conexión al servicio FAM devuelto por fam_open()

Valores devueltos

Devuelve un array conteniendo los códigos de eventos FAM en el elemento 'code', la ruta de este archivo aplicada al elemento 'filename' y opcionalmente un nombre de máquina en el elemento 'hostname'.

Los códigos de eventos posibles están descritos en detalle en la constantes de esta sección.

  • fam_pending() - Verifica si hay eventos FAM pendientes
