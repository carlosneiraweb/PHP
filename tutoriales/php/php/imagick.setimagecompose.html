Imagick::setImageCompose

(PECL imagick 2.0.0)

Imagick::setImageCompose — Establece el operador de composción de una imagen

Descripción

bool Imagick::setImageCompose ( int $compose )

Establece el operador de composción de una imagen, útil para especificar cómo componer la miniatura de la imagen cuando se usa el método Imagick::montageImage().

Parámetros

compose

Valores devueltos

Devuelve TRUE en caso de éxito.

Errores/Excepciones

Lanza ImagickException en caso de error.
