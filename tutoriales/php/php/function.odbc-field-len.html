odbc_field_len

(PHP 4, PHP 5, PHP 7)

odbc_field_len — Obtener la longitud (precisión) de un campo

Descripción

int odbc_field_len ( resource $result_id , int $field_number )

Obtiene la longitud de un campo referenciado por número del indentificador de resultado dado.

Parámetros

result_id

El identificador de resultado.

field_number

El número de campo. La numeración de campos empieza en 1.

Valores devueltos

Devuelve la longitud del campo, o FALSE en caso de error.
