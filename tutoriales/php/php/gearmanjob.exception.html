GearmanJob::exception

(PECL gearman <= 0.5.0)

GearmanJob::exception — Envía una excepción para un trabajo en ejecución (obsoleto)

Descripción

public bool GearmanJob::exception ( string $exception )

Envía la excepción indicada cuando este trabajo está siendo ejecutado.

Nota:

Este método ha sido reemplazado por GearmanJob::sendException() en la versión 0.6.0 de la extensión Gearman.

Parámetros

exception

Descripción de la excepción.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.

Ver también

  • GearmanJob::setReturn() - Establece el valor de retorno
  • GearmanJob::sendStatus() - Envía el estado
  • GearmanJob::sendWarning() - Envía una advertencia
