convert_uuencode

(PHP 5, PHP 7)

convert_uuencode — Codificar mediante uuencode una cadena

Descripción

string convert_uuencode ( string $data )

convert_uuencode() codifica una cadena usando el algoritmo uuencode.

Uuencode traduce todas las cadenas (incluyendo las binarias) a caracteres imprimibles, haciéndolas seguras para transmisiones de red. Los datos codificados de esta forma son aproximadamente un 35% más grandes que el original.

Parámetros

data

Los datos a codificar.

Valores devueltos

Devuelve los datos codificados mediante uuencode o FALSE en caso de error.

Ejemplos

Ejemplo #1 Ejemplo de convert_uuencode()

<?php
$cadena = "prueba\ntexto texto\r\n";

echo convert_uuencode($cadena);
?>
