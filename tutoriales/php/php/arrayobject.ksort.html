ArrayObject::ksort

(PHP 5 >= 5.2.0, PHP 7)

ArrayObject::ksort — Ordena las entradas por clave

Descripción

public void ArrayObject::ksort ( void )

ordenar las entradas por clave, manteniendo la correlación de claves para la entrada. Esto es útil principalmente en arrays asociativos.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

No devuelve ningún valor.

Ejemplos

Ejemplo #1 Ejemplo de ArrayObject::ksort()

<?php
$fruits = array("d" => "limon", "a" => "naranja", "b" => "banana", "c" => "manzana");
$fruitArrayObject = new ArrayObject($fruits);
$fruitArrayObject->ksort();

foreach ($fruitArrayObject as $key => $val) {
    echo "$key = $val\n";
}
 ?>

El resultado del ejemplo sería:

a = naranja 
b = banana 
c = manzana 
d = limon
