SDO_DataObject::getTypeName — Devolver el nombre del tipo de un objeto de datos

Descripción

string SDO_DataObject::getTypeName ( void )
Advertencia

Esta función ha sido declarada EXPERIMENTAL. Su funcionamiento, nombre y la documentación que le acompaña puede cambiar sin previo aviso en futuras versiones de PHP. Utilícela bajo su propia responsabilidad.

Devuelve elnombre del tipo de un objeto de datos. Es un método conveniente correspondiente a SDO_Model_ReflectionDataObject::getType().getName().

Parámetros

Ninguno.

Valores devueltos

El nombre del tipo del objeto de datos.
