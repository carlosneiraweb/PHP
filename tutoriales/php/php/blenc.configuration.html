Configuración en tiempo de ejecución

El comportamiento de estas funciones se ve afectado por la configuración de php.ini.

Opciones de configuración de Blenc
Nombre Por defecto Cambiable Historial de cambios
blenc.key_file /usr/local/etc/blenckeys PHP_INI_ALL

He aquí una breve explicación de las directivas de configuración.

blenc.key_file string

Es la ubicación donde BLENC puede encontrar el fichero que contiene una lista de claves de desencriptación desponible. Este fichero debe ser legible por el servidor Web.
