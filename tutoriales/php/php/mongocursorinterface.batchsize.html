MongoCursorInterface::batchSize

(PECL mongo >=1.5.0)

MongoCursorInterface::batchSize — Limita el número de elementos devueltos en un lote

Descripción

abstract public MongoCursorInterface MongoCursorInterface::batchSize ( int $batchSize )

Un cursor típicamente obtiene un lote de objetos de resultados y los almacena localmente. Este método establece el valor del tamaño del lote para configurar la cantidad de documentos obtenidos de un servidor en un viaje de ida y vuelta.

Parámetros

batchSize

El número de resultados a devolver por lote.

Valores devueltos

Devuelve este cursor.
