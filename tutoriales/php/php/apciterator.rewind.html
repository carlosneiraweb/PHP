APCIterator::rewind

(PECL apc >= 3.1.1)

APCIterator::rewind — Rebobina el iterator

Descripción

public void APCIterator::rewind ( void )

Rebobina el iterador hasta el primer elemento.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

No devuelve ningún valor.
