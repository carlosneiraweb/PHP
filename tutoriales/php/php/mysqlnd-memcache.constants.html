Constantes predefinidas

Estas constantes están definidas por esta extensión y estarán disponibles sólo cuando la extensión haya sido compilada con PHP, o bien sea cargada dinámicamente en ejecución.

Relacionadas con el Complemento de Memcache para MySQL

MYSQLND_MEMCACHE_DEFAULT_REGEXP (string)
Expresión regular predeterminada (estilo PCRE) empleada para hacer coincidir sentencias SELECT que se corresponderán con un punto de acceso al Complemento Memcache de MySQL, si es posible. También es posible emplear mysqlnd_memcache_set(), pero la estrategia predeterminada es emplear esta expresión regular para coincidencias de patrones.

Varias

El número de versión de este comlemento se puede obtener utilizando MYSQLND_MEMCACHE_VERSION o MYSQLND_MEMCACHE_VERSION_ID. MYSQLND_MEMCACHE_VERSION es la representación de tipo string de la versión numérica de MYSQLND_MEMCACHE_VERSION_ID, el cual es un integer como 10000. Los desarrolladores pueden calcular el número de versión como sigue:

Versión (parte) Ejemplo
Mayor*10000 1*10000 = 10000
Menor*100 0*100 = 0
Parche 0 = 0
MYSQLND_MEMCACHE_VERSION_ID 10000

MYSQLND_MEMCACHE_VERSION (string)
String con la versión del complemento, por ejemplo, "1.0.0-alpha".
MYSQLND_MEMCACHE_VERSION_ID (integer)
Número de la versión del complemento, por ejemplo, 10000.
