MongoId::__set_state

(PECL mongo >= 1.0.8)

MongoId::__set_state — Crea un objeto MongoId trivial

Esta extenisón que define este método está obsoleta. En su lugar debe usarse la extensión MongoDB. No hay equivalente para este método en la nueva extensión.

Descripción

public static MongoId MongoId::__set_state ( array $props )

Esta función sólo la utiliza PHP internamente, no debería ser necesario que sea llamada por el usuario.

Es idéntica a la función:

<?php

public static function __set_state($props) {
    return new MongoId("000000000000000000000000");
}

?>

Parámetros

props

En teoría, un array de propiedades utilizado para crear el nueva ID. Sin embargo, ya que las instancias de MongoId no tienen propiedades, no se utiliza.

Valores devueltos

Un nuevo ID con el valor "000000000000000000000000".
