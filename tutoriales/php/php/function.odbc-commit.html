odbc_commit

(PHP 4, PHP 5, PHP 7)

odbc_commit — Envía una transacción ODBC

Descripción

bool odbc_commit ( resource $connection_id )

Envía todas las transacciones pendientes en una conexión.

Parámetros

connection_id

El conector identificador ODBC, ver odbc_connect() para más información.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.
