msql_free_result

(PHP 4, PHP 5, PHP 7)

msql_free_result — Liberar memoria de un resultado

Descripción

bool msql_free_result ( resource $result )

msql_free_result() libera la memoria asociada a query_identifier. Cuando PHP completa una petición, su memoria es liberada automáticamente, así que so solo es necesario llamar a esta función cuando se quiera estar seguro de que no se usa demasiada memoria durante la ejecución del script.

Parámetros

result

El resultado resource que está siendo evaluado. Este resultado proviene de una llamada a msql_query().

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.
