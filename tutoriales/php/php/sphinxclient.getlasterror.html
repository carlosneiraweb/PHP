SphinxClient::getLastError

(PECL sphinx >= 0.1.0)

SphinxClient::getLastError — Obtener el último mensaje de error

Descripción

public string SphinxClient::getLastError ( void )

Devuelve un string con el último mensaje de error. Si no hubo errores durante la previa llamada API, es retornado un string vacío. Este método no restablece el mensaje de error, lo que con seguridad se puede llamar varias veces.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve el último mensaje de error o un string vacío si no hubo errores.
