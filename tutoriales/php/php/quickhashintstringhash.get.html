QuickHashIntStringHash::get

(PECL quickhash >= Unknown)

QuickHashIntStringHash::get — Este método recupera un valor de un hash mediante su clave

Descripción

public mixed QuickHashIntStringHash::get ( int $key )

Este método recupera un valor de un hash mediante su clave.

Parámetros

key

La clave de la entrada a añadir.

Valores devueltos

El valor de la clave si ésta existe, o NULL si la clave no es parte del hash.

Ejemplos

Ejemplo #1 Ejemplo de QuickHashIntStringHash::get()

<?php
$hash = new QuickHashIntStringHash( 8 );
var_dump( $hash->get( 1 ) );

var_dump( $hash->add( 2, "two" ) );
var_dump( $hash->get( 2 ) );

var_dump( $hash->add( 3, 5 ) );
var_dump( $hash->get( 3 ) );
?>

El resultado del ejemplo sería algo similar a:

bool(false)
bool(true)
string(3) "two"
bool(true)
string(1) "5"
