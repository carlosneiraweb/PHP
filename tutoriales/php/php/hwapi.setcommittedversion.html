hw_api::setcommittedversion

(PHP 4, PHP 5 < 5.2.0, PECL hwapi SVN)

hw_api::setcommittedversion — Consolida (commit) una versión diferente que la última versión

Descripción

hw_api_object hw_api::setcommittedversion ( array $parameter )

Consolida una versión de un documento. La versión consolidada es la que es visible para los usuarios con permiso de lectura. Por defecto la última versión es la versión consolidada.

Parámetros

parameter
