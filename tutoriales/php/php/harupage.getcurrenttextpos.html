HaruPage::getCurrentTextPos

(PECL haru >= 0.0.1)

HaruPage::getCurrentTextPos — Obtiene la posición actual para la impresión del texto

Descripción

array HaruPage::getCurrentTextPos ( void )

Obtiene la posición actual para la impresión del texto.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve la posición actual para la impresion del texto como un arreglo de 2 elementos - "x" y "y".

Errores/Excepciones

Lanza una HaruException cuando se produce un error.
