Imagick::getFilename

(PECL imagick 2.0.0)

Imagick::getFilename — El nombre de archivo asociado a una secuencia de imágenes

Descripción

string Imagick::getFilename ( void )

El nombre de archivo asociado a una secuencia de imágenes.

Valores devueltos

Devuelve una cadena en caso de éxito.

Errores/Excepciones

Lanza ImagickException en caso de error.
