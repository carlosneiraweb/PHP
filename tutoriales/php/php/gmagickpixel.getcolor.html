GmagickPixel::getcolor

(PECL gmagick >= Unknown)

GmagickPixel::getcolor — Devuelve el color

Descripción

public mixed GmagickPixel::getcolor ([ bool $as_array [, bool $normalize_array ]] )

Devuelve el color descrito por el objeto GmagickPixel, como un matriz. Si el color tiene establecido un canal de opacidad, éste es proporcionado como un cuarto valor en la lista.

Parámetros

as_array

True para indicar que devuleve una matriz en vez de una cadena

normalize_array

Normaliza los valores de los colores

Valores devueltos

Una matriz de valores de canales, normalizados si se da TRUE como parámetro. Lanza una excepción GmagickPixelException si se produjo un error.
