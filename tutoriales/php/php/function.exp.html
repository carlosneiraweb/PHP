exp

(PHP 4, PHP 5, PHP 7)

exp — Calcula la exponencial de e

Descripción

float exp ( float $arg )

Devuelve e elevado a la potencia de arg.

Nota:

'e' es la base del sistema natural de logaritmos, y su valor aproximado es 2.718282.

Parámetros

arg

El argumento a procesar

Valores devueltos

'e' elevado a la potencia de arg

Ejemplos

Ejemplo #1 Ejemplo de exp()

<?php
echo exp(12) . "\n";
echo exp(5.7);
?>

El resultado del ejemplo sería:

1.6275E+005
298.87
