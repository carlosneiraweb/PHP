MongoCommandCursor::batchSize

(PECL mongo >=1.5.0)

MongoCommandCursor::batchSize — Limita el número de elementos devueltos en un lote

Descripción

public MongoCommandCursor MongoCommandCursor::batchSize ( int $batchSize )

Un cursor normalmente obtiene un lote de objetos de resultado y los almacena localmente. Este método establece el valor de batchSize para configurar la cantidad de documentos recuperados del servidor en viaje de ida y vuelta.

Parámetros

batchSize

El número de resultados a devolver por lote. Cada lote requiere un viaje de ida y vuelta al servidor.

Este parámetro no puede sobrescribir el límite de MongoDB de cuántos datos serán devueltos al cliente (es decir, si se establece el tamaño del lote a 1,000,000,000, MongoDB aún devolverá solamente 4-16 MB de resultados por lote).

Valores devueltos

Devuelve este cursor.

Ejemplos

Ejemplo #1 MongoCommandCursor::batchSize()

<?php
$commandCursor->batchSize(20);
?>
