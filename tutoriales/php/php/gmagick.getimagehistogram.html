Gmagick::getimagehistogram

(PECL gmagick >= Unknown)

Gmagick::getimagehistogram — Obtiene el histograma de la imagen

Descripción

public array Gmagick::getimagehistogram ( void )

Devuelve el histograma de la imagen como una matriz de objetos GmagickPixel. Lanza una excepción de tipo GmagickException si se produjo un error.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve el histograma de la imagen como una matriz de objetos GmagickPixel.

Errores/Excepciones

Arroja un GmagickException en caso de error.
