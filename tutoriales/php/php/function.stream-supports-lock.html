stream_supports_lock

(PHP 5 >= 5.3.0, PHP 7)

stream_supports_lock — Indica si el flujo soporta bloqueo

Descripción

bool stream_supports_lock ( resource $stream )

Indica si el flujo soporta bloqueo a través de flock().

Parámetros

stream

El flujo a comprobar.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.
