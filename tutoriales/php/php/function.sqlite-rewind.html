sqlite_rewind

SQLiteResult::rewind

(PHP 5 < 5.4.0, PECL sqlite >= 1.0.0)

sqlite_rewind -- SQLiteResult::rewind — Busca el primer número de fila

Descripción

bool sqlite_rewind ( resource $result )

Estilo orientado a objetos (método):

bool SQLiteResult::rewind ( void )

sqlite_rewind() vuelve a buscar la primera fila en el conjunto de resultados dado.

Parámetros

result

El recurso de resultados de SQLite. Este parámetro no es necesario cuando se usa el método orientado a objetos.

Nota:

Esta función no puede ser utilizada con manejos de resultados no pasados por buffer.

Valores devueltos

Devuelve FALSE si no existen filas en el conjunto de resultados, TRUE en caso contrario.

Ver también

  • sqlite_next() - Busca el número de fila siguiente
  • sqlite_current() - Obtiene la fila actual como un array de un conjunto de resultado
  • sqlite_seek() - Busca un número de fila en particular en un conjunto de resultados almacenado en buffer
