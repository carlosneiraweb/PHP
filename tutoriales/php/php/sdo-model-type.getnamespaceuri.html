SDO_Model_Type::getNamespaceURI — Obtener el URI del espacio de nombres del tipo

Descripción

string SDO_Model_Type::getNamespaceURI ( void )
Advertencia

Esta función ha sido declarada EXPERIMENTAL. Su funcionamiento, nombre y la documentación que le acompaña puede cambiar sin previo aviso en futuras versiones de PHP. Utilícela bajo su propia responsabilidad.

Devuelve el URI del espacio de nombres del tipo. La combinación del URI del espacio de nombres y el nombre del tipo se usa para identificar el tipo de manera única.

Parámetros

Ninguno.

Valores devueltos

Devuelve el URI del espacio de nombres del tipo.
