event_base_new

(PECL libevent >= 0.0.1)

event_base_new — Crea e inicializa una nueva base de eventos

Descripción

resource event_base_new ( void )

Devuelve una nueva base de eventos, que luego podrá ser usada en event_base_set(), event_base_loop() y otras funciones.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

event_base_new() devuelve una base de eventos válida en caso de éxito o FALSE en caso de error.
