Gmagick::cropthumbnailimage

(PECL gmagick >= Unknown)

Gmagick::cropthumbnailimage — Crea una miniatura recortada

Descripción

public Gmagick Gmagick::cropthumbnailimage ( int $width , int $height )

Crea una miniatura de tamaño fijo ampliando o reduciendo de escala la imagen y recortando un área específica desde el centro.

Parámetros

width

El ancho de la miniatura

height

El alto de la miniatura

Valores devueltos

El objeto Gmagick recortado

Errores/Excepciones

Arroja un GmagickException en caso de error.
