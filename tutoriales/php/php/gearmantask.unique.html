GearmanTask::unique

(PECL gearman >= 0.6.0)

GearmanTask::unique — Obtiene el identificador único de una tarea

Descripción

public string GearmanTask::unique ( void )

Retorna el identificador única para esta tarea. Este se asigna por GearmanClient, a diferencia del manejador de trabajo que es asignado por el servidor de trabajos Gearman.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

El identificador único, o FALSE si no tiene asignado identificador.

Ver también

  • GearmanClient::do() - Ejecuta una sola tarea y retorna el resultado [obsoleto]
  • GearmanClient::addTask() - Añade una tarea para ser ejecutada en paralelo
