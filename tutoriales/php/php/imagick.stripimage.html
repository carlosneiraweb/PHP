Imagick::stripImage

(PECL imagick 2.0.0)

Imagick::stripImage — Elimina todos los perfiles y comentarios de una imagen

Descripción

bool Imagick::stripImage ( void )

Elimina todos los perfiles y comentarios de una imagen.

Valores devueltos

Devuelve TRUE en caso de éxito.

Errores/Excepciones

Lanza ImagickException en caso de error.
