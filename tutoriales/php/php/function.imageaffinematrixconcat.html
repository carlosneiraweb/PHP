imageaffinematrixconcat

(PHP 5 >= 5.5.0, PHP 7)

imageaffinematrixconcat — Concatena dos matrices (como al hacer muchas opciones de una vez)

Descripción

array imageaffinematrixconcat ( array $m1 , array $m2 )

Advertencia

Esta función no está documentada actualmente, solamente se encuentra disponible la lista de parámetros.

Parámetros

m1

Array con las claves 0 a 5.

m2

Array con las claves 0 a 5.

Valores devueltos

Array con las claves 0 a 5 y valores de tipo float o FALSE en caso de error.
