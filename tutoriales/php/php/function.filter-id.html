filter_id

(PHP 5 >= 5.2.0, PHP 7)

filter_id — Indica el ID del nombre de filtro que se indique

Descripción

int filter_id ( string $filtername )

Parámetros

filtername

Nombre del filtro a obtener el ID.

Valores devueltos

ID del filtro en caso de éxito o FALSE si el filtro no existe.
