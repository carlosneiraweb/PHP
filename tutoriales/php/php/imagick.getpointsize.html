(No hay información de versión disponible, podría estar únicamente en SVN)

Imagick::getPointSize — Obtiene el tamaño de punto

Descripción

float Imagick::getPointSize ( void )

Devuelve la propiedad tamaño de punto de los objetos. Este método está disponible si Imagick ha sido compilado con la versión 6.3.7 o superior de ImageMagick.

Valores devueltos

Devuelve un float que contiene el tamaño de punto.
