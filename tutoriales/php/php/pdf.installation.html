Instalación

Esta extensión » PECL no se distribuye con PHP. Se puede encontrar información para la instalación de esta extensión PECL en el capítulo del manual titulado Instalación de extensiones PECL. Se puede encontrar información adicional, tal como nuevos lanzamientos, descargas, ficheros fuente, información de mantenimiento, y un CHANGELOG, aquí: » http://pecl.php.net/package/pdflib.

Para hacer que estas funciones trabajen en PHP < 4.3.9, ha de compilar PHP con --with-pdflib[=DIR] . DIR es el directorio base de instalacion de PDFlib, por defecto es /usr/local.
