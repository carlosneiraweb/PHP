GearmanTask::isKnown

(PECL gearman >= 0.5.0)

GearmanTask::isKnown — Determina si la tarea es conocida

Descripción

public bool GearmanTask::isKnown ( void )

Retorna información sobre el estado para una tarea, sea o no conocida por el servidor de trabajos.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

TRUE si la tarea es conocida, FALSE en otro caso.
