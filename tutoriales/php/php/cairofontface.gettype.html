CairoFontFace::getType

(PECL cairo >= 0.1.0)

CairoFontFace::getType — Recupera el tipo de familia de fuentes

Descripción

public int CairoFontFace::getType ( void )

Esta función devuelve el tipo del entorno usado para crear una familia de fuentes. Véase las constantes de la clase CairoFontType para los tipos disponibles.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Una familia de fuentes que se puede ser cualquiera definida en CairoFontType

Ejemplos

Ejemplo #1 Ejemplo de CairoFontFace::getType()

<?php

// Crea la familia de fuentes
$fontface = new CairoToyFontFace('sans-serif');

// Obtener el tipo de familia de fuentes
var_dump($fontface->getType());

?>

El resultado del ejemplo sería algo similar a:

int(0)
