ReflectionExtension::getClassNames

(PHP 5, PHP 7)

ReflectionExtension::getClassNames — Obtiene nombres de las clases

Descripción

public array ReflectionExtension::getClassNames ( void )

Devuelve una lsita de los nombres de las clases que pertenecen a esta extensión.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Un array con los nombres de las clases, tal como estuvieran definidos en la extensión. Si no hubiera ninguna clase, se devolvería un array vacío.

Ejemplos

Ejemplo #1 Ejemplo de ReflectionExtension::getClassNames()

<?php
$ext = new ReflectionExtension('XMLWriter');
var_dump($ext->getClassNames());
?>

El resultado del ejemplo sería algo similar a:

array(1) {
  [0]=>
  string(9) "XMLWriter"
}
