Introducción

Las funciones de PDF en PHP pueden crear ficheros PDF empleando la biblioteca PDFlib de PDFlib GmbH (» www.pdflib.com). Hay disponible una versión restringida gratuita llamada PDFlib Lite 7, aunque ya no tiene mantenimiento desde 2010. La integración de PDFlib para PHP aún es mantenida activamente por PDFlib GmbH. Está dirigida principalmente a las versiones comerciales de PDFlib, pero se puede emplear para la antigua PDFlib Lite.

Si quiere utlizar PDFlib, le aconsejamos que comience con los ejemplos que se incluyen en todos los paquetes de PDFlib, y que consulte los exhaustivos manuales en formato PDF que también se incluyen. Cómo mínimo, debería echar una ojeada al ejemplo hello.php para aprender la configuración básica para emplear PDFlib.

Nota:

Si está interesado en generadores de PDF gratuitos alternativos que no utilicen bibliotecas PDF externas, vea esta FAQ relacionada.
