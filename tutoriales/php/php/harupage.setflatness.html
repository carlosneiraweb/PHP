HaruPage::setFlatness

(PECL haru >= 0.0.1)

HaruPage::setFlatness — Establecer la planicie para la página

Descripción

bool HaruPage::setFlatness ( float $flatness )

Define la planicie para la página.

Parámetros

flatness

La planicie definida para la página.

Valores devueltos

Devuelve TRUE al tener éxito.

Errores/Excepciones

Lanza una HaruException cuando se produce un error.
