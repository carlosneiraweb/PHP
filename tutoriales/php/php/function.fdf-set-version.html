fdf_set_version

(PHP 4 >= 4.3.0, PHP 5, PHP 7)

fdf_set_version — Asignar número de versión a un fichero FDF

Descripción

bool fdf_set_version ( resource $fdf_document , string $version )

Asigna la version FDF al documento solicitado.

Algunas de las funcionalidades de las que suporta esta extensión sólo están disponibles para las últimas versiones de FDF.

Parámetros

fdf_document

Manejador de documentos de FDF, devuelto por fdf_create(), fdf_open() o por fdf_open_string().

version

Nombre de versión. En el sistema FDF actual 5.0, puede ser o bien 1.2, 1.3 o 1.4.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.
