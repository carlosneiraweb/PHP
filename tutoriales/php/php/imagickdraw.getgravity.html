ImagickDraw::getGravity

(PECL imagick 2.0.0)

ImagickDraw::getGravity — Devuelve la gravedad de emplazamiento del texto

Descripción

int ImagickDraw::getGravity ( void )
Advertencia

Esta función no está documentada actualmente, solamente se encuentra disponible la lista de parámetros.

Devuelve la gravedad de emplazamiento del texto usada cuando se escribe texto.

Valores devueltos

Devuelve una constante GRAVITY_ si se tuvo éxito y 0 si no está establecida la gravedad.
