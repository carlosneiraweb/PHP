Introducción

db++, hecha por la compañía alemanama » Concept asa, es un sistema de base de datos relacional que tiene en mente el alto redimiento y el bajo uso de memoria y disco. Aunque proporciona SQL como una interfaz de lenguaje adicional, no es realmente una base de datos SQL en primera instancia pero proporciona su propio lenguaje de consulta AQL, que está más influenciado por el álgebra relacional que SQL.

Concept asa siempre ha estado interesado en apoyar lenguajes de código abierto, db++ ha tenido Perl y llamadas Tcl durante años y ahora usa Tcl como su lenguaje interno de procedimientos de almacenamiento.

Advertencia

Esta extensión es EXPERIMENTAL. Esto significa que el comportamiento de esta extensión, los nombres de sus funciones y todo lo documentado sobre esta extensión, puede cambiar en una futura edición de PHP sin previo aviso. El uso de esta extensión queda bajo su propia responsabilidad.
