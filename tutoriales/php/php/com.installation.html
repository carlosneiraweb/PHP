Instalación

A partir de PHP 5.3.15 / 5.4.5, esta extensión requiere php_com_dotnet.dll para habilitarla dentro de php.ini para poder usar estas funciones. Las versiones anteriores de PHP habilitaban estas extensiones de manera predeterminada.

Usted tienen la responsabilidada de instalar el soporte para los diversos objetos COM que tenga intención de usar (como MS Word); no incluimos ni podemos incluir todos estos con PHP.
