(PECL gmagick >= Unknown)

Gmagick::despeckleimage — Reduce el ruido granular de una imagen

Descripción

public Gmagick Gmagick::despeckleimage ( void )

Reduce el ruido granular de una imagen mientras preserva los bordes de la imagen original.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

El objeto Gmagick con el ruido granular reducido si se tuvo éxito.

Errores/Excepciones

Arroja un GmagickException en caso de error.
