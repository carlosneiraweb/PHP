Notas

Algunas notas sobre la implementación:

  1. El soporte para MYSQL_TYPE_GEOMETRY fue agregado a la extensión MySQLi en PHP 5.3.

  2. Nótese que existen diferentes implementaciones internas en libmysqlclient y mysqlnd para manejar columnas del tipo MYSQL_TYPE_GEOMETRY. En términos generales, mysqlnd asignará significativamente menos memoria. Por ejemplo, si hay una columna POINT en un set de resultado, libmysqlclient puede pre-asignar hasta 4GB de RAM aunque se necesitan menos de 50 bytes para mantener una comuna POINT en memoria. La asignación de memoria es mucho menor, menos de 50 bytes, si se utiliza mysqlnd.
