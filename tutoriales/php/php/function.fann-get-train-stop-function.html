fann_get_train_stop_function

(PECL fann >= 1.0.0)

fann_get_train_stop_function — Devuelve la función de parada empleada durante el entrenamiento

Descripción

int fann_get_train_stop_function ( resource $ann )

Devuelve la función de parada empleada durante el entrenamiento.

Las funciones de parada están descritas en las constantes de funciones de parada.

La función de parada predeterminada es FANN_STOPFUNC_MSE.

Parámetros

ann

Un resource de red neuronal.

Valores devueltos

La constante de función de parada, o FALSE en caso de error.
