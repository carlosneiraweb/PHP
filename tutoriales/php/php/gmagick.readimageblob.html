Gmagick::readimageblob

(PECL gmagick >= Unknown)

Gmagick::readimageblob — Lee una imagen desde una cadena binaria

Descripción

public Gmagick Gmagick::readimageblob ( string $imageContents [, string $filename ] )

Lee una imagen desde una cadena binaria.

Parámetros

imageContents

El contenido de la imagen

filename

El nombre de archivo de la imagen.

Valores devueltos

El objeto Gmagick si se tuvo éxito

Errores/Excepciones

Arroja un GmagickException en caso de error.
