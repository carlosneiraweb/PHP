Error::getTrace — Obtener la traza de la pila

Descripción

final public array Error::getTrace ( void )

Devuelve la traza de la pila.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve la traza de la pila como un array.

Ejemplos

Ejemplo #1 Ejemplo de Error::getTrace()

<?php
function prueba() {
 throw new Error;
}

try {
 prueba();
} catch(Error $e) {
 var_dump($e->getTrace());
}
?>

El resultado del ejemplo sería algo similar a:

array(1) {
  [0]=>
  array(4) {
    ["file"]=>
    string(22) "/home/bjori/tmp/ex.php"
    ["line"]=>
    int(7)
    ["function"]=>
    string(6) "prueba"
    ["args"]=>
    array(0) {
    }
  }
}
