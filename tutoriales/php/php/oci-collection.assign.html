OCI-Collection::assign

(PHP 5, PHP 7, PECL OCI8 >= 1.1.0)

OCI-Collection::assign — Asigna un valor a la colección desde otra colección existente

Descripción

bool OCI-Collection::assign ( OCI-Collection $from )

Asigna un valor a la colección desde otra colección previamente creada. Ambas colecciones deben crearse con oci_new_collection() antes de utilizarlas.

Parámetros

from

Una instancia de OCI-Collection.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.
