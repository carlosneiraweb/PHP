fdf_save

(PHP 4, PHP 5, PHP 7)

fdf_save — Guardar un documento FDF

Descripción

bool fdf_save ( resource $fdf_document [, string $filename ] )

Guarda un documento FDF.

Parámetros

fdf_document

El gestor de documento FDF, devuelto por fdf_create(), fdf_open() o fdf_open_string().

filename

Si se define, el FDF resultante será escrito en este parámetro. De lo contrario, esta función escribirá el FDF en la secuencia de salida PHP predeterminada.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.

Ver también

  • fdf_close() - Cierra un documento FDF
  • fdf_create() - Crear un nuevo documento FDF
  • fdf_save_string() - Devuelve el documento FDF en forma de string
