SWFDisplayItem::addColor

(PHP 5 <= 5.3.0, PECL ming SVN)

SWFDisplayItem::addColor — Añade el color dado a esta transformación de color del elemento

Descripción

void SWFDisplayItem::addColor ( int $red , int $green , int $blue [, int $a ] )
Advertencia

Esta función ha sido declarada EXPERIMENTAL. Su funcionamiento, nombre y la documentación que le acompaña puede cambiar sin previo aviso en futuras versiones de PHP. Utilícela bajo su propia responsabilidad.

swfdisplayitem::addcolor() añade el color a esta transformación de color del elemento. El color se da en su forma RGB.

El objeto puede ser un swfshape(), un swfbutton(), un swftext() o un swfsprite(). Debe haber sido añadido usando swfmovie::add().

Valores devueltos

No devuelve ningún valor.

Valores devueltos

No devuelve ningún valor.
