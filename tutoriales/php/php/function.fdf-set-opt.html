fdf_set_opt

(PHP 4 >= 4.0.2, PHP 5, PHP 7)

fdf_set_opt — Establecer una opción sobre un campo

Descripción

bool fdf_set_opt ( resource $fdf_document , string $fieldname , int $element , string $str1 , string $str2 )

Establece opciones sobre el campo solicitado.

Parámetros

fdf_document

Manejador de documentos FDF, devuelto por fdf_create(), fdf_open() o por fdf_open_string().

fieldname

Nombre del campo FDF, en forma de string.

element

str1

str2

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.
