GearmanJob::sendComplete

(PECL gearman >= 0.6.0)

GearmanJob::sendComplete — Envía el resultado y el estado completado

Descripción

public bool GearmanJob::sendComplete ( string $result )

Envía los datos que han resultado y la actualización a estado completado para este trabajo.

Parámetros

result

Resultado en formato serializado.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.

Ver también

  • GearmanJob::sendFail() - Envía el estado de fallo
  • GearmanJob::setReturn() - Establece el valor de retorno
