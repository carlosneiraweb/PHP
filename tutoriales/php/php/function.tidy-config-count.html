tidy_config_count

(PHP 5, PHP 7, PECL tidy >= 0.5.2)

tidy_config_count — Devuelve el número de errores de configuración Tidy encontrados en un documento dado

Descripción

int tidy_config_count ( tidy $object )

Devuelve el número de errores encontrados en la configuración del objeto object tidy especificado.

Parámetros

object

El objeto Tidy.

Valores devueltos

Devuelve el número de errores.

Ejemplos

Ejemplo #1 Ejemplo de la función tidy_config_count()

<?php
$html = '<p>test</I>';

$config = array('doctype' => 'bogus');

$tidy = tidy_parse_string($html, $config);

/* Saldrá 1, porque 'bogus' no es un doctype válido */
echo tidy_config_count($tidy);
?>
