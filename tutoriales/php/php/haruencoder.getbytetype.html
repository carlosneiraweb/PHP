HaruEncoder::getByteType

(PECL haru >= 0.0.1)

HaruEncoder::getByteType — Obtiene el tipo de byte en el texto

Descripción

int HaruEncoder::getByteType ( string $text , int $index )

Obtiene el tipo de byte en el texto.

Parámetros

text

El texto.

index

La posición en el texto.

Valores devueltos

Obtiene el tipo de byte en el texto en la posición especificada. El resultado es uno de los siguientes valores:

  • HaruEncoder::BYTE_TYPE_SINGLE - sencillo carácter de byte.
  • HaruEncoder::BYTE_TYPE_LEAD - byte principal de un carácter de doble byte.
  • HaruEncoder::BYTE_TYPE_TRAIL - byte final de un carácter de doble byte.
  • HaruEncoder::BYTE_TYPE_UNKNOWN - codificador invalido o no puede detectar el tipo de byte.

Errores/Excepciones

Lanza una HaruException cuando se produce un error.
