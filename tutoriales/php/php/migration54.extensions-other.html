Otros cambios a extensiones

Cambios en el comportamiento de la extensión, y nuevas características:

  • mysqli - mysqli_result ahora implementa Traversable
  • pdo_mysql - Eliminado el soporte para la vinculación de bibliotecas cliente de MySQL anteriores que 4.1
  • Las extensiones de MySQL mysql, mysqli y PDO_mysql utilizan ahora mysqlnd como la biblioteca predeterminada. Aún es posible usar libmysqlclient especificando una ruta en las opciones de configuración.
  • mysqlnd - Se ha añadido soporte para tuberías con nombre
