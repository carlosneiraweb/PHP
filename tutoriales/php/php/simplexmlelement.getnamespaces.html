SimpleXMLElement::getNamespaces

(PHP 5 >= 5.1.2, PHP 7)

SimpleXMLElement::getNamespaces — Retorna los namespaces usados en el documento

Descripción

public array SimpleXMLElement::getNamespaces ([ bool $recursive = false ] )

Retorna los namespaces usados en el documento

Parámetros

recursive

Si se especifica, retorna todos los namespaces usados en los nodos padre e hijos. De lo contrario, retorna sólo los namespaces usados en el nodo raíz.

Valores devueltos

El método getNamespaces retorna un array de nombres de namespaces con sus URIs asociadas.

Ejemplos

Ejemplo #1 Obtener el namespace del documento en uso

<?php

$xml = <<<XML
<?xml version="1.0" standalone="yes"?>
<gente xmlns:p="http://example.org/ns" xmlns:t="http://example.org/test">
    <p:persona id="1">John Doe</p:persona>
    <p:persona id="2">Susie Q. Public</p:persona>
</gente>
XML;
 
$sxe = new SimpleXMLElement($xml);

$namespaces = $sxe->getNamespaces(true);
var_dump($namespaces);

?>

El resultado del ejemplo sería:

array(1) {
  ["p"]=>
  string(21) "http://example.org/ns"
}
