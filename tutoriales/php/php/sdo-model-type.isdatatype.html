SDO_Model_Type::isDataType — Comprobar si este SDO_Model_Type es un tipo de datos primitivo

Descripción

bool SDO_Model_Type::isDataType ( void )
Advertencia

Esta función ha sido declarada EXPERIMENTAL. Su funcionamiento, nombre y la documentación que le acompaña puede cambiar sin previo aviso en futuras versiones de PHP. Utilícela bajo su propia responsabilidad.

Comprueba si este SDO_Model_Type es un tipo de datos primitivo. Devuelve TRUE si este tipo es un tipo de datos primitivo, si no, devuelve FALSE.

Parámetros

Ninguno.

Valores devueltos

Devuelve TRUE si este tipo es un tipo de datos primitivo, si no, devuelve FALSE.
