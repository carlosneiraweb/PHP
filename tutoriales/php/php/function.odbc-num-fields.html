odbc_num_fields

(PHP 4, PHP 5, PHP 7)

odbc_num_fields — Número de columnas de un resultado

Descripción

int odbc_num_fields ( resource $result_id )

Obtiene el número de campos (columnas) de un resultado ODBC.

Parámetros

result_id

El identificador de resultado devuelto por odbc_exec().

Valores devueltos

Devuelve el número de campos, o -1 si se produjo un error.
