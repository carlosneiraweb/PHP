HaruDoc::setPageMode

(PECL haru >= 0.0.1)

HaruDoc::setPageMode — Indica cómo se debe mostrar el documento

Descripción

bool HaruDoc::setPageMode ( int $mode )

Indica como se debe mostrar el documento.

Parámetros

mode

Lista de valores aceptados:

  • HaruDoc::PAGE_MODE_USE_NONE - muestra el documento sin marco ni miniaturas.
  • HaruDoc::PAGE_MODE_USE_OUTLINE - muestra el documento con marco.
  • HaruDoc::PAGE_MODE_USE_THUMBS - muestra el documento con panel de miniaturas.
  • HaruDoc::PAGE_MODE_FULL_SCREEN - muestra el documento en modo pantalla completa.

Valores devueltos

En caso de éxito, devuelve TRUE.

Errores/Excepciones

Lanza una HaruException cuando se produce un error.
