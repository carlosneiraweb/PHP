ReflectionClass::getFileName

(PHP 5, PHP 7)

ReflectionClass::getFileName — Obtiene el nombre de fichero en el que está definida la clase

Descripción

public string ReflectionClass::getFileName ( void )

Devuelve el nombre del fichero en el que está definida la clase.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve el nombre del fichero en el que está definida la clase. Si la clase estuviera definida en el nucleo de PHP o en una extensión de PHP, se devolvería FALSE.
