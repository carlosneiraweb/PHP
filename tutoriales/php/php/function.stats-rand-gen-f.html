stats_rand_gen_f

(PECL stats >= 1.0.0)

stats_rand_gen_f — Genera una desviación aleatoria

Descripción

float stats_rand_gen_f ( float $dfn , float $dfd )

Genera una desviación aleatoria desde la distribución F (ratio de varianza) con "dfn" grados de libertad en el numerador y "dfd" grados de libertad en el denominador. Método : genera directamente el ratio de las variaciones ji-cuadrado.

Parámetros

dfn

dfd

Valores devueltos

stats_rand_gen_exponential
stats_rand_gen_funiform
