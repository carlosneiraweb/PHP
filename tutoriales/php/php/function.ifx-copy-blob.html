ifx_copy_blob

(PHP 4, PHP <=5.2.0)

ifx_copy_blob — Duplica el objeto blob

Descripción

int ifx_copy_blob ( int $bid )

Duplica el objeto blob dado.

Parámetros

bid

Un identificador BLOB.

Valores devueltos

Devuelve el id del objeto blob, o FALSE en caso de error.
