dbplus_first

(PHP 4 <= 4.1.0, PECL dbplus >= 0.9)

dbplus_first — Obtener la primera tupla de una relación

Descripción

int dbplus_first ( resource $relation , array &$tuple )

Lee la información de la primera tupla de relation, la hace la actual y la devuelve como una matriz asociativa en tuple.

Parámetros

relation

Una relación abierta mediante dbplus_open().

tuple

Valores devueltos

Devuelve DBPLUS_ERR_NOERR si se tuvo éxito o un código de error de db++ si falló.

Notas

Advertencia

Esta función ha sido declarada EXPERIMENTAL. Su funcionamiento, nombre y la documentación que le acompaña puede cambiar sin previo aviso en futuras versiones de PHP. Utilícela bajo su propia responsabilidad.
