ArrayObject::setIteratorClass

(PHP 5 >= 5.1.0, PHP 7)

ArrayObject::setIteratorClass — Establece el nombre de clase iterador para ArrayObject

Descripción

public void ArrayObject::setIteratorClass ( string $iterator_class )

Establecer el nombre de clase iterador que es usado por ArrayObject::getIterator().

Parámetros

iterator_class

El nombre de clase de el array iterador a usar cuando se itera sobre este objeto.

Valores devueltos

No devuelve ningún valor.

Ejemplos

Ejemplo #1 Ejemplo de ArrayObject::setIteratorClass()

<?php
// ArrayIterator personalizado (hereda de ArrayIterator)
class MiArrayIterator extends ArrayIterator {
    // custom implementation
}

// Array de frutas
$fruits = array("limones" => 1, "naranjas" => 4, "bananas" => 5, "manzanas" => 10);

$fruitsArrayObject = new ArrayObject($fruits);

// Establece el nombre de clase del iterador nuevamente.
$fruitsArrayObject->setIteratorClass('MiArrayIterator');
print_r($fruitsArrayObject->getIterator());

?>

El resultado del ejemplo sería:

MyArrayIterator Object
(
    [limones] => 1
    [naranjas] => 4
    [bananas] => 5
    [manzanas] => 10
)
