Introducción

La extensión SQLSRV permite acceder a bases de datos de Microsoft SQL Server y SQL Azure cuando se ejecuta PHP sobre Windows. La versión 3.0 del controlador soporta SQL Server, empezando con SQL Server 2005, incluyendo SQL Server 2012 y SQL Server 2012 LocalDB. (Para más información sobre LocalDB, véase » Controlador de PHP para soporte de SQL Server para LocalDB y » SQL Server 2012 Express LocalDB.)

La extensión SQLSRV está soportada por Microsoft y está disponible para descargala aquí: » http://msdn.microsoft.com/en-us/sqlserver/ff657782.aspx. SQL Server 2012 LocalDB puede descargarse aquí: » http://go.microsoft.com/fwlink/?LinkID=237665.
