XMLReader::getAttribute

(PHP 5 >= 5.1.0, PHP 7)

XMLReader::getAttribute — Obtiener el valor del atributo nombrado

Descripción

public string XMLReader::getAttribute ( string $name )

Devuelve el valor del atributo nombrado o NULL si el atributo no existe o no está posicionado en un eleménto del nodo.

Parámetros

name

El nombre del atributo.

Valores devueltos

El valor del atributo, o NULL si no se encuetra un atributo con el nombre dado por name o no está posicionado en un nodo de elemento.

Historial de cambios

Versión Descripción
5.1.3 Devuelve NULL si no se encuentra ningún atributo. Antes, devolvía una cadena de texto vacía.
