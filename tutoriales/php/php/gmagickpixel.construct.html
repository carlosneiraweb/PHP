GmagickPixel::__construct

(PECL gmagick >= Unknown)

GmagickPixel::__construct — El constructor GmagickPixel

Descripción

public GmagickPixel::__construct ([ string $color ] )

Construye un objeto GmagickPixel. Si se especifica un color, el objeto se construye y se inicializa con el color antes de ser devuelto.

Parámetros

color

La cadena de color opcional que se va a usar como el valor inicial de este objeto.

Valores devueltos

El objeto GmagickPixel si se tuvo éxito.
