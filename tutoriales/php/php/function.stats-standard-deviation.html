stats_standard_deviation

(PECL stats >= 1.0.0)

stats_standard_deviation — Devuelve la desviación estándar

Descripción

float stats_standard_deviation ( array $a [, bool $sample = false ] )

Parámetros

a

El array de datos de los que calcular la desviación estándar. Observed que todos los valores del array serán convertidos a float.

sample

Indica si a representa una muestra de la población; por omisión es FALSE.

Valores devueltos

Devuelve la desviación estándar en caso de éxito; FALSE en caso de error.

Errores/Excepciones

Emite un error de nivel E_WARNING cuando hay menos de dos valores en a.
