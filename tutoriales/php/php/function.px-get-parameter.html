px_get_parameter

(PECL paradox >= 1.1.0)

px_get_parameter — Obtener un parámetro

Descripción

string px_get_parameter ( resource $pxdoc , string $name )

Obtiene varios parámetros.

Parámetros

pxdoc

Identificador de recursos de la base de datos paradox devuelto por la función px_new().

name

El parámetro name puede ser uno de los siguientes:

tablename

El nombre de la tabla, ya que será almacenado en el encabezado de base de datos.

targetencoding

La codificación de la salida. Los datos que son leidos desde los campos de caracteres con la función px_get_record() o con la función px_retrieve_record() se recodifican en la codificación de destino. Si no se establece, entonces los datos se entregarán como fueron almacenados en el archivo de base de datos.

inputencoding

La codificación de los datos de entrada que se va a guardar en la base de datos. Al almacenar los datos de los campos de caracteres en la base de datos, se espera que los datos sean entregados en esta codificación.

Valores devueltos

Devuelve el valor del parámetro o FALSE en caso de error.
