PDF_close

(PHP 4, PECL pdflib >= 1.0.0)

PDF_close — Cerrar un recurso pdf [obsoleta]

Descripción

bool PDF_close ( resource $p )

Cierra el archivo PDF generado y libera todos los recursos del documento relacionados. Devuelve TRUE en caso de éxito o FALSE en caso de error.

Esta función está obsoleta desde la versión 6 de PDFlib, use PDF_end_document() en su lugar.
