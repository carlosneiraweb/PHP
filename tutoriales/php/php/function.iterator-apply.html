iterator_apply

(PHP 5 >= 5.1.0, PHP 7)

iterator_apply — Llama una función para cada elemento en un iterador

Descripción

int iterator_apply ( Traversable $iterator , callable $function [, array $args ] )

Llamar una función para cada elemento en un iterador.

Parámetros

iterator

La clase a ser iterada.

function

La función de llamada de recorno a llamar para cada elemento.

Nota: Esta función debe retornar TRUE para con el fin de seguir con la iterando el iterator.

args

Argumentos que van a ser pasados a la función de llamada de retorno.

Valores devueltos

Devuelve el número de iteraciones.

Ejemplos

Ejemplo #1 Ejemplo de iterator_apply()

<?php
function print_caps(Iterator $iterator) {
    echo strtoupper($iterator->current()) . "\n";
    return TRUE;
}

$it = new ArrayIterator(array("Manzanas", "Bananas", "Cerezas"));
iterator_apply($it, "print_caps", array($it));
?>

El resultado del ejemplo sería:

MANZANAS
BANANAS
CEREZAS
