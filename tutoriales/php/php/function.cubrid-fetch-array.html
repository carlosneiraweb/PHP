cubrid_fetch_array

(PECL CUBRID >=8.3.0)

cubrid_fetch_array — Extrae una fila como array asociativo, numérico o ambos

Descripción

array cubrid_fetch_array ( resource $result [, int $type = CUBRID_BOTH ] )

La función cubrid_fetch_array() se utiliza para consultar una única fila del resultado de la consulta, y lo devuelve como array. El cursor avanza automáticamente a la siguiente fila del resultado.

Parámetros

result

result proviene de una llamada a la función cubrid_execute()

type

Tipo de array del resultado obtenido: CUBRID_NUM, CUBRID_ASSOC, CUBRID_BOTH, CUBRID_OBJECT. Si se operan con objetos lob, se puede usar CUBRID_LOB.

Valores devueltos

Devuelve un array de cadenas que se corresponden a la fila consultada, cuando ésta tenga éxito.

FALSE, cuando no existen más filas; NULL, cuando el proceso no tiene éxito.

El tipo del array devuelto se definirá dependiendo de cómo se haya solicitado. Usando CUBRID_BOTH (por omisión), se obtendrá un array con índices numéricos y asociativos, y se podrá decidir qué tipo de datos usar mediante el parámetro type. En la variable type puede asignarse uno de los siguientes valores:

  • CUBRID_NUM : Array numérico (basado en 0)
  • CUBRID_ASSOC : Array asociativo
  • CUBRID_BOTH : Array numérico y asociativo (predeterminado)

Ejemplos

Ejemplo #1 Ejemplo de cubrid_fetch_array()

<?php
$conexión = cubrid_connect("localhost", 33000, "demodb");
$solicitud = cubrid_execute($conexión, "SELECT name,area,seats,address FROM stadium WHERE nation_code='GRE' AND seats > 10000");

printf("%-40s %-10s %-6s %-20s\n", "name", "area", "seats", "address");
while ($fila = cubrid_fetch_array($solicitud, CUBRID_NUM)) {
    printf("%-40s %-10s %-6s %-20s\n", $fila[0], $fila[1], $fila[2], $fila[3]);
}

// si se operan con objetos lob, se puede usar cubrid_fetch_array($solicitud, CUBRID_NUM | CUBRID_LOB)

cubrid_close_request($solicitud);

cubrid_disconnect($conexión);
?>
