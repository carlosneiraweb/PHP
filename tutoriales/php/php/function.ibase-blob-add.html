ibase_blob_add

(PHP 5, PHP 7)

ibase_blob_add — Añade datos a un nuevo blob

Descripción

void ibase_blob_add ( resource $blob_handle , string $data )

ibase_blob_add() añade datos a un blob creado con ibase_blob_create().

Parámetros

blob_handle

Manejador de blob, abierto con ibase_blob_create().

data

Datos a añadir.

Valores devueltos

No devuelve ningún valor.

Ver también

  • ibase_blob_cancel() - Cancela la creación de un blob
  • ibase_blob_close() - Cierra un blob
  • ibase_blob_create() - Crea un nuevo blob al que añadir datos
  • ibase_blob_import() - Crea un blob, copia un fichero en él, y lo cierra
