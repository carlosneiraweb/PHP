is_nan

(PHP 4 >= 4.2.0, PHP 5, PHP 7)

is_nan — Encuentra si un valor no es un número

Descripción

bool is_nan ( float $val )

Verifica si val no es un numero, como por ejemplo el resultado de acos(1.01).

Parámetros

val

El valor a revisar

Valores devueltos

Devuelve TRUE si val no es un número, en caso contrario devuelve FALSE.

Ejemplos

Ejemplo #1 Ejemplo de is_nan()

<?php
// Invalid calculation, will return a 
// NaN value
$nan = acos(8);

var_dump($nan, is_nan($nan));
?>

El resultado del ejemplo sería:

float(NAN)
bool(true)
