dbplus_lockrel

(PHP 4 <= 4.1.0, PECL dbplus >= 0.9)

dbplus_lockrel — Solictar un bloqueo de escritura en una relación

Descripción

int dbplus_lockrel ( resource $relation )

Solicita un bloqueo de escritura en la relación relation dada.

Los demás clientes pueden seguir consultando la relación, pero no pueden alterarla mientras esté bloqueada.

Parámetros

relation

Una relación abierta con dbplus_open().

Advertencia
