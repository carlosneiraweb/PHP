SDO_DAS_XML::createDataObject

(^)

SDO_DAS_XML::createDataObject — Crea un objeto SDO_DataObject dados un URI de espacio de nombres y un nombre de tipo

Descripción

SDO_DataObject SDO_DAS_XML::createDataObject ( string $namespace_uri , string $type_name )

Crea un objeto SDO_DataObject dados un URI de espacio de nombres y un nombre de tipo. El tipo debería estar definido en el modelo subyacente, o de lo contrario se lanzará una SDO_TypeNotFoundException.

Parámetros

namespace_uri

El URI del espacion de nombres del nombre del tipo.

type_name

El nombre del tipo.

Valores devueltos

Devuelve un objeto SDO_DataObject en caso de éxito.

Errores/Excepciones

SDO_TypeNotFoundException

Lanzada si un tipo no está definido en el modelo subyacente.
