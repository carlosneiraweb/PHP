IntlChar::isalnum

(PHP 7)

IntlChar::isalnum — Comporbar si un punto de código es un carácter alfanumérico

Descripción

public static bool IntlChar::isalnum ( mixed $codepoint )

Determina si el punto de código especificado es un carácter alfanumérico (letra o dígito). TRUE para caracteres con las categorías generales "L" (letras) y "Nd" (números de dígito decimal).

Parámetros

codepoint

El valor de tipo integer del punto de código (p.ej. 0x2603 para U+2603 SNOWMAN), o el carácter codificado como un string UTF-8 (p.ej. "\u{2603}")

Valores devueltos

Devuelve TRUE si codepoint es un caracter alfanumérico, FALSE si no.

Ejemplos

Ejemplo #1 Probar diferentes puntos de código

<?php
var_dump(IntlChar::isalnum("A"));
var_dump(IntlChar::isalnum("1"));
var_dump(IntlChar::isalnum("\u{2603}"));
?>

El resultado del ejemplo sería:

bool(true)
bool(true)
bool(false)
