HaruPage::moveTo

(PECL haru >= 0.0.1)

HaruPage::moveTo — Establece el punto de inicio para la nueva ruta de dibujo

Descripción

bool HaruPage::moveTo ( float $x , float $y )

Define el punto de inicio para la nueva ruta de dibujo.

Parámetros

x

Una nueva coordenada del punto inicial.

y

Una nueva coordenada del punto inicial.

Valores devueltos

Devuelve TRUE en éxito.

Errores/Excepciones

Lanza una HaruException cuando se produce un error.
