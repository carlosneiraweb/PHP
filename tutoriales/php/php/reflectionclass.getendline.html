ReflectionClass::getEndLine

(PHP 5, PHP 7)

ReflectionClass::getEndLine — Consultar línea final

Descripción

public int ReflectionClass::getEndLine ( void )

Devuelve el número de la línea final de una clase definida por el usuario.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Número de la línea final de la clase definida por el usuario, o FALSE si fuera desconocida.

Ejemplos

Ejemplo #1 Ejemplo de ReflectionClass::getEndLine()

<?php
// Clase de prueba
class ClaseDePrueba { }

$rc = new ReflectionClass('ClaseDePrueba');

echo $rc->getEndLine();
?>

El resultado del ejemplo sería:

3
