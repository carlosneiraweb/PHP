OCI-Collection::getElem

(PHP 5, PHP 7, PECL OCI8 >= 1.1.0)

OCI-Collection::getElem — Devuelve el valor de un elemento

Descripción

mixed OCI-Collection::getElem ( int $index )

Devuelve el valor del elemento cuyo índice es index (basado en 0).

Parámetros

index

El índice del elemento. El primer índice es 0.

Valores devueltos

Devuelve FALSE si el elemento no existe; NULL si el elemento es NULL; un string si el elemento es una columna del tipo de datos string, o un número si es un campo numérico.
