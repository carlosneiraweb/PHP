SAMConnection::commit

(PECL sam >= 0.1.0)

SAMConnection::commit — Hace el commit (consolida) la unidad de trabajo actual.

Descripción

bool SAMConnection::commit ( void )

La llamada al método commit en un objeto Connection, hace el commit (consolida) todas las transacciones pendientes de ser consolidadas y que forman parte de la unidad de trabajo actual.

Valores devueltos

Este método devuelve FALSE en caso de error.

Ejemplos

Ejemplo #1 Consolidando la unidad de trabajo actual

<?php
  if (!$conn->commit()) {
    // The commit failed!
    echo "Commit failed ($conn->errno) $conn->error";
  }
?>
