ibase_affected_rows

(PHP 5, PHP 7)

ibase_affected_rows — Devuelve el número de columnas afectadas por la última consulta

Descripción

int ibase_affected_rows ([ resource $link_identifier ] )

Esta función devuelve el número de filas que se vieron afectadas por la última consulta (INSERT, UPDATE o DELETE) que se ejecutó en el contexto de la transacción especificada.

Parámetros

link_identifier

Contexto de transacción. Si link_identifier fuera un recurso de conexión, se utilizará la transacción por omisión.

Valores devueltos

Devuelve el número de filas en forma de número entero.

Ver también

  • ibase_query() - Ejecutar una consulta en una base de datos InterBase
  • ibase_execute() - Ejecutar una consulta previamente preparada
