mssql_connect

(PHP 4, PHP 5, PECL odbtp >= 1.1.1)

mssql_connect — Establece una conexión a un servidor MS SQL

Advertencia

Esta función está OBSOLETA en PHP 7.0.0.

Las alternativas a esta función son:

  • PDO::__construct()
  • sqlsrv_connect()
  • odbc_connect()

Descripción

resource mssql_connect ([ string $servername [, string $username [, string $password [, bool $new_link = false ]]]] )

mssql_connect() establece una conexión a un servidor MS SQL.

La conexión al servidor se cerrará en cuanto finalice la ejecución del script, a menos que ésta sea cerrada antes de forma explícita llamando a la función mssql_close().

Parámetros

servername

El servidor MS SQL. También puede incluir un número de puerto, p.e. hostname:port (Linux), o hostname,port (Windows).

username

El nombre de usuario.

password

La contraseña.

new_link

Si se hace una segunda llamada a la función mssql_connect() con los mismos argumentos, no se establecerá una nueva conexión, pero en vez de eso, devolverá el identificador de la conexión que ya se había abierto. Este parámetro modifica el comportamiento y hace que mssql_connect() siempre abra una nueva conexión, incluso si mssql_connect() se llamó anteriomente con los mismos parámetros.

Valores devueltos

Devuelve un identificador de conexión a MS SQL en caso de éxito, o FALSE en caso de error.

Historial de cambios

Versión Descripción
5.1.0 Se añadió el parámetro new_link .

Ejemplos

Ejemplo #1 Ejemplo con mssql_connect()

<?php
// El servidor con el formato: <computer>\<instance name> o 
// <server>,<port> cuando se use un número de puerto diferente del de defecto
$server = 'KALLESPC\SQLEXPRESS';

// Connect to MSSQL
$link = mssql_connect($server, 'sa', 'phpfi');

if (!$link) {
    die('Algo fue mal mientras se conectaba a MSSQL');
}
?>
