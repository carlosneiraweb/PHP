px_set_parameter

(PECL paradox >= 1.1.0)

px_set_parameter — Establecer un parámetro

Descripción

bool px_set_parameter ( resource $pxdoc , string $name , string $value )

Establece varios parámetros.

Parámetros

pxdoc

Identificador de recursos de la base de datos paradox devuelto por la función px_new().

name

En función del parámetro que desee establecer, el parámetro name puede ser uno de los siguientes.

tablename

El nombre de la tabla que se almacenará en la cabecera de la base de datos.

targetencoding

La codificación de salida. Los datos que sean leidos desde los campos de caracteres se recodifican en la codificación de destino.

inputencoding

La codificación de los datos de entrada que se va a guardar en la base de datos.

value

El valor del parámetro a establecer. Para la codificación de entrada y codificación de destino este debe ser el nombre de la codificación tal como la entiende iconv o recodificada, por ejemplo iso-8859-1, utf-8, cp850.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.
