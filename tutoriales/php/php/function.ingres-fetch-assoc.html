ingres_fetch_assoc

(PECL ingres >= 2.2.2)

ingres_fetch_assoc — Obtienen una fila de resultados en un array asociativo

Descripción

array ingres_fetch_assoc ( resource $result )

Esta función almacena los datos obtenidos desde un consulta ejecutada usando ingres_query() en un array asociativo, usando los nombres de los campos como claves.

Con respecto a la velocidad, la función es idéntica a ingres_fetch_object(), y casi tan rápida como ingres_fetch_row() (la diferencia es indignificante).

De manera predeterminada, los arrays creados por ingres_fetch_assoc() empiezan en la posición 1, y no en la 0 como en otras extensiones de SGBDs. La posición de inicio se puede ajustar a 0 usando el parámetro de configuración ingres.array_index_start.

Nota: Configuraciones relacionadas

Véanse también las directivas ingres.array_index_start, ingres.fetch_buffer_size y ingres.utf8 de la sección Configuración en tiempo de ejecución.

Parámetros

result

El identificador de resultados de la consulta

Valores devueltos

Devuelve un array asociativo que se corresponde con la fila obtenida, o FALSE si no existen más filas

Ejemplos

Ejemplo #1 Obtener una fila en un array asociativo

<?php
$enlace = ingres_connect($base_datos, $usuario, $contraseña);

$resultado = ingres_query($enlace,"select * from table");
while ($fila = ingres_fetch_assoc($resultado)) {
    echo $fila["user_id"];  // usar un array asociativo
    echo $fila["fullname"];
}
?>
