cubrid_lob2_size64

(PECL CUBRID >= 8.4.1)

cubrid_lob2_size64 — Obtiene el tamaño de un objeto LOB

Descripción

string cubrid_lob2_size64 ( resource $lob_identifier )

La función cubrid_lob2_size64() se usa para obtener el tamaño de un objeto LOB. Si el tamaño del objeto LOB es mayor que los datos de tipo integer que se pueden almacenar, se puede usar esta función para devolver el tamaño como string.

Parámetros

lob_identifier

Un identificador de LOB, como un resultado de cubrid_lob2_new() u obtenido de un conjunto de resultados.

Valores devueltos

Devuelve el tamaño del objeto LOB como string cuando el proceso tiene éxito.

FALSE en caso de error.
