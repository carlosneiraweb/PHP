tidy_reset_config

(PECL tidy >= 0.7.0)

tidy_reset_config — Restaura la configuración Tidy a sus valores por omisión

Descripción

bool tidy_reset_config ( void )

Esta función restaura la configuración de Tidy a sus valores por omisión.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.

Notas

Nota: Esta función está disponible sólo en Tidy 1.0. Quedó obsoleta en Tidy 2.0, y por tanto ha sido eliminada.
