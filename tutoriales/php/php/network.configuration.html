He aquí una breve explicación de las directivas de configuración.

define_syslog_variables boolean

Si define o no las diversas variables syslog (por ej., $LOG_PID, $LOG_CRON, etc.). Setearla a off es una buena idea en cuanto a rendimiento se trate. Al momento de ejecución, se pueden definir estas variables al invocar a la función define_syslog_variables().
