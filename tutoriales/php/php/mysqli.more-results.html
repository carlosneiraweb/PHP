mysqli::more_results

mysqli_more_results

(PHP 5, PHP 7)

mysqli::more_results -- mysqli_more_results — Comprueba si hay más resultados de una multi consulta

Descripción

Estilo orientado a objetos

bool mysqli::more_results ( void )

Estilo por procedimientos

bool mysqli_more_results ( mysqli $link )

Indica si hay disponibles algún juego de resultados de alguna llamada previa a mysqli_multi_query().

Parámetros

link

Sólo estilo por procediminetos: Un identificador de enlace devuelto por mysqli_connect() o mysqli_init()

Valores devueltos

Devuelve TRUE si uno o más conjuntos de resultados están disponibles desde una llamada anterior a mysqli_multi_query(), o de lo contrario devuelve FALSE.

Ejemplos

See mysqli_multi_query().

  • mysqli_store_result() - Transfiere un conjunto de resulados de la última consulta
  • mysqli_use_result() - Inicia la recuperación de un conjunto de resultados
