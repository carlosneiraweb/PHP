cyrus_connect

(PHP 4 >= 4.1.0, PECL cyrus 1.0)

cyrus_connect — Conecta a un servidor IMAP Cyrus

Descripción

resource cyrus_connect ([ string $host [, string $port [, int $flags ]]] )

Conectar a un servidor IMAP Cyrus.

Parámetros

host

El nombre del servidor IMAP Cyrus.

port

El número del puerto.

flags

Valores devueltos

Devuelve un controlador de la conexión en caso de éxito o FALSE en caso de error.
