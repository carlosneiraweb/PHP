ReflectionClass::hasProperty

(PHP 5 >= 5.1.0, PHP 7)

ReflectionClass::hasProperty — Comprueba si una propiedad está o no definida

Descripción

public bool ReflectionClass::hasProperty ( string $name )

Comprueba si la propiedad especificada está o no definida.

Parámetros

name

Nombre de la propiedad que se desea comprobar.

Valores devueltos

TRUE si la propiedad está definida, y en cualquier otro caso FALSE

Ejemplos

Ejemplo #1 Ejemplo de ReflectionClass::hasProperty()

<?php
class Foo {
    public    $p1;
    protected $p2;
    private   $p3;

}

$obj = new ReflectionObject(new Foo());

var_dump($obj->hasProperty("p1"));
var_dump($obj->hasProperty("p2"));
var_dump($obj->hasProperty("p3"));
var_dump($obj->hasProperty("p4"));
?>

El resultado del ejemplo sería algo similar a:

bool(true)
bool(true)
bool(true)
bool(false)
