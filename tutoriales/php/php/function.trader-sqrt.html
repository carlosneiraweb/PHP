trader_sqrt

(PECL trader >= 0.2.0)

trader_sqrt — Raíz cuadrada de vectores

Descripción

array trader_sqrt ( array $real )

Calcula la raíz cuadrada de cada valor de real y devuelve el array resultante.

Parámetros

real

Array de valores reales.

Valores devueltos

Devuelve un array con los datos calculados o false en caso de error.
