sqlite_key

SQLiteResult::key

(PHP 5 >= 5.1.0 and < 5.4.0)

sqlite_key -- SQLiteResult::key — Devuelve el índice de la fila actual

Descripción

Estilo orientado a objetos (método):

int SQLiteResult::key ( void )

SQLiteResult::sqlite_key() devuelve el índice de la fila actual del conjunto de resultados almacenado en búfer dado por result.

A diferencia de otras funciones de SQLite, esta no posee una versión procedimental, por lo que solamente puede ser invocada como método de un objeto SQLiteResult.

Valores devueltos

Devuelve el índice de la fila actual del conjunto de resultados almacenado en buffer result.

Ver también

  • sqlite_next() - Busca el número de fila siguiente
  • sqlite_current() - Obtiene la fila actual como un array de un conjunto de resultado
  • sqlite_rewind() - Busca el primer número de fila
