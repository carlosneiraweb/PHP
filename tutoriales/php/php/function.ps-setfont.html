ps_setfont

(PECL ps >= 1.1.0)

ps_setfont — Establecer la fuente a usar para la siguiente impresión

Descripción

bool ps_setfont ( resource $psdoc , int $fontid , float $size )

Establece una fuente, la cual ha de ser cargada antes con la función ps_findfont(). Imprimir texto sin establecer una fuente resultará en un error.

Parámetros

psdoc

El identificador de recursos del fichero postscript, como el devuelto por la función ps_new().

fontid

El identificador de la fuente, como el devuelto por la función ps_findfont().

size

El tamaño de la fuente.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.

Ver también

  • ps_findfont() - Cargar una fuente
