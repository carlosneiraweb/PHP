Imagick::animateImages — Anima una imagen o imágenes

Descripción

bool Imagick::animateImages ( string $x_server )

Este método anima la imagen en un servidor X local o remoto. Este método no está disponible en Windows. Este método está disponible si Imagick ha sido compilado con la versión 6.3.6 o superior de ImageMagick.

Parámetros

x_server

La dirección del servidor X

Valores devueltos

Devuelve TRUE en caso de éxito.
