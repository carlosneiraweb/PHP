fdf_open

(PHP 4, PHP 5, PHP 7)

fdf_open — Abrir un documento FDF

Descripción

resource fdf_open ( string $filename )

Abre un archivo con datos de formulario.

También es posible usar fdf_open_string() para procesar los resultados de una petición POST de formulario PDF.

Parámetros

filename

Ruta al archivo FDF. Este archivo debe contener los datos devueltos de un formulario PDF creado usando fdf_create() y fdf_save().

Valores devueltos

Devuelve un gestor de documento FDF, o FALSE en caso de error.

Ejemplos

Ejemplo #1 Acceso a datos de formulario

<?php
// Guardar los datos FDF en un archivo temporal
$aa_fdf = fopen("prueba.fdf", "w");
fwrite($aa_fdf, $HTTP_FDF_DATA, strlen($HTTP_FDF_DATA));
fclose($aa_fdf);

// Abrir el archivo temporal y evaluar los datos
$fdf = fdf_open("prueba.fdf");
/* ... */
fdf_close($fdf);
?>
