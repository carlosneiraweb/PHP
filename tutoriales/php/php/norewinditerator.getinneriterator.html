NoRewindIterator::getInnerIterator

(PHP 5 >= 5.1.0, PHP 7)

NoRewindIterator::getInnerIterator — Obtiene el iterador interno

Descripción

public iterator NoRewindIterator::getInnerIterator ( void )

Obtiene el iterador interno, que fué pasado a NoRewindIterator.

Advertencia

Esta función no está documentada actualmente, solamente se encuentra disponible la lista de parámetros.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

El iterador interno, como fué pasado a NoRewindIterator::__construct().
