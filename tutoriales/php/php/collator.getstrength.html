Collator::getStrength

collator_get_strength

(PHP 5 >= 5.3.0, PHP 7, PECL intl >= 1.0.0)

Collator::getStrength -- collator_get_strength — Obtener la fuerza de cotejo actual

Descripción

Estilo orientado a objetos

public int Collator::getStrength ( void )

Estilo por procedimientos

int collator_get_strength ( Collator $coll )

Parámetros

coll

Un objeto Collator.

Valores devueltos

Devuelve la fuerza de cotejo actual, o FALSE de tipo boolean en caso de error.

Ejemplos

Ejemplo #1 Ejemplo de collator_get_strength()

<?php
$coll     = collator_create( 'en_US' );
$strength = collator_get_strength( $coll );
?>
