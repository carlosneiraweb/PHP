SWFTextField::setLineSpacing

(PHP 5 <= 5.3.0, PECL ming SVN)

SWFTextField::setLineSpacing — Establece el espaciado entre líneas de un campo de texto

Descripción

void SWFTextField::setLineSpacing ( float $height )
Advertencia

Esta función ha sido declarada EXPERIMENTAL. Su funcionamiento, nombre y la documentación que le acompaña puede cambiar sin previo aviso en futuras versiones de PHP. Utilícela bajo su propia responsabilidad.

swftextfield::setlinespacing() establece el espaciado entre líneas de un campo de texto a la altura de height. Por omisión es 40.

Valores devueltos

No devuelve ningún valor.
