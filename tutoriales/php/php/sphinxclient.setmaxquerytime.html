SphinxClient::setMaxQueryTime

(PECL sphinx >= 0.1.0)

SphinxClient::setMaxQueryTime — Establecer el tiempo máximo de consulta

Descripción

public bool SphinxClient::setMaxQueryTime ( int $qtime )

Establece el tiempo máximo de consulta de búsqueda.

Parámetros

qtime

Tiempo de consulta máximo, en milisegundos. Debe ser un integer no negativo. El valor por defecto es 0, es decir, no hay límite.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.
