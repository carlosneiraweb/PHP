PDOStatement::setAttribute

(PHP 5 >= 5.1.0, PHP 7, PECL pdo >= 0.2.0)

PDOStatement::setAttribute — Establece un atributo de sentencia

Descripción

public bool PDOStatement::setAttribute ( int $attribute , mixed $value )

Establece un atributo de una sentencia. Actualmente, no se establecen atributos genéricos, sino solamente específicos del controlador:

  • PDO::ATTR_CURSOR_NAME (específico de Firebird y ODBC): Establece el nombre del cursor para UPDATE ... WHERE CURRENT OF.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.

Ver también

  • PDO::getAttribute() - Devuelve un atributo de la conexión a la base de datos
  • PDO::setAttribute() - Establece un atributo
  • PDOStatement::getAttribute() - Recupera un atributo de sentencia
