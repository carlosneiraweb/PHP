OAuthProvider::isRequestTokenEndpoint

(PECL OAuth >= 1.0.0)

OAuthProvider::isRequestTokenEndpoint — Establece isRequestTokenEndpoint

Descripción

public void OAuthProvider::isRequestTokenEndpoint ( bool $will_issue_request_token )

Advertencia

Esta función no está documentada actualmente, solamente se encuentra disponible la lista de parámetros.

Parámetros

will_issue_request_token

Establece si emitirá o no una emisión de petición de token, determinando si OAuthProvider::tokenHandler() necesita ser llamado.

Valores devueltos

No devuelve ningún valor.

  • OAuthProvider::setRequestTokenPath() - Establece la ruta de petición del token
