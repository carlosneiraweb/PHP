yp_first

(PHP 4, PHP 5 <= 5.0.5)

yp_first — Devuelve el primer par clave-valor del mapa indicado

Descripción

array yp_first ( string $domain , string $map )

Obtiene el primer par clave-valor del map indicado en el domain dado.

Parámetros

domain

Nombre de dominio NIS.

map

El mapa NIS.

Valores devueltos

Devuelve el primer par clave-valor como un arrego si tiene éxito, o FALSE en caso de error.

Ejemplos

Ejemplo #1 Ejemplo con yp_first()

<?php
$entry = yp_first($domain, "passwd.byname");

$key = key($entry);
$value = $entry[$key];

echo "La primera entrada en el mapa tiene por clave " . $key . " y valor " . $value;
?>
