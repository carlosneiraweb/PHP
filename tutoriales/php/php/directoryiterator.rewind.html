DirectoryIterator::rewind

(PHP 5, PHP 7)

DirectoryIterator::rewind — Robobina DirectoryIterator hasta volver al inicio

Descripción

public void DirectoryIterator::rewind ( void )

Robobinar DirectoryIterator hasta volver al inicio.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

No devuelve ningún valor.

Ejemplos

Ejemplo #1 Ejemplo de DirectoryIterator::rewind()

<?php
$iterator = new DirectoryIterator(dirname(__FILE__));

$iterator->next();
echo $iterator->key(); //1

$iterator->rewind(); //Rebobinándose al inicio
echo $iterator->key(); //0
?>
