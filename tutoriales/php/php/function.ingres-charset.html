ingres_charset

(PECL ingres >= 2.1.0)

ingres_charset — Devuelve el conjunto de caracteres de la instalación

Descripción

string ingres_charset ( resource $link )

ingres_charset() se invoca para determinar el conjunto de caracteres que está usando el cliente Ingres, desde II_CHARSETxx (donce xx es el código de instalación).

Nota:

Se puede anular el valor devuelto usando la función putenv(). Cambiar el valor de II_CHARSETxx en una instalación de Ingres que se esté ejecutando puede corromper los datos.

Parámetros

link

El identificador de enlace de la conexión

Valores devueltos

Devuelve un string con el valor de II_CHARSETxx, o NULL si el valor no se pudo determinar.

Ejemplos

Ejemplo #1 ingres_charset() - Obtener el conjunto de caracteres de la instalación

<?php
$enlace = ingres_connect($base_datos, $usuario, $contraseña);

echo ingres_charset($enlace) . "\n";

ingres_close($enlace);
?>
