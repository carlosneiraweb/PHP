dbplus_freelock

(PHP 4 <= 4.1.0, PECL dbplus >= 0.9)

dbplus_freelock — Libera el bloqueo de escritura en una tupla

Descripción

int dbplus_freelock ( resource $relation , string $tuple )

Libera el bloqueo de escritura en tuple previamente obtenido mediante dbplus_getlock().

Parámetros

relation

Una relación abierta con dbplus_open().

tuple

Valores devueltos

Notas

Advertencia

Esta función ha sido declarada EXPERIMENTAL. Su funcionamiento, nombre y la documentación que le acompaña puede cambiar sin previo aviso en futuras versiones de PHP. Utilícela bajo su propia responsabilidad.
