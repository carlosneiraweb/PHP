Imagick::getSizeOffset

(PECL imagick 2.0.0)

Imagick::getSizeOffset — Devuelve el índice de tamaño

Descripción

int Imagick::getSizeOffset ( void )

Devuelve el índice de tamaño asociado con el objeto Imagick. Este método está disponible si Imagick ha sido compilado con la versión 6.2.9 o superior de ImageMagick.

Valores devueltos

Devuelve el índice de tamaño asociado con el objeto Imagick.

Errores/Excepciones

Lanza ImagickException en caso de error.
