HaruPage::setHorizontalScaling

(PECL haru >= 0.0.1)

HaruPage::setHorizontalScaling — Establece la escala horizontal para la página

Descripción

bool HaruPage::setHorizontalScaling ( float $scaling )

Define la escala horizontal para la página.

Parámetros

scaling

La escala horizontal para el texto mostrado en la página. El valor inicial es 100.

Valores devueltos

Devuelve TRUE en éxito.

Errores/Excepciones

Lanza una HaruException cuando se produce un error.

Ver también

  • HaruPage::getHorizontalScaling() - Obtiene el valor actual de la escala horizontal
