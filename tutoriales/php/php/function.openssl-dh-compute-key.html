openssl_dh_compute_key

(PHP 5 >= 5.3.11, PHP 7)

openssl_dh_compute_key — Computa el secreto compartido para un valor público de una clave DH remota y una clave DH local

Descripción

string openssl_dh_compute_key ( string $pub_key , resource $dh_key )
Advertencia

Esta función no está documentada actualmente, solamente se encuentra disponible la lista de parámetros.

Parámetros

pub_key

Clave pública

dh_key

Clave DH

Valores devueltos

Devuelve la clave computada si se tuvo éxito o FALSE en caso de error.
