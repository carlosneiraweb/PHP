ibase_blob_open

(PHP 5, PHP 7)

ibase_blob_open — Abre un blob para su consulta

Descripción

resource ibase_blob_open ( resource $link_identifier , string $blob_id )
resource ibase_blob_open ( string $blob_id )

Abre un BLOB existente para su lectura.

Parámetros

link_identifier

Identificador de enlace a InterBase. Si se omite, se utilizará el último enlace que se haya abierto.

blob_id

Identificador del BLOB.

Valores devueltos

Devuelve un manejador de BLOB para utilizar con ibase_blob_get() o FALSE en caso de error.

Ver también

  • ibase_blob_close() - Cierra un blob
  • ibase_blob_echo() - Imprime el contenido de un blob
  • ibase_blob_get() - Consulta un determinado número de bytes de un blob
