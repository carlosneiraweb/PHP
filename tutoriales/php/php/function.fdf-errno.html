fdf_errno

(PHP 4 >= 4.3.0, PHP 5, PHP 7)

fdf_errno — Devuelve el código de error de la última operación sobre fdf

Descripción

int fdf_errno ( void )

Devuelve el código de error de la última llamada a una función FDF.

Puede obtenerse una descripción textual del error usando fdf_error().

Valores devueltos

Devuelve el código de error en forma de entero, o cero si no hubo errores.
