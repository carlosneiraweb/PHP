Imagick::decipherImage — Descifra una imagen

Descripción

bool Imagick::decipherImage ( string $passphrase )

Descifra una imagen que ha sido cifrada anteriormente. La iamgen debe ser cifrada usando Imagick::encipherImage(). Este método está disponible si Imagick ha sido compilado con la versión 6.3.9 o superior de ImageMagick.

Parámetros

passphrase

La frase de contraseña

Valores devueltos

Devuelve TRUE en caso de éxito.
