DOMCharacterData::substringData

(PHP 5, PHP 7)

DOMCharacterData::substringData — Extrae un rango de data del nodo

Descripción

string DOMCharacterData::substringData ( int $offset , int $count )

Devuelve la subcadena especificada.

Parámetros

offset

Posición de inicio de la subcadena a extraer.

count

El número de carácteres a extraer.

Valores devueltos

La subcadena especificada. Si la suma de offset y count excede la longitud, entonces todas las unidades de 16-bit hasta el final de la cadena son devueltas.

Errores/Excepciones

DOM_INDEX_SIZE_ERR

Lanzado si offset es negativo o mayor que el numero de unidades de 16-bit en la data, o si count es negativo.
