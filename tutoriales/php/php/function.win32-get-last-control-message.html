win32_get_last_control_message

(PECL win32service SVN)

win32_get_last_control_message — Devuelve el último mensaje de control que fue enviado a este servicio

Descripción

int win32_get_last_control_message ( void )

Devuelve el código de control que fue el último enviado a este proceso de servicio. Cuando se ejecuta como un servicio lo debería comprobar periódicamente para determinar si su servicio necesita detenerse.

Valores devueltos

Devuelve una constante de control que puede ser una de las Constantes de Mensajes de Control de Servicio de Win32Service: WIN32_SERVICE_CONTROL_CONTINUE, WIN32_SERVICE_CONTROL_INTERROGATE, WIN32_SERVICE_CONTROL_PAUSE, WIN32_SERVICE_CONTROL_PRESHUTDOWN, WIN32_SERVICE_CONTROL_SHUTDOWN, WIN32_SERVICE_CONTROL_STOP.
