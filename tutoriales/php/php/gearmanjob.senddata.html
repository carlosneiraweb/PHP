GearmanJob::sendData

(PECL gearman >= 0.6.0)

GearmanJob::sendData — Envía datos para un trabajo en ejecución

Descripción

public bool GearmanJob::sendData ( string $data )

Envía datos al servidor de trabajos (y a cualquier cliente a la escucha) para este trabajo.

Parámetros

data

Datos arbitrarios serializados.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.

Ver también

  • GearmanJob::workload() - Obtiene la carga de trabajo
  • GearmanTask::data() - Obtiene los datos retornados para una tarea
