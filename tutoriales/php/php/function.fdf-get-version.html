fdf_get_version

(PHP 4 >= 4.3.0, PHP 5, PHP 7)

fdf_get_version — Consulta el número de versión de la API o de un fichero FDF

Descripción

string fdf_get_version ([ resource $fdf_document ] )

Devuelve la versión FDF del documento dado, o bien el número de versión de la API si no se utilizó ningún parámetro.

Parámetros

fdf_document

Manejador de documentos FDF, devuelto por fdf_create(), fdf_open() o por fdf_open_string().

Valores devueltos

Devuelve la versión en forma de string. En el sistema FDF actual 5.0 el número de versión de la API es 5.0 y el número de versión de los documentos es o bien 1.2, 1.3 o 1.4.
