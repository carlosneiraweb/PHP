MongoClient::__toString

(PECL mongo >=1.3.0)

MongoClient::__toString — Representación de cadena de esta conexión

Esta extenisón que define este método está obsoleta. En su lugar debe usarse la extensión MongoDB. No hay equivalente para este método en la nueva extensión.

Descripción

public string MongoClient::__toString ( void )

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve el nombre del host y el puerto de esta conexión.
