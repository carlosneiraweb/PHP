sybase_deadlock_retry_count

(PHP 4 >= 4.3.0, PHP 5, PHP 7)

sybase_deadlock_retry_count — Devuelve el número de reintentos en caso de bloqueo (deadlock)

Descripción

void sybase_deadlock_retry_count ( int $retry_count )

Utilizando sybase_deadlock_retry_count(), se puede definir el número de reintentos en caso de bloqueo. Por defecto, cada bloqueo se reintenta un número infinito de veces o hasta que el proceso lo interrumpte Sybase, el script que se está ejecutando se mata (por ejemplo, por set_time_limit()) o la consulta finaliza con éxito.

Parámetros

retry_count

Valores para el parámetro retry_count
-1 Reintentar infinitamente(por defecto)
0 No reintentar
n Reintentar n veces

Valores devueltos

No devuelve ningún valor.

Notas

Nota: Esta función está disponible sólo cuando se utiliza la biblioteca CT de Sybase, y no la biblioteca DB.
