fdf_set_value

(PHP 4, PHP 5, PHP 7)

fdf_set_value — Establecer el valor de un campo

Descripción

bool fdf_set_value ( resource $fdf_document , string $fieldname , mixed $value [, int $isName ] )

Establece el valor para el campo dado.

Parámetros

fdf_document

El gestor de documento FDF, devuelto por fdf_create(), fdf_open() o fdf_open_string().

fieldname

Nombre del campo FDF, como una cadena.

value

Este parámetro será almacenado como una cadena a menos que sea una matriz. En este caso todos los elementos de la matriz serán almacenados como una matriz de valores.

isName

Nota:

En versiones antiguas del conjunto de herramientas FDF, el último parámetro determinada si el valor del campo era convertido a un nombre PDF (= 1) o usado como cadena PDF (= 0).

Este valor ya no es usado en la versión actual, 5.0, del conjunto de herramientas. Por razones de compatibilidad, aun es soportado como un parámetro opcional desde PHP 4.3, pero ignorado internamente.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.

Historial de cambios

Versión Descripción
4.3.0 El soporte para matrices en el parámetro value fue agregado.
