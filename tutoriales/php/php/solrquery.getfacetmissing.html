SolrQuery::getFacetMissing

(PECL solr >= 0.9.2)

SolrQuery::getFacetMissing — Devuelve el estado acutual del parámetro facet.missing

Descripción

public bool SolrQuery::getFacetMissing ([ string $field_override ] )

Devuelve el estado acutual del parámetro facet.missing. Acepta una sobrescritura opcional de campos

Parámetros

field_override

El nombre del campo

Valores devueltos

Devuelve un booleano en caso de éxito y NULL si no se estableció
