SplFixedArray::toArray

(PHP 5 >= 5.3.0, PHP 7)

SplFixedArray::toArray — Devuelve un array PHP de un array fijo

Descripción

public array SplFixedArray::toArray ( void )

Devuelve un array PHP de un array fijo.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve un array PHP, similar al array fijo.

Ejemplos

Ejemplo #1 Ejemplo de SplFixedArray::toArray()

<?php
$fa = new SplFixedArray(3);
$fa[0] = 0;
$fa[2] = 2;
var_dump($fa->toArray());
?>

El resultado del ejemplo sería:

array(3) {
  [0]=>
  int(0)
  [1]=>
  NULL
  [2]=>
  int(2)
}
