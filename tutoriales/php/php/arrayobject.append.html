ArrayObject::append

(PHP 5 >= 5.0.0, PHP 7)

ArrayObject::append — Añade un valor

Descripción

public void ArrayObject::append ( mixed $value )

Añade un nuevo valor como el último elemento.

Nota:

Este método no puede ser llamado cuando la clase ArrayObject fue construida a partir de un objeto. Utilice ArrayObject::offsetSet() en su lugar.

Parámetros

value

El valor a ser añadido.

Valores devueltos

No devuelve ningún valor.

Ejemplos

Ejemplo #1 Ejemplo de ArrayObject::append()

<?php
$arrayobj = new ArrayObject(array('primero','segundo','tercero'));
$arrayobj->append('cuarto');
$arrayobj->append(array('quinto', 'sexto'));
var_dump($arrayobj);
?>

El resultado del ejemplo sería:

object(ArrayObject)#1 (5) {
  [0]=>
  string(5) "primero"
  [1]=>
  string(6) "segundo"
  [2]=>
  string(5) "tercero"
  [3]=>
  string(6) "cuarto"
  [4]=>
  array(2) {
    [0]=>
    string(4) "quinto"
    [1]=>
    string(3) "sexto"
  }
}
