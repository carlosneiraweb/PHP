ImagickPixel::getColorCount

(PECL imagick 2.0.0)

ImagickPixel::getColorCount — Devuelve la cuenta de color asociada con este color

Descripción

int ImagickPixel::getColorCount ( void )
Advertencia

Esta función no está documentada actualmente, solamente se encuentra disponible la lista de parámetros.

Devuelve la cuenta de color asociada con este color.

Valores devueltos

Devuelve la cuenta de color como un entero si se tuvo éxito, lanzando una excepción ImagickPixelException si falló.
