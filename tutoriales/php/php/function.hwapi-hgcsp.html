hwapi_hgcsp

(PHP 4, PHP 5 < 5.2.0, PECL hwapi SVN)

hwapi_hgcsp — Devuelve un objeto de la clase hw_api

Descripción

HW_API hwapi_hgcsp ( string $hostname [, int $port ] )

Abre una conexión al servidor Hyperwave en el host hostname. El protocolo que se usa es HGCSP.

Parámetros

hostname

El nombre del host.

port

Si no se pasa un número de puerto se usa 418.

Valores devueltos

Devuelve una instancia de HW_API.
