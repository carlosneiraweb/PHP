Instalación del Controlador de MongoDB para PHP en OSX con Homebrew

Si se utiliza » Homebrew, el tap de PHP incluye fórmulas para instalar el controlador en varias versiones de PHP.

  • php54-mongodb
  • php55-mongodb
  • php56-mongodb
  • php70-mongodb

Por ejemplo, se podría instalar el controlador para PHP 7.0 con el siguiente comando: 

$ brew install php70-mongodb
