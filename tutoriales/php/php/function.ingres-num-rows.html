ingres_num_rows

(PHP 4 >= 4.0.2, PHP 5 <= 5.0.5, PECL ingres >= 1.0.0)

ingres_num_rows — Obtener el número de filas afectadas o devueltas por una consulta

Descripción

int ingres_num_rows ( resource $result )

Con esta función, principalmente se pretende obtener el número de filas modificadas en la base de datos. Sin embargo, se puede usar para recuperar el número de filas obtenidas por una sentencia SELECT.

Nota:

Si los cursores desplazables están deshabilitados y se llama a esta función antes de usar ingres_fetch_array(), ingres_fetch_object(), o ingres_fetch_row(), el servidor borrará los datos de los resultados, por lo que el script no será capaz de obtenerlos.

En su lugar, se deberían de recuperar los datos de los resultados usando una de estas funciones de obtenición en un buble hasta que devuelvan FALSE, indicando que no hay más resultados disponibles.

Parámetros

result

El identificador de resultados de la consulta

Valores devueltos

Para consultas de eliminación, insertado, o actualización, ingres_num_rows() devuelve el número de filas afectadas por la consulta. Para otras consultas, ingres_num_rows() devuelve el número de filas del resultado de la consulta.

Ver también

  • ingres_fetch_array() - Obtiene una fila de resultados en un array
  • ingres_fetch_assoc() - Obtienen una fila de resultados en un array asociativo
  • ingres_fetch_object() - Obtiene una fila de resultados en un objeto
  • ingres_fetch_row() - Obtiene una fila de resultados en un array enumerado
