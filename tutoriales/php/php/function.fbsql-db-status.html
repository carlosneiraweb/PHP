fbsql_db_status

(PHP 4 >= 4.0.7, PHP 5, PHP 7)

fbsql_db_status — Obtiene el estado de una base de datos dada

Descripción

int fbsql_db_status ( string $database_name [, resource $link_identifier ] )

Obtiene el estado actual de la base de datos especificada.

Parámetros

database_name

El nombre de la base de datos.

link_identifier

Un identificador de enlace FrontBase devuelto por fbsql_connect() o fbsql_pconnect().

Es opcional y si no se especifica, la función intentará encontrar un enlace abierto al servidor FrontBase y de no encontrarse ningún enlace abierto intentará crear uno como si fbsql_connect() fuese invocada sin parámetros.

Valores devueltos

Devuelve un integer con el estado actual. Puede ser una de las siguientes constantes:

  • FALSE - El exec handler del host es inválido. Este error puede ocurrir cuando link_identifier conecta directamente a la base de datos usando un número de puerto. Puede que FBExec esté disponible en el servidor, pero no se ha realizado una conexión para él.
  • FBSQL_UNKNOWN - El estado desconocido.
  • FBSQL_STOPPED - La base de datos no esta en marcha. Use fbsql_start_db() para iniciar la base de datos.
  • FBSQL_STARTING - La base de datos se esta iniciando.
  • FBSQL_RUNNING - La base de datos es en marcha y se pueden realizar operaciones SQL.
  • FBSQL_STOPPING - La base de datos se está deteniendo.
  • FBSQL_NOEXEC - FBExec no está en funcionando en el servidor y no es posible obtener el estado de la base de datos.

Ver también

  • fbsql_start_db() - Inicia una base de datos en un servidor local o remoto
  • fbsql_stop_db() - Detiene una base de datos en un servidor local o remoto
