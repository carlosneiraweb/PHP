(^)

SDO_DAS_DataFactory::getDataFactory — Obtener una instancia de una fábrica de datos

Descripción

SDO_DAS_DataFactory SDO_DAS_DataFactory::getDataFactory ( void )
Advertencia

Esta función ha sido declarada EXPERIMENTAL. Su funcionamiento, nombre y la documentación que le acompaña puede cambiar sin previo aviso en futuras versiones de PHP. Utilícela bajo su propia responsabilidad.

Método estático para obtener una instancia de un SDO_DAS_DataFactory. Esta instancia está configurada inicialmente con los tipos básicos de SDO. Un Servicio de Acceso a Datos (DAS) es el responsable de rellenar el modelo de fábrica de datos y permitir a las aplicaciones de PHP crear SDOs basados en el modelo a través de la interfaz SDO_DataFactory. Las aplicaciones de PHP deberían siempre obtener una fábrica de datos desde un Servicio de Acceso a Datos configurado, no a través de esta interfaz.

Parámetros

None.

Valores devueltos

Devuelve un SDO_DAS_DataFactory.
