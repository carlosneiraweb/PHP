newt_form_run

(PECL newt >= 0.1)

newt_form_run — Ejecuta un formulario

Descripción

void newt_form_run ( resource $form , array &$exit_struct )

Esta función ejecuta el formulario proporcionado.

Parámetros

form

El componente formulario

exit_struct

Un array usado para devolver información después de ejecutar el componente formulario. Las claves y los valores se describen en la siguiente tabla:

Estructura de salida del formulario
Clave índice Tipo de valor Descripción
reason integer El motivo, por qué se ha salido del formulario. Los valores posibles están definidos aquí.
watch resource Enlace a recurso, especificado en newt_form_watch_fd()
key integer Combinación de teclas
component resource Componente, el cual causó la salida del formulario

Valores devueltos

No devuelve ningún valor.
