IntlChar::isULowercase

(PHP 7)

IntlChar::isULowercase — Comprobar si un punto de código tiene la propiedad de Unicode Lowercase

Descripción

public static bool IntlChar::isULowercase ( mixed $codepoint )

Comprueba si un punto de código tiene la propiedad de Unicode Lowercase.

Esto es lo mismo que IntlChar::hasBinaryProperty($codepoint, IntlChar::PROPERTY_LOWERCASE)

Nota:

Esto es diferente que IntlChar::islower() y devolverá TRUE para más caracteres.

Parámetros

codepoint

El valor de tipo integer del punto de código (p.ej. 0x2603 para U+2603 SNOWMAN), o el carácter codificado como un string UTF-8 (p.ej. "\u{2603}")

Valores devueltos

Devuelve TRUE si codepoint tiene la propiedad de Unicode Lowercase, FALSE si no.

Ejemplos

Ejemplo #1 Probar diferentes puntos de código

<?php
var_dump(IntlChar::isULowercase("A"));
var_dump(IntlChar::isULowercase("a"));
var_dump(IntlChar::isULowercase("Φ"));
var_dump(IntlChar::isULowercase("φ"));
var_dump(IntlChar::isULowercase("1"));
?>

El resultado del ejemplo sería:

bool(false)
bool(true)
bool(false)
bool(true)
bool(false)
