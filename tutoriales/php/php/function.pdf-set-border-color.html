PDF_set_border_color

(PHP 4, PECL pdflib >= 1.0.0)

PDF_set_border_color — Establecer el color del borde de las anotaciones [obsoleta]

Descripción

bool PDF_set_border_color ( resource $p , float $red , float $green , float $blue )

Establece el color del borde de todos los tipos de anotaciones. Devuelve TRUE en caso de éxito o FALSE en caso de error.

Esta función está obsoleta desde la versión 6 de PDFlib, use la opción annotcolor en PDF_create_annotation() en su lugar.
