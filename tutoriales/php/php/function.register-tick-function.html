register_tick_function

(PHP 4 >= 4.0.3, PHP 5, PHP 7)

register_tick_function — Registrar una función para que sea ejecutada en cada tick

Descripción

bool register_tick_function ( callable $function [, mixed $arg [, mixed $... ]] )

Registra la función dada por function para ser ejecutada cuando se llame a un tick.

Parámetros

function

El nombre de la función como cadena, o una matriz que consiste en un objeto y un método.

arg

...

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.

Ejemplos

Ejemplo #1 Ejemplo de register_tick_function()

<?php
declare(ticks=1);

// usar una función como llamada de retorno
register_tick_function('mi_función', true);

// usar un objeto->método
$objeto = new mi_clase();
register_tick_function(array(&$objeto, 'mi_método'), true);
?>

Historial de cambios

Versión Descripción
5.3.0 Los ticks ahora están soportados en los módulos de servidores web que usan hilos.

Notas

Advertencia

No debería usarse register_tick_function() con módulos de servidores web que usan hilos con PHP 5.2 o anterior.
