CairoMatrix::initTranslate

cairo_matrix_init_translate

(PECL cairo >= 0.1.0)

CairoMatrix::initTranslate -- cairo_matrix_init_translate — Crea una nueva matriz de traducción

Descripción

Estilo orientado a objetos (método):

public static void CairoMatrix::initTranslate ( float $tx , float $ty )

Estilo por procedimientos:

object cairo_matrix_init_translate ( float $tx , float $ty )

Crea una nueva matriz para una transformación que traduce por tx y ty en las dimensiones X e Y, respectivamente.

Parámetros

tx

cantidad a traducir en la dirección X

ty

cantidad a traducir en la dirección Y

Valores devueltos

Devuelve un nuevo objeto CairoMatrix que puede ser usado con superficies, contextos y patrones.

Ejemplos

Ejemplo #1 Estilo orientado a objetos

<?php
/* Crear una nueva Matriz */
$matriz = CairoMatrix::initTranslate(1.0, 2.0);
?>

Ejemplo #2 Estilo por procedimientos

<?php
/* Crear una nueva Matriz */
$matriz = cairo_matrix_init_translate(1.0, 2.0);
?>
