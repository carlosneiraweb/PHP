GearmanWorker::setOptions

(PECL gearman >= 0.5.0)

GearmanWorker::setOptions — Establece las opciones del trabajador

Descripción

public bool GearmanWorker::setOptions ( int $option )

Establece una o más opciones para el valor dado.

Parámetros

option

Las opciones a establecer

Valores devueltos

Siempre retorna TRUE.

  • GearmanWorker::removeOptions() - Elimina las opciones del trabajador
  • GearmanClient::setOptions() - Establece opciones del cliente
