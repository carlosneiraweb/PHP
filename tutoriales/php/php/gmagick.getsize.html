Gmagick::getsize

(PECL gmagick >= Unknown)

Gmagick::getsize — Devuelve el tamaño asociado con el objeto Gmagick

Descripción

public array Gmagick::getsize ( void )

Devuelve el tamaño asociado con el objeto Gmagick como una matriz con las claves "columns" (columnas) y "rows" (filas).

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve el tamaño asociado con el objeto Gmagick como una matriz con las claves "columns" (columnas) y "rows" (filas).

Errores/Excepciones

Arroja un GmagickException en caso de error.
