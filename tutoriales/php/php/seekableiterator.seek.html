SeekableIterator::seek

(PHP 5 >= 5.1.0, PHP 7)

SeekableIterator::seek — Mueve la posición del apuntador interno

Descripción

abstract public void SeekableIterator::seek ( int $position )

Mueve la posición del apuntador interno en el iterador.

Parámetros

position

La posición para mover el apuntador interno.

Valores devueltos

No devuelve ningún valor.

Errores/Excepciones

Las implementaciones deben lanzar una OutOfBoundsException si la posición position no es válida.

Ejemplos

Ejemplo #1 Ejemplo de SeekableIterator::seek()

Mueve el apuntador de el elemento a la posición 3 en el iterador (ArrayIterator implements SeekableIterator).

<?php
$array = array("manzana", "banana", "cereza", "ciruela", "baya");
$iterator = new ArrayIterator($array);
$iterator->seek(3);
echo $iterator->current();
?>

El resultado del ejemplo sería algo similar a:

ciruela
