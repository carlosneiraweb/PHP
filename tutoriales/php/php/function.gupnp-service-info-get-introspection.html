gupnp_service_info_get_introspection

(PECL gupnp >= 0.1.0)

gupnp_service_info_get_introspection — Obtiene el recurso de introspección de un servicio

Descripción

mixed gupnp_service_info_get_introspection ( resource $proxy [, mixed $callback [, mixed $arg ]] )

Obtiene el recurso de introspección de un servicio o registra un callback si el parámetro correspondiente fue pasado.

Parámetros

proxy

Un identificador de servicio proxy.

callback

La función callback a ser llamada cuando el objeto de introspección está listo. Típicamente, la función callback toma tres parámetros. Siendo el primer parámetro el identificador introspection, El segundo parámetro es el mensaje de error, y el tercero el arg.

arg

Datos de usuario para el callback.

Valores devueltos

Devuelve true si la función callback fue definida. Devuelve el identificador de introspección si la función callback fue omitida.

Errores/Excepciones

Lanza un E_WARNING con una función callback inválida.
