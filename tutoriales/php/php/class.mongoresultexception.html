La clase MongoResultException

(PECL mongo >=1.3.0)

Introducción

Una MongoResultException es lanzada por varios ayudantes de comandos (como MongoCollection::findAndModify()) en caso de que ocurra un error. El documento resultante original está disponible a través de MongoResultException::getDocument().

Sinopsis de la Clase

MongoResultException extends MongoException {
/* Propiedades */
public $document ;
/* Propiedades heredadas */
protected string $message ;
protected int $code ;
protected string $file ;
protected int $line ;
/* Métodos */
public array getDocument ( void )
}

Propiedades

document

Un array con el documento resultante sin tratar.
