Funciones cambiadas

Varias funciones con parámetros opcionales nuevos en PHP 5.4:

PHP Core:

  • Se ha añadido el parámetro opcional limit a debug_backtrace() y debug_print_backtrace(), para limitar la cantidad de marcos de pila devueltos.
  • is_link() ahora funciona apropiadamente con enlaces simbólicos en Windows Vista o posterior. Los sistemas anteriores no soportan enlaces simbólicos.
  • parse_url() ahora reconoce el host al omitir el esquema y está presente un separador componentes está presente al inicio. A partir de PHP 5.4.7.

OpenSSL:

  • Se ha añadido una opción de no relleno a las funciones openssl_encrypt() y openssl_decrypt().

Intl:

  • idn_to_ascii() y idn_to_utf8() ahora toman dos parámetros extra, uno indicando la variante (IDNA 2003 o UTS #46) y otro, pasado por referencia, para devolver detalles sobre la operación en el caso de que se elija UTS #46.
