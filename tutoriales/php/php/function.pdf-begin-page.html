PDF_begin_page

(PHP 4, PECL pdflib >= 1.0.0)

PDF_begin_page — Iniciar una nueva página [obsoleta]

Descripción

bool PDF_begin_page ( resource $pdfdoc , float $width , float $height )

Añade una nueva página al documento. Devuelve TRUE en caso de éxito o FALSE en caso de error.

Esta función está obsoleta desde la versión 6 de PDFlib, use PDF_begin_page_ext() en su lugar.
