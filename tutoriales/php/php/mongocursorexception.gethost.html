MongoCursorException::getHost

(PECL mongo >= 1.0.0)

MongoCursorException::getHost — El nombre de host del servidor que encontró el error

Descripción

public string MongoCursorException::getHost ( void )

Devuelve el nombre de host del sevidor al que se le envió la consulta.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve el nombre de host, o NULL si este es desconocido.
