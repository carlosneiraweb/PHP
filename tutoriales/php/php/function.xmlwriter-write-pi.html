XMLWriter::writePI

xmlwriter_write_pi

(PHP 5 >= 5.1.2, PHP 7, PECL xmlwriter >= 0.1.0)

XMLWriter::writePI -- xmlwriter_write_pi — Escribe un IP

Descripción

Estilo orientado a objetos

bool XMLWriter::writePI ( string $target , string $content )

Estilo por procedimientos

bool xmlwriter_write_pi ( resource $xmlwriter , string $target , string $content )

Escribe una instrucción de procesamiento.

Parámetros

xmlwriter

Sólo para llamadas por procedimientos. El resource XMLWriter que está siendo modificado. Este recurso proviene de una llamada a xmlwriter_open_uri() o xmlwriter_open_memory().

target

El objetivo de la instrucción de procesamiento.

content

El contenido de la instrucción de procesamiento.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.

Ver también

  • XMLWriter::startPI() - Crea la etiqueta PI inicial
  • XMLWriter::endPI() - Finaliza el actual IP
