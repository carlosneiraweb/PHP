ingres_execute

(PECL ingres >= 1.1.0)

ingres_execute — Ejecuta una consulta preparada

Descripción

bool ingres_execute ( resource $result [, array $params [, string $types ]] )

Ejecuta una consulta preparada anterioremente con ingres_prepare().

Nota: Configuraciones relacionadas

Véase también las directivas ingres.describe, ingres.scrollable y ingres.utf8 de la sección Configuración en tiempo de ejecución

Parámetros

result

El identificador de enlace de la conexión

params

Un array con los valores de los parámetros a usar con la consulta

types

Un string que contiene una secuencia con los tipos de los valores de los parámetros proporcionados. Véase el parámetro types de ingres_query() para una lista de los códigos de los tipos.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.
