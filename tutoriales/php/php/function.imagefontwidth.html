imagefontwidth

(PHP 4, PHP 5, PHP 7)

imagefontwidth — Obtener el ancho de la fuente

Descripción

int imagefontwidth ( int $font )

Devuelve el ancho del píxel de un carácter de una fuente.

Parámetros

font

Puede ser 1, 2, 3, 4, 5 para fuentes built-in en latin2 encoding (los números más altos corresponden a large fonts) o cualquier de sus propios identificadores de fuentes registrados con imageloadfont().

Valores devueltos

Devuelve el ancho de píxel de la fuente.

Ejemplos

Ejemplo #1 Usar imagefontwidth() con fuentes incorporadas

<?php
echo 'Alto de la fuente: ' . imagefontwidth(4);
?>

El resultado del ejemplo sería algo similar a:

Font width: 8

Ejemplo #2 Usar imagefontwidth() junto con imageloadfont()

<?php
// Cargar una fuente .gdf
$font = imageloadfont('anonymous.gdf');

echo 'Alto de la fuente: ' . imagefontwidth($font);
?>

El resultado del ejemplo sería algo similar a:

Font width: 23
