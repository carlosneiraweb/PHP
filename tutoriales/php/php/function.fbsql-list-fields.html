fbsql_list_fields

(PHP 4 >= 4.0.6, PHP 5, PHP 7)

fbsql_list_fields — Lista los campos de un resultado de FrontBase

Descripción

resource fbsql_list_fields ( string $database_name , string $table_name [, resource $link_identifier ] )

Obtiene información acerca de la tabla especificada.

Parámetros

database_name

El nombre de la base de datos.

table_name

El nombre de la tabla.

link_identifier

Un identificador de enlace FrontBase devuelto por fbsql_connect() o fbsql_pconnect().

Es opcional y si no se especifica, la función intentará encontrar un enlace abierto al servidor FrontBase y de no encontrarse ningún enlace abierto intentará crear uno como si fbsql_connect() fuese invocada sin parámetros.

Valores devueltos

Devuelve un puntero a resultado que puede usarse con las funciones fbsql_field_xxx , o FALSE si hay error.

Errores/Excepciones

Una cadena con la descripción del error se pondrá en $phperrmsg, y a menos que la función fuese llamada con @fbsql() , esta cadena de error también será mostrada.

Ejemplos

Ejemplo #1 fbsql_list_fields() example

<?php
$link = fbsql_connect('localhost', 'myname', 'secret');

$fields = fbsql_list_fields("database1", "table1", $link);
$columns = fbsql_num_fields($fields);

for ($i = 0; $i < $columns; $i++) {
    echo fbsql_field_name($fields, $i) . "\n";;
}
?>

El resultado del ejemplo sería algo similar a:

field1
field2
field3
...
