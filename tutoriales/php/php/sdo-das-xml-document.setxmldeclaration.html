SDO_DAS_XML_Document::setXMLDeclaration

(^)

SDO_DAS_XML_Document::setXMLDeclaration — Establece la declaración de xml

Descripción

void SDO_DAS_XML_Document::setXMLDeclaration ( bool $xmlDeclatation )

Controla si una declaración XML será generada al inicio del documento XML. Establézcalo a TRUE para generar la declaración XML, orFALSE para omitirla.

Parámetros

xmlDeclatation

Un valor booleano para establecer la declaración XML.

Valores devueltos

Ninguno.
