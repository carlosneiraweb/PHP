ImagickDraw::getStrokeDashOffset

(PECL imagick 2.0.0)

ImagickDraw::getStrokeDashOffset — Devuelve el índice dentro del patrón de discontinuidad para iniciar la discontinuidad

Descripción

float ImagickDraw::getStrokeDashOffset ( void )
Advertencia

Esta función no está documentada actualmente, solamente se encuentra disponible la lista de parámetros.

Devuelve el índice dentro del patrón de discontinuidad para iniciar la discontinuidad.

Valores devueltos

Devuelve un float que representa el índice y 0 si no está establecido.
