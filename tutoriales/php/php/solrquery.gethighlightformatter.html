SolrQuery::getHighlightFormatter

(PECL solr >= 0.9.2)

SolrQuery::getHighlightFormatter — Devuelve el formateador de la salida remarcada

Descripción

public string SolrQuery::getHighlightFormatter ([ string $field_override ] )

Devuelve el formateador de la salida remarcada

Parámetros

field_override

El nombre del campo

Valores devueltos

Devuelve una cadena en caso de éxito y NULL si no se estableció.
