ImagickDraw::getFont

(PECL imagick 2.0.0)

ImagickDraw::getFont — Devuelve la fuente

Descripción

string ImagickDraw::getFont ( void )
Advertencia

Esta función no está documentada actualmente, solamente se encuentra disponible la lista de parámetros.

Devuelve un string especificando la fuente usada cuando se escribe texto.

Valores devueltos

Devuelve un string si se tuvo éxito o false si no está establecida la fuente.
