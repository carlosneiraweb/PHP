HaruDoc::save

(PECL haru >= 0.0.1)

HaruDoc::save — Guarda el documento en el fichero especificado

Descripción

bool HaruDoc::save ( string $file )

Guarda el documento en el fichero especificado.

Parámetros

file

El fichero para guardar el documento.

Valores devueltos

Devuelve TRUE en caso de éxito.

Errores/Excepciones

Lanza una HaruException cuando se produce un error.
