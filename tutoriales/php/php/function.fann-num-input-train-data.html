fann_num_input_train_data

(PECL fann >= 1.0.0)

fann_num_input_train_data — Devuelve el número de entradas de cada patrón de entrenamiento de los datos de entrenamiento

Descripción

int fann_num_input_train_data ( resource $data )

Devuelve el número de entradas de cada patrón de entrenamiento del resource de datos de entrenamiento.

Parámetros

data

Un resource de datos de entrenamiento de red neuronal.

Valores devueltos

El número de entradas, o FALSE en caso de error.
