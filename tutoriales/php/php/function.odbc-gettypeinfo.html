odbc_gettypeinfo

(PHP 4, PHP 5, PHP 7)

odbc_gettypeinfo — Recupera información sobre los tipos de datos soportados por la fuente de datos

Descripción

resource odbc_gettypeinfo ( resource $connection_id [, int $data_type ] )

Recupera información sobre los tipos de datos soportados por la fuente de datos.

Parámetros

connection_id

El conector identificador ODBC, ver odbc_connect() para más información.

data_type

El tipo de datos, el cuál se puede usar para restringir la información a un único tipo de datos.

Valores devueltos

Devuelve un identificador de resultado ODBC o FALSE si se produjo un error.

El conjunto de resultados tiene las siguientes columnas:

  • TYPE_NAME
  • DATA_TYPE
  • PRECISION
  • LITERAL_PREFIX
  • LITERAL_SUFFIX
  • CREATE_PARAMS
  • NULLABLE
  • CASE_SENSITIVE
  • SEARCHABLE
  • UNSIGNED_ATTRIBUTE
  • MONEY
  • AUTO_INCREMENT
  • LOCAL_TYPE_NAME
  • MINIMUM_SCALE
  • MAXIMUM_SCALE

El conjunto de resultados está ordenado por DATA_TYPE y TYPE_NAME.
