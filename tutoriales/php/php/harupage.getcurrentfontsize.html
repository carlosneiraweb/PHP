HaruPage::getCurrentFontSize

(PECL haru >= 0.0.1)

HaruPage::getCurrentFontSize — Obtiene el tamaño de la fuente actual

Descripción

float HaruPage::getCurrentFontSize ( void )

Obtiene el tamaño de la fuente actual.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve el tamaño de la fuente usada.

Errores/Excepciones

Lanza una HaruException cuando se produce un error.
