Thread::detach

(PECL pthreads >= 2.0.0)

Thread::detach — Ejecución

Descripción

public void Thread::detach ( void )

Desconecta el hilo referenciado por el contexto de la llamada, ¡de manera peligrosa!

Advertencia

Este método puede causar una conducta indefinida o insegura. Por lo general no debe ser utilizado. Está presente en los casos de integridad y de uso avanzado.

Parámetros

Esta función no tiene parámetros.
