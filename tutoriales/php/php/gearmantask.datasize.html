GearmanTask::dataSize

(PECL gearman >= 0.5.0)

GearmanTask::dataSize — Obtiene el tamaño de la información retornada

Descripción

public int GearmanTask::dataSize ( void )

Retorna el tamaño de los datos que serán devueltos por una tarea.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

El tamaño de los datos, o FALSE en caso de no haberlos.
