ps_stroke

(PECL ps >= 1.1.0)

ps_stroke — Dibujar el trazado actual

Descripción

bool ps_stroke ( resource $psdoc )

Dibuja el trazado construido previamente con las llamadas a las funciones de dibujo como ps_lineto().

Parámetros

psdoc

El identificador de recursos del fichero postscript, como el devuelto por la función ps_new().

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.

  • ps_fill() - Rellenar el trazado actual
  • ps_fill_stroke() - Rellenar y contornear el trazado actual
