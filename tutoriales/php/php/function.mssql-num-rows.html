mssql_num_rows

(PHP 4, PHP 5, PECL odbtp >= 1.1.1)

mssql_num_rows — Devuelve el número de filas que hay en un resultado

Advertencia

Esta función está OBSOLETA en PHP 7.0.0.

Las alternativas a esta función son:

  • PDOStatement::rowCount()
  • sqlsrv_num_rows()
  • odbc_num_rows()

Descripción

int mssql_num_rows ( resource $result )

mssql_num_rows() devuelve el número de filas que hay en un resultado.

Parámetros

result

El resultado que está siendo evaluado. Este resultado proviene de una llamada a mssql_query().

Valores devueltos

Devuelve el número de filas, como un integer.

Ejemplos

Ejemplo #1 Ejemplo de mssql_num_rows()

<?php
// Conectar a MSSQL y seleccionar la base de datos
$link = mssql_connect('KALLESPC\SQLEXPRESS', 'sa', 'phpfi');
mssql_select_db('php');

// Seleccionar todos los registros de una tabla
$query = mssql_query('SELECT * FROM [php].[dbo].[persons]');

echo 'Núm. registros totales en la tabla: ' . mssql_num_rows($query);

// Limpieza
mssql_free_result($query);
?>
