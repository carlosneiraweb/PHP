SoapVar::SoapVar

(PHP 5 >= 5.0.1, PHP 7)

SoapVar::SoapVar — Constructor de SoapVar

Descripción

SoapVar::SoapVar ( mixed $data , string $encoding [, string $type_name [, string $type_namespace [, string $node_name [, string $node_namespace ]]]] )

Construye un nuevo objecto SoapVar.

Parámetros

data

Datos a pasar o devolver.

encoding

ID de codificación, una de las constantes XSD_....

type_name

Tipo de nombre.

type_namespace

Tipo de espacio de nombres.

node_name

Nombre del nodo XML.

node_namespace

Espacio de nombres del nodo XML.

Ejemplos

Ejemplo #1 Ejemplo de SoapVar::SoapVar()

<?php
class SOAPStruct {
    function SOAPStruct($s, $i, $f) 
    {
        $this->varString = $s;
        $this->varInt = $i;
        $this->varFloat = $f;
    }
}
$client = new SoapClient(null, array('location' => "http://localhost/soap.php",
                                     'uri'      => "http://test-uri/"));
$struct = new SOAPStruct('arg', 34, 325.325);
$soapstruct = new SoapVar($struct, SOAP_ENC_OBJECT, "SOAPStruct", "http://soapinterop.org/xsd");
$client->echoStruct(new SoapParam($soapstruct, "inputStruct"));
?>
