fann_set_weight_array

(PECL fann >= 1.0.0)

fann_set_weight_array — Establecer las conexiones de la red

Descripción

bool fann_set_weight_array ( resource $ann , array $connections )

Establecer las conexiones de la red.

Solamente se pueden cambiar los pesos. Las conexiones y los pesos se ignoran si no existen en la red.

Parámetros

ann

Un resource de red neuronal.

connections

Un array de objetos FANNConnection

Valores devueltos

Devuelve TRUE en caso de éxito, o FALSE de lo contrario.
