Gender\Gender::isNick

(PECL gender >= 0.9.0)

Gender\Gender::isNick — Comprueba si name0 es un alias de name1

Descripción

public array Gender\Gender::isNick ( string $name0 , string $name1 [, integer $country ] )

Comprueba si name0 es un alias de name1.

Parámetros

name0

El nombre a comprobar.

name1

El nombre a comprobar.

country

El Id del país identificado por una constante la clase Gender. Si se omite, se usa ANY_COUNTRY.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.
