fbsql_fetch_row

(PHP 4 >= 4.0.6, PHP 5, PHP 7)

fbsql_fetch_row — Obtiene una fila de resultados como array enumerado

Descripción

array fbsql_fetch_row ( resource $result )

fbsql_fetch_row() obtiene una fila de datos del resultado asociado con el identificador de resultado especificado.

Las siguientes llamadas a fbsql_fetch_row() devolverán la siguiente fila en el resultset, o FALSE si no hay más filas.

Parámetros

result

Un identificador de resultado devuelto por fbsql_query() o fbsql_db_query().

Valores devueltos

Devuelve el array que corresponde a la fila obtenida, donde cada columna se guarda en un offset (comenzando desde 0), o FALSE si no hay más filas.
