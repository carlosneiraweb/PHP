strrev

(PHP 4, PHP 5, PHP 7)

strrev — Invierte una string

Descripción

string strrev ( string $string )

devuelve la cadena, invertida.

Parámetros

cadena

La cadena que será invertida

Valores devueltos

devuelve la cadena invertida.

Ejemplos

Ejemplo #1 Invertir una cadena con strrev()

<?php
echo strrev("Hello world!"); // outputs "!dlrow olleH"
?>
