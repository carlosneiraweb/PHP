Imagick::getImageGravity — Obtiene la gravedad de la imagen

Descripción

int Imagick::getImageGravity ( void )

Obtiene el valor de gravedad actual de la imagen. A diferencia de Imagick::getGravity(), este método devuelve la gravedad definida para la secuencia de imágenes actual. Este método está disponible si Imagick ha sido compilado con la versión 6.4.4 o superior de ImageMagick.

Valores devueltos

Devuelve la propiedad gravedad de las imágenes. Consulte esta lista de constantes de gravedad.
