gupnp_root_device_get_available

(PECL gupnp >= 0.1.0)

gupnp_root_device_get_available — Comprueba si un dispositivo raíz está disponible

Descripción

bool gupnp_root_device_get_available ( resource $root_device )

Obtiene si está disponible o no root_device (anunciando su presencia).

Parámetros

root_device

Un identificador de dispositivo raíz, devuelto por gupnp_root_device_new().

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.
