imap_base64

(PHP 4, PHP 5, PHP 7)

imap_base64 — Decodificar un texto cifrado con BASE64

Descripción

string imap_base64 ( string $text )

Decodifica el text cifrado en BASE-64 dado.

Parámetros

text

El texto cifrado

Valores devueltos

Devuelve el mensaje decodificado como una cadena.
