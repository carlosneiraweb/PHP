imap_msgno

(PHP 4, PHP 5, PHP 7)

imap_msgno — Obtiene el número de secuencia de mensaje para el UID dado

Descripción

int imap_msgno ( resource $imap_stream , int $uid )

Devuelve el número de secuencia de mensaje para el uid dado.

Esta función es la inversa de imap_uid().

Parámetros

imap_stream

IMAP stream devuelto por imap_open().

uid

El UID del mensaje

Valores devueltos

Devuelve el número de secuencia de mensaje para el uid dado.

Ver también

  • imap_uid() - Esta función devuelve el UID del número de secuencia del mensaje dado
