ldap_modify

(PHP 4, PHP 5, PHP 7)

ldap_modify — Modificar una entrada de LDAP

Descripción

bool ldap_modify ( resource $link_identifier , string $dn , array $entry )

Modificar las entradas existentes en el directorio de LDAP. La estructura de la entrada es la misma como en ldap_add().

Parámetros

link_identifier

Un identificador de enlace de LDAP, devuelto por ldap_connect().

dn

El nombre distinguido de una entidad de LDAP.

entry

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.

Notas

Nota: Esta función es segura binariamente.
