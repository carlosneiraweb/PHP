mysqli_fetch

(PHP 5 < 5.4.0)

mysqli_fetch — Alias de mysqli_stmt_fetch()

Descripción

Esta función es un alias de mysqli_stmt_fetch().

Advertencia

Esta función ha sido declarada OBSOLETA desde PHP 5.3.0 y ELIMINADA a partir de PHP 5.4.0.
