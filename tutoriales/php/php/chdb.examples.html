Ejemplos

Ejemplo #1 Crear un archivo chdb

<?php

$data = array(
    'clave1' => 'valor1',
    'clave2' => 'valor2',
    // ...
);
chdb_create('data.chdb', $data);

?>

El ejemplo de arriba generará un archivo chdb llamado data.chdb que contiene los pares clave-valor definidos en $data.

Ejemplo #2 Cargar y examinar un archivo chdb

<?php

$chdb = new chdb('data.chdb');
$valor1 = $chdb->get('clave1');
$valor2 = $chdb->get('clave2');

echo $valor1, PHP_EOL;
echo $valor2, PHP_EOL;

?>

El resultado del ejemplo sería algo similar a:

valor1
valor2
