mysqlnd_ms_fabric_select_shard

(No hay información de versión disponible, podría estar únicamente en SVN)

mysqlnd_ms_fabric_select_shard — Cambiar a los fragmentos

Descripción

array mysqlnd_ms_fabric_select_shard ( mixed $connection , mixed $table_name , mixed $shard_key )
Advertencia

Esta función no está documentada actualmente, solamente se encuentra disponible la lista de parámetros.

Relacionada con MySQL Fabric.

Cambia la conexión a los fragmentos responsables del nombre de tabla y de la clave de fragmento dados.

Parámetros

connection

Un gestor de conexión de MySQL obtenido de cualquier función de conexión de las extensiones mysqli, mysql o PDO_MYSQL.

table_name

El nombre de la tabla sobre la que preguntar a Fabric.

shard_key

La clave de fragmento sobre el que preguntar a Fabric.

Valores devueltos

FALSE en caso de error. De lo contrario, TRUE

Notas

Nota:

mysqlnd_ms_fabric_select_shard() requiere mysqlnd_ms >> 1.6.0 de PECL.
