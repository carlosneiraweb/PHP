stream_get_transports

(PHP 5, PHP 7)

stream_get_transports — Recuperar la lista de transportes de socket registrados

Descripción

array stream_get_transports ( void )

Devuelve una matriz indexada que contiene el nombre de todos los transportes de socket disponibles en el sistema en ejecución.

Valores devueltos

Devuelve una matriz indexada de nombres de transportes de socket.

Ejemplos

Ejemplo #1 Usar stream_get_transports()

<?php
$xportlist = stream_get_transports();
print_r($xportlist);
?>

El resultado del ejemplo sería algo similar a:

Array (
  [0] => tcp
  [1] => udp
  [2] => unix
  [3] => udg
)
