ezmlm_hash

(PHP 4 >= 4.0.2, PHP 5, PHP 7)

ezmlm_hash — Calcula el valor hash que necesita EZMLM

Descripción

int ezmlm_hash ( string $addr )

ezmlm_hash() calcula el valor hash necesario cuando se guardan listas de correo EZMLM en una base de datos MySQL.

Parámetros

addr

La dirección de correo de la que se quiere crear el hash.

Valores devueltos

El valor hash de addr.

Ejemplos

Ejemplo #1 Calcular el hash de un usuario suscriptor.

<?php

$usuario = "joecool@example.com";
$hash = ezmlm_hash($usuario);
$query = sprintf("INSERT INTO sample VALUES (%s, '%s')", $hash, $usuario);
$db->query($query); // usando interface PHPLIB db

?>
