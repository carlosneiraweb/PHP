Judy::offsetExists

(PECL judy >= 0.1.1)

Judy::offsetExists — Comprueba si existe un índice

Descripción

public bool Judy::offsetExists ( mixed $offset )

Comprueba si existe o no un índice.

Parámetros

offset

El índice a ser comprobado.

Valores devueltos

Devuelve TRUE si tiene éxito o FALSE si falla.
