Cambios en funciones

Núcleo de PHP

  • crypt() ahora emitirá un error E_NOTICE si se omite el parámetro salt.
  • substr_compare() ahora aceptará 0 para su parámetro length.
  • unserialize() ahora fallará si se proporcionan datos serializados que hayan sido manipulados para intentar instanciar un objeto sin llamar a su constructor.

cURL

  • Las subidas empleando la sintaxis @file ahora únicamente están soportadas si la opción CURLOPT_SAFE_UPLOAD está establecida a FALSE. Debería utilizarse en su lugar CURLFile.

Mcrypt

  • El valor predeterminado del parámetro source de mcrypt_create_iv() ahora es MCRYPT_DEV_URANDOM en vez de MCRYPT_DEV_RANDOM.

OpenSSL

  • stream_socket_enable_crypto() ahora permite que el parámetro crypto_type sea opcional si el contexto SSL del flujo incluye la nueva opción crypto_type.

PostgreSQL

  • pg_insert(), pg_select(), pg_update() y pg_delete() ya no son experimentales.
  • pg_send_execute(), pg_send_prepare(), pg_send_query() y pg_send_query_params() ya no bloquearán hasta que termine la escritura de la consulta si el flujo de socket subyacente para la conexión a la base de datos está establecido al modo de no bloqueo.

Reflection

  • ReflectionClass::newInstanceWithoutConstructor() ahora permite que se instancien clases internas no finales.

XMLReader

  • XMLReader::getAttributeNs() y XMLReader::getAttributeNo() ahora devuelven NULL si no se pudo encontrar el atributo, al igual que XMLReader::getAttribute().
