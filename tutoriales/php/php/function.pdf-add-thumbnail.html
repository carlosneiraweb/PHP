PDF_add_thumbnail

(PHP 4 >= 4.0.5, PECL pdflib >= 1.0.0)

PDF_add_thumbnail — Añadir una imagen en miniatura a la página actual

Descripción

bool PDF_add_thumbnail ( resource $pdfdoc , int $image )

Añade una imgen existente como miniatura a la página actual. Devuelve TRUE en caso de éxito o FALSE en caso de error.
