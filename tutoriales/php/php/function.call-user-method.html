call_user_method

(PHP 4, PHP 5)

call_user_method — Llamar a un método de usuario sobre un objeto específico

Advertencia

Esta función está OBSOLETA en PHP 4.1.0, por lo tanto, será ELIMINADA en PHP 7.0.0.

Las alternativas a esta función son:

  • call_user_func()

Descripción

mixed call_user_method ( string $method_name , object &$obj [, mixed $parameter [, mixed $... ]] )

Parámetros

method_name

El nombre del método a llamar.

obj

El object que está siendo llamado desde method_name.

parameter ...

Los parámetros opcionales.

Ejemplos

Ejemplo #1 Alternativa a call_user_method()

<?php
call_user_func(array($obj, $method_name), $parameter /* , ... */);
?>
