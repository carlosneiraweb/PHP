FilesystemIterator::getFlags

(PHP 5 >= 5.3.0, PHP 7)

FilesystemIterator::getFlags — Obtiene el manejo de flags

Descripción

public int FilesystemIterator::getFlags ( void )

Obtener el manejo de flags, como está establecido en FilesystemIterator::__construct() o FilesystemIterator::setFlags().

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Un valor integer con las flags establecidas.

Ver también

  • FilesystemIterator::__construct() - Construye un nuevo iterador filesystem
  • FilesystemIterator::setFlags() - Establece las flags de manejo
