dbplus_errcode

(PHP 4 <= 4.1.0, PECL dbplus >= 0.9)

dbplus_errcode — Obtener la cadena de error de un código de error dado o del último error

Descripción

string dbplus_errcode ([ int $errno ] )

Devuelve una cadena de error limpia del código de error dado.

Parámetros

errno

El código de error. Si no se proporcionó, se asume el código resultante de la última operación de db++.

Valores devueltos

Devuelve el mensaje de error.

Notas

Advertencia

Esta función ha sido declarada EXPERIMENTAL. Su funcionamiento, nombre y la documentación que le acompaña puede cambiar sin previo aviso en futuras versiones de PHP. Utilícela bajo su propia responsabilidad.
