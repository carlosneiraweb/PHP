mcrypt_module_self_test

(PHP 4 >= 4.0.2, PHP 5, PHP 7)

mcrypt_module_self_test — Esta función ejecuta una prueba automática sobre el módulo especificado

Descripción

bool mcrypt_module_self_test ( string $algorithm [, string $lib_dir ] )

Esta función ejecuta la prueba automática sobre el algoritmo especificado.

Parámetros

algorithm

Una de las constantes MCRYPT_nombredelcifrado, o el nombre del algoritmo como string.

lib_dir

El parámetro opcional lib_dir puede contener la ubicación donde el módulo de algoritmos se encuentra dentro del sistema.

Valores devueltos

Esta fucnión devuelve TRUE si la prueba automática es exitosa, o FALSE cuando falla.

Ejemplos

Ejemplo #1 Ejemplo de mcrypt_module_self_test()

<?php
var_dump(mcrypt_module_self_test(MCRYPT_RIJNDAEL_128)) . "\n";
var_dump(mcrypt_module_self_test(MCRYPT_BOGUS_CYPHER));
?>

El resultado del ejemplo sería:

bool(true)
bool(false)
