win32_ps_list_procs

(PECL win32ps >= 1.0.1)

win32_ps_list_procs — Listar los procesos en ejecución

Descripción

array win32_ps_list_procs ( void )

Recupera las estadísticas sobre todos los procesos en ejecución.

Valores devueltos

Devuelve FALSE si falló, o una matriz consistente en las estadísticas de los procesos como devuelve win32_ps_stat_proc() para todos los procesos en ejecución si se tuvo éxito.
