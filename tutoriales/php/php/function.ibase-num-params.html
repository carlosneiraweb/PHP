ibase_num_params

(PHP 5, PHP 7)

ibase_num_params — Devuelve el número de parámetros de una sentencia preparada

Descripción

int ibase_num_params ( resource $query )

Esta función devuelve el número de parámetros de la sentencia preparada especificada en query. Este es el número de argumentos que se deben proporcionar al invocar a ibase_execute().

Parámetros

query

Manejador de consulta preparada.

Valores devueltos

Devuelve el número de parámetros en forma de entero.

  • ibase_prepare() - Preparar una consulta para su asociación con parámetros de sustitución y ejecución posterior
  • ibase_param_info() - Devuelve información sobre un parámetro de una sentencia preparada
