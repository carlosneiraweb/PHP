EventBuffer::enableLocking

(PECL event >= 1.2.6-beta)

EventBuffer::enableLocking —

Descripción

public void EventBuffer::enableLocking ( void )

Activa el bloqueo de EventBuffer de tal forma que puede utilizarse de forma segura por múltiples líneas de ejecución a la vez. Cuando se activa el bloqueo, éste se mantiene cuando se invoquen las llamadas de vuelta. Esto puede dar lugar a un punto muerto (o 'deadlock') si no se es cuidadoso. Planificar de forma adecuada!

Parámetros

Esta función no tiene parámetros.

Valores devueltos

No devuelve ningún valor.
