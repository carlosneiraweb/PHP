ArrayIterator::setFlags

(PHP 5 >= 5.1.0, PHP 7)

ArrayIterator::setFlags — Definir opciones de comportamiento

Descripción

public void ArrayIterator::setFlags ( string $flags )

Define opciones de comportamiento.

Advertencia

Esta función no está documentada actualmente, solamente se encuentra disponible la lista de parámetros.

Parámetros

flags

Un bitmask como se indica:

  • 0 = Propiedades del objeto tienen su normal comportamiento cuando son obtenidas como lista (var_dump, foreach, etc.).
  • 1 = Índicies de array pueden ser accedidos como propiedades de lectura/escritura.

Valores devueltos

No devuelve ningún valor.
