ibase_rollback

(PHP 5, PHP 7)

ibase_rollback — Deshace una transacción

Descripción

bool ibase_rollback ([ resource $link_or_trans_identifier = NULL ] )

Deshace una transacción.

Parámetros

link_or_trans_identifier

Si se invoca sin argumentos, esta función deshace la transacción por omisión del enlace por omisión. Si el argumento fuera un identificador de conexión, se desharía su transacción por omisión. Si, por contra, fuera un identificador de transacción, ésta se desharía.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.
