wddx_serialize_vars

(PHP 4, PHP 5, PHP 7)

wddx_serialize_vars — Serializa variables en un paquete WDDX

Descripción

string wddx_serialize_vars ( mixed $var_name [, mixed $... ] )

Crea un paquete WDDX con una estructura que contiene la representación serializada de las variables pasadas.

Parámetros

Esta función toma un número variables de parámetros.

var_name

Pude ser una cadena, una variable o un array conteniendo una cadena, una variable u otro array, etc.

...

Valores devueltos

Devuelve el paquete WDDX, o FALSE en caso de error.

Ejemplos

Ejemplo #1 Ejemplo wddx_serialize_vars()

<?php
$a = 1;
$b = 5.5;
$c = array("blue", "orange", "violet");
$d = "colors";

$clvars = array("c", "d");
echo wddx_serialize_vars("a", "b", $clvars);
?>

El resultado del ejemplo sería:

<wddxPacket version='1.0'><header/><data><struct><var name='a'><number>1</number></var>
<var name='b'><number>5.5</number></var><var name='c'><array length='3'>
<string>blue</string><string>orange</string><string>violet</string></array></var>
<var name='d'><string>colors</string></var></struct></data></wddxPacket>
