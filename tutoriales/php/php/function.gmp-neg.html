gmp_neg

(PHP 4 >= 4.0.4, PHP 5, PHP 7)

gmp_neg — Número negativo

Descripción

GMP gmp_neg ( GMP $a )

Devuelve el valor negativo de un número.

Parámetros

a

resource de número de GMP en PHP 5.5 y anterior, o un objeto de GMP en PHP 5.6 y posterior, o un string numérico siempre que sea posible convertirlo a un número.

Valores devueltos

Devuelve -a, como un número GMP.

Ejemplos

Ejemplo #1 Ejemplo degmp_neg()

<?php
$neg1 = gmp_neg("1");
echo gmp_strval($neg1) . "\n";
$neg2 = gmp_neg("-1");
echo gmp_strval($neg2) . "\n";
?>

El resultado del ejemplo sería:

-1
1
