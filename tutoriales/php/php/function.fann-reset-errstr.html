fann_reset_errstr

(PECL fann >= 1.0.0)

fann_reset_errstr — Reinicia el string del último error

Descripción

void fann_reset_errstr ( resource $errdat )

Reinicia el string del último error.

Parámetros

errdat

Un resource de red neuronal o un resource de datos de entrenamiento de red neuronal.

Valores devueltos

No se devuelve ningún valor.
