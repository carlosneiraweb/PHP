ibase_drop_db

(PHP 5, PHP 7)

ibase_drop_db — Elimina una base de datos

Descripción

bool ibase_drop_db ([ resource $connection = NULL ] )

Esta función elimina una base de datos abierta con ibase_connect() o con ibase_pconnect(). La base de datos se cierra y se elimina del servidor.

Parámetros

connection

Identificador de enlace a InterBase. Si se omite, se utilizará el último enlace que se haya abierto.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.

Ver también

  • ibase_connect() - Abrir una conexión a una base de datos
  • ibase_pconnect() - Abrir una conexión persistente con una base de datos InterBase
