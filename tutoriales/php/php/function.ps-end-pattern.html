ps_end_pattern

(PECL ps >= 1.2.0)

ps_end_pattern — Finalizar un patrón

Descripción

bool ps_end_pattern ( resource $psdoc )

Finaliza un patrón que fue creado con la función ps_begin_pattern(). Una vez que el patrón ha sido finalizado se puede usar como un color para rellenar áreas.

Parámetros

psdoc

El identificador de recursos del fichero postscript, como el devuelto por la función ps_new().

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.
