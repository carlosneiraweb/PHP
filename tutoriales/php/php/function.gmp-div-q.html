gmp_div_q

(PHP 4 >= 4.0.4, PHP 5, PHP 7)

gmp_div_q — Divide los números

Descripción

GMP gmp_div_q ( GMP $a , GMP $b [, int $round = GMP_ROUND_ZERO ] )

Divide a con b y devuelve el entero resultante.

Parámetros

a

El número que sera dividido.

resource de número de GMP en PHP 5.5 y anterior, o un objeto de GMP en PHP 5.6 y posterior, o un string numérico siempre que sea posible convertirlo a un número.

b

El número que a esta siendo dividido.

resource de número de GMP en PHP 5.5 y anterior, o un objeto de GMP en PHP 5.6 y posterior, o un string numérico siempre que sea posible convertirlo a un número.

round

El redondeo resultante es definido por el round, el cual puede tener uno de los siguientes valores:

  • GMP_ROUND_ZERO: El resultante es truncado hacia o.
  • GMP_ROUND_PLUSINF:El resultado es redondeado hacia +infinity.
  • GMP_ROUND_MINUSINF: El resultado es redondeado hacia -infinity.

resource de número de GMP en PHP 5.5 y anterior, o un objeto de GMP en PHP 5.6 y posterior, o un string numérico siempre que sea posible convertirlo a un número.

Valores devueltos

resource de número de GMP en PHP 5.5 y anterior, o un objeto GMP en PHP 5.6 y posterior.

Ejemplos

Ejemplo #1 Ejemplo de gmp_div_q()

<?php
$div1 = gmp_div_q("100", "5");
echo gmp_strval($div1) . "\n";

$div2 = gmp_div_q("1", "3");
echo gmp_strval($div2) . "\n";

$div3 = gmp_div_q("1", "3", GMP_ROUND_PLUSINF);
echo gmp_strval($div3) . "\n";

$div4 = gmp_div_q("-1", "4", GMP_ROUND_PLUSINF);
echo gmp_strval($div4) . "\n";

$div5 = gmp_div_q("-1", "4", GMP_ROUND_MINUSINF);
echo gmp_strval($div5) . "\n";
?>

El resultado del ejemplo sería:

20
0
1
0
-1

Notas

Nota:

This function can also be called as gmp_div().
