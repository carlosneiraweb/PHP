tidy::root

tidy_get_root

(PHP 5, PHP 7, PECL tidy 0.5.2-1.0.0)

tidy::root -- tidy_get_root — Devuelve un objeto tidyNode que representa la raíz del árbol analizado por tidy

Descripción

Estilo orientado a objetos

tidyNode tidy::root ( void )

Estilo por procedimientos

tidyNode tidy_get_root ( tidy $object )

Devuelve un objeto tidyNode que representa la raíz del arbol a ser analizada por tidy.

Parámetros

object

El objeto Tidy.

Valores devueltos

Devuelve el objeto tidyNode.

Ejemplos

Ejemplo #1 tidy::root() ejemplo

<?php

$html = <<< HTML
<html><body>

<p>paragraph</p>
<br/>

</body></html>
HTML;

$tidy = tidy_parse_string($html);
dump_nodes($tidy->root(), 1);


function dump_nodes($node, $indent) {

    if($node->hasChildren()) {
        foreach($node->child as $child) {
            echo str_repeat('.', $indent*2) . ($child->name ? $child->name : '"'.$child->value.'"'). "\n";

            dump_nodes($child, $indent+1);
        }
    }
}
?>

El resultado del ejemplo sería:

..html
....head
......title
....body
......p
........"paragraph"
......br

Notas

Nota: Esta función está disponible sólo con Zend Engine 2 (PHP >= 5.0.0).
