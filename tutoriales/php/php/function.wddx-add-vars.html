wddx_add_vars

(PHP 4, PHP 5, PHP 7)

wddx_add_vars — Agrega variables a un paquete WDDX con un ID específico

Descripción

bool wddx_add_vars ( resource $packet_id , mixed $var_name [, mixed $... ] )

Serializa las variables pasadas y agrega el resultado al paquete determinado.

Parámetros

Esta función toma un número variable de parámetros.

packet_id

Un paquete WDDX, devuelto por wddx_packet_start().

var_name

Pude ser una cadena, una variable o un array conteniendo una cadena, una variable u otro array, etc.

...

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.
