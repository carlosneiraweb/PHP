MongoCommandCursor::next

(PECL mongo >=1.5.0)

MongoCommandCursor::next — Avanza el cursor al siguiente resultado

Descripción

public void MongoCommandCursor::next ( void )

Parámetros

Esta función no tiene parámetros.

Valores devueltos

NULL.

Errores/Excepciones

Lanza una MongoConnectionException si no se puede conectar con la base de datos, y una MongoCursorTimeoutException si se excede el tiempo límite.
