CairoContext::hasCurrentPoint

cairo_has_current_point

(PECL cairo >= 0.1.0)

CairoContext::hasCurrentPoint -- cairo_has_current_point — El propósito de hasCurrentPoint

Descripción

Estilo orientado a objetos (método):

public bool CairoContext::hasCurrentPoint ( void )

Estilo por procedimientos:

bool cairo_has_current_point ( CairoContext $context )

Devuelve si está definido un punto actual sobre el trazado actual. Véase CairoContext::getCurrentPoint() para detalles sobre el punto actual.

Parámetros

context

Un objeto CairoContext válido.

Valores devueltos

Si un punto actual está definido

Ejemplos

Ejemplo #1 Estilo orientado a objetos

<?php

$s = new CairoImageSurface(CairoFormat::ARGB32, 100, 100);
$c = new CairoContext($s);

var_dump($c->hasCurrentPoint());

$c->moveTo(10, 10);

var_dump($c->hasCurrentPoint());

?>

El resultado del ejemplo sería:

bool(false)
bool(true)

Ejemplo #2 Estilo por procedimientos

<?php

$s = cairo_image_surface_create(CAIRO_SURFACE_TYPE_IMAGE, 100, 100);
$c = cairo_create($s);

var_dump(cairo_has_current_point($c));

cairo_move_to($c, 10, 10);

var_dump(cairo_has_current_point($c));

?>

El resultado del ejemplo sería:

bool(false)
bool(true)
