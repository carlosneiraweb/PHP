ps_new

(PECL ps >= 1.1.0)

ps_new — Crear un nuevo objeto de documento PostScript

Descripción

resource ps_new ( void )

Crea una nueva instancia de documento. No crea el fichero en disco o en memoria, simplemente establece todo. ps_new() normalmente es seguido de una llamada a la función ps_open_file() para crear realmente el documento postscript.

Valores devueltos

Un recurso de un documento PostScript, o FALSE en caso de error. El valor devuelto es pasado a todas las demás funciones como primer argumento.
