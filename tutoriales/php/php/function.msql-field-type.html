msql_field_type

(PHP 4, PHP 5, PHP 7)

msql_field_type — Obtiene el tipo de campo

Descripción

string msql_field_type ( resource $result , int $field_offset )

msql_field_type() obtiene el tipo del índice del campo especificado.

Parámetros

result

El resultado resource que está siendo evaluado. Este resultado proviene de una llamada a msql_query().

field_offset

El número del campo a buscar. El valor de field_offset comienza en 1.

Valores devueltos

El tipo de campo. Alguno de los siguientes: int, char, real, ident, null o unknown. Estas funciones devolverán FALSE en caso de error.
