ifx_textasvarchar

(PHP 4, PHP <=5.2.0)

ifx_textasvarchar — Establecer el modo de texto predeterminado

Descripción

bool ifx_textasvarchar ( int $mode )

Establece el modo de texto predeterminado para todas las consultas "SELECT".

Parámetros

mode

El modo "0" devolverá un identificador BLOB y el modo "1" devolverá un VARCHAR que contenga el texto.

Valores devueltos

Devuelve TRUE en caso de éxito, o FALSE en caso de error.
