GearmanClient::doJobHandle

(PECL gearman >= 0.5.0)

GearmanClient::doJobHandle — Obtiene el manejador de trabajo para la tarea en ejecución

Descripción

public string GearmanClient::doJobHandle ( void )

Retorna el manejador de trabajo para la tarea en ejecución. Debería usarse entre llamadas repetidas a GearmanClient::doNormal(). El manejador puede ser usado para obtener información de la tarea.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

El manejador de trabajo para la tarea en ejecución.

  • GearmanClient::jobStatus() - Obtiene el estado de un trabajo en segundo plano
