MongoInt64::__construct

(PECL mongo >= 1.0.9)

MongoInt64::__construct — Crea un nuevo entero de 64 bits

Esta extensión que define esta clase está obsoleta. En su lugar debe usarse la extensión MongoDB. No existe un equivalente para esta clase en la nueva extensión.

En su lugar, la nueva extensión elige el tipo de base de datos apropiado según el valor del entero.

Descripción

public MongoInt64::__construct ( string $value )

Crea un nuevo número de 64 bits con el valor proporcionado.

Parámetros

value

Un número.

Valores devueltos

Devuelve un nuevo entero.
