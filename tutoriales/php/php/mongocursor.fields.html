MongoCursor::fields

(PECL mongo >=1.0.6)

MongoCursor::fields — Establece los campos para una consulta

Descripción

public MongoCursor MongoCursor::fields ( array $f )

Los cambpo se especifican mediante "nombre_campo" : bool. TRUE indica que se debería devolver un campo, FALSE indica que no debería ser devuelto. También se puede usar 1 y 0 en lugar de TRUE y FALSE.

Así, para devolver solamente el campo "summary", sería:

<?php

$cursor->fields(array("summary" => true));

?>

Para devolver todos los campos excepto el campo "hidden":

<?php

$cursor->fields(array("hidden" => false));

?>

Parámetros

f

Los campo a devolver (o no).

Valores devueltos

Devuelve este cursor.

Errores/Excepciones

Lanza una MongoCursorException si este cursor ha empezado a iterar o se proporciona un argumento escalar.
