HaruDoc::addPage

(PECL haru >= 0.0.1)

HaruDoc::addPage — Agrega una nueva página al documento

Descripción

object HaruDoc::addPage ( void )

Agrega una nueva página al documento.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve una nueva instancia de HaruPage.

Errores/Excepciones

Lanza una HaruException cuando se produce un error.

Ver también

  • HaruDoc::insertPage() - Inserta una nueva página solo antes de la página especificada
