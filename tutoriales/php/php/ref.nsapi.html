NSAPI implementa un subconjunto de funciones del módulo Apache para una máxima compatibilidad.

Funciones Apache implementadas por NSAPI
Funciones Apache (sólo como alias) Funciones NSAPI Descripción

apache_response_headers() nsapi_response_headers() Recupera todas las cabeceras de respuesta HTTP
getallheaders() nsapi_request_headers() Recupera todas las cabeceras de petición HTTP
virtual() nsapi_virtual() Hace una sub-petición NSAPI
