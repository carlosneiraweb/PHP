gmp_clrbit

(PHP 4 >= 4.0.4, PHP 5, PHP 7)

gmp_clrbit — Limpia un bit

Descripción

void gmp_clrbit ( GMP $a , int $index )

Limpia el bit (lo establece a 0) index en a. El índice empieza 0.

Parámetros

a

resource de número de GMP en PHP 5.5 y anterior, o un objeto de GMP en PHP 5.6 y posterior, o un string numérico siempre que sea posible convertirlo a un número.

index

El índice del bit a limpiar. El índice 0 representa el bit menos significantivo.

Valores devueltos

resource de número de GMP en PHP 5.5 y anterior, o un objeto GMP en PHP 5.6 y posterior.

Ejemplos

Ejemplo #1 Ejemplo de gmp_clrbit()

<?php
$a = gmp_init("0xff");
gmp_clrbit($a, 0); // index starts at 0, least significant bit
echo gmp_strval($a) . "\n";
?>

El resultado del ejemplo sería:

254

Notas

Nota:

A diferencia de otras funciones GMP, La gmp_clrbit() tiene que ser llamada con un recurso GMP que ya exista (usando gmp_init() por ejemplo). No se va a crear automáticamente.

Ver también

  • gmp_setbit() - Establece el bit
  • gmp_testbit() - Prueba si un bit es establecido
