px_close

(PECL paradox >= 1.0.0)

px_close — Cerrar una base de datos paradox

Descripción

bool px_close ( resource $pxdoc )

Cierra la base de datos paradox. Esta función no cierra el archivo. Deberá llamar seguidamente a la función fclose().

Parámetros

pxdoc

Identificador de recursos de la base de datos paradox devuelto por la función px_new().

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.
