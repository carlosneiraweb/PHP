is_array

(PHP 4, PHP 5, PHP 7)

is_array — Comprueba si una variable es un array

Descripción

bool is_array ( mixed $var )

Comprueba si la variable es un array.

Parámetros

var

La variable a ser evaluada.

Valores devueltos

Devuelve TRUE si var es un array, o de lo contrario FALSE.

Ejemplos

Ejemplo #1 Comprobar que la variable es un array

<?php
$si = array('esto', 'es', 'un array');

echo is_array($si) ? 'Array' : 'No es un array';
echo "\n";

$no = 'esto es un string';

echo is_array($no) ? 'Array' : 'No es un array';
?>

El resultado del ejemplo sería:

Array
No es un array
