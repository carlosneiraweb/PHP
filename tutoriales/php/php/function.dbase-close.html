dbase_close

(PHP 4, PHP 5, PHP 7)

dbase_close — Cierra una base de datos

Descripción

bool dbase_close ( int $dbase_identifier )

Cierra el identificador de enlace de la base de datos dado.

Parámetros

dbase_identifier

El identificador de enlace de la base de datos, devuelto por dbase_open() o dbase_create()..

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.

Ejemplos

Ejemplo #1 Cerrar un archivo de base de datos de dBase

<?php

// abrir en modo sólo lectura
$db = dbase_open('/tmp/test.dbf', 0);

if ($db) {
  // leer alguna información ..
  
  dbase_close($db);
}

?>
