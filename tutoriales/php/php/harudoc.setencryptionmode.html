HaruDoc::setEncryptionMode

(PECL haru >= 0.0.1)

HaruDoc::setEncryptionMode — Establece el modo de cifrado para el documento

Descripción

bool HaruDoc::setEncryptionMode ( int $mode [, int $key_len = 5 ] )

Establece el modo de cifrado para el documento. El modo de cifrado no puede ser establecido antes de establecer la contraseña.

Parámetros

mode

El modo de encripción a utilizar. Puede ser uno de los siguientes:

  • HaruDoc::ENCRYPT_R2 - utilizar el algoritmo "revision2".
  • HaruDoc::ENCRYPT_R3 - utilizar el algoritmo "revision3". El uso de esta constante es incompatible con PDF 1.4.

key_len

La longitud de la clave de cifrado en bytes. Este parámetro es opcional y solo será utilizado cuando el modo sea HaruDoc::ENCRYPT_R3. El valor por defecto es 5 (40bit).

Valores devueltos

Devuelve TRUE on success.

Errores/Excepciones

Lanza una HaruException cuando se produce un error.
