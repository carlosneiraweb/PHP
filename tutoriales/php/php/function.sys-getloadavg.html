sys_getloadavg

(PHP 5 >= 5.1.3, PHP 7)

sys_getloadavg — Obtiene la carga media del sistema

Descripción

array sys_getloadavg ( void )

Devuelve tres muestras que representan la carga media del sistema (el número de procesos en la cola de ejecución del sistema) sobre los últimos 1, 5 y 15 minutos, respectivamente.

Valores devueltos

Devuelve un array con tres muestras (últimos 1, 5 y 15 minutos).

Ejemplos

Ejemplo #1 Ejemplo de sys_getloadavg()

<?php
$carga = sys_getloadavg();
if ($carga[0] > 0.80) {
    header('HTTP/1.1 503 Ocupado, pruebe más tarde');
    die('Servidor ocupado. Por favor, inténtelo de nuevo más tarde.');
}
?>

Notas

Nota: Esta función no está implementada en plataformas Windows.
