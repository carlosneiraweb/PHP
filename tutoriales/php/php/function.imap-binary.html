imap_binary

(PHP 4, PHP 5, PHP 7)

imap_binary — Convertir una cadena 8bit a una cadena base64

Descripción

string imap_binary ( string $string )

Convertir una cadena 8bit a una cadena base64 de acuerdo al documento » RFC2045, Sección 6.8.

Parámetros

string

La cadena 8bit

Valores devueltos

Devuelve una cadena codificada mediante base64.
