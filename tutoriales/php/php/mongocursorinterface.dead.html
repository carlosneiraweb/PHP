MongoCursorInterface::dead

(PECL mongo >=1.5.0)

MongoCursorInterface::dead — Comprueba si existen resultados que aún no han sido enviados desde la base de datos

Descripción

abstract public bool MongoCursorInterface::dead ( void )

Este método comprueba si el cursor ha finalizado y la base de datos no tiene más resultados que enviar al cliente. Un cursor "muerto" no significa necesariamente que no hayan más resultados disponibles que iterar.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve TRUE si hay más resutaldos que aún no han sido enviados al cliente, y FALSE en caso contrario.
