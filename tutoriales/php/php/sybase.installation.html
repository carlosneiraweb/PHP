Instalación

Para habilitar Sybase-DB se debe configurar PHP con la opción --with-sybase[=DIR] . DIR es el directorio principal Sybase, por omisión es /home/sybase. Para habilitar Sybase-CT se debe configurar PHP con la opción --with-sybase-ct[=DIR] . DIR es el directorio principal Sybase, por omisión es /home/sybase.

Nota:

A partir de PHP 5.3.0, la extensión Sybase ha sido reemplazada por la extensión sybase_ct y ya no se mantiene. Para continuar usando sybase debe utilizar la extensión sybase_ct.
