HaruDoc::getStreamSize

(PECL haru >= 0.0.1)

HaruDoc::getStreamSize — Obtiene el tamaño de la secuencia temporal

Descripción

int HaruDoc::getStreamSize ( void )

Obtiene el tamaño de la secuencia temporal.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve el tamaño de los datos en la secuencia temporal del documento. El tamaño es cero si el documento no fue salvado en la secuencia temporal.

Errores/Excepciones

Lanza una HaruException cuando se produce un error.

Ver también

  • HaruDoc::saveToStream() - Guarda el documento en una secuencia temporal
  • HaruDoc::resetStream() - Rebobina la secuencia temporal
  • HaruDoc::readFromStream() - Lee los datos de la corriente temporal
