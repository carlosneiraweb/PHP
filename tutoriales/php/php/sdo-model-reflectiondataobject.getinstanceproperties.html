SDO_Model_ReflectionDataObject::getInstanceProperties — Obtener las propiedades de instancia del SDO_DataObject

Descripción

array SDO_Model_ReflectionDataObject::getInstanceProperties ( void )
Advertencia

Esta función ha sido declarada EXPERIMENTAL. Su funcionamiento, nombre y la documentación que le acompaña puede cambiar sin previo aviso en futuras versiones de PHP. Utilícela bajo su propia responsabilidad.

Obtiene las propiedades de instancia del SDO_DataObject. Las propiedades de instancia consisten en todas las propiedades definidas en el tipo del objeto de datos, más cualquier propiedad de instancia del contenido abierto (si el objeto de datos es de tipo abierto).

Parámetros

Ninguno.

Valores devueltos

Un array de objetos SDO_Model_Property.
