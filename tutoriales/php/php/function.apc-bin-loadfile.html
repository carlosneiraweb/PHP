apc_bin_loadfile

(PECL apc >= 3.1.4)

apc_bin_loadfile — Cargar una copia binaria desde un fichero a la caché de fichero/usuario de APC

Descripción

bool apc_bin_loadfile ( string $filename [, resource $context = NULL [, int $flags = 0 ]] )

Carga una copia binaria desde un fichero a la caché de fichero/usuario de APC.

Parámetros

filename

El nombre de fichero que contiene la copia, probablemente desde apc_bin_dumpfile().

context

El contexto de ficheros.

flags

Puede ser APC_BIN_VERIFY_CRC32, APC_BIN_VERIFY_MD5, o ambos.

Valores devueltos

Devuelve TRUE en caso de éxito, si no devuelve FALSE. Las razones por la que devuelve FALSE incluyen que APC no está habilitado, filename no es un nombre de fichero válido o está vacío, no se puede abrir filename, la copia del fichero no se puede completar, o si data no es una copia binaria de APC válida (p.ej., tamaño no esperado).
