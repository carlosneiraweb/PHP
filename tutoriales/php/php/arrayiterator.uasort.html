ArrayIterator::uasort

(PHP 5 >= 5.2.0, PHP 7)

ArrayIterator::uasort — Ordenado definido por el usuario

Descripción

public void ArrayIterator::uasort ( string $cmp_function )

Ordena las entradas por los valores usando una función definida por el usuario.

Advertencia

Esta función no está documentada actualmente, solamente se encuentra disponible la lista de parámetros.

Parámetros

cmp_function

La función de comparación usada para el ordenado.

Valores devueltos

No devuelve ningún valor.
