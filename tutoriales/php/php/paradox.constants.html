Constantes predefinidas

Estas constantes están definidas por esta extensión y estarán disponibles sólo cuando la extensión haya sido compilada con PHP, o bien sea cargada dinámicamente en ejecución.

Las dos siguientes tablas listan todas las constantes definido por la extensión paradox.

Contantes para tipos de campo
Nombre Significado
PX_FIELD_ALPHA Datos de tipo carácter de longitud fija
PX_FIELD_DATE Fecha, el número de días desde el 1.1.0000
PX_FIELD_SHORT Entero de tamaño corto (2 Bytes)
PX_FIELD_LONG Entero de tamaño largo (4 Bytes)
PX_FIELD_CURRENCY Igual que PX_FIELD_NUMBER
PX_FIELD_NUMBER Double
PX_FIELD_LOGICAL Booleano
PX_FIELD_MEMOBLOB Objeto grande binario
PX_FIELD_BLOB Objeto grande binario (No se admite)
PX_FIELD_FMTMEMOBLOB Objeto grande binario
PX_FIELD_OLE Objeto OLE (básicamente un blob, no se admite)
PX_FIELD_GRAPHIC Gráfico (básicamente un blob, no se admite)
PX_FIELD_TIME tiempo, número de milisegundos desde la medianoche
PX_FIELD_TIMESTAMP fecha y hora, número de milisegundos desde el 1.1.0000
PX_FIELD_AUTOINC Incremento automático de enteros (como PX_FIELD_LONG)
PX_FIELD_BCD Número decimal almacenado en formato BCD (no se admite)
PX_FIELD_BYTES Matriz de bytes con un máximo de 255 bytes (no se admite)
PX_KEYTOLOWER Convertir todos los nombres de campo en minúsculas
PX_KEYTOUPPER Convertir todos los nombres de campo en mayúsculas
Contantes para tipos de archivos
Nombre Significado
PX_FILE_INDEX_DB Base de datos indexada
PX_FILE_PRIM_INDEX Índice primario
PX_FILE_NON_INDEX_DB Ninguna base de datos indexada
PX_FILE_NON_INC_SEC_INDEX Ningún índice secundario incremental
PX_FILE_SEC_INDEX Índice secundario
PX_FILE_INC_SEC_INDEX Índice secundario incremental
PX_FILE_NON_INC_SEC_INDEX_G Índice secundario no incremental
PX_FILE_SEC_INDEX_G Índice secundario
PX_FILE_INC_SEC_INDEX_G Índice secundario no incremental
