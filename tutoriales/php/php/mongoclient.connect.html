MongoClient::connect

(PECL mongo >=1.3.0)

MongoClient::connect — Conecta a un servidor de bases de datos

Esta extenisón que define este método está obsoleta. En su lugar debe usarse la extensión MongoDB. No hay equivalente para este método en la nueva extensión.

Descripción

public bool MongoClient::connect ( void )

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Si la conexión se realizó con éxito.

Errores/Excepciones

Lanza una MongoConnectionException si falla al conectarse a la base de datos.
