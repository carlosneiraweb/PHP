Memcached::isPersistent

(PECL memcached >= 2.0.0)

Memcached::isPersistent — Comprueba si se está utilizando una conexión persistente a memcache

Descripción

public bool Memcached::isPersistent ( void )

Memcached::isPersistent() comprueba si las conexiones a los servidores de memcache son conexiones persistentes.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve true si la instancia de Memcache usa una conexión persistente, de lo contrario devuelve false.
