LimitIterator::rewind

(PHP 5 >= 5.1.0, PHP 7)

LimitIterator::rewind — Rebobina el iterador a un punto específicado

Descripción

public void LimitIterator::rewind ( void )

Rebobina el iterador a un índice de inicio específicado en LimitIterator::__construct().

Parámetros

Esta función no tiene parámetros.

Valores devueltos

No devuelve ningún valor.
