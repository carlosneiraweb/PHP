px_numrecords

(PECL paradox >= 1.0.0)

px_numrecords — Devuelve el número de registros en una base de datos

Descripción

int px_numrecords ( resource $pxdoc )

Obtener el número de registros en un archivo de base de datos.

Parámetros

pxdoc

Identificador de recursos de la base de datos paradox devuelto por la función px_new().

Valores devueltos

Devuelve el número de registros en un archivo de base de datos. El valor de retorno de esta función es idéntico al elemento numrecords en la matriz devuelta por la función px_get_info().
