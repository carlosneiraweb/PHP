mcrypt_module_is_block_mode

(PHP 4 >= 4.0.2, PHP 5, PHP 7)

mcrypt_module_is_block_mode — Devuelve si el modo especificado retorna bloques o no

Descripción

bool mcrypt_module_is_block_mode ( string $mode [, string $lib_dir ] )

Esta función devuelve TRUE si el modo especificado retorna bloques de bytes o FALSE si retorna tan solo bytes. (por ej., TRUE para cbc y ecb, y FALSE para cfb y stream).

Parámetros

mode

Una de las constantes MCRYPT_MODE_nombredelmodo, o una de las siguientes strings: "ecb", "cbc", "cfb", "ofb", "nofb" o "stream".

lib_dir

El parámetro opcional lib_dir puede contener la ubicación donde el módulo de algoritmos se encuentra dentro del sistema.

Valores devueltos

Esta función devuelve TRUE si el modo especificado retorna bloques de bytes o FALSE si retorna tan solo bytes. (por ej., TRUE para cbc y ecb, y FALSE para cfb y stream).
