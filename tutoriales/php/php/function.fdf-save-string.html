fdf_save_string

(PHP 4 >= 4.3.0, PHP 5, PHP 7)

fdf_save_string — Devuelve el documento FDF en forma de string

Descripción

string fdf_save_string ( resource $fdf_document )

Devuelve el documento FDF en forma de string.

Parámetros

fdf_document

Manejador de documentos FDF, devuelto por fdf_create(), fdf_open() o por fdf_open_string().

Valores devueltos

Devuelve el documento en forma de string, o FALSE en caso de error.

Ejemplos

Ejemplo #1 Salvando un FDF como string

<?php
$fdf = fdf_create();
fdf_set_value($fdf, "foo", "bar");
$str = fdf_save_string($fdf);
fdf_close($fdf);
echo $str;
?>
