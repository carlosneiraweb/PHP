fann_get_sarprop_step_error_threshold_factor

(PECL fann >= 1.0.0)

fann_get_sarprop_step_error_threshold_factor — Devuelve el factor de umbral del error de paso de sarprop

Descripción

float fann_get_sarprop_step_error_threshold_factor ( resource $ann )

Devuelve el factor de umbral del error de paso de sarprop.

El factor predeterminado es 0.1.

Parámetros

ann

Un resource de red neuronal.

Valores devueltos

El factor de umbral del error de paso de sarprop, o FALSE en caso de error.
