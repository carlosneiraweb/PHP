fbsql_field_type

(PHP 4 >= 4.0.6, PHP 5, PHP 7)

fbsql_field_type — Obtiene el tipo de campo especificado en un resultado

Descripción

string fbsql_field_type ( resource $result [, int $field_offset ] )

fbsql_field_type() es similar a la función fbsql_field_name(), pero se devuelve el tipo de campo en su lugar.

Parámetros

result

Un identificador de resultado devuelto por fbsql_query() o fbsql_db_query().

field_offset

La posición numérica (offset) del campo. El índice comienza por 0.

Valores devueltos

Devuelve el tipo de campo, como string.

Puede ser: int, real, string, blob, además de otras, como se detalla en la » documentación de FrontBase.

Ejemplos

Ejemplo #1 Ejemplo de fbsql_field_type()

<?php

fbsql_connect("localhost", "_SYSTEM", "");
fbsql_select_db("wisconsin");
$result = fbsql_query("SELECT * FROM onek;");
$fields = fbsql_num_fields($result);
$rows   = fbsql_num_rows($result);
$i = 0;
$table = fbsql_field_table($result, $i);
echo "La tabla '" . $table . "' tiene " . $fields . " campos y " . $rows . " registros <br />";
echo "La tabla tiene los siguientes campos: <br />";
while ($i < $fields) {
    $type  = fbsql_field_type($result, $i);
    $name  = fbsql_field_name($result, $i);
    $len   = fbsql_field_len($result, $i);
    $flags = fbsql_field_flags($result, $i);
    echo $type . " " . $name . " " . $len . " " . $flags . "<br />";
    $i++;
}
fbsql_close();

?>
