dba_close

(PHP 4, PHP 5, PHP 7)

dba_close — Cerrar una base de datos DBA

Descripción

void dba_close ( resource $handle )

dba_close() cierra la base de datos establecida y libera todos los recursos del gestor de base de datos especificado.

Parámetros

handle

El gestor de base de datos, devuelto por dba_open() o dba_popen().

Valores devueltos

No devuelve ningún valor.

Ver también

  • dba_open() - Abrir una base de datos
  • dba_popen() - Abrir una base de datos de forma persistente
