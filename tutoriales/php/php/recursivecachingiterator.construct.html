RecursiveCachingIterator::__construct

(PHP 5 >= 5.1.0, PHP 7)

RecursiveCachingIterator::__construct — Construct

Descripción

public RecursiveCachingIterator::__construct ( Iterator $iterator [, string $flags = self::CALL_TOSTRING ] )

Construye un nuevo RecursiveCachingIterator, que consiste en pasar un iterator.

Advertencia

Esta función no está documentada actualmente, solamente se encuentra disponible la lista de parámetros.

Parámetros

iterator

El iterador a ser usado.

flags

Las flags. Use CALL_TOSTRING para llamar a RecursiveCachingIterator::__toString() para cada elemento (por omisión), y/o CATCH_GET_CHILD para capturar excepciones cuando se intenta obtener un hijo.

Valores devueltos

La clase RecursiveCachingIterator.
