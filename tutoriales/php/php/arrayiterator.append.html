ArrayIterator::append

(PHP 5 >= 5.0.0, PHP 7)

ArrayIterator::append — Añade un elemento

Descripción

public void ArrayIterator::append ( mixed $value )

Añade el valor cómo el último elemento.

Advertencia

Esta función no está documentada actualmente, solamente se encuentra disponible la lista de parámetros.

Parámetros

value

El valor a añadir.

Valores devueltos

No devuelve ningún valor.

Notas

Nota:

Este método no puede ser llamado cuando ArrayIterator hace referencia a un objeto.
