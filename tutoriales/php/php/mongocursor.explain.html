MongoCursor::explain

(PECL mongo >=0.9.2)

MongoCursor::explain — Devuelve una explicación de la consulta, útil para la optimización y depuración

Descripción

public array MongoCursor::explain ( void )

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve la explicación de la consulta.

Ejemplos

Ejemplo #1 Ejemplo de MongoCursor::explain()

<?php

$cursor = $collection->find(array("x"=>1), array("y"));
$cursor->sort(array("z" => 1))->limit(4)->skip(5);

var_dump($cursor->explain());

?>
