ibase_errcode

(PHP 5, PHP 7)

ibase_errcode — Devuelve un código de error

Descripción

int ibase_errcode ( void )

Devuelve un código de error correspondiente a la última llamada a una función InterBase.

Valores devueltos

Devuelve el código de error en forma de entero, o FALSE si no hubo ningún error.
