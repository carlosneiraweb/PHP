Imagick::getFont

(PECL imagick 2.1.0)

Imagick::getFont — Obtiene la fuente

Descripción

string Imagick::getFont ( void )

Devuelve la propiedad fuente de los objetos. Este método está disponible si Imagick ha sido compilado con la versión 6.3.7 o superior de ImageMagick.

Valores devueltos

Devuelve la cadena que contiene el nombre de la fuente o FALSE si no está establecida la fuente.

Ver también

  • Imagick::setFont() - Establece el tipo de letra
  • ImagickDraw::setFont() - Establece la fuente especificada completamente para usarla cuando se escribe texto
  • ImagickDraw::getFont() - Devuelve la fuente
