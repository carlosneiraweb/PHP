Introducción

Estas funciones trabajan en conjunto con la API MCVE (Monetra) (libmonetra, formalmente conocida como libmcve), permitiendo trabajar directamente con MCVE/Monetra desde sus scripts de PHP. MCVE/Monetra es una solución de Main Street Softworks para dirigir el procesamiento de tarjetas de crédito/débito/regalo para Linux/Unix/MacOSX/Windows ( » http://www.mainstreetsoftworks.com/ ). Le permite tratar directamente con repositorios de tarjetas de crédito mediante su caja *nix, módem y/o conexión a internet (evitando la necesidad de un servicio adicional como Authorize.Net o Pay Flow Pro). Al usar el módulo de PHP MCVE/Monetra puede procesar tarjetas de crédito directamente a través de MCVE/Monetra mediante sus scripts de PHP. Las siguientes referencias explicarán brevemente el proceso.

Nota:

MCVE/Monetra es el sustituto del CCVS de RedHat. Hicieron un contrato con RedHat a finales de 2001 para migrar toda la clientela a la plataforms MCVE.

Nota:

Esta extensión se ha movido al repositorio de » PECL y ya no se distribuye con PHP a partir de PHP 5.1.0.

Nota: Esta extensión no está disponible en plataformas Windows.
