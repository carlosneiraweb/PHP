OCI-Collection::trim

(PHP 5, PHP 7, PECL OCI8 >= 1.1.0)

OCI-Collection::trim — Elimina elementos del final de la colección

Descripción

bool OCI-Collection::trim ( int $num )

Elimina el número de elementos dado por num del final de la colección.

Parámetros

num

El número de elementos a eliminar.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.
