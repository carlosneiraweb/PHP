msql_result

(PHP 4, PHP 5, PHP 7)

msql_result — Obtención de datos del resultado

Descripción

string msql_result ( resource $result , int $row [, mixed $field ] )

msql_result() devuelve el contenido de una celda de un conjunto de resultados mSQL.

Cuando se trabaja con conjuntos de resultados grandes, se debe barajar la posibilidad de usar una de las funciones que buscan una fila completa (especificadas abajo). Como estas funciones devuelven el contenido de varias celdas en una llamada a la función, a menudo son mucho más rápidas que msql_result().

Recomendaciones de alto rendimiento: msql_fetch_row(), msql_fetch_array(), y msql_fetch_object().

Parámetros

result

El resultado resource que está siendo evaluado. Este resultado proviene de una llamada a msql_query().

row

La fila del conjunto de resultados que está siendo recuperada.

field

Puede ser el campo de desplazamiento, o el nombre del campo o el campo de la tabla punto nombre del campo (nombre-de-tabla.nombre-de-campo). Si el nombre de la columna se ha abreviado ('select foo as bar from ...'), use la abreviatura en lugar del nombre de la columna.

Nota:

Especificar un campo numérico de tipo desplazamiento es mucho más rápido que especificar un nombre de campo (fieldname) o su ruta relativa dentro de la tabla (nombre-de-tabla.nombre-de-campo).

Valores devueltos

Devuelve el contenido de la celda en la fila y el desplazamiento del conjunto de resultados mSQL especificado. Returns the contents of the cell at the row and offset in the specified mSQL result set.
