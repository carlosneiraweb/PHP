dbase_numfields

(PHP 4, PHP 5, PHP 7)

dbase_numfields — Obtiene el número de campos de una base de datos

Descripción

int dbase_numfields ( int $dbase_identifier )

Obtiene el número de campos (columnas) de la base de datos especificada.

Nota:

El número de campos está entre 0 y dbase_numfields($db)-1, mientras que el número de registro está entre 1 y dbase_numrecords($db).

Parámetros

dbase_identifier

El identificador de enlace de la base de datos, devuelto por dbase_open() o dbase_create()..

Valores devueltos

El número de campos de la base de datos, o FALSE si ocurrió un error.

Ejemplos

Ejemplo #1 Ejemplo de dbase_numfields()

<?php

$reg = dbase_get_record($db, $recno);
$nc  = dbase_numfields($db);
for ($i = 0; $i < $nc; $i++) {
  echo $reg[$i], "\n";
}

?>
