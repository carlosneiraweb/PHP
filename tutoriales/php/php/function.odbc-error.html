odbc_error

(PHP 4 >= 4.0.5, PHP 5, PHP 7)

odbc_error — Obtener el último código de error

Descripción

string odbc_error ([ resource $connection_id ] )

Devuelve el estado de ODBC mediante un número de seis dígitos, o una cadena vacía si no hubo errores.

Parámetros

connection_id

El conector identificador ODBC, ver odbc_connect() para más información.

Valores devueltos

Si se especifica connection_id, se devuelve el último estado de esta conexión, si no, se devuleve el último estado de cualquier conexión.

Esta función devuelve un valor significativo sólo si la última consultaodbc falló (es decir, odbc_exec() devolvió FALSE).
