SolrQuery::getFacetDateOther

(PECL solr >= 0.9.2)

SolrQuery::getFacetDateOther — Devuelve el valor del parámetro facet.date.other

Descripción

public array SolrQuery::getFacetDateOther ([ string $field_override ] )

Devuelve el valor del parámetro facet.date.other. Este método acepta una sobrescritura opcional de campos.

Parámetros

field_override

El nombre del campo

Valores devueltos

Devuelve una cadena en caso de éxito y NULL si no se estableció.
