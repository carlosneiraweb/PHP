SCA::createDataObject

(PECL SDO >= 0.5.0)

SCA::createDataObject — Crear un SDO

Descripción

SDO_DataObject SCA::createDataObject ( string $type_namespace_uri , string $type_name )
Advertencia

Esta función ha sido declarada EXPERIMENTAL. Su funcionamiento, nombre y la documentación que le acompaña puede cambiar sin previo aviso en futuras versiones de PHP. Utilícela bajo su propia responsabilidad.

Este método se usa dentro de un componente SCA que necesite crear un SDO que devolver. Los parámetros son la URI del espacio de nombres del SDO deseada y el nombre del tipo. El espacio de nombres y el tipo deben estar definidos en uno de los ficheros de esquemas que están especificados en la anotación @types dentro del componente.

Parámetros

type_namespace_uri

El espacio de nombres del tipo.

type_name

El nombre del tipo.

Valores devueltos

Devuelve el recién creado SDO_DataObject.

Errores/Excepciones

SDO_TypeNotFoundException

Lanzada si namespaceURI y typeName no se corresponden con un tipo de ningún fichero de esquemas especificado en las anotaciones @types.
