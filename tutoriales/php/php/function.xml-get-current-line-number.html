xml_get_current_line_number

(PHP 4, PHP 5, PHP 7)

xml_get_current_line_number — Obtiene el número de línea actual para un intérprete XML

Descripción

int xml_get_current_line_number ( resource $parser )

Obtiene el número de línea actual para el intérprete XML dado.

Parámetros

parser

Identificador del recurso del intérprete XML sobre el que se va a obtener el número de línea.

Valores devueltos

Devuelve el número de línea en el que se haya el intérprete actualmente en el buffer de datos, o devuelve FALSE si parser no se refiere a un intérprete válido.
