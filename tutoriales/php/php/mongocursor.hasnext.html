MongoCursor::hasNext

(PECL mongo >=0.9.0)

MongoCursor::hasNext — Comprueba si existe algún elemento más en este cursor

Descripción

public bool MongoCursor::hasNext ( void )

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve si existe otro elemento.

Errores/Excepciones

Lanza una MongoConnectionException si no se puede acceder a la base de datos, y una MongoCursorTimeoutException si se excede el tiempo de espera.
