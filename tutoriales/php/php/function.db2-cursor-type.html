db2_cursor_type

(PECL ibm_db2 >= 1.0.0)

db2_cursor_type — Determinar el tipo de cursor utilizado en una consulta

Descripción

int db2_cursor_type ( resource $stmt )

Devuelve el tipo de cursor utilizado en una consulta. Se puede utilizar para determinar si se está trabajando con un curso de tipo "forward-only" (sin recorrido) o "scrollable" (con recorrido).

Parámetros

stmt

Un recurso de sentencia válido.

Valores devueltos

Devuelve DB2_FORWARD_ONLY si la consulta utiliza un cursor de tipo "forward-only" (sin recorrido) o DB2_SCROLLABLE si utiliza un cursor de tipo "scrollable" (con recorrido).
