ingres_autocommit

(PHP 4 >= 4.0.2, PHP 5 <= 5.0.5, PECL ingres >= 1.0.0)

ingres_autocommit — Activa o desactiva la autoconsigna

Descripción

bool ingres_autocommit ( resource $link )

ingres_autocommit() se invoca antes de abrir una transacción (antes de la primera llamada a ingres_query(), o justo después de una llamada a ingres_rollback() o a ingres_commit()) para activar o desactivar el modo autoconsigna del servidor (cuando se inicia el script, el modo autoconsigan está desactivado).

Cuando el modo autoconsigna está activado, cada consulta es automáticamente consignada por el servidor, igual que si ingres_commit() fuera invocada después de cada llamada a ingres_query(). Para ver si la autoconsigna está habilitada, use ingres_autocommit_state().

Por omisión, Ingres revertirá cualquier transacción no consignada al final de una petición. Use esta función o ingres_commit() para asegurarse que los datos son consignados a la base de datos.

Parámetros

link

El identificador de enlace de la conexión

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.

Ver también

  • ingres_autocommit_state() - Comprueba si la conexión usa la autoconsigna
  • ingres_query() - Envía una consulta SQL a Ingres
  • ingres_rollback() - Revierte una transacción
  • ingres_commit() - Consigna una trasacción
