SplFileInfo::getSize

(PHP 5 >= 5.1.2, PHP 7)

SplFileInfo::getSize — Obtiene el tamaño de el fichero

Descripción

public int SplFileInfo::getSize ( void )

Devuelve el tamaño de el fichero en bytes para el fichero referenciado.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

El tamaño del fichero en bytes.

Errores/Excepciones

Lanza una RuntimeException si el fichero no existe o en caso de error.
