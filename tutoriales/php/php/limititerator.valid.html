LimitIterator::valid

(PHP 5 >= 5.1.0, PHP 7)

LimitIterator::valid — Comprobar si el elemento actual es válido

Descripción

public bool LimitIterator::valid ( void )

Comprobar si el elemento actual es válido.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.
