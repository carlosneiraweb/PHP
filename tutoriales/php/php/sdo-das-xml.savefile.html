SDO_DAS_XML::saveFile

(^)

SDO_DAS_XML::saveFile — Guarda el objeto SDO_DAS_XML_Document en un fichero

Descripción

void SDO_DAS_XML::saveFile ( SDO_XMLDocument $xdoc , string $xml_file [, int $indent ] )

Guarda el objeto SDO_DAS_XML_Document en un fichero.

Parámetros

xdoc

El objeto SDO_DAS_XML_Document.

xml_file

El fichero xml.

indent

Argumento opcional para especificar que el xml será formateado. Un valor entero no negativo es la cantidad de indentación de cada nivel del xml. Así, el entero 2, por ejemplo, indentará el xml, por lo que cada elemento contenido estará dos espacios a la derecha que su elemento contenedor. El entero 0 causará que el xml esté completamente alieado a la izquierda. El entero -1 significa sin formato - el xml saldrá como una única línea larga.

Valores devueltos

Ninguno.

Errores/Excepciones

SDO_DAS_XML_FileException

Lanzada si el fichero especificado no se puede encontrar.
