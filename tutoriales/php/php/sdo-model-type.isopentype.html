SDO_Model_Type::isOpenType — Comprobar si este tipo es un tipo abierto

Descripción

bool SDO_Model_Type::isOpenType ( void )
Advertencia

Esta función ha sido declarada EXPERIMENTAL. Su funcionamiento, nombre y la documentación que le acompaña puede cambiar sin previo aviso en futuras versiones de PHP. Utilícela bajo su propia responsabilidad.

Comrpuebas si este tipo es un tipo abierto. Devuelve TRUE si este tipo es abierto, si no, devuelve FALSE. Un SDO_DataObject cuyo tipo es abierto puede tener propiedades añadidas que no están descritas por el tipo. Esta capacidad se usa para soportar trabajar con documentos XML cuyo esquema soporta contenido abierto, tal como el definido por un elemento 

<xsd:any>
.

Parámetros

None.

Valores devueltos

Devuelve TRUE si este tipo es abierto, si no, devuelve FALSE.
