Introducción

La extensión MaxDB permite acceder a la funcionalidad proporcionada por MaxDB 7.5.0 o superior. Más información sobre el servidor de bases de datos MaxDB puede encontrarse en » http://www.sdn.sap.com/irj/sdn/maxdb.

La extensión MaxDB de PHP es compatible con la extensión MySQL mysqli. The MaxDB PHP extension is compatible to the MySQL mysqli extension. Sólo hay pequeñas diferencias en el comportamiento de algunas funciones, por las diferencias de los servidores de base de datos interna, MaxDB y MySQL.

Las principales diferencias de mysqli se encuentran en las siguientes funciones:

  • maxdb_character_set_name() - Devuelve sólo ascii o Unicode.
  • maxdb_get_client_info() - Devuelve un string de versión diferente.
  • maxdb_get_client_version() - Devuelve un string de versión diferente.
  • maxdb_get_host_info() - Devuelve localhost o hostname.
  • maxdb_get_server_info() - Devuelve un string de versión diferente.
  • maxdb_get_server_version() - Devuelve un string de versión diferente.
  • maxdb_kill() - Sólo desconecta el período de sesiones.
  • maxdb_multi_query() - No puede manejar múltiples sentencias SQL.
  • maxdb_next_result() - Esta función devuelve siempre false.
  • maxdb_options() - Compatible con un conjunto diferente de opciones.
  • maxdb_report() - Soporta un modo de informe diferente.
  • maxdb_stat() - Devuelve un string de estado del sistema diferente.
  • maxdb_stmt_store_result() - No es necesario para MaxDB.
  • maxdb_store_result() - No es necesario para MaxDB.

Documentación para MaxDB se puede encontrar en » http://maxdb.sap.com/documentation/.
