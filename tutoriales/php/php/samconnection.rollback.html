SAMConnection::rollback

(PECL sam >= 0.1.0)

SAMConnection::rollback — Cancela (revierte) una unidad de trabajo pendiente de ser consolidada.

Descripción

bool SAMConnection::rollback ( void )

Revierte una unidad de trabajo pendiente de ser consolidada.

Valores devueltos

Este método devuelve FALSE en caso de error.

Ejemplos

Ejemplo #1 Cancelación de una unidad de trabajo no consolidada

<?php
if (!$conn->rollback()) {
  // El rollback falló!
  echo "Rollback fallido ($conn->errno) $conn->error";
}
?>
