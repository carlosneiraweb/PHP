SDO_DAS_Relational::createRootDataObject

(^)

SDO_DAS_Relational::createRootDataObject — Devuelve el objeto raíz especial en un grafo de datos en principio vacío. Usado para crear un grafo de datos desde cero.

Descripción

SDODataObject SDO_DAS_Relational::createRootDataObject ( void )
Advertencia

Esta función ha sido declarada EXPERIMENTAL. Su funcionamiento, nombre y la documentación que le acompaña puede cambiar sin previo aviso en futuras versiones de PHP. Utilícela bajo su propia responsabilidad.

Devuelve el objeto raíz especial del inicio de un grafo de datos en principio vacío. Esta llamada se usa cuando una aplicación quiere crear un grafo de datos desde cero, sin tener que llamar a executeQuery() para crear un grafo de datos.

El objeto raíz especial tiene una propiedad de contención polievaluada, con un nombre del tipo raíz de aplicación que fue pasado cuando se construtó el DAS Relacional. La propiedad solo puede tomar valores de ese tipo. Lo único que la aplicación puede hacer de manera útil con el tipo raíz es llamar a createDataObject() sobre él, pasando el nombre del tipo raíz de aplicación, para crear un objeto de datos de su propio tipo de aplicación.

Parámetros

Ninguno.

Valores devueltos

El objeto raíz.

Errores/Excepciones

Ninguno.
