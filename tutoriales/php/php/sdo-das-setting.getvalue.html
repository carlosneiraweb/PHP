SDO_DAS_Setting::getValue — Obtener el valor antiguo de una propiedad cambiada

Descripción

mixed SDO_DAS_Setting::getValue ( void )
Advertencia

Esta función ha sido declarada EXPERIMENTAL. Su funcionamiento, nombre y la documentación que le acompaña puede cambiar sin previo aviso en futuras versiones de PHP. Utilícela bajo su propia responsabilidad.

Devuelve el valor antiguo de la propiedad cambiada. Puede ser usado por un Servicio de Acceso a Datos (DAS)cuando se escriben actualizaciones en un origen de datos. El DAS utiliza el valor antiguo para detectar conflictos comparándolo con el valor actual del origen de datos. Si no coinciden, el origen de datos ha sido actualizado ya que el objeto de datos fue rellenado originalmente, y así al escribir cualquier nueva actualización se arriesga el compromiso de integridad de datos.

Parámetros

Ninguno.

Valores devueltos

Devuelve el valor antiguo de una propiedad cambiada.
