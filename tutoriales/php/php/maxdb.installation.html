Instalación

Usando la opción de configuración --with-maxdb[=DIR] habilita a PHP para acceder a bases de MaxDB. [DIR] apunta al directorio que contiene el paquete instalado MaxDB SQLDBC.

Los usuarios de Windows necesitan activar php_maxdb.dll en el interior de php.ini.
