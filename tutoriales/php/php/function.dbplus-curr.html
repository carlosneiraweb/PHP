dbplus_curr

(PHP 4 <= 4.1.0, PECL dbplus >= 0.9)

dbplus_curr — Get current tuple from relation

Descripción

int dbplus_curr ( resource $relation , array &$tuple )

Lee la información de la tupla actual de la relación relation dada.

Parámetros

relation

Una relación abierta con dbplus_open().

tuple

La información será devuelta a este parámetro como una matriz asociativa.

Valores devueltos

La función devolverá cero (es decir, DBPLUS_ERR_NOERR) si se tuvo éxito o un código de error de db++ si falló.

Notas

Advertencia

Esta función ha sido declarada EXPERIMENTAL. Su funcionamiento, nombre y la documentación que le acompaña puede cambiar sin previo aviso en futuras versiones de PHP. Utilícela bajo su propia responsabilidad.
