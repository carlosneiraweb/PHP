Other changes

  • El conjunto de caracteres predeterminado para htmlspecialchars() y htmlentities() ahora es UTF-8, en lugar de ISO-8859-1. Observe que si se cambia el conjunto de caracteres de salida mediante el ajuste de configuración default_charset no afecta a htmlspecialchars/htmlentities a menos que se proporcione "" (una cadena vacía) como parámetro de codificación a las llamadas a htmlspecialchars()/htmlentities(). Generealmente no se recomienda hacer esto ya que se debería de poder cambiar el conjunto de caracteres de salida sin afectar al conjunto de caracteres en tiempo de ejecución usado por estas funciones.these functions. El enfoque más rápido es establecer explícitamente el conjunto de caracteres en cada llamada a htmlspecialchars() y htmlentities().
  • E_ALL ahora incluye los errores de nivel E_STRICT en la directiva de configuración error_reporting.
  • SNMP ahora tiene una API para POO. Las funciones ahora devuelven FALSE en cada condición de error incluyendo las relacionadas con SNMP (no existe tal instancia, fin de MIB, etc). Así, en particular, se rompe con el comportamiento de que las funciones get/walk devuelvan una cadena vacía en errores relacionados con SNMP. Las consultas multi-OID get/getnext/set ahora son soportadas. Compatibilidad con código Dropped UCD-SNMP, considere actualizar a net-snmp v5.3+, y se requiere Net-SNMP v5.4+ para la versión de Windows. Ahora el soporte para la resolución de nombres DNS de IPv6 DNS de agentes SNMP remotos SNMP (pares) se realiza por extensión, y ya no por la biblioteca Net-SNMP.
  • OpenSSL ahora soporta AES.
  • La SAPI CLI ya no finaliza con errores fatales al usar el modo interactivo con soporte de readline.
  • Se ha añadido $_SERVER['REQUEST_TIME_FLOAT'] para incluir una precisión de microsegundos.
  • Se ha añadido nuevos algoritmos hash: fnv132, fnv164, joaat
  • Los índices encadencados de strings - p.ej. $a[0][0] donde $a es un string - ahora funcionan.
  • La conversión de arrays desde SimpleXMLElement ahora siempre contiene todos los nodos en lugar del sólo el primero nodo coincidente. Todos los hijos de SimpleXMLElement ahora siempre se imprimen al usar var_dump(), var_export() y print_r().
  • Ahora es posible forzar los argumentos de __construct de una clase en un constructor abstracto en la clase base.
