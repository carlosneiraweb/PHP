MultipleIterator::containsIterator

(PHP 5 >= 5.3.0, PHP 7)

MultipleIterator::containsIterator — Comprueba si un iterador está adjunto

Descripción

public bool MultipleIterator::containsIterator ( Iterator $iterator )

Comprueba si un iterador está adjunto o no.

Advertencia

Esta función no está documentada actualmente, solamente se encuentra disponible la lista de parámetros.

Parámetros

iterator

El iterador a comprobar.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.
