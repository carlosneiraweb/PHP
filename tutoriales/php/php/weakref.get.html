Weakref::get

(PECL weakref >= 0.1.0)

Weakref::get — Devuelve el objeto apuntado por una referencia débil

Descripción

public object Weakref::get ( void )

Devuelve el objeto apuntado por una referencia débil.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve el objeto si la referencia aún es válida, o NULL en caso contrario.

  • Weakref::valid() - Comprueba si el objeto referenciado aún existe
