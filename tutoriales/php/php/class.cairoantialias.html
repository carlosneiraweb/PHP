La clase CairoAntialias

(No hay información de versión disponible, podría estar únicamente en SVN)

Introducción

Clase de tipo Enum que especifica el tipo de antialias a realizar cuando se renderiza texto o formas.

Sinopsis de la Clase

CairoAntialias {
/* Constantes */
const integer MODE_DEFAULT = 0 ;
const integer MODE_NONE = 1 ;
const integer MODE_GRAY = 2 ;
const integer MODE_SUBPIXEL = 3 ;
}

Constantes predefinidas

CairoAntialias::MODE_DEFAULT

Usar el altialias predeterminado para el subsistema y el dispositivo objetivo

CairoAntialias::MODE_NONE

Usar una máscara alfa de dos niveles

CairoAntialias::MODE_GRAY

Realizar un antialias de color sencillo (usando sombras de gris para texto negro sobre un fondo blanco, por ejemplo).

CairoAntialias::MODE_SUBPIXEL

Realizar un antialias tomando ventaja del orden de los elementos de subpíxel en dispositivos tales como paneles LCD.
