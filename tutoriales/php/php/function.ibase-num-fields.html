ibase_num_fields

(PHP 5, PHP 7)

ibase_num_fields — Consulta el número de campos de un resultado

Descripción

int ibase_num_fields ( resource $result_id )

Obtiene el número de campos de un juego de resultados.

Parámetros

result_id

Identificador de resultado de InterBase.

Valores devueltos

Devuelve el número de campos en forma de entero.

Ejemplos

Ejemplo #1 Ejemplo de ibase_num_fields()

<?php
$rs = ibase_query("SELECT * FROM tablename");
$coln = ibase_num_fields($rs);
for ($i = 0; $i < $coln; $i++) {
    $col_info = ibase_field_info($rs, $i);
    echo "name: " . $col_info['name'] . "\n";
    echo "alias: " . $col_info['alias'] . "\n";
    echo "relation: " . $col_info['relation'] . "\n";
    echo "length: " . $col_info['length'] . "\n";
    echo "type: " . $col_info['type'] . "\n";
}
?>
