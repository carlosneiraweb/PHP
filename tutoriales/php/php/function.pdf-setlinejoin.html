PDF_setlinejoin

(PHP 4, PECL pdflib >= 1.0.0)

PDF_setlinejoin — Establecer el parámetro linejoin

Descripción

bool PDF_setlinejoin ( resource $p , int $value )

Establece el parámetro linejoin para especificar la forma de las esquinas del trazado que está contorneado. Devuelve TRUE en caso de éxito o FALSE en caso de error.
