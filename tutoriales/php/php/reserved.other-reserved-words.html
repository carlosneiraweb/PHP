Listado de otras palabras reservadas

Las siguientes palabras no se pueden utilizar para dar nombre a una clase, interfaz o rasgo, y están también prohibidas emplearlas en espacios de nombres.

Palabras reservadas
int (a partir de PHP 7) float (a partir de PHP 7) bool (a partir de PHP 7) string (a partir de PHP 7)
true (a partir de PHP 7) false (a partir de PHP 7) null (a partir de PHP 7)

Las palabras de la siguiente lista han estado ligeramente reservadas. Aunque todavía se puden emplear como nombres de clases interfaces y rasgos, (así como en espacio de nombres), su uso está totalmente desaconsejado debido a que podrían ser utilizadas en versiones futuras de PHP.

Palabras ligeramente reservadas
resource (a partir de PHP 7) object (a partir de PHP 7) mixed (a partir de PHP 7) numeric (a partir de PHP 7)
