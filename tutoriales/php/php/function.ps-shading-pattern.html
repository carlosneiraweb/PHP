ps_shading_pattern

(PECL ps >= 1.3.0)

ps_shading_pattern — Crear un patrón basado en un sombreado

Descripción

int ps_shading_pattern ( resource $psdoc , int $shadingid , string $optlist )

Crea un patrón basado en un sombreado, el cual ha de ser creado antes con la función ps_shading(). Los patrones de sombreado se pueden usar como patrones normales.

Parámetros

psdoc

El identificador de recursos del fichero postscript, como el devuelto por la función ps_new().

shadingid

El identificador de un sombreado previamente creado con la función ps_shading().

optlist

Este argumento actualmente no se usa.

Valores devueltos

El identificador del patrón o FALSE en caso de error.
