imap_undelete

(PHP 4, PHP 5, PHP 7)

imap_undelete — Desmarcar el mensaje que está marcado para borrar

Descripción

bool imap_undelete ( resource $imap_stream , int $msg_number [, int $flags = 0 ] )

Elimina la bandera de borrado de un mensaje especificado, que está establecida mediante imap_delete() o imap_mail_move().

Parámetros

imap_stream

IMAP stream devuelto por imap_open().

msg_number

EL número de mensaje

flags

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.

Ver también

  • imap_delete() - Marcar un mensaje para su borrado del buzón actual
  • imap_mail_move() - Mover los mensajes especificados a un buzón
