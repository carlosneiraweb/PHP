Gmagick::queryformats

(PECL gmagick >= Unknown)

Gmagick::queryformats — Devuelve los formatos soportados por Gmagick

Descripción

public array Gmagick::queryformats ([ string $pattern = "*" ] )

Devuelve los formatos soportados por Gmagick.

Parámetros

pattern

Especifica un puntero a una cadena de texto que contiene un patrón.

Valores devueltos

El objeto Gmagick si se tuvo éxito

Errores/Excepciones

Arroja un GmagickException en caso de error.
