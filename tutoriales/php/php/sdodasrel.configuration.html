Configuración en tiempo de ejecución

Esta extensión no tiene directivas de configuración definidas en php.ini.

Rastreo

Puede interesar ver las sentencias SQL generadas para poder aplicar cambios en la base de datos. Al principio del SDO/DAS/Relational.php se encontrarán varias constantes que controlan si el proceso de construcción y ejecución de sentencias SQL es rastreado. Intente establecer DEBUG_EXECUTE_PLAN a TRUE para ver las sentencias SQL generadas.
