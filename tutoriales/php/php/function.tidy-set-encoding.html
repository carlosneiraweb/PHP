tidy_set_encoding

(PECL tidy >= 0.5.2)

tidy_set_encoding — establece el juego de caracteres de entrada/salida del analizador de etiquetas

Descripción

bool tidy_set_encoding ( string $encoding )

Establece la codificación de entrada/salida del documento.

Parámetros

encoding

El parámetro de codificación encoding establece la codificación de la entrada/salida de los documentos. Los posibles valores de codificación son : ascii, latin0, latin1, raw, utf8, iso2022, mac, win1252, ibm858, utf16, utf16le, utf16be, big5, y shiftjis.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.

Notas

Nota: Esta función está disponible sólo en Tidy 1.0. Quedó obsoleta en Tidy 2.0, y por tanto ha sido eliminada.
