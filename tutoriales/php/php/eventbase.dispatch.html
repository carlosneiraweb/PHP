EventBase::dispatch

(PECL event >= 1.2.6-beta)

EventBase::dispatch — Despacha eventos pendientes

Descripción

public void EventBase::dispatch ( void )

Espara a que los eventos estén activos y ejecuta sus llamadas de retorno. Hace lo mismo que EventBase::loop() pero sin flags definidas.

Advertencia

Do NOT destroy the EventBase object as long as resources of the associated Event objects are not released. Otherwise, it will lead to unpredictable results!

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve TRUE en caso de éxito. FALSE de lo contrario.
