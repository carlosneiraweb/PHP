Instalación

Se necesita compilar PHP con el parámetro --with-mhash[=DIR] para habilitar esta extensión. DIR es el directorio de instalación de mhash.

A partir de PHP 5.3.0, la extensión mhash esta emulada a través de la extensión Hash. Esto hace que el directorio de instalación de mhash no tenga efecto, y requiera de la extensión hash habilitada para habilitar el soporte de mhash.
