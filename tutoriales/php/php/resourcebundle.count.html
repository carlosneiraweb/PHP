ResourceBundle::count

resourcebundle_count

(PHP >= 5.3.2, PECL intl >= 2.0.0)

ResourceBundle::count -- resourcebundle_count — Obtener el número de elementos de un paquete

Descripción

Estilo orientado a objetos

public int ResourceBundle::count ( void )

Estilo por procedimientos

int resourcebundle_count ( ResourceBundle $r )

Obtiene el número de elementos de un paquete.

Parámetros

r

Un objeto de la clase ResourceBundle.

Valores devueltos

Devuelve el número de elementos de un paquete.

Ejemplos

Ejemplo #1 Ejemplo de resourcebundle_count()

<?php
$r = resourcebundle_create( 'es', "/usr/share/data/myapp");
echo resourcebundle_count($r);
?>

Ejemplo #2 Ejemplo orientado a objetos

<?php
$r = new ResourceBundle( 'es', "/usr/share/data/myapp");
echo $r->count();
?>

El resultado del ejemplo sería:

42
