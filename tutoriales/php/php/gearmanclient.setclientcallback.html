GearmanClient::setClientCallback

(PECL gearman <= 0.5.0)

GearmanClient::setClientCallback — Función de llamada de retorno cuando hay un paquete de datos para una tarea (obsoleto)

Descripción

public void GearmanClient::setClientCallback ( callable $callback )

Establece la función de llamada de retorno que acepta paquetes de datos para una tarea. La función de llamada de retorno debe tener un solo argumento, un objeto GearmanTask.

Nota:

Este método ha sido reemplazado por GearmanClient::setDataCallback() en la versión 0.6.0 de la extensión Gearman..

Parámetros

callback

Una función o método a llamar

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.
