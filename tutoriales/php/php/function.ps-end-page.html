ps_end_page

(PECL ps >= 1.1.0)

ps_end_page — Finalizar una página

Descripción

bool ps_end_page ( resource $psdoc )

Finaliza un página que ha sido empezada con la función ps_begin_page(). Al finalizar una página se abandonará el contexto de dibujo actual, lo que, por ejemplo, requiere recargar la fuentes si fueron cargadas dentro de la página, y establecer muchos otros parámetros como el ancho de línea, color...

Parámetros

psdoc

El identificador de recursos del fichero postscript, como el devuelto por la función ps_new().

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.
