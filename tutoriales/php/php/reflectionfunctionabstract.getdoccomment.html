ReflectionFunctionAbstract::getDocComment

(PHP 5 >= 5.1.0, PHP 7)

ReflectionFunctionAbstract::getDocComment — Obtiene comentarios de documentación

Descripción

public string ReflectionFunctionAbstract::getDocComment ( void )

Devuelve los comentarios de documentación de una función.

Advertencia

Esta función no está documentada actualmente, solamente se encuentra disponible la lista de parámetros.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Si existieran, comentarios de documentación. En caso contrario, FALSE.
