Yaf_Response_Abstract::clearBody

(Yaf >=1.0.0)

Yaf_Response_Abstract::clearBody — El propósito de clearBody

Descripción

public bool Yaf_Response_Abstract::clearBody ([ string $key ] )

Limpia el contenido existente.

Parámetros

key

La clave del contenido, si no se especifica, todo el contenido será limpiado.

Nota:

Este parámetro se introdujo a partir de la versión 2.2.0
