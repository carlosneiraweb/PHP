ming_setswfcompression

(PHP 5.2.1-5.3.0, PHP 7, PECL ming SVN)

ming_setswfcompression — EStablece la comprensión de salida de SWF

Descripción

void ming_setswfcompression ( int $level )

Establece el nivel de comprensión de salida de SWF.

Parámetros

level

El nuevo nivel de comprensión. Debería ser un valor entre 1 y 9 inclusive.

Valores devueltos

No devuelve ningún valor.
