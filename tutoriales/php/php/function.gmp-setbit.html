gmp_setbit

(PHP 4 >= 4.0.4, PHP 5, PHP 7)

gmp_setbit — Establece el bit

Descripción

void gmp_setbit ( GMP &$a , int $index [, bool $bit_on = true ] )

Establece el index del bit en a.

Parámetros

a

El valor a modificar.

resource de número de GMP en PHP 5.5 y anterior, o un objeto de GMP en PHP 5.6 y posterior, o un string numérico siempre que sea posible convertirlo a un número.

index

El índice del bit a establecer. El índice 0 representa el bit menos significativo.

bit_on

Verdadero para establecer el bit (establecer a 1/on); falso para limpiar el bit (establecer a 0/off).

Valores devueltos

resource de número de GMP en PHP 5.5 y anterior, o un objeto GMP en PHP 5.6 y posterior.

Ejemplos

Ejemplo #1 Ejemplo de gmp_setbit() - índice 0

<?php
$a = gmp_init("2"); //
echo gmp_strval($a), ' -> 0b', gmp_strval($a, 2), "\n";
gmp_setbit($a, 0); // 0b10 ahora es 0b11
echo gmp_strval($a), ' -> 0b', gmp_strval($a, 2), "\n";
?>

El resultado del ejemplo sería:

2 -> 0b10
3 -> 0b11

Ejemplo #2 Ejemplo de gmp_setbit() - índice 1

<?php
$a = gmp_init("0xfd");
echo gmp_strval($a), ' -> 0b', gmp_strval($a, 2), "\n";
gmp_setbit($a, 1); // el índice empieza en 0
echo gmp_strval($a), ' -> 0b', gmp_strval($a, 2), "\n";
?>

El resultado del ejemplo sería:

253 -> 0b11111101
255 -> 0b11111111

Ejemplo #3 Ejemplo de gmp_setbit() - limpiar un bit

<?php
$a = gmp_init("0xff");
echo gmp_strval($a), ' -> 0b', gmp_strval($a, 2), "\n";
gmp_setbit($a, 0, false); // limpiar el bit en el índice 0
echo gmp_strval($a), ' -> 0b', gmp_strval($a, 2), "\n";
?>

El resultado del ejemplo sería:

255 -> 0b11111111
254 -> 0b11111110

Notas

Nota:

A diferencia de otras funciones GMP, gmp_setbit() debe ser llamado con un recurso GMP que ya exista (usando gmp_init() por ejemplo). No puede ser automáticamente creado.

Ver también

  • gmp_clrbit() - Limpia un bit
  • gmp_testbit() - Prueba si un bit es establecido
