DOMElement::hasAttributeNS

(PHP 5, PHP 7)

DOMElement::hasAttributeNS — Comprueba si un atributo existe

Descripción

public bool DOMElement::hasAttributeNS ( string $namespaceURI , string $localName )

Indica si un atributo en el espacio de nombres namespaceURI llamado localName existe como miembro del elemento.

Parámetros

namespaceURI

La URI del espacio de nombres.

localName

El nombre local.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.
