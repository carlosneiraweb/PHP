ReflectionMethod::isPublic

(PHP 5, PHP 7)

ReflectionMethod::isPublic — Comprueba si el método es público

Descripción

public bool ReflectionMethod::isPublic ( void )

Comprueba si el método es público.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

TRUE si el método es público, o si no FALSE
