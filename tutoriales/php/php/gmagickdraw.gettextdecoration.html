GmagickDraw::gettextdecoration

(PECL gmagick >= Unknown)

GmagickDraw::gettextdecoration — Devuelve la decoración del texto

Descripción

public int GmagickDraw::gettextdecoration ( void )

Devuelve la decoración aplicada cuando se anota texto.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve una de las constantes DECORATION_ y 0 si no está establecida la decoración.
