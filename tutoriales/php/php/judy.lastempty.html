Judy::lastEmpty

(PECL judy >= 0.1.1)

Judy::lastEmpty — Busca el último índice ausente del array Judy

Descripción

public int Judy::lastEmpty ([ int $index = -1 ] )

Busca el último índice ausente (incluido) que sea igual o menor al índice pasado.

Parámetros

index

El índice puede ser un entero o un string correspondiente al índice donde empezar a buscar.

Valores devueltos

Devuelve el correspondiente índice del array.
