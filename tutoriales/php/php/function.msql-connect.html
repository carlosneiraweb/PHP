msql_connect

(PHP 4, PHP 5, PHP 7)

msql_connect — Abre una conexión mSQL

Descripción

resource msql_connect ([ string $hostname ] )

msql_connect() establece una conexión a un servidor mSQL.

Si se produjera una segunda llamada a msql_connect() con los mismos parámetros, no se establecerá un nuevo enlace, pero en su lugar, el identificador del enlace ya abierto será devuelto.

El enlace al servidor se cerrará en cuanto la ejecución del script termine, a menos que haya sido cerrado previamente por una llamada explícita a la función msql_close().

Parámetros

hostname

El hostname también puede incluir un número de un puerto, por ejemplo: hostname,port.

Si no se especifica, la conexión se establece por medio de un dominio socket de Unix, siendo más eficaz que una conexión TCP al servidor local.

Nota:

Mientras esta función acepte los dos puntos (:) como un separador de host/puerto, una coma (,) será el método preferido.

Valores devueltos

Devuelve un identificador positivo de enlace mSQL en caso de éxito, o FALSE en caso de error.

Ver también

  • msql_pconnect() - Abrir una conexión mSQL persistente
  • msql_close() - Cierra la conexión mSQL
