fdf_set_encoding

(PHP 4 >= 4.0.7, PHP 5, PHP 7)

fdf_set_encoding — Asignar codificación de caracteres de FDF

Descripción

bool fdf_set_encoding ( resource $fdf_document , string $encoding )

Establece la codificación de caracteres del documento FDF.

Parámetros

fdf_document

Manejador de documentos FDF, devuelto por fdf_create(), fdf_open() o por fdf_open_string().

encoding

Nombre de codicación. Los siguientes tipos están soportados: "Shift-JIS", "UHC", "GBK" y "BigFive".

Si se usa una cadena vacía, se restablecerá el esquema de codificación por omisión PDFDocEncoding/Unicode.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.
