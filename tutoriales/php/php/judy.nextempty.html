Judy::nextEmpty

(PECL judy >= 0.1.1)

Judy::nextEmpty — Busca el sigueinte índice ausente del array Judy

Descripción

public int Judy::nextEmpty ( int $index )

Busca el siguiente índice ausente (no incluido) que sea mayor que el índice pasado.

Parámetros

index

El índice puede ser un integer o un string correspondiente al índice donde comenzar la búsqueda.

Valores devueltos

Devuelve el correspondiente índice del array.
