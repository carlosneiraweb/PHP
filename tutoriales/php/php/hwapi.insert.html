hw_api::insert

(PHP 4, PHP 5 < 5.2.0, PECL hwapi SVN)

hw_api::insert — Inserta un objeto nuevo

Descripción

hw_api_object hw_api::insert ( array $parameter )

Inserta un objeto nuevo. El tipo del objeto puede ser "user", "group", "document" o "anchor". Dependiendo del tipo se deberán configurar otros atributos del objeto.

Parámetros

parameter

El parámetro es un array que contiene los elementos obligatorios 'object' y 'content' (si el objeto es un documento) y los parámetros opcionales 'parameters', 'mode' y 'attributeSelector'. 'object' debe contener todos los atributos del objeto. 'parameters' es un objeto a su vez, que contiene otros atributos como la destinación (atributo clave 'Parent'). 'content' es el contenido del documento. 'mode' puede ser una combinación de los siguientes señaladores:

HW_API_INSERT_NORMAL
El objeto ha sido insertado en el servidor.
HW_API_INSERT_FORCE_VERSION_CONTROL
HW_API_INSERT_AUTOMATIC_CHECKOUT
HW_API_INSERT_PLAIN
HW_API_INSERT_KEEP_TIME_MODIFIED
HW_API_INSERT_DELAY_INDEXING
