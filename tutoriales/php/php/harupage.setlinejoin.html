HaruPage::setLineJoin

(PECL haru >= 0.0.1)

HaruPage::setLineJoin — Establece el estilo de la línea de unión para la página

Descripción

bool HaruPage::setLineJoin ( int $join )

Define el estilo de la línea de unión para la página.

Parámetros

join

Debe ser uno de los siguientes valores:

  • HaruPage::MITER_JOIN
  • HaruPage::ROUND_JOIN
  • HaruPage::BEVEL_JOIN

Valores devueltos

Devuelve TRUE en éxito.

Errores/Excepciones

Lanza una HaruException cuando se produce un error.
