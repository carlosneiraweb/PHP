MongoGridFSCursor::key

(PECL mongo >=0.9.0)

MongoGridFSCursor::key — Devuelve el nombre de fichero del resultado actual

Descripción

public string MongoGridFSCursor::key ( void )

Parámetros

Esta función no tiene parámetros.

Valores devueltos

El _id del resultado actual como cadena.

Historial de cambios

Versión Descripción
1.3.0 El _id del documento es devuelto como un valor de tipo string, ya que la clave debería ser única. Antes de la versión 1.3.0, se devolvía filename.
