sybase_result

(PHP 4, PHP 5, PHP 7)

sybase_result — Obtiene un valor dentro de un resultado Sybase

Descripción

string sybase_result ( resource $result , int $row , mixed $field )

Devuelve el contenido de la celda que se encuentra en la fila y posición especificada dentro del resultado Sybase.

Cuando se trabaja con conjuntos de resultados muy grandes, se debe considerar la utilización de una de las funciones que devuelven una fila entera (especificada abajo). Como estas funciones devuelven el contenido de múltiples celdas en una única llamada a la función, son MUCHO más rápidas que sybase_result(). También, notar que especificar un índice numérico para el parámetro field es mucho más rápido que especificar un paránetro fieldname o tablename.fieldname.

Alternativas recomendadas con un rendimiento mejor: sybase_fetch_row(), sybase_fetch_array() y sybase_fetch_object().

Parámetros

result

row

field

El parámetro field puede ser el índice del campo, o el nombre del campo, o la tabla del campo punto el nombre del campo (tablename.fieldname). Si se ha definido un alias para el nombre de la columna ('select foo as bar from...'), se ha de utilizar el alias en vez del nombre de la columna.

Valores devueltos

sybase_result() devuelve el contenido de una celda de un conjunto de resultado Sybase.
