ibase_prepare

(PHP 5, PHP 7)

ibase_prepare — Preparar una consulta para su asociación con parámetros de sustitución y ejecución posterior

Descripción

resource ibase_prepare ( string $query )
resource ibase_prepare ( resource $link_identifier , string $query )
resource ibase_prepare ( resource $link_identifier , string $trans , string $query )

Preparar una consulta para su asociación con parámetros de sustitución y ejecución posterior (mediante ibase_execute()).

Parámetros

query

Una consulta InterBase.

Valores devueltos

Devuelve un gestor de consulta preparado, o FALSE en caso de fallo.
