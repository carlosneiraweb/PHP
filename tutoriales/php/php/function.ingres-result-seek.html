ingres_result_seek

(PECL ingres >= 2.1.0)

ingres_result_seek — Establece la posición de una fila antes de obtener datos

Descripción

bool ingres_result_seek ( resource $result , int $position )

Esta función se usa para posicionar el cursor asociado al recurso de resultados antes de realizar una obtención. Si ingres.array_index_start está establecido a 0, la primera fila es 0, y no 1. ingres_result_seek() se puede usar solamente con consultas que hacen uso de cursores desplazables. No se puede usar con ingres_unbuffered_query().

Nota: Related Configurations

Véanse también las directivas ingres.scrollable y ingres.array_index_start de la sección Configuración en tiempo de ejcución.

Parámetros

result

El identificador de resultados de la consulta

position

La fila donde posicionar el cursor. Si ingres.array_index_start está establecido a 0, la primera fila es 0, y no 1

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.

Ejemplos

Ejemplo #1 Posicionar el cursor a la 3ª fila

<?php

$resultado=ingres_query($enlace, "select * from airport where ap_ccode = 'ES' order by ap_place asc");

/* ir a la fila 3 */
if (!ingres_result_seek($resultado, 3))
{
    echo ingres_errno() . " - " . ingres_error . "\n";
    die("he muerto");
}
else
{
    $aeropuerto = ingres_fetch_object ($resultado);
    {
        echo $aeropuerto->ap_iatacode . " - " .  $aeropuerto->ap_name . "\n";
    }
}

ingres_commit($enlace);

?>
