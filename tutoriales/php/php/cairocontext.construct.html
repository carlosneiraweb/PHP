CairoContext::__construct

(PECL cairo >= 0.1.0)

CairoContext::__construct — Crea un nuevo CairoContext

Descripción

public CairoContext::__construct ( CairoSurface $surface )

Crea un nuevo objeto CairoContext para dibujar

Parámetros

surface

Un objeto CairoSurface válido como CairoImageSurface o CairoPdfSurface

Valores devueltos

Un objeto CairoContext

Ejemplos

Ejemplo #1 Ejemplo de CairoContext::__construct()

<?php
// La superficie sobre la que se va a trabajar
$superficie = new CairoImageSurface(CairoFormat::ARGB32, 50, 50);

// Crear un nuevo CairoContext para la CairoSurface
$contexto = new CairoContext($superficie);

?>
