HaruPage::drawImage

(PECL haru >= 0.0.1)

HaruPage::drawImage — Mostrar una imagen en la página

Descripción

bool HaruPage::drawImage ( object $image , float $x , float $y , float $width , float $height )

Mostrar una imagen en la página.

Parámetros

image

Valida instacia de HaruImage.

x

El borde izquierdo del área donde la imagen es mostrada.

y

El borde mas bajo de el área donde la imagen es mostrada.

width

El ancho del área de la imagen.

height

El alto del área de la imagen.

Valores devueltos

Devuelve TRUE en éxito.

Errores/Excepciones

Lanza una HaruException cuando se produce un error.
