mb_ereg_search_getregs

(PHP 4 >= 4.2.0, PHP 5, PHP 7)

mb_ereg_search_getregs — Recupera el resultado de la última comparación de expresión regular multibyte

Descripción

array mb_ereg_search_getregs ( void )

Recupera el resultado de la última comparación de expresión regular multibyte.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Un array que incluye el substring de la parte coincidente mediante la última llamada a mb_ereg_search(), mb_ereg_search_pos(), mb_ereg_search_regs(). Si existieran algunas coincidencias, el primer elemento contendrá el substring coincidente, el segundo elemento contendrá el primer agrupamiento entre paréntesis, el tercer elemento contendrá el segundo agrupamiento entre paréntesis, etc. Devuelve FALSE en caso de error.

Notas

Nota:

La codificación interna o la codificación especificada por mb_regex_encoding() será usada en esta función.
