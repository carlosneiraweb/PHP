HaruDoc::setPermission

(PECL haru >= 0.0.1)

HaruDoc::setPermission — Establece los permisos para el documento

Descripción

bool HaruDoc::setPermission ( int $permission )

Define los permisos para el documento.

Parámetros

permission

Los valores son la combinación de estas banderas:

  • HaruDoc::ENABLE_READ - el usuario puede leer el documento.
  • HaruDoc::ENABLE_PRINT - el usuario puede imprimir el documento.
  • HaruDoc::ENABLE_EDIT_ALL - el usuario puede editar el contenido del documento más que anotaciones y campos de formulario.
  • HaruDoc::ENABLE_COPY - el usuario puede copiar el texto y los gráficos del documento.
  • HaruDoc::ENABLE_EDIT - el usuario puede agregar o modificar las anotaciones y los campos de formulario del documento.

Valores devueltos

Devuelve TRUE en éxito.

Errores/Excepciones

Lanza una HaruException cuando se produce un error.

  • HaruDoc::setEncryptionMode() - Establece el modo de cifrado para el documento
