fdf_set_file

(PHP 4, PHP 5, PHP 7)

fdf_set_file — Establecer el documento PDF en donde desplegar datos FDF

Descripción

bool fdf_set_file ( resource $fdf_document , string $url [, string $target_frame ] )

Selecciona un documento PDF diferente para desplegar los resultados de formulario desde el formulario en donde se originaron.

Parámetros

fdf_document

El gestor de documento FDF, devuelto por fdf_create(), fdf_open() o fdf_open_string().

url

Debe ser una URL absoluta.

target_frame

Use este parámetro para especificar el marco en el cual desplegar el documento. También es posible definir el valor predeterminado para este parámetro usando fdf_set_target_frame().

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.

Ejemplos

Ejemplo #1 Paso de datos FDF a un segundo formulario

<?php
  /* establecer tipo de contenido para Adobe FDF */
  fdf_header();

  /* iniciar nuevo fdf */
  $fdf = fdf_create();

  /* definir campo "foo" al valor "bar" */
  fdf_set_value($fdf, "foo", "bar");

  /* decir al cliente que despliegue datos FDF usando "formulario_fdf.pdf" */
  fdf_set_file($fdf, "http://www.example.com/formulario_fdf.pdf");

  /* producir salida fdf */
  fdf_save($fdf);

  /* limpiar */
  fdf_close($fdf);
?>
