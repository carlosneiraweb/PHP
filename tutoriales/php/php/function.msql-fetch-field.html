msql_fetch_field

(PHP 4, PHP 5, PHP 7)

msql_fetch_field — Extraer información de campo

Descripción

object msql_fetch_field ( resource $result [, int $field_offset = 0 ] )

msql_fetch_field() puede ser usado para extraer información sobre los campos en un determinado resultado de consulta.

Parámetros

result

El resultado resource que está siendo evaluado. Este resultado proviene de una llamada a msql_query().

field_offset

El campo de desplazamiento. Si no se especifica, el siguiente campo que no haya sido recuperado por msql_fetch_field() será recuperado.

Valores devueltos

Devuelve un objeto que contiene la información de campo. Las propiedades del objeto son:

  • name - nombre de columna

  • table - nombre de la tabla a la cual pertenece la columna

  • not_null - 1 si la columna no puede ser NULL

  • unique - 1 si la columna es una clave única

  • type - el tipo de la columna
