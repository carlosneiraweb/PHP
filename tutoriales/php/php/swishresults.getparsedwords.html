SwishResults::getParsedWords

(PECL swish >= 0.1.0)

SwishResults::getParsedWords — Obtener un array de palabras irrelevantes

Descripción

array SwishResults::getParsedWords ( string $index_name )
Advertencia

Esta función ha sido declarada EXPERIMENTAL. Su funcionamiento, nombre y la documentación que le acompaña puede cambiar sin previo aviso en futuras versiones de PHP. Utilícela bajo su propia responsabilidad.

Parámetros

indexi_name

EL nombre del índice utilizado para inicializar el objeto Swish.

Valores devueltos

Un array de palabras irrelevantes eliminadas. La lista de palabras irrelevantes puede ser útil para destacar términos de búsqueda en los resultados.

Ejemplos

Ejemplo #1 Ejemplo básico de SwishResults::getParsedWords()

<?php

try {

    $swish = new Swish("index.swish-e");
    $results = $swish->query("'some characters' and numbers");

    var_dump($results->getParsedWords("index.swish-e"));
    var_dump($results->indexes[0]['parsed_words']); //mismo resultado de forma diferente

} catch (SwishException $e) {
    echo "Error: ", $e->getMessage(), "\n";
}

?>
