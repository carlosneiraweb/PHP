msql_field_name

(PHP 4, PHP 5, PHP 7)

msql_field_name — Obtiene el nombre del campo especificado en un resultado

Descripción

string msql_field_name ( resource $result , int $field_offset )

msql_field_name() da el nombre del índice del campo especificado.

Parámetros

result

El resultado resource que está siendo evaluado. Este resultado proviene de una llamada a msql_query().

field_offset

El número del campo a buscar. El valor de field_offset comienza en 1.

Valores devueltos

El nombre del campo o FALSE en caso de error.
