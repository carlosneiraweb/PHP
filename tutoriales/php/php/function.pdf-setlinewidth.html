PDF_setlinewidth

(PHP 4, PECL pdflib >= 1.0.0)

PDF_setlinewidth — Establecer el ancho de línea

Descripción

bool PDF_setlinewidth ( resource $p , float $width )

Establece el ancho actual de línea. Devuelve TRUE en caso de éxito o FALSE en caso de error.
