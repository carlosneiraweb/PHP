sybase_min_client_severity

(PHP 4, PHP 5, PHP 7)

sybase_min_client_severity — Fija la severidad mínima del cliente Sybase

Descripción

void sybase_min_client_severity ( int $severity )

sybase_min_client_severity() fija el nivel mínimo de severidad del cliente.

Parámetros

severity

Valores devueltos

No devuelve ningún valor.

Notas

Nota: Esta función está disponible sólo cuando se utiliza la biblioteca CT de Sybase, y no la biblioteca DB.
