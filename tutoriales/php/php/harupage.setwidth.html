HaruPage::setWidth

(PECL haru >= 0.0.1)

HaruPage::setWidth — Establece el ancho de la página

Descripción

bool HaruPage::setWidth ( float $width )

Establece el ancho de la página.

Parámetros

width

Define el ancho de la página.

Valores devueltos

Devuelve TRUE en éxito.

Errores/Excepciones

Lanza una HaruException cuando se produce un error.
