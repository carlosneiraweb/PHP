Cairo::availableFonts

cairo_available_fonts

(PECL cairo >= 0.1.0)

Cairo::availableFonts -- cairo_available_fonts — Recupera los tipos de fuente disponibles

Descripción

Estilo orientado a objetos:

public static array Cairo::availableFonts ( void )

Estilo por procedimientos:

array cairo_available_fonts ( void )

Devuelve un array con los entornos de fuentes disponibles

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Una matriz tipo lista con todos los entornos de fuentes disponibles.

Ejemplos

Ejemplo #1 Estilo orientado a objetos

<?php

/* Estilo orientado a objetos */
var_dump(Cairo::availableFonts());

?>

El resultado del ejemplo sería algo similar a:

array(2) {
  [0]=>
  string(5) "WIN32"
  [1]=>
  string(4) "USER"
}

Ejemplo #2 Estilo por procedimientos

<?php

/* Estilo procedural */
var_dump(cairo_available_fonts());

?>

El resultado del ejemplo sería algo similar a:

array(2) {
  [0]=>
  string(5) "WIN32"
  [1]=>
  string(4) "USER"
}
