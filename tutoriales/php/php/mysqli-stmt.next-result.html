mysqli_stmt::next_result

mysqli_stmt_next_result

(PHP 5 >= 5.3.0, PHP 7)

mysqli_stmt::next_result -- mysqli_stmt_next_result — Lee el siguiente resultado de una consulta múltiple

Descripción

Estilo orientado a objetos (method):

public bool mysqli_stmt::next_result ( void )

Estilo por procedimientos:

bool mysqli_stmt_next_result ( mysql_stmt $stmt )

Lee el siguiente resultado de una consulta múltiple.

Parámetros

stmt

Sólo estilo por procediminetos: Un identificador de declaraciones devuelto por mysqli_stmt_init().

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.

Errores/Excepciones

Emite un error de nivel E_STRICT si no existe un conjunto de resultados, y sugiere el uso de mysqli_stmt::more_results() en estos casos, antes de llamar a mysqli_stmt::next_result().

MySQL Native Driver Only

Disponible sólo con mysqlnd.
