Constantes predefinidas

Estas constantes están definidas por esta extensión y estarán disponibles sólo cuando la extensión haya sido compilada con PHP, o bien sea cargada dinámicamente en ejecución.

YPERR_ACCESS (integer)
Violación de acceso (ha sido añadido recientemente y está sólo disponible paa PECL SVN por ahora)
YPERR_BADARGS (integer)
Los argumentos de la función son incorrectos
YPERR_BADDB (integer)
Base de datos YP es incorrecta
YPERR_BUSY (integer)
Base de datos ocupada
YPERR_DOMAIN (integer)
No puede enlazar al servidor sobre ese dominio
YPERR_KEY (integer)
Clave no se encuentra en el mapa
YPERR_MAP (integer)
Mapa no se encuentra en ese dominio
YPERR_NODOM (integer)
Dominio local no establecido
YPERR_NOMORE (integer)
No hay más registros en la base de datos
YPERR_PMAP (integer)
No puede comunicarse con portmapper
YPERR_RESRC (integer)
Error en la asignación de recursos
YPERR_RPC (integer)
Fallo RPC - dominio no ha sido contactado
YPERR_YPBIND (integer)
No logra comunicarse con ypbind
YPERR_YPERR (integer)
Error interno en servidor yp o error de cliente
YPERR_YPSERV (integer)
No logra comunicarse con ypserv
YPERR_VERS (integer)
La versión YP no coincide
