px_delete

(PECL paradox >= 1.0.0)

px_delete — Eliminar los recursos de la base de datos paradox

Descripción

bool px_delete ( resource $pxdoc )

Elimina el recurso del archivo paradox y libera toda la memoria.

Parámetros

pxdoc

Identificador de recursos de la base de datos paradox devueltos por la función px_new().

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.
