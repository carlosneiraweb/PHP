LimitIterator::getInnerIterator

(PHP 5 >= 5.1.0, PHP 7)

LimitIterator::getInnerIterator — Obtiene el iterador interno

Descripción

public Iterator LimitIterator::getInnerIterator ( void )

Obtener el Iterator interno.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

El iterador interno pasado a LimitIterator::__construct().
