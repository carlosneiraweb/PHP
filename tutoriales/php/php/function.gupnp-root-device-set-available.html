gupnp_root_device_set_available

(PECL gupnp >= 0.1.0)

gupnp_root_device_set_available — Establece si un dispositivo raíz estará disponible o no

Descripción

bool gupnp_root_device_set_available ( resource $root_device , bool $available )

Controla si un dispositivo raíz estará diponible o no (anunciando su presencia).

Parámetros

root_device

Un identificador de dispositivo raíz, devuelto por gupnp_root_device_new().

available

Establece TRUE si root_device estará disponible.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.
