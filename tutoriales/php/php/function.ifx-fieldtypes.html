ifx_fieldtypes

(PHP 4, PHP <=5.2.0)

ifx_fieldtypes — Listar los tipos de dato de los campos SQL de Informix

Descripción

array ifx_fieldtypes ( resource $result_id )

Devuelve un arreglo asociativo con los nombres de las campos como llave y sus tipos de dato como valores para la consulta asociada con result_id.

Parámetros

result_id

result_id es el identificador válido devuelto por ifx_query() o ifx_prepare() (sólo para consultas "SELECT").

Valores devueltos

Devuelve un arreglo asociativo con los nombres de las campos como llave y sus tipos de dato como valores para la consulta asociada con result_id. Devuelve FALSE en caso de error.

Ejemplos

Ejemplo #1 Campos y sus tipos de dato SQL

<?php
$types = ifx_fieldtypes($resultid);
if (is_array($types)) {
    foreach ($types as $fname => $val) {
        echo "$fname:\t tipo = $val\n";
    }
}
?>
