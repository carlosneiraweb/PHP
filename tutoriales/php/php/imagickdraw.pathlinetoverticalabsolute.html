ImagickDraw::pathLineToVerticalAbsolute

(PECL imagick 2.0.0)

ImagickDraw::pathLineToVerticalAbsolute — Dibuja un línea vertical

Descripción

bool ImagickDraw::pathLineToVerticalAbsolute ( float $y )
Advertencia

Esta función no está documentada actualmente, solamente se encuentra disponible la lista de parámetros.

Dibuja un trazado de línea vertical desde el punto actual al punto objetivo usando coordenadas absolutas. El punto objetivo entonces se convierte en el nuevo punto actual.

Parámetros

y

coordenada y

Valores devueltos

No devuelve ningún valor.
