geoip_netspeedcell_by_name

(PECL geoip >= 1.1.0)

geoip_netspeedcell_by_name — Obtener la velocidad de la conexión a Internet

Descripción

string geoip_netspeedcell_by_name ( string $hostname )

La función geoip_netspeedcell_by_name() devolverá el tipo y la velocidad de conexión awill Internet correspondiente a un nombre de host o a una dirección IP.

Esta función solamente está disponible si se utiliza la versión 1.4.8 o superior de la Biblioteca de GeoIP.

Esta función sólo está disponible actualmente a usuarios que hayan comprado una edición comercial de GeoIP Domain Edition. Se emitirá una advertencia si no se puede localizar la propia base de datos.

El valor devuelto es un string, cuyos valores más comunes son:

  • Cable/DSL
  • Dialup
  • Cellular
  • Corporate

Parámetros

hostname

El nombre de host o la dirección IP.

Valores devueltos

Devuelve la velocidad de conexión en caso de éxito, o FALSE si la dirección no se puede encontrar en la base de datos.

Ejemplos

Ejemplo #1 Un ejemplo de geoip_netspeedcell_by_name()

Mostrar la velocidad de conexión del host example.com.

<?php
$netspeed = geoip_netspeedcell_by_name('www.example.com');

if ($netspeed) {
    echo 'El tipo de conexión es: '. $netspeed;
}
?>

El resultado del ejemplo sería:

El tipo de conexión es: Corporate
