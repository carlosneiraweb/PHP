geoip_asnum_by_name

(PECL geoip >= 1.1.0)

geoip_asnum_by_name — Obtiene los Números de sistemas autónomos (ASN)

Descripción

string geoip_asnum_by_name ( string $hostname )

La función geoip_asnum_by_name() retorna los Números de sistemas autónomos (ASN) asociados con una dirección IP.

Parámetros

hostname

El nombre del host o la dirección IP.

Valores devueltos

Retorna el ASN en caso de éxito, o FALSE si la dirección no se encuentra en la base de datos.

Ejemplos

Ejemplo #1 Un ejemplo de geoip_asnum_by_name()

Este ejemplo mostrará el ASN del host www.example.com.

<?php
$asn = geoip_asnum_by_name('www.example.com');

if ($asn) {
    echo 'El ASN es: ' $asn;
}
?>

El resultado del ejemplo sería:

El ASN es: AS15133 EdgeCast Networks, Inc
