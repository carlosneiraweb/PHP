IntlChar::isdigit

(PHP 7)

IntlChar::isdigit — Comprobar si un punto de código es un carácter de dígito

Descripción

public static bool IntlChar::isdigit ( mixed $codepoint )

Determina si el punto de código especificado es un carácter de dígito.

TRUE para caracteres con categoría general "Nd" (números de dígito decimal). Desde Unicode 4, esto es lo mismo que probar el Numeric_Type de Decimal.

Parámetros

codepoint

El valor de tipo integer del punto de código (p.ej. 0x2603 para U+2603 SNOWMAN), o el carácter codificado como un string UTF-8 (p.ej. "\u{2603}")

Valores devueltos

Devuelve TRUE si codepoint es un carácter de dígito, FALSE si no.

Ejemplos

Ejemplo #1 Probar diferentes puntos de código

<?php
var_dump(IntlChar::isdigit("A"));
var_dump(IntlChar::isdigit("1"));
var_dump(IntlChar::isdigit("\t"));
?>

El resultado del ejemplo sería:

bool(false)
bool(true)
bool(false)
