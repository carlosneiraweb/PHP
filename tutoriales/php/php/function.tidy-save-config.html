tidy_save_config

(PECL tidy >= 0.5.2)

tidy_save_config — Guarda la configuración actual en un archivo

Descripción

bool tidy_save_config ( string $filename )

Guarda la configuración actual en un archivo específicado. Sólo los valores distintos a los valores por mosión serán guardados.

Parámetros

filename

Ruta al archivo de configuración.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.

Notas

Nota: Esta función está disponible sólo en Tidy 1.0. Quedó obsoleta en Tidy 2.0, y por tanto ha sido eliminada.
