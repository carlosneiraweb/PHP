GearmanTask::taskNumerator

(PECL gearman >= 0.5.0)

GearmanTask::taskNumerator — Retorna el numerador del tanto por ciento completado de la tarea, expresado como una fracción

Descripción

public int GearmanTask::taskNumerator ( void )

Retorna el numerador del tanto por ciento completado de la tarea, expresado como una fracción.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Un número entre 0 y 100 o FALSE si no se puede determinar.
