Gmagick::flopimage

(PECL gmagick >= Unknown)

Gmagick::flopimage — Crea una imagen espejo horizontal

Descripción

public Gmagick Gmagick::flopimage ( void )

Crea una imagen espejo horizontal reflejando los píxeles alrededor del eje y central.

Valores devueltos

The objeto Gmagick volteado.

Errores/Excepciones

Arroja un GmagickException en caso de error.
