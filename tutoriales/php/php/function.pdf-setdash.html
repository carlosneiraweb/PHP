PDF_setdash

(PHP 4, PECL pdflib >= 1.0.0)

PDF_setdash — Establecer el patrón de discontinuidad simple

Descripción

bool PDF_setdash ( resource $pdfdoc , float $b , float $w )

Establece el patrón de discontinuidad actual a unidades de b negro y w blanco. Devuelve TRUE en caso de éxito o FALSE en caso de error.
