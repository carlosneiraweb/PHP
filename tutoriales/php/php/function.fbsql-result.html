fbsql_result

(PHP 4 >= 4.0.6, PHP 5, PHP 7)

fbsql_result — Obtiene los datos de resultado

Descripción

mixed fbsql_result ( resource $result [, int $row [, mixed $field ]] )

Devuelve el contenido de una celda de un set de result FrontBase .

Si se trabaja con sets de resultados grandes, se debería considerar usar una de las funciones que obtienen una fila completa (especificadas más abajo). Como estas funciones devuelve los contenidos de múltiples celdas en una sola llamada, son MUCHO más rápidas que fbsql_result().

Las llamadas a fbsql_result() no deberían mezclarse con llamadas a otras funciones que trabajan con el set de resultados.

Parámetros

result

Un identificador de resultado devuelto por fbsql_query() o fbsql_db_query().

row

field

Puede ser la posición o el nombre del campo, o el nombre de la tabla punto nombre del campo (nombredetabla.nombredelcampo).

Si se ha creado un alias para la columna ('select foo as bar from...'), use el alias en lugar del nombre de la columna.

Nota:

Especificar una posición numérica como argumento es mucho más rápido que especificando un argumento como nombre de campo o nombredetabla.nombredecampo.

Ver también

Alternativas de alto rendimiento:

  • fbsql_fetch_row() - Obtiene una fila de resultados como array enumerado
  • fbsql_fetch_array() - Obtiene una fila de resultados como array asociativo, array numérico, o ambos
  • fbsql_fetch_assoc() - Obtiene una fila de resultados como array asociativo
  • fbsql_fetch_object() - Obtiene una fila de resultados como objeto
