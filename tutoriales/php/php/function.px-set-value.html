px_set_value

(PECL paradox >= 1.1.0)

px_set_value — Establecer un valor

Descripción

bool px_set_value ( resource $pxdoc , string $name , float $value )

Establece varios valores.

Parámetros

pxdoc

Identificador de recursos de la base de datos paradox devuelto por la función px_new().

name

El parámetro name puede ser uno de los siguientes.

numprimkeys

El número de claves primarias. Las bases de datos paradox utilizan siempre los primeros campos numprimkeys para el índice principal.

value

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.
