ReflectionFunctionAbstract::getNamespaceName

(PHP 5 >= 5.3.0, PHP 7)

ReflectionFunctionAbstract::getNamespaceName — Obtiene el nombre del espacio de nombres

Descripción

public string ReflectionFunctionAbstract::getNamespaceName ( void )

Devuelve el nombre del espacio de nombres en el que está definida la clase.

Advertencia

Esta función no está documentada actualmente, solamente se encuentra disponible la lista de parámetros.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Nombre del espacio de nombres.
