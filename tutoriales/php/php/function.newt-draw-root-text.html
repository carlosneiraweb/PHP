newt_draw_root_text

(PECL newt >= 0.1)

newt_draw_root_text — Muestra el texto de cadena en la posición indicada

Descripción

void newt_draw_root_text ( int $left , int $top , string $text )

Muestra el texto de cadena en la posición indicada.

Parámetros

left

Número de columna

Nota:

Si es negativo, la posición se mide desde el lado opuesto de la pantalla.

top

Número de línea

Nota:

Si es negativo, la posición se mide desde el lado opuesto de la pantalla.

text

El texto a mostrar.

Valores devueltos

No devuelve ningún valor.

Ejemplos

Ejemplo #1 Un ejemplo de newt_draw_root_text()

Este código muestra el dibjujo de títulos en ambas esquinas de la pantalla.

<?php
 newt_init();
 newt_cls();

 newt_draw_root_text (2, 0, "Algún texto raíz");
 newt_refresh();
 sleep(1);

 newt_draw_root_text (-30, 0, "Texto raíz en la otra esquina");
 newt_refresh();
 sleep(1);

 newt_finished();
?>
