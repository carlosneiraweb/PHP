La clase CairoContent

(No hay información de versión disponible, podría estar únicamente en SVN)

Introducción

CairoContent se usa para describir el contenido que contendrá una superficie, si la información de color, la información alfa (translúcida contra opaca), o ambas.

Nota: Los valores grandes están diseñados aquí para mantener los valores de CairoContent bien diferenciados de loa valores de CairoContent y así la implementación puede detectar el error si los usuarios confunden los dos tipos.

Sinopsis de la Clase

CairoContent {
/* Constantes */
const integer COLOR = 4096 ;
const integer ALPHA = 8192 ;
const integer COLOR_ALPHA = 12288 ;
}

Constantes predefinidas

CairoContent::COLOR

La superficie sólo tendrá el contenido de color.

CairoContent::ALPHA

La superficie sólo tendrá el contenido alfa.

CairoContent::COLOR_ALPHA

La superficie tendrá el contenido alfa y de color.
