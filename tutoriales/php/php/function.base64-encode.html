base64_encode

(PHP 4, PHP 5, PHP 7)

base64_encode — Codifica datos con MIME base64

Descripción

string base64_encode ( string $data )

Codifica data en base64.

Este tipo de codificación está diseñado para que datos binarios sobrepasen capas de transporte que no son de 8-bits 100%, como por ejemplo el cuerpo de un E-Mail.

La codificación en Base64 hace que los datos sean un 33% más largos que los datos el originales.

Parámetros

data

Datos a codificar.

Valores devueltos

Los datos codificados, como un string o FALSE en caso de error.

Ejemplos

Ejemplo #1 Ejemplo de base64_encode()

<?php
$str = 'This is an encoded string';
echo base64_encode($str);
?>

El resultado del ejemplo sería:

VGhpcyBpcyBhbiBlbmNvZGVkIHN0cmluZw==
