QuickHashStringIntHash::exists — Este método comprueba si una clave es parte de un hash

Descripción

public bool QuickHashStringIntHash::exists ( string $key )

Este método comprueba si una entrada con la clave proporcionada existe en el hash.

Parámetros

key

La clave de la entrada para comprobar si existe en el hash.

Valores devueltos

Devuelve TRUE si la entrada se encuentra en el hash, o FALSE si no.
