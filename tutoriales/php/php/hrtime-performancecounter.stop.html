HRTime\PerformanceCounter::stop

(PECL hrtime >= 0.4.3)

HRTime\PerformanceCounter::stop — Detener la medición del tiempo

Descripción

public void HRTime\PerformanceCounter::stop ( void )

Detiene la medición del tiempo para el intervalo anteriormente iniciado.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

No devuelve nada.
