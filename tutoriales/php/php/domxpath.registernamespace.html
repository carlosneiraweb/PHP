DOMXPath::registerNamespace

(PHP 5, PHP 7)

DOMXPath::registerNamespace — Registra el espacio de nombres con el objeto DOMXPath

Descripción

public bool DOMXPath::registerNamespace ( string $prefix , string $namespaceURI )

Registra namespaceURI y prefix con el objeto DOMXPath.

Parámetros

prefix

El prefijo.

namespaceURI

La URI del espacio de nombres.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.
