SplPriorityQueue::insert

(PHP 5 >= 5.3.0, PHP 7)

SplPriorityQueue::insert — Inserta un elemento en la cola en la parte de arriba

Descripción

public void SplPriorityQueue::insert ( mixed $value , mixed $priority )

Inserta un value con la prioridad priority en la cola.

Parámetros

value

El valor a insertar.

priority

La prioridad asociada.

Valores devueltos

No devuelve ningún valor.
