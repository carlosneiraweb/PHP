udm_cat_path

(PHP 4 >= 4.0.6, PHP 5 <= 5.0.5, PECL mnogosearch >= 1.0.0)

udm_cat_path — Obtiene la ruta a la categoría actual

Descripción

array udm_cat_path ( resource $agent , string $category )

Devuelve un array que describe la ruta en las categorías árbol desde el árbol raíz hasta la actual, especificada por category.

Parámetros

agent

Un link al agente, recibido después de llamar a udm_alloc_agent().

category

Valores devueltos

El array devuelto consiste en parejas. Los elementos con índices pares contienen rutas de categorías, los impares contienen el nombre de la categoría correspondiente.

Por ejemplo, la llamada $array=udm_cat_path($agent, '02031D'); devuelve el siguiente array: 

 $array[0] will contain ''
 $array[1] will contain 'Root'
 $array[2] will contain '02'
 $array[3] will contain 'Sport'
 $array[4] will contain '0203'
 $array[5] will contain 'Auto'
 $array[4] will contain '02031D'
 $array[5] will contain 'Ferrari'

Ejemplos

Ejemplo #1 Especificar la ruta a la categoría actual en el siguiente formato: '> Root > Sport > Auto > Ferrari'

<?php
  $cat_path_arr = udm_cat_path($udm_agent, $cat);
  $cat_path = '';
  for ($i=0; $i<count($cat_path_arr); $i+=2) {
    $path = $cat_path_arr[$i];
    $name = $cat_path_arr[$i+1];
    $cat_path .= " > <a href=\"$_SERVER[PHP_SELF]?cat=$path\">$name</a> ";
  }
?>
