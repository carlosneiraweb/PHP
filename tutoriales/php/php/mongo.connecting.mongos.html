Fragmentación

Para conectarse a un clúster fragmentado se ha de especificar la dirección de una o más instancias de mongos en la cadena de conexión. Se pueden delimitar múltiples servidores mediante comas.

<?php
// Usar un servidor como lista de semillas
$m = new MongoClient("mongodb://mongos1.example.com:27017");

// Usar múltiples servidores como lista de semillas
$m = new MongoClient("mongodb://mongos1.example.com:27017,mongos2.example.com:27017"));
?>

Independientemente de si cada fragmento es un servidor mongod independiente o un conjunto de réplicas complemto, el proceso de conexión del controlador es el mismo. Toda comunicación con la base de datos será enviada a través de mongos.

Para más información sobre la fragmentación con MongoDB, véase la » documentación de fragmentación.
