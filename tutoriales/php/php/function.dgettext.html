dgettext

(PHP 4, PHP 5, PHP 7)

dgettext — Sobrescribe el dominio actual

Descripción

string dgettext ( string $domain , string $message )

La función dgettext() le permite sobrescribir el domain actual en una búsqueda en un mensaje único.

Parámetros

domain

El dominio

message

El mensaje

Valores devueltos

Un string si es correcto.
