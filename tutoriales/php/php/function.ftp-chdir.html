ftp_chdir

(PHP 4, PHP 5, PHP 7)

ftp_chdir — Cambia el directorio actual en un servidor FTP

Descripción

bool ftp_chdir ( resource $ftp_stream , string $directory )

Cambia el directorio actual al especificado.

Parámetros

ftp_stream

El identificador de enlace de la conexión FTP.

directory

El directorio de destino.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error. Si al cambiar de directorio falla, PHP también lanzará una advertencia.

Ejemplos

Ejemplo #1 Ejemplo de ftp_chdir()

<?php

// configurar conexión básica
$conn_id = ftp_connect($ftp_server); 

// iniciar sesión con nombre de usuario y contraseña
$login_result = ftp_login($conn_id, $ftp_user_name, $ftp_user_pass); 

// check connection
if ((!$conn_id) || (!$login_result)) {
    die("La conexión FTP ha fallado!");
}

echo "Directorio actual: " . ftp_pwd($conn_id) . "\n";

// intenta cambiar el directorio a somedir
if (ftp_chdir($conn_id, "somedir")) {
    echo "El directorio actual es: " . ftp_pwd($conn_id) . "\n";
} else { 
    echo "No se pudo cambiar al directorio\n";
}

// cerrar la conexión ftp
ftp_close($conn_id);
?>
