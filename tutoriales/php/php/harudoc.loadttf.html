HaruDoc::loadTTF

(PECL haru >= 0.0.1)

HaruDoc::loadTTF — Carga fichero de fuente TTF

Descripción

string HaruDoc::loadTTF ( string $fontfile [, bool $embed = false ] )

Carga el fichero TTF dado y (opcionalmente) incrusta sus datos dentro del documento.

Parámetros

fontfile

El fichero TTF para cargar.

embed

Cuando se establece a TRUE, los datos glifos de la fuente son encrustado dentro del fichero PDF, de lo contrario sólo la matriz de datos están incluidos.

Valores devueltos

Devuelve el nombre de la fuente cargada en caso de éxito.

Errores/Excepciones

Lanza una HaruException cuando se produce un error.
