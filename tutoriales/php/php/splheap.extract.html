SplHeap::extract

(PHP 5 >= 5.3.0, PHP 7)

SplHeap::extract — Extrae un nodo de la parte superior del montón

Descripción

public mixed SplHeap::extract ( void )

Parámetros

Esta función no tiene parámetros.

Valores devueltos

El valor del nodo a ser extraído.

Errores/Excepciones

Lanza una excepción RuntimeException cuando la estructura de datos está vacía.
