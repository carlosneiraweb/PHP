DirectoryIterator::getExtension

(PHP 5 >= 5.3.6, PHP 7)

DirectoryIterator::getExtension — Obtiene la extensión de un fichero

Descripción

public string DirectoryIterator::getExtension ( void )

Recupera la extensión de un fichero.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve un string que contiene la extensión del ficher, o un string vacío si el fichoer no tiene extensión.

Ejemplos

Ejemplo #1 Ejemplo de DirectoryIterator::getExtension()

<?php

$directory = new DirectoryIterator(__DIR__);
foreach ($directory as $fileinfo) {
    if ($fileinfo->isFile()) {
        echo $fileinfo->getExtension() . "\n";
    }
}

?>

El resultado del ejemplo sería algo similar a:

php
txt
jpg
gz

Notas

Nota:

Este método sólo está disponible a partir de PHP 5.3.6. Otra manera de obtener la extensión es usar la función pathinfo().

<?php
$extension = pathinfo($fileinfo->getFilename(), PATHINFO_EXTENSION);
?>
