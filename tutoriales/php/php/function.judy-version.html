judy_version

(PECL judy >= 0.1.1)

judy_version — Devolver o imprimir la versión actual Judy de PHP

Descripción

string judy_version ( void )

Devolver una cadena con la versión Judy de PHP. Si el valor de retorno no se utiliza, la cadena será impresa.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devolver una cadena con la versión Judy de PHP.
