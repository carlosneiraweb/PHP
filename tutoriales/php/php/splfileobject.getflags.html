SplFileObject::getFlags

(PHP 5 >= 5.1.0, PHP 7)

SplFileObject::getFlags — Obtener las flags de SplFileObject

Descripción

public int SplFileObject::getFlags ( void )

Obtiene las flags establecidas en una instancia de SplFileObject como un integer.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve un integer que representa las flags.

Ejemplos

Ejemplo #1 Ejemplo de SplFileObject::getFlags()

<?php
$file = new SplFileObject(__FILE__, "r");

if ($file->getFlags() & SplFileObject::SKIP_EMPTY) {
    echo "Saltando líneas vacías\n";
} else {
    echo "Sin saltar líneas vacías\n";
}

$file->setFlags(SplFileObject::SKIP_EMPTY);

if ($file->getFlags() & SplFileObject::SKIP_EMPTY) {
    echo "Saltando líneas vacías\n";
} else {
    echo "Sin saltar líneas vacías\n";
}
?>

El resultado del ejemplo sería algo similar a:

Sin saltar líneas vacías
Saltando líneas vacías
