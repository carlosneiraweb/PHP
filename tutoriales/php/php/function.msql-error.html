msql_error

(PHP 4, PHP 5, PHP 7)

msql_error — Devuelve un mensaje de error de la última llamada msql

Descripción

string msql_error ( void )

msql_error() retorna el último error devuelto por el servidor mSQL. Téngase en cuenta que sólo el último mensaje de error es accesible con msql_error().

Valores devueltos

Devuelve el último mensaje de error, o una cadena vacía si ningún error había sido devuelto anteriormente.
