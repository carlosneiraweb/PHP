HaruDoc::loadType1

(PECL haru >= 0.0.1)

HaruDoc::loadType1 — Carga fuente Type1

Descripción

string HaruDoc::loadType1 ( string $afmfile [, string $pfmfile ] )

Carga fuente Type1 de el fichero dado y la registra en el documento PDF.

Parámetros

afmfile

Ruta a un fichero AFM.

pfmfile

La ruta a un fichero PFA/PFB, opcional. Si no es establecida solo los datos del glifo de la fuente son embebidos dentro del documento PDF.

Valores devueltos

Devuelve el nombre de la fuente cargada en caso de éxito.

Errores/Excepciones

Lanza una HaruException cuando se produce un error.
