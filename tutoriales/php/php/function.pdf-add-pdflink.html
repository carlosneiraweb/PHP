PDF_add_pdflink

(PHP 4, PECL pdflib >= 1.0.0)

PDF_add_pdflink — Añadir una anotación de enlace de página a la página actual [obsoleta]

Descripción

bool PDF_add_pdflink ( resource $pdfdoc , float $bottom_left_x , float $bottom_left_y , float $up_right_x , float $up_right_y , string $filename , int $page , string $dest )

Añade una anotación de enlace de página a un PDF objetivo. Devuelve TRUE en caso de éxito o FALSE en caso de error.

Esta función está obsoleta desde la versión 6 de PDFlib, use PDF_create_action() con type=GoToR y PDF_create_annotation() con type=Link en su lugar.
