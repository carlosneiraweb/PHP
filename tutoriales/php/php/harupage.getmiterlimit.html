HaruPage::getMiterLimit

(PECL haru >= 0.0.1)

HaruPage::getMiterLimit — Obtiene el valor del límite del inglete

Descripción

float HaruPage::getMiterLimit ( void )

Obtiene el valor del límite del inglete.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve el valor del límite del inglete.

Errores/Excepciones

Lanza una HaruException cuando se produce un error.

Ver también

  • HaruPage::setMiterLimit() - Establece el valor actual del inglete limite de la página
