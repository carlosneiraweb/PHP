fdf_open_string

(PHP 4 >= 4.3.0, PHP 5, PHP 7)

fdf_open_string — Consulta un documento FDF a partir de un string

Descripción

resource fdf_open_string ( string $fdf_data )

Consulta los datos de un formulario a partir de un string.

Puede usar fdf_open_string() junto con $HTTP_FDF_DATA para procesar un formulario de entrada FDF que procede de un equipo cliente remoto.

Parámetros

fdf_data

Datos tal y como los devolvió un formulario PDF, o como fueron creados al usar fdf_create() y fdf_save_string().

Valores devueltos

Devuelve un manejador de documentos FDF, o FALSE en caso de error.

Ejemplos

Ejemplo #1 Accediendo a los datos de un formulario

<?php
$fdf = fdf_open_string($HTTP_FDF_DATA);
/* ... */
fdf_close($fdf);
?>
