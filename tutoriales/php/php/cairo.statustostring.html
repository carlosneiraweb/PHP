Cairo::statusToString

cairo_status_to_string

(PECL cairo >= 0.1.0)

Cairo::statusToString -- cairo_status_to_string — Recupera el estado acutal como cadena

Descripción

Estilo orientado a objetos (método):

public static string Cairo::statusToString ( int $status )

Estilo por procedimientos:

string cairo_status_to_string ( int $status )

Recupera el estado actual como una cadena legible

Parámetros

status

Un código de estado válido dado por cairo_status() o CairoContext::status()

Valores devueltos

Una cadena que contiene el estado actual de un objeto CairoContext

Ejemplos

Ejemplo #1 Estilo orientado a objetos

<?php
$superficie = new CairoImageSurface(CairoFormat::ARGB32, 50, 50);
$contexto = new CairoContext($superficie);

var_dump(Cairo::statusToString($contexto->status()));
?>

El resultado del ejemplo sería algo similar a:

string(7) "success"

Ejemplo #2 Estilo por procedimientos

<?php
$superficie = cairo_image_surface_create(CAIRO_FORMAT_ARGB32, 50, 50);
$contexto = cairo_create($superficie);

var_dump(cairo_status_to_string(cairo_status($contexto)));
?>

El resultado del ejemplo sería algo similar a:

string(7) "success"
