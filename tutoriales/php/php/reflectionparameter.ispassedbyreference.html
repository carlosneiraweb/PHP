ReflectionParameter::isPassedByReference

(PHP 5, PHP 7)

ReflectionParameter::isPassedByReference — Comprueba si es pasado por referencia

Descripción

public bool ReflectionParameter::isPassedByReference ( void )

Comprueba si el parámetro es pasado por referencia.

Advertencia

Esta función no está documentada actualmente, solamente se encuentra disponible la lista de parámetros.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

TRUE si el parámetro es pasado por referencia, en caso contrario FALSE
