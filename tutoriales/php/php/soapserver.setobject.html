SoapServer::setObject

(PHP 5 >= 5.2.0, PHP 7)

SoapServer::setObject — Define el objecto que será usado para controlar las peticiones SOAP

Descripción

public void SoapServer::setObject ( object $object )

Define un objecto específico como controlador para las peticiones SOAP, en lugar de solo una clase como en SoapServer::setClass().

Parámetros

object

El objecto a controlar las peticiones.

Valores devueltos

No devuelve ningún valor.
