HaruPage::getGrayFill

(PECL haru >= 0.0.1)

HaruPage::getGrayFill — Obtiene el color de relleno actual

Descripción

float HaruPage::getGrayFill ( void )

Obtiene el color de relleno actual.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve el color de relleno actual.

Errores/Excepciones

Lanza una HaruException cuando se produce un error.
