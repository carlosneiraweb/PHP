GearmanClient::timeout

(PECL gearman >= 0.6.0)

GearmanClient::timeout — Obtiene el tiempo máximo de inactividad de entrada/salida para el socket

Descripción

public int GearmanClient::timeout ( void )

Retorna el tiempo máximo de inactividad de entrada/salida en milisegundos para el socket.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Tiempo máximo en milisegundos a esperar para actividad de entrada/salida. Un valor negativo significa tiempo infinito.

  • GearmanClient::setTimeout() - Establece el tiempo máximo de inactividad para actividad de entrada/salida en el socket
