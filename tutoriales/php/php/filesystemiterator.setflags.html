FilesystemIterator::setFlags

(PHP 5 >= 5.3.0, PHP 7)

FilesystemIterator::setFlags — Establece las flags de manejo

Descripción

public void FilesystemIterator::setFlags ([ int $flags ] )

Establecer las flags de manejo.

Parámetros

flags

las flags de manejo a establecer. Véase las constantes FilesystemIterator.

Valores devueltos

No devuelve ningún valor.

Ejemplos

Ejemplo #1 Ejemplo de FilesystemIterator::key()

Este ejemplo demuestra las diferencias entre las flags FilesystemIterator::KEY_AS_PATHNAME y FilesystemIterator::KEY_AS_FILENAME.

<?php
$iterator = new FilesystemIterator(dirname(__FILE__), FilesystemIterator::KEY_AS_PATHNAME);
echo "Clave como nombre de rura:\n";
foreach ($iterator as $key => $fileinfo) {
    echo $key . "\n";
}

$iterator->setFlags(FilesystemIterator::KEY_AS_FILENAME);
echo "\nClave como nombre de fichero:\n";
foreach ($iterator as $key => $fileinfo) {
    echo $key . "\n";
}
?>

El resultado del ejemplo sería algo similar a:

Clave como nombre de rura:
/www/examples/apple.jpg
/www/examples/banana.jpg
/www/examples/example.php

Clave como nombre de fichero:
apple.jpg
banana.jpg
example.php
