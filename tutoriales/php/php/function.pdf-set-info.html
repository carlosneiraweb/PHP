PDF_set_info

(PHP 4 >= 4.0.1, PECL pdflib >= 1.0.0)

PDF_set_info — Rellenar el campo de información de un documento

Descripción

bool PDF_set_info ( resource $p , string $key , string $value )

Rellena el campo de información de un documento key con value. Devuelve TRUE en caso de éxito o FALSE en caso de error.
