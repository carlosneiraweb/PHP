trader_tan

(PECL trader >= 0.2.0)

trader_tan — Tangente trigonométrica de vectores

Descripción

array trader_tan ( array $real )

Calcula la tangente para cada valor de real y devuelve el array resultante.

Parámetros

real

Array de valores reales.

Valores devueltos

Devuelve un array con los datos calculados o false en caso de error.
