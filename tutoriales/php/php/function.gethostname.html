gethostname

(PHP >= 5.3.0)

gethostname — Obtiene el nombre de host

Descripción

string gethostname ( void )

gethostname() obtiene el nombre de host estándar para el equipo local.

Valores devueltos

Devuelve un string con el nombre de host en caso de éxito, FALSE en caso contrario.

Ejemplos

Ejemplo #1 Un ejemplo simple de gethostname()

<?php
echo gethostname(); //puede imprimir: sandie

// O, una opción que también funciona antes de PHP 5.3
echo php_uname('n'); //puede imprimir: sandie
?>

  • gethostbyname() - Obtener la dirección IPv4 que corresponde a un nombre de host de Internet dado
  • gethostbyaddr() - Obtener el nombre del host de Internet correspondiente a una dirección IP dada
