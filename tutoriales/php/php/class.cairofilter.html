La clase CairoFilter

(No hay información de versión disponible, podría estar únicamente en SVN)

Introducción

Un CairoFilter se usa para indicar qué filtración debería ser aplicada cuando se leen valores de píxeles desde patrones. Véase CairoPattern::setSource() o cairo_pattern_set_source() para indicar el filtro deseado que se va a usar con un patrón en particular.

Sinopsis de la Clase

CairoFilter {
/* Constantes */
const integer FAST = 0 ;
const integer GOOD = 1 ;
const integer BEST = 2 ;
const integer NEAREST = 3 ;
const integer BILINEAR = 4 ;
const integer GAUSSIAN = 5 ;
}

Constantes predefinidas

CairoFilter::FAST

Un filtro de alto rendimiento, con calidad similar a CairoFilter::NEAREST

CairoFilter::GOOD

Un filtro con un rendimiento razonable, con calidad similar a CairoFilter::BILINEAR

CairoFilter::BEST

La más alta calidad disponible, el rendimiento no puede ser adecuado para uso interactivo.

CairoFilter::NEAREST

Filtrado de adyacente más cercano

CairoFilter::BILINEAR

Interpolación linear en dos dimensiones

CairoFilter::GAUSSIAN

Este valor de filtro no está actualmente implementado, y no debería usarse en el código actual.
