fdf_get_attachment

(PHP 4 >= 4.3.0, PHP 5, PHP 7)

fdf_get_attachment — Extraer un fichero subido por FDF

Descripción

array fdf_get_attachment ( resource $fdf_document , string $fieldname , string $savepath )

Extrae un fichero subido por medio del campo "selección de fichero" fieldname, y lo almacena en savepath.

Parámetros

fdf_document

Manejador de documentos FDF, devuelto por fdf_create(), fdf_open() o por fdf_open_string().

fieldname

savepath

Puede ser o bien el nombre de un fichero, o bien el de un directorio existente en el que creará usando el nombre original del fichero. Si hubiera algún otro fichero con ese nombre, se scobrescribiría.

Nota:

Aparentemente el único método para consultar el nombre original del fichero consiste en almacenar el archivo usando un directorio como savepath para después consultar el nombre (basename) con el que se guardó.

Valores devueltos

El array devuelto contiene los siguientes campos:

  • path - ruta donde se ha almacenado
  • size - tamaño del fichero en bytes
  • type - Si se proporcionó en el FDF, mimetype

Ejemplos

Ejemplo #1 Guardando un fichero subido por FDF

<?php
  $fdf = fdf_open_string($HTTP_FDF_DATA);
  $data = fdf_get_attachment($fdf, "filename", "/tmpdir");
  echo "El fichero subido se ha guardado en $data[path]";
?>
