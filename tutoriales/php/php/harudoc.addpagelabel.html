HaruDoc::addPageLabel

(PECL haru >= 0.0.1)

HaruDoc::addPageLabel — Establece el estilo de numeración para el rango especificado de páginas

Descripción

bool HaruDoc::addPageLabel ( int $first_page , int $style , int $first_num [, string $prefix = "" ] )

Establece el estilo numérico para el rango especificado de páginas.

Parámetros

first_page

La primera página incluida dentro del rango de la etiqueta.

style

El estilo de numeración. Los siguientes valores son permitidos:

  • HaruPage::NUM_STYLE_DECIMAL - etiqueta de página es mostrada usando numeración decimal.
  • HaruPage::NUM_STYLE_UPPER_ROMAN - etiqueta de página es mostrada usando numeración Romana en mayúsculas.
  • HaruPage::NUM_STYLE_LOWER_ROMAN - etiqueta de página es mostrada usando numeración Romana en minúsculas.
  • HaruPage::NUM_STYLE_UPPER_LETTER - etiqueta de página es mostrada usando letras en mayúsculas (de la A a la Z).
  • HaruPage::NUM_STYLE_LOWER_LETTERS - etiqueta de página es mostrada usando letras en minúsculas (de la A a la Z).

first_num

El número de la primera página en éste rango.

prefix

El prefijo para la etiqueta de página.

Valores devueltos

Devuelve TRUE en éxito.

Errores/Excepciones

Lanza una HaruException cuando se produce un error.
