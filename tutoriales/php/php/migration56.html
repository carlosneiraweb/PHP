La mayoria de las mejoras de PHP 5.6.x no tienen impacto en el código existente. Hay varias incompatibilidades y nuevas características que deberían ser consideradas, ya que el código debería probarse antes de cambiar de versión de PHP en ambientes de producción.

Véanse también las guías de migración para versiones de PHP 5.0.x, 5.1.x, 5.2.x, 5.3.x, 5.4.x y 5.5.x.
