px_put_record

(PECL paradox >= 1.0.0)

px_put_record — Almacenar registro en la base de datos paradox

Descripción

bool px_put_record ( resource $pxdoc , array $record [, int $recpos = -1 ] )

Almacena un registro en una base de datos Paradox. El registro se añade siempre al final de la base de datos, independientemente de las ranuras libres. Utilice la función px_insert_record() para agregar un nuevo registro en la primera ranura libre en la base de datos.

Parámetros

pxdoc

Identificador de recursos de la base de datos paradox devuelto por la función px_new().

record

Matriz asociado o indexada que contiene los valores de campo como, por ejemplo devuelto por la función px_retrieve_record().

recpos

Este parámetro opcional puede ser usado para especificar un número de registro mayor que el número actual de registros en la base de datos. La función añadirá tantos registros vacíos según sea necesario. There is hardly Apenas hay necesidad de utilizar este parámetro.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.
