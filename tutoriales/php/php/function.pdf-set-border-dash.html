PDF_set_border_dash

(PHP 4 >= 4.0.1, PECL pdflib >= 1.0.0)

PDF_set_border_dash — Establecer el estilo de borde discontinuo de las anotaciones [obsoleta]

Descripción

bool PDF_set_border_dash ( resource $pdfdoc , float $black , float $white )

Establece el estilo de borde discontinuo para todos los tipos de anotaciones. Devuelve TRUE en caso de éxito o FALSE en caso de error.

Esta función está obsoleta desde la versión 6 de PDFlib, use la opción dasharray en PDF_create_annotation() en su lugar.
