Instalación

Se puede encontrar información para la instalación de esta extensión PECL en el capítulo del manual titulado Instalación de extensiones PECL. Se puede encontrar información adicional, tal como nuevos lanzamientos, descargas, ficheros fuente, información de mantenimiento, y un CHANGELOG, aquí: » http://pecl.php.net/package/chdb

Se necesita » cmph (versión 0.9) para instalar esta extensión. Si cmph está instalado en un directorio no estándar, use el conmutador --with-libcmph-dir=DIR , donde DIR es el prefijo de instalación de cmph.
