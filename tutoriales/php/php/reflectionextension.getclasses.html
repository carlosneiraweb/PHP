ReflectionExtension::getClasses

(PHP 5, PHP 7)

ReflectionExtension::getClasses — Obtiene clases

Descripción

public array ReflectionExtension::getClasses ( void )

Devuelve la lista de clases que pertenecen a una extensión.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Un array de objetos ReflectionClass, uno por cada clase que incluya la extensión. Si no se hubiera definido ninguna, se devolvería un array vacío.

Ejemplos

Ejemplo #1 Ejemplo de ReflectionExtension::getClasses()

<?php
$ext = new ReflectionExtension('XMLWriter');
var_dump($ext->getClasses());
?>

El resultado del ejemplo sería algo similar a:

array(1) {
  ["XMLWriter"]=>
  &object(ReflectionClass)#2 (1) {
    ["name"]=>
    string(9) "XMLWriter"
  }
}
