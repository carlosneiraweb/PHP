GearmanTask::recvData

(PECL gearman >= 0.5.0)

GearmanTask::recvData — Guarda un trabajo o los datos resultantes de una tarea en un buffer

Descripción

public array GearmanTask::recvData ( int $data_len )

Advertencia

Esta función ha sido declarada EXPERIMENTAL. Su funcionamiento, nombre y la documentación que le acompaña puede cambiar sin previo aviso en futuras versiones de PHP. Utilícela bajo su propia responsabilidad.

Parámetros

data_len

Longitud de datos a leer.

Valores devueltos

Un array cuyo primer elemento es la longitud de los datos leídos y el segundo son los datos. Retorna FALSE si falla la lectura.
