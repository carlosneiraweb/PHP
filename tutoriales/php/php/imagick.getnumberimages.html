Imagick::getNumberImages

(PECL imagick 2.0.0)

Imagick::getNumberImages — Devuelve el número de imágenes del objeto

Descripción

int Imagick::getNumberImages ( void )

Devuelve el número de imágenes asociadas con el objeto Imagick.

Valores devueltos

Devuelve el número de imágenes asociadas con el objeto Imagick.

Errores/Excepciones

Lanza ImagickException en caso de error.
