La clase CairoHintMetrics

(No hay información de versión disponible, podría estar únicamente en SVN)

Introducción

Especifica si retocar la métrica de la fuente; retocar la métrica de la fuente significa cuantizarla, por lo que son valores enteros en el espacio del dispositivo. Al realizar esto se mejora la consistencia de la letra y el espacio entre líneas, aunque también significa que el texto será diseñado de forma diferente en diferentes factores de zoom.

Sinopsis de la Clase

CairoHintMetrics {
/* Constantes */
const integer METRICS_DEFAULT = 0 ;
const integer METRICS_OFF = 1 ;
const integer METRICS_ON = 2 ;
}

Constantes predefinidas

CairoHintMetrics::METRICS_DEFAULT

Retocar las métricas de la manera predeterminda para el entorno de fuente y el dispositivo objetivo

CairoHintMetrics::METRICS_OFF

No realizar el retoque de las métricas de fuente

CairoHintMetrics::METRICS_ON

Retocar las métricas de la fuente
