db2_close

(PECL ibm_db2 >= 1.0.0)

db2_close — Cierra una conexión a base de datos

Descripción

bool db2_close ( resource $connection )

Esta función cierra una conexión cliente DB2 creada con db2_connect() y devuelve el recurso correspondiente al servidor de base de datos.

Si se tratara de cerrar una conexión de cliente DB2 persistente creada con db2_pconnect(), la petición de cierre se ignoraría y la conexión de cliente DB2 permanecería disponibel para la siguiente llmada.

Parámetros

connection

Especifica la conexión a cliente DB2 activa.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.

Ejemplos

Ejemplo #1 Cierre de una conexión

El siguiente ejemplo demuestra un intento de cierre exitoso de una conexión a una base de datos IBM DB2, Cloudscape, o Apache Derby.

<?php
$conn = db2_connect('SAMPLE', 'db2inst1', 'ibmdb2');
$rc = db2_close($conn);
if ($rc) {
    echo "Conexión cerrada con éxito.";
}
?>

El resultado del ejemplo sería:

Conexión cerrada con éxito.
