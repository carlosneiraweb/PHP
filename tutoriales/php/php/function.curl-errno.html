curl_errno

(PHP 4 >= 4.0.3, PHP 5, PHP 7)

curl_errno — Devuelve el último número de error

Descripción

int curl_errno ( resource $ch )

Devuelve el número de error de la última operación cURL.

Parámetros

ch

El recurso cURL devuelto por curl_init().

Valores devueltos

Devuelve el número de error ó 0 (cero) si no ocurrió ningún error.

Ejemplos

Ejemplo #1 curl_errno() ejemplo

<?php
// Crear un recurso curl en una dirección no existente
$ch = curl_init('http://404.php.net/');

// Ejecutar
curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);
curl_exec($ch);

// Comprobar si occurió algún error
if(curl_errno($ch))
{
    echo 'Curl error: ' . curl_error($ch);
}

// Cerrar recurso
curl_close($ch);
?>
