Constantes predefinidas

Estas constantes están definidas por esta extensión y estarán disponibles sólo cuando la extensión haya sido compilada con PHP, o bien sea cargada dinámicamente en ejecución.

LDAP_DEREF_NEVER (integer)
LDAP_DEREF_SEARCHING (integer)
LDAP_DEREF_FINDING (integer)
LDAP_DEREF_ALWAYS (integer)
LDAP_OPT_DEREF (integer)
Especifica reglas alternativas para el seguimiento de aliases en el servidor.
LDAP_OPT_SIZELIMIT (integer)

Especifica el número máximo de entradas que pueden ser devueltas en una operación de búsqueda.

Nota: El límite de tamaño actual de las operaciones también está delimitado por el número máximo de entradas de retorno configuradas en el servidor. El menor de estos dos valores es el límite de tamaño actual.

LDAP_OPT_TIMELIMIT (integer)
Especifica el número de segundos de espera para la búsqueda de resultados.

Nota: El límite de tiempo para las operaciones también está delimitado por el tiempo máximo configurado en el servidor. El menor de estos dos valores es el límite de tiempo actual.

LDAP_OPT_NETWORK_TIMEOUT (integer)
Opción para ldap_set_option() que permite ajustar el tiempo de espera de la red. (Disponible a partir de PHP 5.3.0)
LDAP_OPT_PROTOCOL_VERSION (integer)
Especifica el protocolo LDAP a ser utilizado (V2 ó V3).
LDAP_OPT_ERROR_NUMBER (integer)
LDAP_OPT_REFERRALS (integer)
Especifica si se deben seguir automáticamente las referencias devueltas por el servidor LDAP.
LDAP_OPT_RESTART (integer)
LDAP_OPT_HOST_NAME (integer)
LDAP_OPT_ERROR_STRING (integer)
LDAP_OPT_MATCHED_DN (integer)
LDAP_OPT_SERVER_CONTROLS (integer)
Especifica una lista predeterminada de controles de servidor que se envía con cada solicitud.
LDAP_OPT_CLIENT_CONTROLS (integer)
Especifica una lista predeterminada de controles de cliente a ser procesada con cada solicitud.
LDAP_OPT_DEBUG_LEVEL (integer)
Especifica un nivel de bit por bit para depurar trazas.
GSLC_SSL_NO_AUTH (integer)
GSLC_SSL_ONEWAY_AUTH (integer)
GSLC_SSL_TWOWAY_AUTH (integer)
