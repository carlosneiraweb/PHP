gmp_gcd

(PHP 4 >= 4.0.4, PHP 5, PHP 7)

gmp_gcd — Calcula el máximo común divisor

Descripción

GMP gmp_gcd ( GMP $a , GMP $b )

Calcula el máximo común divisor de a y b. El resultado es siempre positivo aun si cualquiera de, o ambos, operadores fueran negativos.

Parámetros

a

resource de número de GMP en PHP 5.5 y anterior, o un objeto de GMP en PHP 5.6 y posterior, o un string numérico siempre que sea posible convertirlo a un número.

b

resource de número de GMP en PHP 5.5 y anterior, o un objeto de GMP en PHP 5.6 y posterior, o un string numérico siempre que sea posible convertirlo a un número.

Valores devueltos

Un número positivo GMP que divide dentro a ambos a y b.

Ejemplos

Ejemplo #1 Ejemplo degmp_gcd()

<?php
$gcd = gmp_gcd("12", "21");
echo gmp_strval($gcd) . "\n";
?>

El resultado del ejemplo sería:

3
