MongoCursor::skip

(PECL mongo >=0.9.0)

MongoCursor::skip — Omite un número de resultados

Descripción

public MongoCursor MongoCursor::skip ( int $num )

Parámetros

num

El número de resultados a omitir.

Valores devueltos

Returns this cursor.

Errores/Excepciones

Lanza una MongoCursorException si este cursor ha empezado a iterar.

  • MongoCursor::limit() - Limita el número de resultados devueltos
