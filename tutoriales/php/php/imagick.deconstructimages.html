Imagick::deconstructImages

(PECL imagick 2.0.0)

Imagick::deconstructImages — Devuelve ciertas diferencias de píxeles entre imágenes

Descripción

Imagick Imagick::deconstructImages ( void )

Compara cada imagen con la siguiente en una secuencia y devuelve la región circundante máxima de cualquier diferencia de píxeles que se descubra.

Valores devueltos

Devuelve un nuevo objeto Imagick si se tuvo éxito.

Errores/Excepciones

Lanza ImagickException en caso de error.
