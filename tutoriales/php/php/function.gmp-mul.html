gmp_mul

(PHP 4 >= 4.0.4, PHP 5, PHP 7)

gmp_mul — Multiplicación de números

Descripción

GMP gmp_mul ( GMP $a , GMP $b )

Multiplicación de a por b y devuelve el resultado.

Parámetros

a

Un número que va a ser multiplicado por b.

resource de número de GMP en PHP 5.5 y anterior, o un objeto de GMP en PHP 5.6 y posterior, o un string numérico siempre que sea posible convertirlo a un número.

b

Un número que va a ser multiplicado por a.

resource de número de GMP en PHP 5.5 y anterior, o un objeto de GMP en PHP 5.6 y posterior, o un string numérico siempre que sea posible convertirlo a un número.

Valores devueltos

resource de número de GMP en PHP 5.5 y anterior, o un objeto GMP en PHP 5.6 y posterior.

Ejemplos

Ejemplo #1 Ejemplo degmp_mul()

<?php
$mul = gmp_mul("12345678", "2000");
echo gmp_strval($mul) . "\n";
?>

El resultado del ejemplo sería:

24691356000
