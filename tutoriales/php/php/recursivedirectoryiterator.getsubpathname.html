RecursiveDirectoryIterator::getSubPathname

(PHP 5 >= 5.1.0, PHP 7)

RecursiveDirectoryIterator::getSubPathname — Obtiene la sub ruta y nombre de fichero

Descripción

public string RecursiveDirectoryIterator::getSubPathname ( void )

Obtiene la sub ruta y nombre de fichero.

Advertencia

Esta función no está documentada actualmente, solamente se encuentra disponible la lista de parámetros.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

La sub ruta (sub directorio) y nombre del fichero.
