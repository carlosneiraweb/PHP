sybase_select_db

(PHP 4, PHP 5, PHP 7)

sybase_select_db — Selecciona una base de datos Sybase

Descripción

bool sybase_select_db ( string $database_name [, resource $link_identifier ] )

sybase_select_db() fija la base de datos actualmente activa en el servidor que está asociada con el identificador de conexión link_identifier especificado.

Cada siguiente llamada a sybase_query() se realizará en la base de datos activa.

Parámetros

database_name

link_identifier

Si no se especifica un identificador de conexión, se asumirá la última conexión abierta. Si no hay ninguna conexión abierta, la función intentará establecer una conexión como si se hubiera llamado a sybase_connect(), y la utilizará.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.

Ver también

  • sybase_connect() - Establece una conexión a un servidor Sybase
  • sybase_pconnect() - Abre una conexión persistente a un servidor Sybase
  • sybase_query() - Envía una consulta a una base Sybase
