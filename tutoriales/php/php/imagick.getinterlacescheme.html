Imagick::getInterlaceScheme

(PECL imagick 2.0.0)

Imagick::getInterlaceScheme — Obtiene la combinación de entrelazado de un objeto

Descripción

int Imagick::getInterlaceScheme ( void )

Obtiene la combinación del entrelazado de un objeto.

Valores devueltos

Obtiene la varita mágica combinación de entrelazado.

Errores/Excepciones

Lanza ImagickException en caso de error.
