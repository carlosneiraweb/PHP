mssql_fetch_batch

(PHP 4 >= 4.0.4, PHP 5, PECL odbtp >= 1.1.1)

mssql_fetch_batch — Devuelve el siguiente lote de registros

Advertencia

Esta función fue ELILMINADA en PHP 7.0.0.

Descripción

int mssql_fetch_batch ( resource $result )

Devuelve el siguiente lote de registros.

Parámetros

result

Los resultados que se están evaluando. Este resultado proviene de una llamada a la función mssql_query().

Valores devueltos

Devuelve el número de filas del lote devuelto.

Ejemplos

Ejemplo #1 Ejemplo con mssql_fetch_batch()

<?php
// Conectar a MSSQL y seleccionar la base de datos
$link = mssql_connect('MANGO\SQLEXPRESS', 'sa', 'phpfi');
mssql_select_db('php', $link);

// Enviar una consulta
$result = mssql_query('SELECT * FROM [php].[dbo].[people]', $link, 100);
$records = 10;

while ($records >= 0) {
    while ($row = mssql_fetch_assoc($result)) {
        // Tratar las filas leídas...
    }

    --$records;
}

if ($batchsize = mssql_fetch_batch($result)) {
    // $batchsize es el número de registros que quedan 
    // del resultado, pero que todavía no se han mostrado
}
?>
