La clase CairoFontOptions

(PECL cairo >= 0.1.0)

Introducción

Una estructura opaca mantiene todas las opciones que se usan al renderizar fuentes.

Las característica indivuduales de un cairo_font_options_t se puede establecer o acceder usando funciones llamadas cairo_font_options_set_nombre_característica y cairo_font_options_get_nombre_característica, como cairo_font_options_set_antialias() y cairo_font_options_get_antialias().

Se pueden añadir nuevas características a CairoFontOptions en el futuro. Por esta razón CairoFontOptions::copy(), CairoFontOptions::equal(), CairoFontOptions::merge(), CairoFontOptions::hash() (cairo_font_options_copy(), cairo_font_options_equal(), cairo_font_options_merge(), y cairo_font_options_hash() de la manera procedural) deberían usarse para copiar, verificar igualdad, fusionar, o computar un valor hash value de objetos CairoFontOptions.
