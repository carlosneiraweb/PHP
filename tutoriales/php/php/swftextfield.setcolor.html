SWFTextField::setColor

(PHP 5 <= 5.3.0, PECL ming SVN)

SWFTextField::setColor — Establece el color del campo de texto

Descripción

void SWFTextField::setColor ( int $red , int $green , int $blue [, int $a = 255 ] )
Advertencia

Esta función ha sido declarada EXPERIMENTAL. Su funcionamiento, nombre y la documentación que le acompaña puede cambiar sin previo aviso en futuras versiones de PHP. Utilícela bajo su propia responsabilidad.

swftextfield::setcolor() establece el color del campo de texto. Por defecto es negro completamente opaco. El color es representado usando el sistema RGB.

Parámetros

Estos parámetros son enteros entre 0 y 255 o hexadecimales entre 0x00 y 0xFF:

red

Valor del componente rojo

green

Valor del componente verde

blue

Valor del componente azul

a

Valor del componente alfa

Valores devueltos

No devuelve ningún valor.
