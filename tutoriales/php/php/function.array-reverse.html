array_reverse

(PHP 4, PHP 5, PHP 7)

array_reverse — Devuelve un array con los elementos en orden inverso

Descripción

array array_reverse ( array $array [, bool $preserve_keys = false ] )

Toma un valor array y devuelve un nuevo array con el orden de los elementos invertido.

Parámetros

array

El array de entrada.

preserve_keys

Si se establece en TRUE las claves numéricas se conservarán. Las claves no numéricas no se ven afectadas por este ajuste y se conservarán siempre.

Valores devueltos

Devuelve el array en orden inverso.

Ejemplos

Ejemplo #1 Ejemplo de array_reverse()

<?php
$input  = array("php", 4.0, array("verde", "rojo"));
$reversed = array_reverse($input);
$preserved = array_reverse($input, true);

print_r($input);
print_r($reversed);
print_r($preserved);
?>
