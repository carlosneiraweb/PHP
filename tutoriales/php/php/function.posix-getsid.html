posix_getsid

(PHP 4, PHP 5, PHP 7)

posix_getsid — Obtener el sid actual del proceos

Descripción

int posix_getsid ( int $pid )

Devuelve el identificador de sesión del proceso pid. El identificador de sesión de un proceso es el id de grupo del proceso del líder de la sesión.

Parámetros

pid

Eñ identificador de proceso. Si se establece a 0, se asume el proceso actual. Si se especifica un pid no válido, se devuelve FALSE y se establece un error que puede ser verificado con posix_get_last_error().

Valores devueltos

Devuelve el identificador, como valor de tipo integer.

Ejemplos

Ejemplo #1 Ejemplo de uso de posix_getsid()

<?php
$pid = posix_getpid();
echo posix_getsid($pid); //8805
?>
