dbx_close

(PHP 4 >= 4.0.6, PHP 5 <= 5.0.5, PECL dbx >= 1.1.0)

dbx_close — Cierra una conexión/base de datos abierta

Descripción

int dbx_close ( object $link_identifier )

Parámetros

link_identifier

El objeto de enlace de DBX a cerrar.

Valores devueltos

Devuelve 1 si se tuvo éxito o 0 si ocurrió un error.

Ejemplos

Ejemplo #1 Ejemplo de dbx_close()

<?php
$enlace = dbx_connect(DBX_MYSQL, "localhost", "db", "username", "password")
    or die("No se pudo conectar");

echo "Conectado satisfactoriamente";
dbx_close($enlace);
?>
