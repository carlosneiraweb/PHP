imap_mail_copy

(PHP 4, PHP 5, PHP 7)

imap_mail_copy — Copiar los mensajes especificados a un buzón

Descripción

bool imap_mail_copy ( resource $imap_stream , string $msglist , string $mailbox [, int $options = 0 ] )

Copia los mensajes de correo especificados por msglist al buzón especificado.

Parámetros

imap_stream

IMAP stream devuelto por imap_open().

msglist

msglist es un rango, no números de mensajes (como está descrito en » RFC2060).

mailbox

El nombre del buzón, véase imap_open() para más información

options

options es una máscara de bits de un o más de

  • CP_UID - los números de secuencia conetienen UIDS
  • CP_MOVE - Borrar los mensajes del buzón actual después de copiar

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.
