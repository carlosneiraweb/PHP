fbsql_free_result

(PHP 4 >= 4.0.6, PHP 5, PHP 7)

fbsql_free_result — Libera la memoria de resultados

Descripción

bool fbsql_free_result ( resource $result )

Libera toda la memoria asociada al identificador de result recibido como parámetro.

Solo es necesario llamar fbsql_free_result() si se está preocupado sobre cuánta memoria se está usando en consultas que devuelven result-sets grandes. Toda la memoria de resultado a asociada se libera de forma automática al final de la ejecución del script.

Parámetros

result

Un identificador de resultado devuelto por fbsql_query() o fbsql_db_query().

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.
