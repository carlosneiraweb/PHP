is_int

(PHP 4, PHP 5, PHP 7)

is_int — Comprueba si el tipo de una variable es integer

Descripción

bool is_int ( mixed $var )

Comprueba si el tipo de variable dado es integer.

Nota:

Para comprobar si una variable es un número o un string numérico (por ejemplo, un formulario de entrada, que es siempre un string), debe utilizar is_numeric().

Parámetros

var

La variable a ser evaluada.

Valores devueltos

Devuelve TRUE si var es de tipo integer, o FALSE de lo contrario.

Ejemplos

Ejemplo #1 Ejemplo de is_int()

<?php
$values = array(23, "23", 23.5, "23.5", null, true, false);
foreach ($values as $value) {
    echo "is_int(";
    var_export($value);
    echo ") = ";
    var_dump(is_int($value));
}
?>

El resultado del ejemplo sería:

is_int(23) = bool(true)
is_int('23') = bool(false)
is_int(23.5) = bool(false)
is_int('23.5') = bool(false)
is_int(NULL) = bool(false)
is_int(true) = bool(false)
is_int(false) = bool(false)
