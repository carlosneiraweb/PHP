imagepsextendfont

(PHP 4, PHP 5)

imagepsextendfont — Extender o condensar una fuente

Advertencia

Esta función fue ELILMINADA en PHP 7.0.0.

Descripción

bool imagepsextendfont ( resource $font_index , float $extend )

Extiende o condensa una fuente (font_index), si el valor del parámetro extend es menor que 1 se condensará la fuente.

Parámetros

font_index

Un recurso de fuente, devuelto por imagepsloadfont().

extend

Valor de extensión, debe ser mayor que 0.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.

Ejemplos

Ejemplo #1 Ejemplo de imagepsextendfont()

<?php
// Cargar un archivo de fuente .pfb
$fuente = imagepsloadfont('./px3l.pfb');

// Extender la fuente por 2.5
imagepsextendfont($fuente, 2.5);

// Hacer cualesquiera operaciones con la fuente aquí

// Liberar la fuente de la memoria
imagepsfreefont($fuente);
?>

Historial de cambios

Versión Descripción
7.0.0 Se elimió el soporte para T1Lib de PHP, y así la eliminación de esta función.

Notas

Nota: Esta función sólo está disponible si PHP fue compilado usando --with-t1lib[=DIR] .
