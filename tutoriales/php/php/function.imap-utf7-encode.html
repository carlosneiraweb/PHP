imap_utf7_encode

(PHP 4, PHP 5, PHP 7)

imap_utf7_encode — Convierte una cadena ISO-8859-1 a un texto UTF-7 modifiicado

Descripción

string imap_utf7_encode ( string $data )

Convierte data a texto UTF-7 modificado.

Esto es necesario para codificar nombres de buzones que contienen ciertos caracteres que no están en el conjunto de caracters imprimibles de ASCII.

Parámetros

data

Una cadena ISO-8859-1.

Valores devueltos

Devuelve data codificado con la codificación UTF-7 modificada como está definido en » RFC 2060, sección 5.1.3 (el UTF-7 original fue definido en » RFC1642).
