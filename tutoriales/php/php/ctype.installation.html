Instalación

Desde PHP 4.2.0 estas funciones están habilitadas por omisión. Para versiones antiguas tiene que configurar y compilar PHP con --enable-ctype . Puede deshabilitar el soporte para ctype con --disable-ctype .

La versión de PHP para Windows tiene soporte nativo para esta extensión. No se requiere cargar extensiones adicionales para utilizar estas funciones.

Nota: El soporte interno para ctype está disponible con PHP 4.3.0.
