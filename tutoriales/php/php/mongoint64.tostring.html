MongoInt64::__toString

(PECL mongo >= 1.0.9)

MongoInt64::__toString — Devuelve la representación en forma de texto de este entero de 64 bits

Esta extensión que define esta clase está obsoleta. En su lugar debe usarse la extensión MongoDB. No existe un equivalente para esta clase en la nueva extensión.

En su lugar, la nueva extensión elige el tipo de base de datos apropiado según el valor del entero.

Descripción

public string MongoInt64::__toString ( void )

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve la representación en forma de texto de este entero.
