mcrypt_list_modes

(PHP 4 >= 4.0.2, PHP 5, PHP 7)

mcrypt_list_modes — Obtiene un array con todos los modos soportados

Descripción

array mcrypt_list_modes ([ string $lib_dir = ini_get("mcrypt.modes_dir") ] )

Obtiene una lista de todos modos soportados por el parámetro lib_dir.

Parámetros

lib_dir

Especifica el directorio donde todos los modos están ubicados. Si no se especifica, se utiliza el valor de la directiva mcrypt.modes_dir del fichero php.ini.

Valores devueltos

Devuelve un array con todos los modos soportados.

Ejemplos

Ejemplo #1 Ejemplo de mcrypt_list_modes()

<?php
    $modes = mcrypt_list_modes();

    foreach ($modes as $mode) {
        echo "$mode <br />\n";
    }
?>

El ejemplo anterior producirá una lista con todos los algoritmos soportados en el directorio de modos por defecto. Si no es seteado con la directiva mcrypt.modes_dir del fichero php.ini, se utiliza el directorio por defecto de mcrypt (que es /usr/local/lib/libmcrypt).
