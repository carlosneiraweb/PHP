msql_field_len

(PHP 4, PHP 5, PHP 7)

msql_field_len — Ontener longitud del campo

Descripción

int msql_field_len ( resource $result , int $field_offset )

msql_field_len() devuelve la longitud del campo especificado.

Parámetros

result

El resultado resource que está siendo evaluado. Este resultado proviene de una llamada a msql_query().

field_offset

El número del campo a buscar. El valor de field_offset comienza en 1.

Valores devueltos

Devuelve la longitud del campo especificado o FALSE en caso de error.
