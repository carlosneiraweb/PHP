HaruDoc::resetError

(PECL haru >= 0.0.1)

HaruDoc::resetError — Reinicia el estado de error de el documento manejado

Descripción

bool HaruDoc::resetError ( void )

Una vez que un código de error se establece, la mayoría de las operaciones, incluyendo funciones de procesamiento I/O no se pueden realizar. En el caso de que se desee continuar después del error causado ha sido solventado, se tendrá que invocar ésta función en orden para reiniciar el estado de error de el documento.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Siempre tiene éxito y devuelve TRUE.
