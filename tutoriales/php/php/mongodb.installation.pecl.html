Instalación del Controlador de MongoDB para PHP con PECL

Se puede encontrar información para la instalación de esta extensión PECL en el capítulo del manual titulado Instalación de extensiones PECL. Se puede encontrar información adicional, tal como nuevos lanzamientos, descargas, ficheros fuente, información de mantenimiento, y un CHANGELOG, aquí: » http://pecl.php.net/package/mongodb

Los usuarios de Linux, Unix, y OS X pueden ejecutar el siguiente comando para instalar el controlador: 

$ sudo pecl install mongodb

Añada la siguiente línea al fichero php.ini: 

extension=mongodb.so
