Memcached::decrementByKey

(PECL memcached >= 2.0.0)

Memcached::decrementByKey — Decrementar un valor numérico de un ítem almacenado en un servidor específico

Descripción

public int Memcached::decrementByKey ( string $server_key , string $key [, int $offset = 1 [, int $initial_value = 0 [, int $expiry = 0 ]]] )

Memcached::decrementByKey() decrementa un valor numérico de un ítem por el offset especificado. Si el valor del ítem no es numérico, generará un error. Si la operación requiere decrementar un valor inferior a 0, el nuevo valor será 0. Memcached::decrement() establecerá el ítem al parámetro initial_value si la clave no existe.

Parámetros

server_key

La clave que identifica al servidor donde guardar o recuperar el valor. En lugar de realizar un hash en la clave real para el ítem, se realiza en la clave del servidor cuando se decide con qué servidor de memcached conversar. Esto permite que los ítems relacionados sean agrupados en un único servidor para una mayor eficacia con operaciones múltiples.

key

La clave del ítem a decrementar.

offset

La cantidad por la que decrementar el valor del ítem.

initial_value

El valor para establecer el ítem si actualmente no existe.

expiry

El periodo de expiración a establecer para el ítem.

Valores devueltos

Devuelve el nuevo valor del ítem en caso de éxito o FALSE en caso de error.
