OAuthProvider::setParam

(PECL OAuth >= 1.0.0)

OAuthProvider::setParam — Establece un parámetro

Descripción

final public bool OAuthProvider::setParam ( string $param_key [, mixed $param_val ] )

Establece un parámetro.

Advertencia

Esta función no está documentada actualmente, solamente se encuentra disponible la lista de parámetros.

Parámetros

param_key

La clave del parámetro.

param_val

El valor opcional del parámetro.

Para excluir un parámetro de la verificación de firmas, colocar este valor a NULL.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.
