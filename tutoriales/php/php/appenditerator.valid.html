AppendIterator::valid

(PHP 5 >= 5.1.0, PHP 7)

AppendIterator::valid — Comprueba la validación del elemento actual

Descripción

public bool AppendIterator::valid ( void )

Comprueba la validación del elemento actual.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve TRUE si la iteración actual es válida, FALSE en caso contrario.
