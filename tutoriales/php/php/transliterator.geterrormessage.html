Transliterator::getErrorMessage

transliterator_get_error_message

(PHP >= 5.4.0, PECL intl >= 2.0.0)

Transliterator::getErrorMessage -- transliterator_get_error_message — Obtener el mensaje del último error

Descripción

Estilo orientado a objetos

public string Transliterator::getErrorMessage ( void )

Estilo por procedimientos

string transliterator_get_error_message ( void )

Obtiene el mensaje del último error para este transliterador.

Advertencia

Esta función no está documentada actualmente, solamente se encuentra disponible la lista de parámetros.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

El código del error en caso de éxito, o FALSE si no existe o en caso de error.
