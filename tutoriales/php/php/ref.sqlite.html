SQLite Funciones

Clases predefinidas

SQLiteDatabase

Representa una base de datos SQLite abierta.

Constructor

  • __construct - Construir un objeto nuevo SQLiteDatabase

Métodos

  • query - Ejecutar un query
  • queryExec - Ejecutar un resultado menos-consulta
  • arrayQuery - Ejecuta una consulta y devolver el resultado como una matriz
  • singleQuery - Ejecuta una consulta y devuelve o una matriz para una columna o el valor de la primera fila
  • unbufferedQuery - Ejecuta una consulta sin búfer
  • lastInsertRowid - Devuelve el identificador de la última fila insertada
  • changes - Devuelve el número de registros cambiados por la declaración más reciente
  • createAggregate - Registra una FDU agregación para su uso en sentencias SQL
  • createFunction - Registrar una UDF para su uso en sentencias SQL
  • busyTimeout - Establece o deshabilita la duración del temporizador de ocupado
  • lastError - Devuelve el último código de error del error más reciente encontrado
  • fetchColumnTypes - Devuelve una matriz de tipos de columnas de una tabla en particular

SQLiteResult

Representa un buffer conjunto SQLite resultado.

Métodos

  • fetch - Busca la siguiente fila del conjunto de resultados como un arreglo
  • fetchObject - Busca la siguiente fila del conjunto de resultados como un objeto
  • fetchSingle - Obtiene la primera columna del conjunto de resultados como una cadena
  • fetchAll - Obtiene todas las filas del conjunto de resultados como un arreglo de arreglos
  • column - Obtiene una columna de la fila actual del conjunto de resultados
  • numFields - Devuelve el número de campos en el conjunto de resultados
  • fieldName - Devuelve el nombre de un campo determinado en el conjunto de resultados
  • current - Obtiene la fila actual del conjunto de resultados como una arreglo
  • key - Devuelve el índice de la fila actual
  • next - Se desplaza hasta el siguiente número de fila
  • valid - Indica si existen más filas disponibles
  • rewind - Se desplaza hasta el primer número de fila del conjunto
  • prev - Se desplaza hasta el número de la fila anterior del conjunto de resultados
  • hasPrev - Devuelve si una fila anterior está disponible
  • numRows - Devuelve el número de filas en el conjunto de resultados
  • seek - Se desplaza hasta un determinado número de filas

SQLiteUnbuffered

Representa un conjunto de búfer SQLite resultado. Conjuntos de resultados sin búfer son secuenciales, con visión de buscar solamente.

Métodos

  • fetch - Busca la siguiente fila del conjunto de resultados como un arreglo
  • fetchObject - Busca la siguiente fila del conjunto de resultados como un objeto
  • fetchSingle - Obtiene la primera columna del conjunto de resultados como una cadena
  • fetchAll - Obtiene todas las filas del conjunto de resultados como una matriz de matrices
  • column - Obtiene una columna de la fila actual del conjunto de resultados
  • numFields - Devuelve el número de campos en el conjunto de resultados
  • fieldName - Devuelve el nombre de un campo determinado en el conjunto de resultados
  • current - Obtiene la fila actual del conjunto de resultados como un arreglo
  • next - Se desplaza hasta el siguiente número de fila
  • valid - Indica si existen más filas disponibles
