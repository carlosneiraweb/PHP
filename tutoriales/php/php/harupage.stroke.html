HaruPage::stroke

(PECL haru >= 0.0.1)

HaruPage::stroke — Imprime una ruta actual

Descripción

bool HaruPage::stroke ([ bool $close_path = false ] )

Imprime una ruta actual.

Parámetros

close_path

Cierra la ruta actual si se establece a TRUE.

Valores devueltos

Devuelve TRUE en éxito.

Errores/Excepciones

Lanza una HaruException cuando se produce un error.
