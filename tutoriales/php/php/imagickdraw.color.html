ImagickDraw::color

(PECL imagick 2.0.0)

ImagickDraw::color — Dibuja color en una imagen

Descripción

bool ImagickDraw::color ( float $x , float $y , int $paintMethod )
Advertencia

Esta función no está documentada actualmente, solamente se encuentra disponible la lista de parámetros.

Dibuja color en una imagen usando el color de relleno actual, comenzando por la posición especificada y usando el método de pintura especificado.

Parámetros

x

coordenada x de la pintura

y

coordenada y de la pintura

paintMethod

una de las constantes PAINT_

Valores devueltos

No devuelve ningún valor.
