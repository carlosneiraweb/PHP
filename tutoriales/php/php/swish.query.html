Swish::query

(PECL swish >= 0.1.0)

Swish::query — Realizar una consulta y devolver el objeto de resultados

Descripción

object Swish::query ( string $query )
Advertencia

Esta función ha sido declarada EXPERIMENTAL. Su funcionamiento, nombre y la documentación que le acompaña puede cambiar sin previo aviso en futuras versiones de PHP. Utilícela bajo su propia responsabilidad.

Un método rápido para realizar una búsqueda con parámetros por omisión.

Parámetros

query

string de la consulta.

Valores devueltos

Devuelve un objeto SwishResults.

Errores/Excepciones

Lanza una SwishException en caso de error.

Ejemplos

Ejemplo #1 Ejemplo básico de Swish::query()

<?php

try {

    $swish = new Swish("index.swish-e");
    $results = $swish->query("test query");

    echo "Encontradas: ", $results->hits, " visitas\n";

} catch (SwishException $e) {
    echo $e->getMessage(), "\n";
}

?>
