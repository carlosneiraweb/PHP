SAMConnection::errno

(PECL sam >= 0.1.0)

SAMConnection::errno — Contiene el código de error numérico de la última operación SAM ejecutada

Description

int $SAMConnection->errno;

Contiene el código de error numérico de la última operación SAM ejecutada en esa conexión. Si la última operación se completa con éxito esta propiedad contiene un 0.

Valores devueltos

Un entero mayor de cero indica el último error encontrado en la conexión. Cero indica que la última operación de esa conexión se completó con éxito.

Ejemplos

Ejemplo #1 Utilizando las propiedades número y descripción de error

<?php
$conn = new SAMConnection();
$conn->connect(SAM_WMQ, array(SAM_HOST => 'localhost', SAM_PORT => 1506));
$msg = new SAMMessage('Este es un elemento simple de texto');
if (!$conn->send('topic://test', $msg)) {
    // El Send falló!
    echo "Send fallido ($conn->errno) $conn->error";
}
?>
