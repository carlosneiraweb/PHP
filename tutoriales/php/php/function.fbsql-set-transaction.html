fbsql_set_transaction

(PHP 4 >= 4.2.0, PHP 5, PHP 7)

fbsql_set_transaction — Asigna el bloqueo y aislamiento de la transacción

Descripción

void fbsql_set_transaction ( resource $link_identifier , int $locking , int $isolation )

Asigna el locking y isolation de la transacción.

Parámetros

link_identifier

Un identificador de enlace FrontBase devuelto por fbsql_connect() o fbsql_pconnect().

Es opcional y si no se especifica, la función intentará encontrar un enlace abierto al servidor FrontBase y de no encontrarse ningún enlace abierto intentará crear uno como si fbsql_connect() fuese invocada sin parámetros.

locking

El tipo de bloqueo a establecer. Puede ser una de las siguientes constantes: FBSQL_LOCK_DEFERRED, FBSQL_LOCK_OPTIMISTIC, or FBSQL_LOCK_PESSIMISTIC.

isolation

El tipo de aislamiento a establecer. Puede ser una de las siguientes constantes: FBSQL_ISO_READ_UNCOMMITTED, FBSQL_ISO_READ_COMMITTED, FBSQL_ISO_REPEATABLE_READ, FBSQL_ISO_SERIALIZABLE, or FBSQL_ISO_VERSIONED.

Valores devueltos

No devuelve ningún valor.
