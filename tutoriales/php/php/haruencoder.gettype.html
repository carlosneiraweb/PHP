HaruEncoder::getType

(PECL haru >= 0.0.1)

HaruEncoder::getType — Obtiene el tipo del decodificador

Descripción

int HaruEncoder::getType ( void )

Obtiene el tipo del decodificador.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve el tipo del decodificador. El resultado es uno de los siguientes valores:

  • HaruEncoder::TYPE_SINGLE_BYTE - the encoder is for single byte characters.
  • HaruEncoder::TYPE_DOUBLE_BYTE - el codificador es para sencillos caracteres de byte.
  • HaruEncoder::TYPE_UNINITIALIZED - el codificador no está iniciado.
  • HaruEncoder::UNKNOWN - el codificador es invalido.

Errores/Excepciones

Lanza una HaruException cuando se produce un error.
