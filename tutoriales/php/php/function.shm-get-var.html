shm_get_var

(PHP 4, PHP 5, PHP 7)

shm_get_var — Devuelve una variable desde la memoria compartida

Descripción

mixed shm_get_var ( resource $shm_identifier , int $variable_key )

shm_get_var() devuelve la variable con la clave dada por variable_key, del segmento de memoria compartida dado. La variable todavía está presente en la memoria compartida.

Parámetros

shm_identifier

Segmento de memoria compartida, obtenido desde shm_attach().

variable_key

La clave de la variable.

Valores devueltos

Devuelve la variable con la clave dada.

Ver también

  • shm_has_var() - Verificar si existe una entrada específica
  • shm_put_var() - Inserta o actualiza una variable en la memoria compartida
