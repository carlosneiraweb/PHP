Requerimientos

Se necesita instalar libharu primero para que pueda ser posible usar PECL/haru. PECL/haru está probada con libharu 2.0.8, versiones más viejas tal vez o no trabajen con ello. PECL/hary también necesita PHP 5.1.3 o superior.
