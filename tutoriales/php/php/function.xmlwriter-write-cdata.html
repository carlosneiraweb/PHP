XMLWriter::writeCData

xmlwriter_write_cdata

(PHP 5 >= 5.1.2, PHP 7, PECL xmlwriter >= 0.1.0)

XMLWriter::writeCData -- xmlwriter_write_cdata — Escribe una etiqueta completa del CDATA

Descripción

Estilo orientado a objetos

bool XMLWriter::writeCData ( string $content )

Estilo por procedimientos

bool xmlwriter_write_cdata ( resource $xmlwriter , string $content )

Escribe un CDATA completo.

Parámetros

xmlwriter

Sólo para llamadas por procedimientos. El resource XMLWriter que está siendo modificado. Este recurso proviene de una llamada a xmlwriter_open_uri() o xmlwriter_open_memory().

content

El contenido del CDATA.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.

  • XMLWriter::endCData() - Fin del actual CDATA
