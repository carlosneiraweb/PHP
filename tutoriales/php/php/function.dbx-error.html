dbx_error

(PHP 4 >= 4.0.6, PHP 5 <= 5.0.5, PECL dbx >= 1.1.0)

dbx_error — Reporta un mensaje de error de la última llamada a la función en el módulo

Descripción

string dbx_error ( object $link_identifier )

dbx_error() devuelve el último mensaje de error.

Parámetros

link_identifier

El objeto de enlace de DBX devuelto por dbx_connect()

Valores devueltos

Devuelve una cadena que contiene el mensaje de error de la última llamada a la función del módulo abstracto (p.ej. el módulo mysql). Si existen múltiples conexiones en el mismo módulo sólo se da el último error. Si existen conexiones en diferentes módulos, se devuelve el último error del módulo especificado por el parámetro link_identifier.

Ejemplos

Ejemplo #1 Ejemplo de dbx_error()

<?php
$enlace = dbx_connect(DBX_MYSQL, "localhost", "db", "username", "password")
    or die("No se pudo conectar");

$resultado = dbx_query($enlace, "select id from non_existing_table");
if ($resultado == 0) {
    echo dbx_error($enlace);
}
dbx_close($enlace);
?>

Notas

Nota:

Consulte siempre la documentación específica del módulo también.

El mensaje de error para Microsoft SQL Server es en realidad el resultado de la función mssql_get_last_message().

El mensaje de error para Oracle (oci8) no está aún implementado.
