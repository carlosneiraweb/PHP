ingres_close

(PHP 4 >= 4.0.2, PHP 5 <= 5.0.5, PECL ingres >= 1.0.0)

ingres_close — Cierra una conexión a una base de datos de Ingres

Descripción

bool ingres_close ( resource $link )

ingres_close() cierra la conexión al servidor Ingres que está asociada con el enlace especificado.

ingres_close() normalmente no es necesaria, ya que no cerrará las conexiones persistentes, y todas las conexiones no persistentes serán automáticamente cerradas al finalizar el script.

Parámetros

link

El identificador de enlace de la conexión

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.
