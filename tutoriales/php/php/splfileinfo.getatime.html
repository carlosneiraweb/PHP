SplFileInfo::getATime

(PHP 5 >= 5.1.2, PHP 7)

SplFileInfo::getATime — Obtiene la hora del último acceso al fichero

Descripción

public int SplFileInfo::getATime ( void )

Obtiene la hora del último acceso al fichero.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve el tiempo del ultimo acceso al fichero.

Errores/Excepciones

Lanza RunTimeException en caso de error.
