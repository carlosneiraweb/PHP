mssql_fetch_field

(PHP 4, PHP 5, PECL odbtp >= 1.1.1)

mssql_fetch_field — Obtiene información del campo

Advertencia

Esta función está OBSOLETA en PHP 7.0.0.

Las alternativas a esta función son:

  • PDOStatement::getColumnMeta()
  • sqlsrv_field_metadata()
  • Las funciones odbc_field_* del controlador de ODBC unificado.

Descripción

object mssql_fetch_field ( resource $result [, int $field_offset = -1 ] )

mssql_fetch_field() puede utilizarse para obtener información sobre los campos de una determinada consulta realizada.

Parámetros

result

El resultado que está siendo evaluado. Este resultado proviene de una llamada a la función mssql_query().

field_offset

El número de posición del campo. Si no se especifica, se mostrará el próximo campo que todavía no se haya mostrado. El parámetro field_offset empieza en 0.

Valores devueltos

Devuelve un objeto que contiene la información de los campos.

Las propiedades del objeto son las siguientes:

  • name - nombre de columna. Si la columna es el resultado de una función, esta propiedad toma el valor computed#N, donde #N es un número de serie.
  • column_source - la tabla de la que proviene la columna.
  • max_length - longitud máxima de la columna.
  • numeric - 1 si la columna es numérica.
  • type - el tipo de la columna.

Ejemplos

Ejemplo #1 mssql_fetch_field() example

<?php
// Conectar a MSSQL y seleccionar la base de datos
mssql_connect('MANGO\SQLEXPRESS', 'sa', 'phpfi');
mssql_select_db('php');

// Enviar una selección a MSSQL
$query = mssql_query('SELECT * FROM [php].[dbo].[persons]');

// Construir la tabla
echo '<h3>Estructura de tabla para \'persons\'</h3>';
echo '<table border="1">';

// Cabecera de la tabla
echo '<thead>';
echo '<tr>';
echo '<td>Field name</td>';
echo '<td>Data type</td>';
echo '<td>Max length</td>';
echo '</tr>';
echo '</thead>';

// Descargar los campos
echo '<tbody>';

for ($i = 0; $i < mssql_num_fields($query); ++$i) {
    // Leer la información del campo
    $field = mssql_fetch_field($query, $i);

    // Mostrar la fila
    echo '<tr>';
    echo '<td>' . $field->name . '</td>';
    echo '<td>' . strtoupper($field->type) . '</td>';
    echo '<td>' . $field->max_length . '</td>';
    echo '</tr>';
}

echo '</tbody>';
echo '</table>';

// Liberar el recurso
mssql_free_result($query);
?>
