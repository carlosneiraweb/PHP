ibase_fetch_object

(PHP 5, PHP 7)

ibase_fetch_object — Consulta un objeto de una base de datos InterBase

Descripción

object ibase_fetch_object ( resource $result_id [, int $fetch_flag = 0 ] )

Captura una fila en forma de pseudo-objeto, a partir de un identificador de resultado.

El resto de llamadas a ibase_fetch_object() devolverán la siguiente fila del juego de resultados.

Parámetros

result_id

Identificador de resultado de InterBase obtenido por ibase_query() o por ibase_execute().

fetch_flag

fetch_flag es una combinación OR de las constantes IBASE_TEXT y IBASE_UNIXTIME. Si se pasara IBASE_TEXT provocaría que esta función devolviera el contenido de los BLOB en lugar de sus identificadores. Si se pasara IBASE_UNIXTIME provocaría que esta función devolviera las fechas/hora en formato Unix en lugar de en forma de cadena de texto.

Valores devueltos

Devuelve un objeto con información de la siguiente fila, o FALSE si no hubiera más filas.

Ejemplos

Ejemplo #1 Ejemplo de ibase_fetch_object()

<?php
$dbh = ibase_connect($host, $username, $password);
$stmt = 'SELECT * FROM tblname';
$sth = ibase_query($dbh, $stmt);
while ($row = ibase_fetch_object($sth)) {
    echo $row->email . "\n";
}
ibase_close($dbh);
?>
