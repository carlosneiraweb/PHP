cyrus_bind

(PHP 4 >= 4.1.0, PECL cyrus 1.0)

cyrus_bind — Enlaza llamadas de retorno a una conexión IMAP Cyrus

Descripción

bool cyrus_bind ( resource $connection , array $callbacks )

Enlaza llamadas de retorno a una conexión Cyrus IMAP.

Parámetros

connection

El controlador de la conexión.

callbacks

Un array de llamadas de retorno.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.
