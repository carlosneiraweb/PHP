newt_push_help_line

(PECL newt >= 0.1)

newt_push_help_line — Guarda la línea actual de ayuda en una pila, y muestra la nueva línea

Descripción

void newt_push_help_line ([ string $text ] )

Guarda la línea actual de ayuda en una pila, y muestra la nueva línea.

Parámetros

text

El nuevo mensaje de texto de ayuda.

Nota:

Si no se especifica, se elimina limpia la línea de ayuda.

Valores devueltos

No devuelve ningún valor.
