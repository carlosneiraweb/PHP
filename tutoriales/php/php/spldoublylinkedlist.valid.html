SplDoublyLinkedList::valid

(PHP 5 >= 5.3.0, PHP 7)

SplDoublyLinkedList::valid — Comprueba si la lista doblemente enlazada contiene más nodos

Descripción

public bool SplDoublyLinkedList::valid ( void )

Comprueba si la lista doblemente enlazada contiene más nodos.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve TRUE si la lista doblemente enlazada contiene más nodos, o FALSE en caso contrario.
