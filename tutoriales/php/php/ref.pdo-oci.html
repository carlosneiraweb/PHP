Funciones de Oracle (PDO_OCI)

Instalación

Utilice --with-pdo-oci[=DIR] para instalar la extensión PDO Oracle OCI, donde el opcional [=DIR] es el directorio base de la instalación de Oracle. [=DIR] por defecto es la variable de entorno $ORACLE_HOME.

Utilice --with-pdo-oci=instantclient,prefijo,versión para un cliente de la Instancia Oracle SDK, donde prefijo y versión se configuran. 

// Usando $ORACLE_HOME
$ ./configure --with-pdo-oci

// Usando OCI para Linux con la versión 10.2.0.3 con un prefijo /usr 
$ ./configure --with-pdo-oci=instantclient,/usr,10.2.0.3
