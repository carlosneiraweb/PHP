fbsql_select_db

(PHP 4 >= 4.0.6, PHP 5, PHP 7)

fbsql_select_db — Selecciona una base de datos FrontBase

Descripción

bool fbsql_select_db ([ string $database_name [, resource $link_identifier ]] )

Asigna la base de datos activa para el identificador de enlace especificado.

El cliente contacta con FBExec para obtener el puerto a usar con la conexión a la base de datos. Si en el nombre de la base de datos se especifica un número, este se usará como puerto y no se consultará FBExec para obtener el puerto. El servidor FrontBase server puede iniciarse como FRontBase -FBExec=No -port=<port number> <database name>.

Every subsequent call to fbsql_query() will be made on the active database.

Parámetros

database_name

El nombre de la base de datos a seleccionar.

Si la base de datos está protegida con contraseña, se tendrá que llamar a fbsql_database_password() antes de seleccionar esta base de datos.

link_identifier

Un identificador de enlace FrontBase devuelto por fbsql_connect() o fbsql_pconnect().

Es opcional y si no se especifica, la función intentará encontrar un enlace abierto al servidor FrontBase y de no encontrarse ningún enlace abierto intentará crear uno como si fbsql_connect() fuese invocada sin parámetros.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.

Ver también

  • fbsql_pconnect() - Abrir una conexión persistente a un servidor FrontBase
  • fbsql_database_password() - Asigna o recupera el password de una base de datos FrontBase
  • fbsql_query() - Realiza una consulta FrontBase
