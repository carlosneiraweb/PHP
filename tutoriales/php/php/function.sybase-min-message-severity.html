sybase_min_message_severity

(PHP 4, PHP 5, PHP 7)

sybase_min_message_severity — Fija la severidad mínima del cliente para los mensajes

Descripción

void sybase_min_message_severity ( int $severity )

sybase_min_message_severity() fija el nivel mínimo de severidad del cliente para los mensajes.

Parámetros

severity

Valores devueltos

No devuelve ningún valor.

Notas

Nota: Esta función está disponible sólo cuando se utiliza la biblioteca DB de Sybase, y no la biblioteca CT.
