streamWrapper::unlink

(PHP 5, PHP 7)

streamWrapper::unlink — Borrar un archivo

Descripción

public bool streamWrapper::unlink ( string $path )

Este método es llamado en respuesta a unlink().

Nota:

Para que el mensaje de error apropiado sea devuelto, este método no debería ser definido si la envoltura no soporta la eliminación de archivos.

Parámetros

path

LA URL del archivo que debería ser borrado.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.

Errores/Excepciones

Emite un E_WARNING si la llamada a este método falla (por ej., por no estar implementado).

Notas

Nota:

La propiedad streamWrapper::$context es actualizada si se pasa un contexto válido en la función invocante.
