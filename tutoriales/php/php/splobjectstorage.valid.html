SplObjectStorage::valid

(PHP 5 >= 5.1.0, PHP 7)

SplObjectStorage::valid — Comprobar si la entrada actual del iterador es válida

Descripción

public bool SplObjectStorage::valid ( void )

Devuelve si la entrada actual del iterador es válida.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve TRUE si la entrada actual del iterador es válida, en caso contrario FALSE.

Ejemplos

Ejemplo #1 Ejemplo de SplObjectStorage::valid()

<?php
$s = new SplObjectStorage();

$o1 = new StdClass;
$o2 = new StdClass;

$s->attach($o1, "d1");
$s->attach($o2, "d2");

$s->rewind();
while($s->valid()) {
    echo $s->key()."\n";
    $s->next();
}
?>
