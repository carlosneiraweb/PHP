MongoLog::getCallback

(PECL mongo >= 1.3.0)

MongoLog::getCallback — Obtiene la función de retrollamada establecida anteriormente

Descripción

public static callable MongoLog::getCallback ( void )

Recupera la función de retrollamada.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve la función de retrollamada, o FALSE si aún no se ha establecido.

Notas

Precaución

Esta función solamente está disponible con PHP 5.3.0 y posterior.
