HaruAnnotation::setOpened

(PECL haru >= 0.0.1)

HaruAnnotation::setOpened — Define el estado inicial de la anotación

Descripción

bool HaruAnnotation::setOpened ( bool $opened )

Define si la anotación se muestra inicialmente abierta. Ésta función puede ser usada con textos de anotación solamente.

Parámetros

opened

TRUE significa que la anotación se muestra inicialmente abierta, FALSE - significa que esta cerrada.

Valores devueltos

Devuelve TRUE en éxito.

Errores/Excepciones

Lanza una HaruException cuando se produce un error.
