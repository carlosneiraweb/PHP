IntlChar::ispunct

(PHP 7)

IntlChar::ispunct — Comprobar si un punto de código es un carácter de puntuación

Descripción

public static bool IntlChar::ispunct ( mixed $codepoint )

Determina si el punto de código especificado es un carácter de puntuación.

TRUE para caracteres con categoría general "P" (puntuación).

Parámetros

codepoint

El valor de tipo integer del punto de código (p.ej. 0x2603 para U+2603 SNOWMAN), o el carácter codificado como un string UTF-8 (p.ej. "\u{2603}")

Valores devueltos

Devuelve TRUE si codepoint es un carácter de puntuación, FALSE si no.

Ejemplos

Ejemplo #1 Probar diferentes puntos de código

<?php
var_dump(IntlChar::ispunct("."));
var_dump(IntlChar::ispunct(","));
var_dump(IntlChar::ispunct("\n"));
var_dump(IntlChar::ispunct("$"));

El resultado del ejemplo sería:

bool(true)
bool(true)
bool(false)
bool(false)
