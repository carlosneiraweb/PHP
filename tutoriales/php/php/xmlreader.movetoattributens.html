XMLReader::moveToAttributeNs

(PHP 5 >= 5.1.0, PHP 7)

XMLReader::moveToAttributeNs — Mover el cursor a un atributo dado

Descripción

public bool XMLReader::moveToAttributeNs ( string $localName , string $namespaceURI )

Posiciona el cursor en el atributo dado en el espacio de nombres especificado.

Parámetros

localName

El nombre local.

namespaceURI

El URI namespace.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.
