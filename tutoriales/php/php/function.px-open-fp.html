px_open_fp

(PECL paradox >= 1.0.0)

px_open_fp — Abrir base de datos paradox

Descripción

bool px_open_fp ( resource $pxdoc , resource $file )

Abre un archivo de base de datos paradox existente. El archivo actual tiene que ser abierto antes con la función fopen(). Esta función también puede utilizarse para abrir el índice primario del archivo y pasar ellos como una base de datos paradox. Esto es aprobado para aquellos que quieran investigar un índice primario. Esto no se puede utilizar para acelerar el acceso a un archivo de base de datos.

Parámetros

pxdoc

Identificador de recursos de la base de datos paradox devuelto por la función px_new().

file

El parámetro file es el valor devuelto desde la función fopen() con el archivo de base de datos actual como parámetro. Asegúrese de que la base de datos se puede escribir si va a actualizar o insertar registros.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.
