fbsql_database_password

(PHP 4 >= 4.0.6, PHP 5, PHP 7)

fbsql_database_password — Asigna o recupera el password de una base de datos FrontBase

Descripción

string fbsql_database_password ( resource $link_identifier [, string $database_password ] )

Asigna o recupera el database password de la base de datos usado por la conexión. Si una base de datos está protegida por un password, el usuario debe llamar esta función antes de llamar fbsql_select_db().

Si no hay enlace abierto, la función intentará establecer un enlace como si se hubiese llamado a fbsql_connect(), y lo usará.

Esta función no modifica el database password en la propia base de datos ni tampoco puede usarse para obtener el database password de una base de datos.

Parámetros

link_identifier

Un identificador de enlace FrontBase devuelto por fbsql_connect() o fbsql_pconnect().

Es opcional y si no se especifica, la función intentará encontrar un enlace abierto al servidor FrontBase y de no encontrarse ningún enlace abierto intentará crear uno como si fbsql_connect() fuese invocada sin parámetros.

database_password

El database password, en un string. Si se proporicona, la función asigna el database password para el identificador de enlace especificado.

Valores devueltos

Devuelve el database password asociado con el identificador de enlace.

Ejemplos

Ejemplo #1 Ejemplo de fbsql_database_password()

<?php
$link = fbsql_pconnect("localhost", "_SYSTEM", "secret")
    or die("No se puede conectar");
fbsql_database_password($link, "secret db password");
fbsql_select_db($database, $link);
?>
