fbsql_field_name

(PHP 4 >= 4.0.6, PHP 5, PHP 7)

fbsql_field_name — Obtiene el nombre de un campo especificado en un resultado

Descripción

string fbsql_field_name ( resource $result [, int $field_index ] )

Devuelve el nombre del campo en la posición especificada.

Parámetros

result

Un puntero a resultado, devuelto por fbsql_list_fields().

field_index

El offset del campo. El índice de campos comienza en 0.

Valores devueltos

Devuelve el nombre como una cadena, o FALSE si el campo no existe.

Ejemplos

Ejemplo #1 Ejemplo de fbsql_field_name()

<?php
// La tabla usuarios consiste en tres campos:
//   user_id
//   username
//   password.

$res = fbsql_db_query("users", "select * from users", $link);

echo fbsql_field_name($res, 0) . "\n";
echo fbsql_field_name($res, 2);
?>

El resultado del ejemplo sería:

user_id
password
