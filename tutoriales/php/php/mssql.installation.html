Instalación

La extensión MSSQL se activa añadiendo extension=php_mssql.dll en el fichero php.ini.

Para que funcionen estas funciones, se ha de compilar PHP con --with-mssql[=DIR] , donde DIR es el prefijo de instalación FreeTDS. Y FreeTDS debe compilarse utilizando --enable-msdblib .

Advertencia

Las funciones MS SQL son alias a funciones Sybase, si PHP está compilado con la extensión Sybase y sin extensión MS SQL.
