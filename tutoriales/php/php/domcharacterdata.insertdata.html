DOMCharacterData::insertData

(PHP 5, PHP 7)

DOMCharacterData::insertData — Inserta una cadena en la posición de 16-bit especificada

Descripción

void DOMCharacterData::insertData ( int $offset , string $data )

Inserta la cadena data en la posición offset.

Parámetros

offset

La posición en la cual se insertará.

data

La cadena a insertar.

Valores devueltos

No devuelve ningún valor.

Errores/Excepciones

DOM_INDEX_SIZE_ERR

Lanzado si offset es negativo o mayor que el número de unidades de 16-bit en la data.
