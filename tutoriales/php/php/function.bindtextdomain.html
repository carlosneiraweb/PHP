bindtextdomain

(PHP 4, PHP 5, PHP 7)

bindtextdomain — Establece la ruta del dominio

Descripción

string bindtextdomain ( string $domain , string $directory )

La función bindtextdomain() establece la ruta para el dominio.

Parámetros

domain

El dominio

directory

La ruta del directorio

Valores devueltos

La ruta completa para el domain actual establecido.

Ejemplos

Ejemplo #1 Ejemplo de bindtextdomain()

<?php

$domain = 'myapp';
echo bindtextdomain($domain, '/usr/share/myapp/locale');

?>

El resultado del ejemplo sería:

/usr/share/myapp/locale
