Requerimientos

PHP: esta extensión requiere PHP 5.4+, la versión 5.4.4 o posterior de PHP. Las extensiones de PHP requeridas son PCRE (habilitada de forma predeterminada), y la verisón 2.0.x de la extensión memcached.

El complemento de Memcache mysqlnd_memcache admite todas las aplicaciones de PHP y todas las extensión de MySQL para PHP disponibles (mysqli, mysql, PDO_MYSQL). La extensión de MySQL para PHP debe cofigurarse con soporte para mysqlnd.

Para acceder a tablas InnoDB, esta extensión de PHP requiere MySQL Server 5.6.6 o posterior con el Complemento InnoDB Memcache Daemon habilitado.

Para acceder a tablas MySQL Cluster NDB, esta extensión de PHP requiere MySQL Cluster 7.2 o posterior con los nodos de la API NDB Memcache habilitados.
