ReflectionFunctionAbstract::isGenerator

(PHP 5 >= 5.5.0, PHP 7)

ReflectionFunctionAbstract::isGenerator — Devuelve si la función es un generador

Descripción

public bool ReflectionFunctionAbstract::isGenerator ( void )

Advertencia

Esta función no está documentada actualmente, solamente se encuentra disponible la lista de parámetros.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve TRUE si la función es un generador, FALSE si no lo es, o NULL en caso de error.
