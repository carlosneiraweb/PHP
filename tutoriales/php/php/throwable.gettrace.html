Throwable::getTrace

(PHP 7)

Throwable::getTrace — Obtener la traza de la pila

Descripción

abstract public array Throwable::getTrace ( void )

Devuelve la traza de la pila como un array.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve la traza de la pila como un array con el mismo formato que en debug_backtrace().
