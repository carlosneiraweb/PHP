SDO_DAS_XML::create

(^)

SDO_DAS_XML::create — Para crar un objeto SDO_DAS_XML para un fichero de esquema dado

Descripción

SDO_DAS_XML SDO_DAS_XML::create ([ mixed $xsd_file [, string $key ]] )

Este es el único método estático de la clase SDO_DAS_XML. Se usa para instanciar objetos SDO_DAS_XML.

Parámetros

xsd_file

Ruta al fichero de esquema XSD. Este parámetro es opcional. Si se omite se creará un DAS que tiene únicamente los tipos base de SDO definidos. Lo ficheros de esquema se pueden cargar entonces con el método addTypes(). Puede ser una cadena o un array de valores.

key

Valores devueltos

Devuelve un objeto SDO_DAS_XML en caso de éxito, o de otro modo lanza una excepción tal como se describe abajo.

Errores/Excepciones

SDO_TypeNotFoundException

Lanzada si un tipo no está definido en el modelo subyacente.

SDO_DAS_XML_ParserException

Lanzada por cualqier problema al analizar el fichero XSD dado.

SDO_DAS_XML_FileException

Lanzada si el fichero especificado no se puede encontrar.
