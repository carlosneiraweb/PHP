fbsql_db_query

(PHP 4 >= 4.0.6, PHP 5, PHP 7)

fbsql_db_query — Ejecuta una consulta a FrontBase

Descripción

resource fbsql_db_query ( string $database , string $query [, resource $link_identifier ] )

Selecciona una base de datos y ejecuta una consulta sobre ella.

Parámetros

database

La base de datos a seleccionar.

query

La consulta SQL a ejecutar.

Nota:

La consulta deberá acabar siempre con un punto y coma (;).

link_identifier

Un identificador de enlace FrontBase devuelto por fbsql_connect() o fbsql_pconnect().

Es opcional y si no se especifica, la función intentará encontrar un enlace abierto al servidor FrontBase y de no encontrarse ningún enlace abierto intentará crear uno como si fbsql_connect() fuese invocada sin parámetros.

Valores devueltos

Devuelve el identificador de resultado positivo de la consulta, o FALSE si es error.

  • fbsql_connect() - Abre una conexión a un servidor FrontBase
