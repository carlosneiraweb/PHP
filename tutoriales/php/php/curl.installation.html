Instalación

Para usar el soporte para cURL de PHP, se debe compilar PHP con --with-curl[=DIR] , donde DIR es la ubicación del directorio que contiene los directorios lib e include. En el directorio include, debería haber una carpeta llamada curl que debería contener los ficheros easy.h y curl.h. Debería haber un fichero llamado libcurl.a ubicado en el directorio lib. Desde PHP 4.3.0, se puede configurar PHP para usar cURL para los flujos de URL con --with-curlwrappers . Esta característica ha sido movida a PECL a partir de PHP 5.5.0.

Nota: Nota para los usuarios de Win32
Para poder habilitar este módulo en un entorno Windows, los ficheros libeay32.dll y ssleay32.dll deben estar presentes en la PATH. No se require libcurl.dll del sitio de cURL.
