Configuración en tiempo de ejecución

El comportamiento de estas funciones se ve afectado por la configuración de php.ini.

Opciones de configuración de mysqlnd_memcache
Nombre Por defecto Cambiable Historial de cambios
mysqlnd_memcache.enable 1 PHP_INI_SYSTEM Disponible desde 1.0.0

He aquí una breve explicación de las directivas de configuración.

mysqlnd_memcache.enable integer

Habilita o deshabilita el complemento. Si se deshabilita, la extensión no se conectará a mysqlnd para representar a las llamadas internas a la API mysqlnd en C.

Nota:

Esta opción es empleada principalmente por desarrolladores para construir esta extensión de forma estática en PHP. A los demás usuarios se les recomienda construir esta extensión como un objeto compartido, y no cargarla en absoluto cuando no sea necesaria.
