HaruDoc::readFromStream

(PECL haru >= 0.0.1)

HaruDoc::readFromStream — Lee los datos de la corriente temporal

Descripción

string HaruDoc::readFromStream ( int $bytes )

Lee los datos de el flujo temporal.

Parámetros

bytes

El parámetro bytes se especifica como muchos otros bytes a leer, aunque el flujo contenga menos bytes que los requeridos.

Valores devueltos

Devuelve los datos de el flujo temporal.

Errores/Excepciones

Lanza una HaruException cuando se produce un error.

Ver también

  • HaruDoc::saveToStream() - Guarda el documento en una secuencia temporal
  • HaruDoc::resetStream() - Rebobina la secuencia temporal
  • HaruDoc::getStreamSize() - Obtiene el tamaño de la secuencia temporal
