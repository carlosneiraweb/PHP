La clase CairoHintStyle

(No hay información de versión disponible, podría estar únicamente en SVN)

Introducción

Especifica el tipo de "hinting" (retoque) a realizar sobre los contornos de las fuentes. "Hinting" es el proceso de ajustar los contornos a la cuadrícula de píxeles para mejorar la apariencia del resultado. Ya que retocar los contornos involucra distorsionarlos, también reduce la fidelidad de las formas de los contornos originales. No todos los estilos de retoque de contornos están soportados por todas los entornos de fuentes.

Sinopsis de la Clase

CairoHintStyle {
/* Constantes */
const integer STYLE_DEFAULT = 0 ;
const integer STYLE_NONE = 1 ;
const integer STYLE_SLIGHT = 2 ;
const integer STYLE_MEDIUM = 3 ;
const integer STYLE_FULL = 4 ;
}

Constantes predefinidas

CairoHintStyle::STYLE_DEFAULT

Usar el estilo de retoque predeterminado para los entornos de fuentes y el dispositivo objetivo

CairoHintStyle::STYLE_NONE

No retocar los contornos

CairoHintStyle::STYLE_SLIGHT

Retocar los contornos ligeramente para mejorar el contraste mientras se mantiene una buena fidelidad a las formas originales.

CairoHintStyle::STYLE_MEDIUM

Retocar los contornos con intensidad media dando un compromiso entre la fidelidad de las formas originales y el contraste

CairoHintStyle::STYLE_FULL

Retocar los contornos para maximizar el contraste
