OCI-Lob::writeTemporary

(PHP 5, PHP 7, PECL OCI8 >= 1.1.0)

OCI-Lob::writeTemporary — Escribe un objeto grande temporal

Descripción

bool OCI-Lob::writeTemporary ( string $data [, int $lob_type = OCI_TEMP_CLOB ] )

Crea un objeto grande temporal y escribe los datos dados por data en él.

Debería usarse OCI-Lob::close al finalizar de usar este objeto.

Parámetros

data

Los datos a escribir.

lob_type

Puede ser una de las siguientes constantes:

  • OCI_TEMP_BLOB se utiliza para crear BLOBs temporales
  • OCI_TEMP_CLOB se utiliza para crear CLOBs temporales

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.
