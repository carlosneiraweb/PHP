ZipArchive::setPassword

(PHP 5 >= 5.6.0, PHP 7, PECL zip >= 1.12.4)

ZipArchive::setPassword — Establece la contraseña para el archivo activo

Descripción

public bool ZipArchive::setPassword ( string $password )

Establece la contraseña para el archivo activo.

Parámetros

password

La contraseña a emplear para el archivo.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.

Notas

Nota:

Esta función solamente establece la contraseña para descomprimir el archivo; no convierte un ZipArchive no protegido con contraseña en un ZipArchive protegido con contraseña.
