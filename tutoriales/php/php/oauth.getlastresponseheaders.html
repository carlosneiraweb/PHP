OAuth::getLastResponseHeaders — Obtener encabezados de la última respuesta

Descripción

public string OAuth::getLastResponseHeaders ( void )

Obtiene los encabezados de la última respuesta.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Un string que contiene los encabezados de la última respuesta o FALSE en caso de error
