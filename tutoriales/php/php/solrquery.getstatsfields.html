SolrQuery::getStatsFields

(PECL solr >= 0.9.2)

SolrQuery::getStatsFields — Devuelve todas las estadísticas de los campos

Descripción

public array SolrQuery::getStatsFields ( void )

Devuelve todas las estadísticas de los campos

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve una matriz en caso de éxito y NULL si no se estableció.
