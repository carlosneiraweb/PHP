Constantes predefinidas

Estas constantes están definidas por esta extensión y estarán disponibles sólo cuando la extensión haya sido compilada con PHP, o bien sea cargada dinámicamente en ejecución.

SAM_AUTO (string)
Comportamiento automático
SAM_BOOLEAN (string)
Especificador de tipo utilizado en la configuración de propiedades en objetos SAM_Message.
SAM_BUS (string)
Atributo de conexión utilizado para definir el nombre del Enterprise Service Bus al que se va a conectar.
SAM_BYTE (string)
Especificador de tipo utilizado en la configuración de las propiedades en objetos SAM_Message.
SAM_BYTES (string)
Descriptor de tipo de cuerpo de mensaje.
SAM_CORRELID (string)
Atributo utilizado en las solicitudes de recepción, envio y eliminación, para identificar mensajes específicos.
SAM_DELIVERYMODE (string)
Propiedad de cabecera de mensaje.
SAM_DOUBLE (string)
Especificador de tipo utilizado en la configuración de las propiedades de objetos SAM_Message .
SAM_ENDPOINTS (string)
Atributo de conexión utilizado para definir los posibles "endpoints" a los que conectar.
SAM_FLOAT (string)
Especificador de tipo utilizado en la configuración de las propiedades de los objetos SAM_Message.
SAM_HOST (string)
Atributo de conexión utilizado para definir el nombre de host del servidor de mensajería requerido.
SAM_INT (string)
Especificador de tipo utilizado en la configuración de propiedades de los objetos SAM_Message.
SAM_LONG (string)
Especificador de tipo utilizado en la configuración de las propiedades de los objetos SAM_Message.
SAM_MANUAL (string)
Comportamiento manual (controlado por script).
SAM_MESSAGEID (string)
Atributo utilizado en las solicitudes de recepción y eliminación para identificar mensajes específicos.
SAM_MQTT (string)
Definición del protocolo de conexión para seleccionar el protocolo MQTT (MQ Telemetry Transport).
SAM_MQTT_CLEANSTART (bool)
Opción de conexión opcional para indicarle a un servidor MQTT que todas las conexiones previas para este cliente así como las suscripciones, deben ser eliminadas cuando el cliente se desconecte explícitamente o de forma inesperada.
SAM_NON_PERSISTENT (string)
Atributo de conexión utilizado para solicitar que los mensajes no sean persistentes en el servidor de mensajería.
SAM_PASSWORD (string)
Atributo de conexión utilizado para definir la contraseña que ha de usarse para la cuenta de usuario con la que se conecta a un servidor de mensajería que requiere autorización para las conexiones.
SAM_PERSISTENT (string)
Atributo de conexión utilizado para solicitar que los mensajes sean persistentes en el servidor de mensajería, para proteger contra la pérdida de mensajes en caso de fallo.
SAM_PORT (string)
Atributo de conexión utilizado para dar valor al número de puerto en el que comunicar con el servidor de mensajería.
SAM_PRIORITY (string)
Nombre de opción utilizado en las solicitudes de envío para especificar un valor de prioridad de la distribución.
SAM_REPLY_TO (string)
Propiedad del mensaje utilizada para especificar la identificación de la cola en la que el script espera respuesta o donde los mensajes de respuesta serán publicados.
SAM_RTT (string)
Definición del protocolo de conexión para seleccionar el protocolo IBM Realtime Transport para la comunicación con un servidor de mensajería integrado.
SAM_STRING (string)
Especificador de tipo utilizado en la configuración de las propiedades de los objetos SAM_Message.
SAM_TARGETCHAIN (string)
Atributo de conexión utilizado para definir el identificador de la cadena objetivo requerida.
SAM_TEXT (string)
Descriptor de tipo de cuerpo de mensaje.
SAM_TIMETOLIVE (string)
Opción de envío de mensaje utilizado para especificar (en milisegundos) cuando tiempo debe ser retenido un mensaje.
SAM_TRANSACTIONS (string)
Atributo de conexión utilizado para definir el comportamiento transaccional. Puede tomar el valor SAM_AUTO (por defecto) o SAM_MANUAL.
SAM_USERID (string)
Atributo de conexión utilizado para definir la cuenta que será usada para conectar al servidor de mensajería que requiere autorización para las conexiones.
SAM_WAIT (string)
Propiedad de recepción utilizada para especificar el timeout de espera que se utilizará cuando se reciba un mensaje en una cola o suscripción.
SAM_WMQ (string)
Definición del protocolo de conexión para seleccionar el protocolo IBM WebSphere MQSeries para la comunicación con el servicio de mensajería deseado.
SAM_WMQ_BINDINGS (string)
Definición del protocolo de conexión para seleccionar el protocolo IBM WebSphere MQSeries para la comunicación con un servidor de mensajería local.
SAM_WMQ_CLIENT (string)
Definición del protocolo de conexión para seleccionar el protocolo de comunicación IBM WebSphere MQSeries para la comunicación con un servidor remoto de mensajería.
SAM_WMQ_TARGET_CLIENT (string)
Opción utilizada en las solicitudes de envío para especificar el modo de cliente objetivo. Puede tomar los valores 'jms' (por defecto), o 'mq'. El valor por defecto es 'jms' que significa que una cabecera RFH2 será enviada con el mensaje mientras que el valor 'mq' significa que no se incluye cabecera RFH2.
SAM_WPM (string)
Definición del protocolo de conexión para seleccionar el protocolo de mensajería IBM WebSphere Platform, para la comunicación con un servidor de mensajería WebSphere Application Server.
