GearmanJob::sendFail

(PECL gearman >= 0.6.0)

GearmanJob::sendFail — Envía el estado de fallo

Descripción

public bool GearmanJob::sendFail ( void )

Envía el estado de fallo para este trabajo, indicando que el trabajo ha fallado de forma conocida (al contrario que un fallo debido una excepción).

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.

  • GearmanJob::setReturn() - Establece el valor de retorno
  • GearmanJob::sendStatus() - Envía el estado
  • GearmanJob::sendWarning() - Envía una advertencia
