GearmanTask::sendWorkload

(PECL gearman >= 0.6.0)

GearmanTask::sendWorkload — Envía datos a una tarea

Descripción

public int GearmanTask::sendWorkload ( string $data )

Advertencia

Esta función ha sido declarada EXPERIMENTAL. Su funcionamiento, nombre y la documentación que le acompaña puede cambiar sin previo aviso en futuras versiones de PHP. Utilícela bajo su propia responsabilidad.

Parámetros

data

Datos a enviar al trabajador.

Valores devueltos

Longitud de los datos enviados o FALSE si falla el envío.
