HaruPage::setDash

(PECL haru >= 0.0.1)

HaruPage::setDash — Establece el patrón de guía para la página

Descripción

bool HaruPage::setDash ( array $pattern , int $phase )

Define el patrón de guía para la página.

Parámetros

pattern

Un arreglo (8 elementos máximo) el cual contiene un patrón guía y los espacios usados para las líneas de la página.

phase

La fase en la cual el patrón empieza.

Valores devueltos

Devuelve TRUE en éxito.

Errores/Excepciones

Lanza una HaruException cuando se produce un error.
