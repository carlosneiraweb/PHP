ArrayObject::natcasesort

(PHP 5 >= 5.2.0, PHP 7)

ArrayObject::natcasesort — Ordena un array usando un algoritmo orden natural

Descripción

public void ArrayObject::natcasesort ( void )

Este método es la versión insensible de ArrayObject::natsort.

Este método implementa un algoritmo que ordena string alfanuméricos en la forma en que un humano debe mantener la asociación clave/valor.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

No devuelve ningún valor.

Ejemplos

Ejemplo #1 Ejemplo de ArrayObject::natcasesort()

<?php
$array = array('IMG0.png', 'img12.png', 'img10.png', 'img2.png', 'img1.png', 'IMG3.png');

$arr1 = new ArrayObject($array);
$arr2 = clone $arr1;

$arr1->asort();
echo "Orden estándar\n";
print_r($arr1);

$arr2->natcasesort();
echo "\nOrden natural (mayúsculas/minúsculas)\n";
print_r($arr2);
?>

El resultado del ejemplo sería:

Orden estándar
ArrayObject Object
(
    [0] => IMG0.png
    [5] => IMG3.png
    [4] => img1.png
    [2] => img10.png
    [1] => img12.png
    [3] => img2.png
)

Orden natural (mayúsculas/minúsculas)
ArrayObject Object
(
    [0] => IMG0.png
    [4] => img1.png
    [3] => img2.png
    [5] => IMG3.png
    [2] => img10.png
    [1] => img12.png
)
