geoip_country_code3_by_name

(PECL geoip >= 0.2.0)

geoip_country_code3_by_name — Obtiene la abreviación de tres letras del país

Descripción

string geoip_country_code3_by_name ( string $hostname )

La función geoip_country_code3_by_name() devolverá la abreviación de tres letras del país correspondientes al sitio web o dirección IP.

Parámetros

hostname

El sitio web o dirección IP cuya localización quiere comprobarse.

Valores devueltos

Devuelve la abreviación de tres letras del país en caso de éxito, o FALSE si la dirección no puede encontrarse en la base de datos.

Ejemplos

Ejemplo #1 Ejemplo de geoip_country_code3_by_name()

Mostrará donde está localizado el sitio web example.com.

<?php
$pais = geoip_country_code3_by_name('www.example.com');
if ($country) {
    echo 'Este sitio web está localizado en: ' . $pais;
}
?>

El resultado del ejemplo sería:

Este sitio web está localizado en: USA
