odbc_fetch_into

(PHP 4, PHP 5, PHP 7)

odbc_fetch_into — Traer una fila de resultados en una matriz

Descripción

int odbc_fetch_into ( resource $result_id , array &$result_array [, int $rownumber ] )

Trae una fila de resultados en un array.

Parámetros

result_id

El resource del resultado.

result_array

El array resultante que puede ser de cualquier tipo ya que será convertido a tipo array. La matriz contendrá los valores de columnas comenzando en el índice de matriz 0.

rownumber

El número de fila.

Valores devueltos

Devuelve el número de columnas del resultado; FALSE si se produjo un error.

Ejemplos

Ejemplo #1 Ejemplos de odbc_fetch_into()

<?php
$rc = odbc_fetch_into($res_id, $mi_array);
?>

o

<?php
$rc = odbc_fetch_into($res_id, $mi_array, 2);
?>
