get_declared_traits

(PHP 5 >= 5.4.0, PHP 7)

get_declared_traits — Devuelve un array de todos los traits declarados

Descripción

array get_declared_traits ( void )

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve un array con los nombres de todos los traits declarados en valores. Devuelve NULL en caso de error.

  • class_uses() - Devolver los "traits" usados por la clase dada
