(PHP 4, PHP 5, PHP 7)

quotemeta — Escapa meta caracteres

Descripción

string quotemeta ( string $str )

Devuelve una versión de str con una barra invertida (\) antes de cada caracter que sea uno de los siguientes:

. \ + * ? [ ^ ] ( $ )

Parámetros

str

El string de entrada.

Valores devueltos

Devuelve el string con los meta caracteres escapados o FALSE si un string vacío es dado como str.

Notas

Nota: Esta función es segura binariamente.
