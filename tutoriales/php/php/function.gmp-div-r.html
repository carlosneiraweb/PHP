gmp_div_r

(PHP 4 >= 4.0.4, PHP 5, PHP 7)

gmp_div_r — El resto de la división de los números

Descripción

GMP gmp_div_r ( GMP $n , GMP $d [, int $round = GMP_ROUND_ZERO ] )

Calcula el resto de la división entera de n por d. El tiene el signo del argumento n, si no es cero.

Parámetros

n

El número que es dividido.

resource de número de GMP en PHP 5.5 y anterior, o un objeto de GMP en PHP 5.6 y posterior, o un string numérico siempre que sea posible convertirlo a un número.

d

El número que es dividio por n.

resource de número de GMP en PHP 5.5 y anterior, o un objeto de GMP en PHP 5.6 y posterior, o un string numérico siempre que sea posible convertirlo a un número.

round

Ver la función gmp_div_q() para la descripción del argumento round.

Valores devueltos

The remainder, as a GMP number.

Ejemplos

Ejemplo #1 gmp_div_r() example

<?php
$div = gmp_div_r("105", "20");
echo gmp_strval($div) . "\n";
?>

El resultado del ejemplo sería:

5
