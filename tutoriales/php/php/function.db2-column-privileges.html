db2_column_privileges

(PECL ibm_db2 >= 1.0.0)

db2_column_privileges — Obtiene la lista de columnas y permisos asociados a una tabla

Descripción

resource db2_column_privileges ( resource $connection [, string $qualifier [, string $schema [, string $table-name [, string $column-name ]]]] )

Obtiene la lista de columnas y permisos asociados a una tabla.

Parámetros

connection

Una conexión a IBM DB2, Cloudscape, o Apache Derby.

qualifier

Un calificador para bases de datos DB2 que se ejecutan en servidores OS/390 o z/OS. Para otras bases de datos, use NULL o una cadena vacía.

schema

El esquema que contiene la tabla indicada. Para buscar en todos los esquemas, use NULL o una cadena vacía.

table-name

El nombre de la tabla o vista. Para consultar todas las tablas en la base de datos, use NULL o una cadena vacía.

column-name

El nombre de la columna. Para consultar todas las columnas en la tabla, use NULL o una cadena vacía.

Valores devueltos

Devuelve una lista de registros que describen los permisos de las columnas especificadas a través de los parámetros entregados a la función. Los registros contienen los siguientes campos:

Columna Descripción
TABLE_CAT Nombre del catálogo. Puede ser NULL si esta tabla no tiene catálogo.
TABLE_SCHEM Nombre del esquema.
TABLE_NAME Nombre de la tabla o vista.
COLUMN_NAME Nombre de la columna.
GRANTOR Identificador del usuario que otorgó el permiso.
GRANTEE Identificador del usuario a quien se le dió el permiso.
PRIVILEGE Permisos de la columna.
IS_GRANTABLE Informa si el usuario que obtiene el permiso puede otorgárselo a otros usuarios.
