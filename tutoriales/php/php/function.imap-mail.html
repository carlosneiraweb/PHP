imap_mail

(PHP 4, PHP 5, PHP 7)

imap_mail — Enviar un mensaje de correo electrónico

Descripción

bool imap_mail ( string $to , string $subject , string $message [, string $additional_headers = NULL [, string $cc = NULL [, string $bcc = NULL [, string $rpath = NULL ]]]] )

Esta función permite enviar emails con el manejo correcto de los destinatarios Cc y Bcc.

Los parámetros to, cc y bcc son todos cadenas y son analizadas como la lista de dierccioes » RFC822.

Parámetros

to

El destinatario

subject

El asunto del correo

message

El cuerpo del correo, véase imap_mail_compose()

additional_headers

Como cadena con cabeceras adicionales para ser establecidas en el correo

cc

bcc

Los destinatarios especificados en bcc recibirán el correo, pero son excluidos de las cabeceras.

rpath

Use este parámetro para especificar la ruta de vuelta si la entrega del corro falla. Esto es útil al usar PHP como un cliente de correo para múltiples usuarios.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.
