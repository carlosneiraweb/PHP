Transliterator::__construct

(PHP >= 5.4.0, PECL intl >= 2.0.0)

Transliterator::__construct — Constructor privado para denegar la instanciación

Descripción

Transliterator::__construct ( void )

Este método no debería ser llamada. Su único propósito es denegar la instanciación con el operador new.

En su lugar use los métodos de fábrica Transliterator::create() o Transliterator::createFromRules().

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Este método no debería ser ejecutado. Si lo es (p.ej. a través de reflexión), devuelve un valor no especificado.

Ver también

  • Transliterator::create() - Crear un transliterador
  • Transliterator::createFromRules() - Crear un transliterador desdes normas
