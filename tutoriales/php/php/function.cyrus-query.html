cyrus_query

(PHP 4 >= 4.1.0, PECL cyrus 1.0)

cyrus_query — Envía una petición a un servidor IMAP Cyrus

Descripción

array cyrus_query ( resource $connection , string $query )

Envía una petición a un servidor IMAP Cyrus.

Parámetros

connection

El controlador de la conexión.

query

La petición.

Valores devueltos

Devuelve un array asociativo con las siguientes claves: text, msgno, y keyword.
