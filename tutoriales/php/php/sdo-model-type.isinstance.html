SDO_Model_Type::isInstance — Comprobar si un SDO_DataObject es una instancia de este SDO_Model_Type

Descripción

bool SDO_Model_Type::isInstance ( SDO_DataObject $data_object )
Advertencia

Esta función ha sido declarada EXPERIMENTAL. Su funcionamiento, nombre y la documentación que le acompaña puede cambiar sin previo aviso en futuras versiones de PHP. Utilícela bajo su propia responsabilidad.

Comprueba si un SDO_DataObject es una instancia de este SDO_Model_Type. Devuelve TRUE si el SDO_DataObject proporcionado es una instancia de este SDO_Model_Type, o un tipo derivado, si no, devuelve FALSE.

Parámetros

data_object

El SDO_DataObject a ser comprobado.

Valores devueltos

Devuelve TRUE si el SDO_DataObject proporcionado es una instancia de este SDO_Model_Type, o un tipo derivado, si no, devuelve FALSE.
