MongoLog::getLevel

(PECL mongo >= 1.2.3)

MongoLog::getLevel — Obtiene el nivel o niveles que están siendo registrados

Descripción

public static int MongoLog::getLevel ( void )

Esta función se puede usar para conocer qué niveles de registro están habilitados actualmente. El valor de tipo integer devuelto puede ser comparado con las constantes de nivel de MongoLog empleando operadores a nivel de bit para comprobar niveles de registro específicos.

<?php

if (MongoLog::getLevel() & MongoLog::FINE) {
    echo "un montón de logs\n";
}

if (MongoLog::getLevel() ^ MongoLog::NONE) {
    echo "logging, al menos un poco\n";
}

if (MongoLog::getLevel() == MongoLog::ALL) {
    echo "logging al más alto nivel\n";
}

?>

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve el nivel o niveles que están siendo registrados acutalemente.
