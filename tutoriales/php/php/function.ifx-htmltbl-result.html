ifx_htmltbl_result

(PHP 4, PHP <=5.2.0)

ifx_htmltbl_result — Formatea todas las filas de una consulta en una tabla HTML

Descripción

int ifx_htmltbl_result ( resource $result_id [, string $html_table_options ] )

Formatea e imprime todas las filas de la consulta result_id en una tabla HTML.

Parámetros

result_id

result_id es un resultid valido devuelto por ifx_query() o ifx_prepare() (¡consultas de selección de tipo único!).

html_table_options

Este argumento opcional es un string de la etiqueta de opciones <table>.

Valores devueltos

Devuelve el número de filas recuperadas, o FALSE en caso de error.

Ejemplos

Ejemplo #1 Informix resultados como tabla HTML

<?php
$rid = ifx_prepare ("select * from emp where name like " . $name,
                     $connid, IFX_SCROLL);
if (! $rid) {
    /* ... error ... */
}
$rowcount = ifx_affected_rows ($rid);
if ($rowcount > 1000) {
    printf ("Too many rows in result set (%d)\n<br />", $rowcount);
    die ("Please restrict your query<br />\n");
}
if (! ifx_do($rid)) {
    /* ... error ... */
}

ifx_htmltbl_result ($rid, "border=\"2\"");

ifx_free_result($rid);
?>
