Yaf_Response_Abstract::response

(Yaf >=1.0.0)

Yaf_Response_Abstract::response — Envía una respuesta

Descripción

public void Yaf_Response_Abstract::response ( void )

Envía una respuesta.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Ejemplos

Ejemplo #1 Ejemplo de Yaf_Response_Abstract::response()

<?php
$respuesta = new Yaf_Response_Http();

$respuesta->setBody("Hola")->setBody(" Mundo", "footer");

?>
