dbplus_rchperm

(PHP 4 <= 4.1.0, PECL dbplus >= 0.9)

dbplus_rchperm — Cambiar los permisos de una relación

Descripción

int dbplus_rchperm ( resource $relation , int $mask , string $user , string $group )

Cambia los permisos de acceso de la manera especificada por mask, user y group. Los valores para estos parámetros son específicos del sistema operativo.

Parámetros

relation

Una relación abierta con dbplus_open().

mask

user

group

Valores devueltos

Notas

Advertencia

Esta función ha sido declarada EXPERIMENTAL. Su funcionamiento, nombre y la documentación que le acompaña puede cambiar sin previo aviso en futuras versiones de PHP. Utilícela bajo su propia responsabilidad.
