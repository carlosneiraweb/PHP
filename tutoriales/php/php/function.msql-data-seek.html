msql_data_seek

(PHP 4, PHP 5, PHP 7)

msql_data_seek — Mueve el puntero de fila interno

Descripción

bool msql_data_seek ( resource $result , int $row_number )

msql_data_seek() mueve el puntero de fila interno del resultado mSQL asociado, con el identificador de consulta especificado para indicar el número de fila especificado. La siguiente llamada a msql_fetch_row() devolverá esta fila.

Parámetros

result

El resultado resource que está siendo evaluado. Este resultado proviene de una llamada a msql_query().

row_number

El número de fila buscado.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.

  • msql_fetch_object() - Extraer la fila como un objeto
  • msql_fetch_row() - Extrae la fila como un array ordenado
  • msql_result() - Obtención de datos del resultado
