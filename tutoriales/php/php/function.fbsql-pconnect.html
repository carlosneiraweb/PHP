fbsql_pconnect

(PHP 4 >= 4.0.6, PHP 5, PHP 7)

fbsql_pconnect — Abrir una conexión persistente a un servidor FrontBase

Descripción

resource fbsql_pconnect ([ string $hostname = ini_get("fbsql.default_host") [, string $username = ini_get("fbsql.default_user") [, string $password = ini_get("fbsql.default_password") ]]] )

Establece una conexión persisten a un servidor FrontBase.

Para definir el número de puerto, use fbsql_select_db().

fbsql_pconnect() funciona de forma similar a fbsql_connect() con dos grandes diferencias:

Primero, al conectarse, la función tratará de encontrar un enlace (persistente) que ya esté abierto con el mismo host, usuario y contraseña. Si se encuentra uno, se devolverá un identificador en lugar de abrir una nueva conexión.

Segundo, la conexión con el servidor SQL no se cerrará al finalizar la ejecución del script. En su lugar, el enlace permanecerá abierto para su futuro uso.

Por esta razón este tipo de enlaces es llamado 'persistente'.

Parámetros

hostname

Nombre del servidor.

username

El nombre de usuario para la conexión.

password

El password para la conexión.

Valores devueltos

Devuelve un enlace persistente positivo FrontBase en caso de éxito, o FALSE en caso de error.
