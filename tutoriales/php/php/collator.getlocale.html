Collator::getLocale

collator_get_locale

(PHP 5 >= 5.3.0, PHP 7, PECL intl >= 1.0.0)

Collator::getLocale -- collator_get_locale — Obtener el nombre de la configuración regional del cotejador

Descripción

Estilo orientado a objetos

public string Collator::getLocale ( int $type )

Estilo por procedimientos

string collator_get_locale ( Collator $coll , int $type )

Obtiene el nombre de la configuración regional del cotejador.

Parámetros

coll

Un objeto de la clase Collator.

type

Se puede escoger entre una configuración regional válida y real ( Locale::VALID_LOCALE y Locale::ACTUAL_LOCALE, respectivamente).

Valores devueltos

El nombre de la configuración regional real de donde viene la información del cotejador. Si el cotejador fue instanciado desde normas u ocurrió un error, devuelve FALSE de tipo boolean.

Ejemplos

Ejemplo #1 Ejemplo collator_get_locale()

<?php
$coll    = collator_create( 'en_US_California' );
$res_val = collator_get_locale( $coll, Locale::VALID_LOCALE );
$res_act = collator_get_locale( $coll, Locale::ACTUAL_LOCALE );
printf( "Nombre regional válido: %s\nNombre regional real: %s\n",
         $res_val, $res_act );
?>

El resultado del ejemplo sería:

Requested locale name: en_US_California
Nombre regional válido: en_US
Nombre regional real: en
