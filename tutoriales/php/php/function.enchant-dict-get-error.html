enchant_dict_get_error

(PHP 5 >= 5.3.0, PHP 7, PECL enchant >= 0.1.0 )

enchant_dict_get_error — Devuelve el último error de la sesión ortográfica actual

Descripción

string enchant_dict_get_error ( resource $dict )

Devuelve el último error de la sesión ortográfica actual

Parámetros

dict

Recurso de diccionario

Valores devueltos

Devuelve un mensaje de error como cadena o FALSE si no se produjo ningún error.
