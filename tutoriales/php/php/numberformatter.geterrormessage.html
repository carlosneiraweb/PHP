NumberFormatter::getErrorMessage

numfmt_get_error_message

(PHP 5 >= 5.3.0, PHP 7, PECL intl >= 1.0.0)

NumberFormatter::getErrorMessage -- numfmt_get_error_message — Obtener el último mensaje de error del formateador

Descripción

Estilo orientado a objetos

public string NumberFormatter::getErrorMessage ( void )

Estilo por procedimientos

string numfmt_get_error_message ( NumberFormatter $fmt )

Obtener mensaje de error de la última función realizada por el formateador.

Parámetros

fmt

Objeto NumberFormatter.

Valores devueltos

Devuelve el mensaje de error de la última llamada al formateador.

Ejemplos

Ejemplo #1 Ejemplo de numfmt_get_error_message()

<?php
$fmt = numfmt_create( 'de_DE', NumberFormatter::DECIMAL );
$datos = numfmt_format($fmt, 1234567.891234567890000);
var_dump(numfmt_get_error_message($fmt));
?>

Ejemplo #2 Ejemplo orientado a objetos

<?php
$fmt = new NumberFormatter( 'de_DE', NumberFormatter::DECIMAL );
$fmt->format(1234567.891234567890000);
var_dump(numfmt_get_error_message($fmt));
?>
