imap_mail_move

(PHP 4, PHP 5, PHP 7)

imap_mail_move — Mover los mensajes especificados a un buzón

Descripción

bool imap_mail_move ( resource $imap_stream , string $msglist , string $mailbox [, int $options = 0 ] )

Mueve los mensajes de correo especificados por msglist al buzón especificado por mailbox.

Parámetros

imap_stream

IMAP stream devuelto por imap_open().

msglist

msglist es un rango, y no números de mensajes (como está descrito en » RFC2060).

mailbox

El nombre del buzón, véase imap_open() para más información

options

options es una máscara de bits y puede contener la única opción:

  • CP_UID - los números de secuencia que contienen UIDS

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.

Notas

Nota:

imap_mail_move() marcará el correo original con una bandera de eliminación, para eliminarlo con éxito se debe hacer una llamada a imap_expunge().
