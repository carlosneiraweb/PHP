SolrQuery::getFacetDateHardEnd

(PECL solr >= 0.9.2)

SolrQuery::getFacetDateHardEnd — Devuelve el valor del parámetro facet.date.hardend

Descripción

public string SolrQuery::getFacetDateHardEnd ([ string $field_override ] )

Devuelve el valor del parámetro facet.date.hardend. Acepta una sobrescritura opcional de campos

Parámetros

field_override

El nombre del campo

Valores devueltos

Devuelve una cadena en caso de éxito y NULL si no se estableció
