cubrid_num_rows

(PECL CUBRID >= 8.3.0)

cubrid_num_rows — Obtener el número de filas del conjunto de resultados

Descripción

int cubrid_num_rows ( resource $result )

La función cubrid_num_rows() se usa para obtener el número de filas del resultado de la consulta. Se puede usar sólo cuando la consulta ejecutada es una sentencia SELECT. Cuando se quiere conocer dicho valor para consultas INSERT, UPDATE, o DELETE, se tiene que usar la función cubrid_affected_rows().

Nota: La función cubrid_num_rows() sólo puede ser usada para consultas síncronas; devuelve 0 cuando se usa para consultas asíncronas.

Parámetros

result

result viene de una llamada a cubrid_execute(), cubrid_query() y cubrid_prepare()

Valores devueltos

El número de filas, cuando el proceso tiene éxito.

0 cuando la consulta fue hecha en modo asíncrono.

-1, si la sentencia SQL no es SELECT.

FALSE cuando el proceso no tiene éxito.

Ejemplos

Ejemplo #1 Ejemplo de cubrid_num_rows()

<?php
$conn = cubrid_connect("localhost", 33000, "demodb", "dba");

$req = cubrid_execute($conn, "SELECT * FROM code");

$row_num = cubrid_num_rows($req);
$col_num = cubrid_num_cols($req);

printf("Row Num: %d\nColumn Num: %d\n", $row_num, $col_num);

cubrid_disconnect($conn);
?>

El resultado del ejemplo sería:

Row Num: 6
Column Num: 2
