HaruPage::setWordSpace

(PECL haru >= 0.0.1)

HaruPage::setWordSpace — Establece el espacio de la palabra para la página

Descripción

bool HaruPage::setWordSpace ( float $word_space )

Establece el espacio de la palabra para la página.

Parámetros

word_space

Define el espacio de la palabra para la página.

Valores devueltos

Devuelve TRUE en éxito.

Errores/Excepciones

Lanza una HaruException cuando se produce un error.
