Introducción

Este módulo permite leer y escribir bases de datos Paradox, archivos primary index y archivos blob. El soporte de escritura ha demostrado ser muy fiable, aunque debido a la falta de documentación de los archivos producidos no podrán en ningún caso, ser leídos por otras aplicaciones. Las bases de datos cifradas pueden ser ser leídas sin especificar una contraseña si se utiliza pxlib >= 0.5.0.

Nota:

Este módulo también está en desarrollo y puede cambiar, aunque no se espera que haya cambios importantes en la API.

Advertencia

Esta extensión es EXPERIMENTAL. Esto significa que el comportamiento de esta extensión, los nombres de sus funciones y todo lo documentado sobre esta extensión, puede cambiar en una futura edición de PHP sin previo aviso. El uso de esta extensión queda bajo su propia responsabilidad.
