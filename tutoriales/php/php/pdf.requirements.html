Requerimientos

PDFlib Lite está disponible como código abierto. Sin embargo, la licencia de PDFlib Lite permite el uso gratuito sólo bajo ciertas condiciones. PDFlib Lite soporta un subconjunto de fucionalidadse de PDFlib; por favor, vea la página web de PDFlib para más detalles. La versión completa de PDFlib está disponible para descargar en » http://www.pdflib.com/products/pdflib-family/, pero requiere que adquiera una licencia para uso comercial.

Cuestiones sobre versiones antiguas de PDFlib

Cualquier versión de PHP 4 después del 9 de marzo del 2000 no soporta versiones de PDFlib anteriores a 3.0.

PDFlib 4.0 o posterior es soportada por PHP 4.3.0 o posterior.
