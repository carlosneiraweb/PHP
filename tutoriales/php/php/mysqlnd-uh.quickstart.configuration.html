Puesta en marcha

El complemento está implementado como una extensión de PHP. Véanse las instrucciones de instalación para instalar la extensión » PECL/mysqlnd_uh. Luego, se ha de cargar la estensión en PHP y activar el complemento en el fichero de configuración de PHP usando la directiva de configuración de PHP llamada mysqlnd_uh.enable. El ejemplo de abajo muestra la configuración predeterminada de la extensión.

Ejemplo #1 Habilitar el complemento (php.ini)

mysqlnd_uh.enable=1
mysqlnd_uh.report_wrong_types=1
