ReflectionClass::getExtension

(PHP 5, PHP 7)

ReflectionClass::getExtension — Obtiene un objeto ReflectionExtension para la extensión que definió la clase

Descripción

public ReflectionExtension ReflectionClass::getExtension ( void )

Obtiene un objeto ReflectionExtension para la extensión que definió la clase.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Un objeto ReflectionExtension que representa la extensión que definió la clase, o NULL para clases definidas por el usuario.

Ejemplos

Ejemplo #1 Uso básico de ReflectionClass::getExtension()

<?php
$clase = new ReflectionClass('ReflectionClass');
$extensión = $clase->getExtension();
var_dump($extensión);
?>

El resultado del ejemplo sería:

object(ReflectionExtension)#2 (1) {
  ["name"]=>
  string(10) "Reflection"
}
