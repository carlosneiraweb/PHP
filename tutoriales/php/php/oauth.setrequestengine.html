OAuth::setRequestEngine

(PECL OAuth >= 1.0.0)

OAuth::setRequestEngine — El propósito de setRequestEngine

Descripción

public void OAuth::setRequestEngine ( int $reqengine )

Establece el motor de peticiones, que estará enviando las peticiones HTTP.

Advertencia

Esta función no está documentada actualmente, solamente se encuentra disponible la lista de parámetros.

Parámetros

reqengine

El motor de peticiones deseado. Poner a OAUTH_REQENGINE_STREAMS para usar el Stream PHP, o OAUTH_REQENGINE_CURL para usar Curl.

Valores devueltos

No devuelve ningún valor.

Errores/Excepciones

Emite una excepción OAuthException si un motor de peticiones inválido es escogido.

Ejemplos

Ejemplo #1 Ejemplo de OAuth::setRequestEngine()

<?php
$consumer = new OAuth();

$consumer->setRequestEngine(OAUTH_REQENGINE_STREAMS);
?>
