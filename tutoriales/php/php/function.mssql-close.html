mssql_close

(PHP 4, PHP 5, PECL odbtp >= 1.1.1)

mssql_close — Cierra una conexión MS SQL Server

Advertencia

Esta función está OBSOLETA en PHP 7.0.0.

Las alternativas a esta función son:

  • sqlsrv_close()
  • odbc_close()
  • odbc_close_all()

Descripción

bool mssql_close ([ resource $link_identifier ] )

Cierra la conexión a la base MS SQL Server que está asociada con el identificador de conexión especificado. Si el identificador de conexión no está especificado, se se cerrará la última conexión abierta.

Notar que habitualmente no es necesario, puesto que las conexiones abiertas no persistentes se cierran automáticamente al final de la ejecución del script.

Parámetros

link_identifier

Un identificador de conexión MS SQL, devuelto por la función mssql_connect().

Esta función no cerrará las conexiones persistentes generadas por mssql_pconnect().

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.

Ejemplos

Ejemplo #1 Ejemplo con mssql_close()

<?php
// Conectar a MSSQL
$link = mssql_connect('KALLESPC\SQLEXPRESS', 'sa', 'phpfi');

// Realizar las operaciones SQL aquí ....

// Cerrar la conexión a MSSQL
mssql_close($link);
?>
