ImagickDraw::pathCurveToSmoothAbsolute

(PECL imagick 2.0.0)

ImagickDraw::pathCurveToSmoothAbsolute — Dibuja una curva cúbica de Bézier

Descripción

bool ImagickDraw::pathCurveToSmoothAbsolute ( float $x2 , float $y2 , float $x , float $y )
Advertencia

Esta función no está documentada actualmente, solamente se encuentra disponible la lista de parámetros.

Dibuja una curva cúbica de Bézier desde el punto actual a (x,y) usando coordenadas absolutas. Se asume que el primer punto de control es la reflexión del segundo punto de control del comando anterior relativo a punto actual. (Si no hay comando previo o el comando previo no es DrawPathCurveToAbsolute, DrawPathCurveToRelative, DrawPathCurveToSmoothAbsolute o DrawPathCurveToSmoothRelative, se asume que el primer punto de control coincide con el punto actual.) (x2,y2) es el segundo punto de control (es decir, el punto de control al final de la curva). Al final del comando, el nuevo punto actual se convierte en el par de coordenadas final (x, y) usado en el Bezígono.

Parámetros

x2

coordenada x del segundo punto de control

y2

coordenada y del segundo punto de control

x

coordenada x del punto final

y

coordenada y del punto final

Valores devueltos

No devuelve ningún valor.
