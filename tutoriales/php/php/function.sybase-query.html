sybase_query

(PHP 4, PHP 5, PHP 7)

sybase_query — Envía una consulta a una base Sybase

Descripción

mixed sybase_query ( string $query [, resource $link_identifier ] )

sybase_query() envia una consulta a la base de datos activa en ese momento en el servidor y que está asociada con el identificador de conexión link_identifier especificado.

Parámetros

query

link_identifier

Si no se especifica el parámetro link_identifier, se utiliza la última conexión establecida. Si no hay ninguna conexión abierta, la función intenta establecer una conexión como si se hubiera llamado a sybase_connect() , y la utiliza.

Valores devueltos

Devuelve un identificador de resultado Sybase en caso de éxito, FALSE en caso de error, o TRUE si la consulta se ejecutó satisfactoriamente pero no devolvió ninguna columna.

Ver también

  • sybase_select_db() - Selecciona una base de datos Sybase
  • sybase_connect() - Establece una conexión a un servidor Sybase
