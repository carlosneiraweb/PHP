CairoSurface::__construct

(PECL cairo >= 0.1.0)

CairoSurface::__construct — El propósito de __construct

Descripción

public CairoSurface::__construct ( void )

CairoSurface es un tipo abstracto y, como tal, no debería ser instanciado en los scripts de PHP.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

No devuelve ningún valor

Notas

Nota:

Si está usando una versión de PHP anterior a 5.3, debe destruir el CairoContext asociado con CairoSurface después de destruir el CairoSurface. De otro modo, perderá información de referencia circular en la memoria.
