sqlite_column

SQLiteResult::column

SQLiteUnbuffered::column

(PHP 5 < 5.4.0, PECL sqlite >= 1.0.0)

sqlite_column -- SQLiteResult::column -- SQLiteUnbuffered::column — Obtiene una columna de la fila actual de un conjunto de resultados

Descripción

mixed sqlite_column ( resource $result , mixed $index_or_name [, bool $decode_binary = true ] )
mixed SQLiteResult::column ( mixed $index_or_name [, bool $decode_binary = true ] )
mixed SQLiteUnbuffered::column ( mixed $index_or_name [, bool $decode_binary = true ] )

Obtiene el valor de una columna llamada index_or_name (si es un string), o del número ordinal de columna index_or_name (si es un integer) de la fila actual del manejador del resultado result de una consulta.

Parámetros

result

El recurso del resultado de SQLite. Este parámetro no es requerido cuando se usa el método orientado a objetos.

index_or_name

El índice o nombre de la columna a obtener.

decode_binary

Cuando el parámetro decode_binary es seteado a TRUE (el valor por defecto), PHP decodificará la codificación binaria que se aplicó a los datos si éstos fueron codificados usando la función sqlite_escape_string(). Normalmente debería dejar este valor por defecto, a menos que esté interoperando con bases de datos creadas por otras aplicaciones sqlite.

Valores devueltos

Devuelve el valor de la columna.

Notas

Nota:

Usar esta función cuando se esté iterando un conjunto de resultado grande con muchas columnas, o con columnas que contengan una gran cantidad de datos.
