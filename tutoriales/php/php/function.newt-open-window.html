newt_open_window

(PECL newt >= 0.1)

newt_open_window — Abre una ventana con el tamaño y posición especificados

Descripción

int newt_open_window ( int $left , int $top , int $width , int $height [, string $title ] )

Abre una ventana con el tamaño y posición especificados.

Parámetros

left

La ubicación de la esquina superior izquierda de la ventana (el número de columna)

top

La ubicación de la esquina inferior derecha de la ventana (el número de fila)

width

El ancho de la ventana

height

El alto de la ventana

title

El título de la ventana

Valores devueltos

Devuelve 1 en caso de éxito, 0 en caso de error.

Ver también

  • newt_pop_window() - Elimina la ventana superior de la exposición
  • newt_centered_window() - Abrir una ventana centrada de un tamaño específico
