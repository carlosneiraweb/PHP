filepro

(PHP 4, PHP 5 <= 5.1.6, PHP 7)

filepro — Lee y verifica el mapa del fichero

Descripción

bool filepro ( string $directory )

Lee y verifica el mapa del fichero, guardando el número de campos e información.

No se produce ningún bloqueo, así que se debería evitar modificar la base de datos cuando es abierto con PHP.

Nota: Cuando el modo seguro está habilitado, PHP comprueba si los ficheros o directorios sobre los que se opera tienen el mismo UID (propietario) que el script que está siendo ejecutado.

Parámetros

directory

El directorio del mapa.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.
