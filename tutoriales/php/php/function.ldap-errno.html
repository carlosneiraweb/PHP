ldap_errno

(PHP 4, PHP 5, PHP 7)

ldap_errno — Devuelve el número de error LDAP del último comando LDAP

Descripción

int ldap_errno ( resource $link_identifier )

Devuelve el número de error estándar devuelto por el último comando LDAP. Este número se puede convertir a un mensaje de error textual utilizando ldap_err2str().

Parámetros

link_identifier

Un identificador de enlace de LDAP, devuelto por ldap_connect().

Valores devueltos

Devolver el número de error de LDAP del último comando LDAP para este enlace.

Ejemplos

A menos que reduzca suficientemente el nivel de advertencia en su php.ini o le anteponga una arroba (@) a sus comandos LDAP para suprimir la salida de advertencia, los errores generados también se mostrarán en la salida HTML.

Ejemplo #1 Generación y captura de un error

<?php
// Este ejemplo contiene un error, el cual capturaremos.
$ld = ldap_connect("localhost");
$bind = ldap_bind($ld);
// Error de sintáxis en la expresión de filtro (errno 87),
// debe ser un "objectclass=*" para que funcione.
$res =  @ldap_search($ld, "o=Myorg, c=DE", "objectclass");
if (!$res) {
    echo "Errno de LDAP: " . ldap_errno($ld) . "<br />\n";
    echo "Errno de LDAP: " . ldap_error($ld) . "<br />\n";
    die("Ouch!<br />\n");
}
$info = ldap_get_entries($ld, $res);
echo $info["count"] . " pareando entradas.<br />\n";
?>
