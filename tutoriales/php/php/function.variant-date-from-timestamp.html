variant_date_from_timestamp

(PHP 5, PHP 7)

variant_date_from_timestamp — Devuelve una representación de variante de fecha de una fecha Unix

Descripción

variant variant_date_from_timestamp ( int $timestamp )

Convierte timestamp desde un valor de fecha unix a una variante de tipo VT_DATE. Permite la interpolación más sencilla entre las partes unix de PHP y COM.

Parámetros

timestamp

Una fecha unix.

Valores devueltos

Devuelve una variante VT_DATE.

Ver también

  • variant_date_to_timestamp() - Convierte un valor de una variante de fecha/hora en una fecha Unix
  • mktime() - Obtener la marca de tiempo Unix de una fecha
  • time() - Devuelve la fecha Unix actual
