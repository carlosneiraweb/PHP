dbplus_add

(PHP 4 <= 4.1.0, PECL dbplus >= 0.9)

dbplus_add — Añade una tupla a una relación

Descripción

int dbplus_add ( resource $relation , array $tuple )

Añade una tupla a relation.

Parámetros

relation

tuple

Una matriz de pares atributo/valor que se va a insertar en la relación relation dada.

Después de una ejecución con éxito, esta matriz contendrá la información completa de la recién creada tupla, incluyendo todos los campos de dominio establecidos implícitamente como secuencias.

Valores devueltos

La función devolverá DBPLUS_ERR_NOERR si se tuvo éxito o un código de error de db++ si falló.

Notas

Advertencia

Esta función ha sido declarada EXPERIMENTAL. Su funcionamiento, nombre y la documentación que le acompaña puede cambiar sin previo aviso en futuras versiones de PHP. Utilícela bajo su propia responsabilidad.
