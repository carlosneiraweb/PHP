ibase_blob_get

(PHP 5, PHP 7)

ibase_blob_get — Consulta un determinado número de bytes de un blob

Descripción

string ibase_blob_get ( resource $blob_handle , int $len )

Esta función devuelve hasta len bytes de un BLOB que se haya abierto para lectura mediante ibase_blob_open().

Nota:

No se podrá leer de un BLOB que se haya abierto en modo escritura mediante ibase_blob_create().

Parámetros

blob_handle

Manejador de BLOB, abierto por ibase_blob_open().

len

Tamaño de la lectura.

Valores devueltos

Devuelve hasta len bytes del BLOB, o FALSE en caso de error.

Ejemplos

Ejemplo #1 Ejemplo de ibase_blob_get()

<?php
$result    = ibase_query("SELECT blob_value FROM table");
$data      = ibase_fetch_object($result);
$blob_data = ibase_blob_info($data->BLOB_VALUE);
$blob_hndl = ibase_blob_open($data->BLOB_VALUE);
echo         ibase_blob_get($blob_hndl, $blob_data[0]);
?>
A pesar de que este ejemplo no hace nada que no haga 'ibase_blob_echo($data->BLOB_VALUE)', ilustra cómo consultar información en una variable para poder manipularla posteriormente.
