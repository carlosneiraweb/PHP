filter_list

(PHP 5 >= 5.2.0, PHP 7)

filter_list — Devuelve una lista de todos los filtros soportados

Descripción

array filter_list ( void )

Valores devueltos

Retorna un array con los nombres de todos los filtros soportados; el array estará vacío si no hay filtros. Los índices de este array no son los IDs de los filtros, estos se obtienen a partir del nombre y la función filter_id().

Ejemplos

Ejemplo #1 Ejemplo de filter_list()

<?php
print_r(filter_list());
?>

El resultado del ejemplo sería algo similar a:

Array
(
    [0] => int
    [1] => boolean
    [2] => float
    [3] => validate_regexp
    [4] => validate_url
    [5] => validate_email
    [6] => validate_ip
    [7] => string
    [8] => stripped
    [9] => encoded
    [10] => special_chars
    [11] => unsafe_raw
    [12] => email
    [13] => url
    [14] => number_int
    [15] => number_float
    [16] => magic_quotes
    [17] => callback
)
