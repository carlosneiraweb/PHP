sqlite_prev

SQLiteResult::prev

(PHP 5 < 5.4.0)

sqlite_prev -- SQLiteResult::prev — Busca el número de fila anterior de un conjunto de resultados

Descripción

bool sqlite_prev ( resource $result )

Estilo orientado a objetos (método):

bool SQLiteResult::prev ( void )

sqlite_prev() vuelve a buscar en el gestor result una fila anterior.

Parámetros

result

El recurso de resultados de SQLite. Este parámetro no es necesario cuando se usa el método orientado a objetos.

Nota:

Esta función no puede ser utilizada con manejos de resultados no pasados por buffer.

Valores devueltos

Devuelve TRUE en caso de éxito, o FALSE si no hay más filas anteriores.

Ver también

  • sqlite_has_prev() - Comprueba si está o no disponible una fila anterior
  • sqlite_rewind() - Busca el primer número de fila
  • sqlite_next() - Busca el número de fila siguiente
