streamWrapper::rename

(PHP 5, PHP 7)

streamWrapper::rename — Renombra un archivo o directorio

Descripción

public bool streamWrapper::rename ( string $path_from , string $path_to )

Este método es llamado en respuesta a rename().

Debería intentar renombrar path_from a path_to

Nota:

Para que el mensaje de error apropiado sea devuelto, este método no debería ser definido si la envoltura no soporta el renombramiento de archivos.

Parámetros

path_from

La URL al archivo actual.

path_to

La URL que debería ser renombrada por path_from.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.

Errores/Excepciones

Emite un E_WARNING si la llamada a este método falla (por ej., por no estar implementado).

Notas

Nota:

La propiedad streamWrapper::$context es actualizada si se pasa un contexto válido en la función invocante.
