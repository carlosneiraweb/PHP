ReflectionExtension::info

(PHP 5, PHP 7)

ReflectionExtension::info — Imprime información sobre la extensión

Descripción

public void ReflectionExtension::info ( void )

Imprime el bloque de "phpinfo()" correspondiente a la extensión dada.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Información sobre la extensión.

Ejemplos

Ejemplo #1 Ejemplo de ReflectionExtension::info()

<?php
$ext = new ReflectionExtension('mysqli');
$ext->info();
?>
