PDF_set_border_style

(PHP 4, PECL pdflib >= 1.0.0)

PDF_set_border_style — Establecer el estilo de borde de las anotaciones [obsoleta]

Descripción

bool PDF_set_border_style ( resource $pdfdoc , string $style , float $width )

Establece el estilo de borde de todos los tipos de anotaciones. Devuelve TRUE en caso de éxito o FALSE en caso de error.

Esta función está obsoleta desde la versión 6 de PDFlib, use las opciones borderstyle y linewidth en PDF_create_annotation() en su lugar.
