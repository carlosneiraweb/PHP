Instalación

Desde PHP 4.3.0 estas funciones están activadas por defecto. Para versiones anteriores se tiene que configurar y compilar PHP con --enable-tokenizer . Se puede deshabilitar el soporte del tokenizer con --disable-tokenizer .

La versión de PHP para Windows tiene soporte nativo para esta extensión. No se requiere cargar extensiones adicionales para utilizar estas funciones.

Nota: Soporte interno para el tokenizer está disponible como en PHP 4.3.0.
