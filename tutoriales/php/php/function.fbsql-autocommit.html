fbsql_autocommit

(PHP 4 >= 4.0.6, PHP 5, PHP 7)

fbsql_autocommit — Activa o desactiva el autocommit

Descripción

bool fbsql_autocommit ( resource $link_identifier [, bool $OnOff ] )

Devuelve el estado actual del autocommit.

Parámetros

link_identifier

Un identificador de enlace FrontBase devuelto por fbsql_connect() o fbsql_pconnect().

Es opcional y si no se especifica, la función intentará encontrar un enlace abierto al servidor FrontBase y de no encontrarse ningún enlace abierto intentará crear uno como si fbsql_connect() fuese invocada sin parámetros.

OnOff

Si se recibe este parametro opcional, se cambiará el estado del autocommit

Con OnOff a TRUE cada sentencia tendrá un commit automático en caso de no encontrarse errores.

Con OnOff a FALSE el usuario deberá hacer commit o rollback de la transacción, usando fbsql_commit() o fbsql_rollback() respectivamente.

Valores devueltos

Devuelve el estado actual del autocommit, como boolean.

Ver también

  • fbsql_commit() - Ejecuta un commit de la transacción sobre la base de datos
  • fbsql_rollback() - Retrocede una transacción a la base de datos
