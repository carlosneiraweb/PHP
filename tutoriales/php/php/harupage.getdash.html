HaruPage::getDash

(PECL haru >= 0.0.1)

HaruPage::getDash — Obtiene el actual patrón guía

Descripción

array HaruPage::getDash ( void )

Obtiene el actual patrón guía. Ver HaruPage::setDash() para más información de patrón guía.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve el actual patrón guía como un arreglo de dos elementos - "patrón" y "fase" o FALSE si el patrón guía no fuera establecido.

Errores/Excepciones

Lanza una HaruException cuando se produce un error.

Ver también

  • HaruPage::setDash() - Establece el patrón de guía para la página
