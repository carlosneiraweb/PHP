session_unregister

(PHP 4, PHP 5 < 5.4.0)

session_unregister — Deja de registrar una variable global de la sesión actual

Descripción

bool session_unregister ( string $name )

session_unregister() deja de registrar una variable global name de la sesión actual.

Advertencia

Esta función ha sido declarada OBSOLETA desde PHP 5.3.0 y ELIMINADA a partir de PHP 5.4.0.

Parámetros

name

El nombre de la variable.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.

Notas

Nota:

Si se usa $_SESSION (o $HTTP_SESSION_VARS para PHP 4.0.6 o anterior), utilice unset() para dejar de registrar una variable de sesión. No use unset() con $_SESSION ya que deshabilitará la función especial de la matriz superglobal $_SESSION.

Precaución

Esta función no destruye la variable global correspondiente para name, sólo impide que la variable sea guardada como parte de la sesión. Se debe llamar a unset() para eliminar la variable global correspondiente.

Precaución

Si está usando $_SESSION (o $HTTP_SESSION_VARS), no use session_register(), session_is_registered() y session_unregister().
