mb_parse_str

(PHP 4 >= 4.0.6, PHP 5, PHP 7)

mb_parse_str — Analiza datos GET/POST/COOKIE y establece varialbes globales

Descripción

bool mb_parse_str ( string $encoded_string [, array &$result ] )

Analiza datos GET/POST/COOKIE y establece variables globales. Ya que PHP no proporciona datos POST/COOKIE sin tratar, solamente se puede usar para datos GET por ahora. Analiza los datos del URL codificado, detecta la codificación, convierte la codificación a la codificación interna y establece los valores del array result o de las variables globales.

Parámetros

encoded_string

Los datos del URL codificado.

result

Un array que contiene la decodificación y los valores convertidos de la codificación de caracteres.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.
