HaruPage::setMiterLimit

(PECL haru >= 0.0.1)

HaruPage::setMiterLimit — Establece el valor actual del inglete limite de la página

Descripción

bool HaruPage::setMiterLimit ( float $limit )

Establece el valor actual del inglete limite de la página.

Parámetros

limit

Define el valor actual del inglete limite de la página.

Valores devueltos

Devuelve TRUE en éxito.

Errores/Excepciones

Lanza una HaruException cuando se produce un error.
