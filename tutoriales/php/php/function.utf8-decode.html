utf8_decode

(PHP 4, PHP 5, PHP 7)

utf8_decode — Convierte una cadena con los caracteres codificados ISO-8859-1 con UTF-8 a un sencillo byte ISO-8859-1

Descripción

string utf8_decode ( string $data )

Ésta función decodifica los data, asumidos a ser codificados por UTF-8, a ISO-8859-1.

Parámetros

data

Una cadena codificada UTF-8.

Valores devueltos

Devuelve la traducción ISO-8859-1 de los data.

Ver también

  • utf8_encode() - Codifica un string ISO-8859-1 a UTF-8 (contiene una explicación de la codificación UTF-8)
