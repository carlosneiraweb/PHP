sem_remove

(PHP 4 >= 4.1.0, PHP 5, PHP 7)

sem_remove — Eliminar un semáforo

Descripción

bool sem_remove ( resource $sem_identifier )

sem_remove() elimina el semáforo dado.

Después de eliminar el semáforo, éste ya no está accesible.

Parámetros

sem_identifier

Un identificador de recurso de semáforo devuelto por sem_get().

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.

  • sem_release() - Liberar un semáforo
  • sem_acquire() - Adquirir un semáforo
