ImagickDraw::getTextAntialias

(PECL imagick 2.0.0)

ImagickDraw::getTextAntialias — Devuelve la configuración de antialias del texto actual

Descripción

bool ImagickDraw::getTextAntialias ( void )
Advertencia

Esta función no está documentada actualmente, solamente se encuentra disponible la lista de parámetros.

Devuelve la configuración de antialias del texto actual, que determina si el texto tiene antialias o no. El texto tiene antialias por defecto.

Valores devueltos

Devuelve TRUE si el texto tiene antialias y false si no.
