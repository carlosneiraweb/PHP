RecursiveTreeIterator::key

(PHP 5 >= 5.3.0, PHP 7)

RecursiveTreeIterator::key — Obtiene la clave de el elemento actual

Descripción

public string RecursiveTreeIterator::key ( void )

Obtiene la clave con prefiro y postfijo.

Advertencia

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve la clave actual con prefijo y postfijo.
