HaruPage::getHorizontalScaling

(PECL haru >= 0.0.1)

HaruPage::getHorizontalScaling — Obtiene el valor actual de la escala horizontal

Descripción

float HaruPage::getHorizontalScaling ( void )

Obtiene el valor actual de la escala horizontal.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve el valor actual de de la escala horizontal.

Errores/Excepciones

Lanza una HaruException cuando se produce un error.

Ver también

  • HaruPage::setHorizontalScaling() - Establece la escala horizontal para la página
