(^)

SDO_DataObject::getTypeNamespaceURI — Devolver el URI del espacio de nombre del tipo de un objeto de datos.

Descripción

string SDO_DataObject::getTypeNamespaceURI ( void )
Advertencia

Esta función ha sido declarada EXPERIMENTAL. Su funcionamiento, nombre y la documentación que le acompaña puede cambiar sin previo aviso en futuras versiones de PHP. Utilícela bajo su propia responsabilidad.

Devuelve el URI del espacio de nombres del tipo de un objeto de datos. Un método conveniente correspondiente a SDO_Model_ReflectionDataObject::getType().getNamespaceURI().

Parámetros

Ninguno.

Valores devueltos

El URI del espacio de nombres del tipo del objeto de datos
