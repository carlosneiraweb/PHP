Introducción

NIS (anteriormente conocidocomo Páginas Amarillas) permite la gestión de redes de importantes archivos administrativos (cómo archivos de contraseñas). Para más información consulte la página de manual de NIS y »  The Linux NIS(YP)/NYS/NIS+ HOWTO. también hay un libro llamado » Managing NFS and NIS de Hal Stern.

Nota:

Esta extensión se ha movido al repositorio de » PECL y ya no se distribuye con PHP a partir de PHP 5.1.0.

Nota: Esta extensión no está disponible en plataformas Windows.
