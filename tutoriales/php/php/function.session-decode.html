session_decode

(PHP 4, PHP 5, PHP 7)

session_decode — Decodifica la información de sesión desde una cadena de sesión codificada

Descripción

bool session_decode ( string $data )

session_decode() decodifica la información de sesión serializada proporcianda en $data, y rellena la variable superglobal $_SESSION con el resultado.

Observe que el método de deserialización no es el mismo que unserialize(). El método de serialización es interno a PHP y se puede establecer usando session.serialize_handler.

Parámetros

data

La información codificada que va a ser alamcenada.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.

  • session.serialize_handler
