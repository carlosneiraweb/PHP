SAMConnection::unsubscribe

(PECL sam >= 0.1.0)

SAMConnection::unsubscribe — Cancela una suscripción a un tema específico.

Descripción

bool SAMConnection::unsubscribe ( string $subscriptionId [, string $targetTopic ] )

El método "unsubscribe" se utiliza para eliminar una suscripción existente a un tema específico.

Parámetros

subscriptionId

El identificador de una suscripción existente tal y como es devuelto por una llamada al método 'subscribe'.

Valores devueltos

Este método devuelve FALSE en caso de error.

Ejemplos

Ejemplo #1 Eliminar una suscripción

<?php
if (!$conn->unsubscribe($subid)) {
    // La eliminación de la suscripción falló!
    echo "Eliminación de la suscripción fallida ($conn->errno) $conn->error";
}
?>
