HaruPage::getLineCap

(PECL haru >= 0.0.1)

HaruPage::getLineCap — Obtiene el estilo actual del extremo de línea

Descripción

int HaruPage::getLineCap ( void )

Obtiene el estilo actual del extremo de línea.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve el estilo actual del extremo de línea. El valor resultate es uno de los siguientes:

  • HaruPage::BUTT_END - the line is squared off at the endpoint of the path.
  • HaruPage::ROUND_END - the end of the line becomes a semicircle with center in the end point of the path.
  • HaruPage::PROJECTING_SCUARE_END - the line continues to the point that exceeds half of the stroke width the end point.

Errores/Excepciones

Lanza una HaruException cuando se produce un error.
