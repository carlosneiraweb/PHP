Migración de ficheros de configuración

Dado que los módulos ISAPI cambiaron sus nombres, de php4xxx a php5xxx, es necesario hacer algunos cambios en los ficheros de configuración. Hubo también cambios en los nombres de ficheros de CLI y CGI. Por favor, revise la sección correspondiente para más información.

Migrar la configuración de Apache es muy sencillo. Vea el ejemplo de abajo para conocer qué cambios realizar:

Ejemplo #1 Migración en Apache de la configuración de PHP 5

# cambie esta línea:    
LoadModule php4_module /php/sapi/php4apache2.dll

# por esta otra:
LoadModule php5_module /php/php5apache2.dll

Si el servidor web está ejecutando PHP en modo CGI, debe tenerse en cuenta que la versión CGI ha cambiado su nombre de php.exe a php-cgi.exe. En Apache se debe hacer algo tal que así:

Ejemplo #2 Migración en Apache de la configuración a PHP 5 en modo CGI

# cambie esta línea:
Action application/x-httpd-php "/php/php.exe" 

# por esta otra:
Action application/x-httpd-php "/php/php-cgi.exe"

En otros servidores web se debe cambiar el nombre del CGI o del módulo ISAPI correspondiente.
