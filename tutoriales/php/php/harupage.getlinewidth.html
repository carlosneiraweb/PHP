HaruPage::getLineWidth

(PECL haru >= 0.0.1)

HaruPage::getLineWidth — Obtiene el ancho actual de la línea

Descripción

float HaruPage::getLineWidth ( void )

Obtiene el ancho actual de la línea.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve el ancho actual de la línea.

Errores/Excepciones

Lanza una HaruException cuando se produce un error.

Ver también

  • HaruPage::setLineWidth() - Establece el ancho de la línea para la página
