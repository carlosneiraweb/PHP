SDO_DAS_XML::loadFile

(^)

SDO_DAS_XML::loadFile — Devuelve un objeto SDO_DAS_XML_Document para la ruta dada al documento de instancia xml

Descripción

SDO_XMLDocument SDO_DAS_XML::loadFile ( string $xml_file )

Construye el árbol de of SDO_DataObjects desde la dirección dada al documento de instancia xml. Devuelve un objeto SDO_DAS_XML_Document. Use el método SDO_DAS_XML_Document::getRootDataObject para obtener el objeto de datos raíz.

Parámetros

xml_file

La ruta al documento instancia. Puede ser una ruta a un fichero local o un URL.

Valores devueltos

Devuelve un objeto SDO_DAS_XML_Document en caso de éxito o lanza una excepción como se describe abajo.

Errores/Excepciones

SDO_TypeNotFoundException

Lanzada si un tipo no está definido en el modelo subyacente.

SDO_PropertyNotFoundException

Lanzada si una propiedad dentro de un tipo no está definida en el modelo subyacente.

SDO_DAS_XML_ParserException

Lanzada por cualquier problema al analizar el fichero XSD dado.

SDO_DAS_XML_FileException

Lanzada si el fichero especificado no se puede encontrar.
