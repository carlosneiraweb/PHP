GearmanClient::setWorkloadCallback

(PECL gearman >= 0.5.0)

GearmanClient::setWorkloadCallback — Establece una llamada de retorno para aceptar actualizaciones incrementales de datos

Descripción

public bool GearmanClient::setWorkloadCallback ( callable $callback )

Establece una función a ser llamada cuando un trabajador necesita enviar datos antes de completar el trabajo. Un trabajador puede realizar esta acción cuando necesita enviar actualizaciones, enviar resultados parciales o enviar datos durante trabajados de larga duración. La función acepta un único argumento, un objeto GearmanTask.

Parámetros

callback

Función a llamar

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.
