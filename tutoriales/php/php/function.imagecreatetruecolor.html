imagecreatetruecolor

(PHP 4 >= 4.0.6, PHP 5, PHP 7)

imagecreatetruecolor — Crear una nueva imagen de color verdadero

Descripción

resource imagecreatetruecolor ( int $width , int $height )

imagecreatetruecolor() devuelve un identificador de imagen que representa una imagen en negro del tamaño especificado.

Dependiendo de las versiones de PHP y GD esta función está definida o no. Con PHP 4.0.6 hasta 4.1.x esta función siempre existe si el módulo GD está cargado, pero al llamarla sin que GD2 esté instalado, PHP emitirá un error fatal y saldrá. Con PHP 4.2.x este comportamiento es diferente, al emitir una advertencia en vez de un error. Otras versiones sólo definen esta función, si la versión correcta de GD está instalada.

Parámetros

width

Ancho de la imagen.

height

Alto de la imagen.

Valores devueltos

En caso de éxito, devuelve un identificador de recurso de imagen, y FALSE en caso de error.

Ejemplos

Ejemplo #1 Crear un nuevo flujo a una imagen GD e imprimir una imagen.

<?php
header ('Content-Type: image/png');
$im = @imagecreatetruecolor(120, 20)
      or die('No se puede Iniciar el nuevo flujo a la imagen GD');
$color_texto = imagecolorallocate($im, 233, 14, 91);
imagestring($im, 1, 5, 5,  'A Simple Text String', $color_texto);
imagepng($im);
imagedestroy($im);
?>
