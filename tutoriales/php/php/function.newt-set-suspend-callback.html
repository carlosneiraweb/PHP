newt_set_suspend_callback

(PECL newt >= 0.1)

newt_set_suspend_callback — Establece una función de llamada de retorno que se invoca cuando el usuario presiona la tecla de suspensión

Descripción

void newt_set_suspend_callback ( callable $function , mixed $data )

Establece una función de llamada de retorno que se invoca cuando el usuario presiona la tecla de suspensión (normalmente ^Z). Si no está registrada una llamda de retorno de suspensión, la pulsación de la tecla de suspensión se ignora.

Parámetros

function

Una función de llamada de retorno, la cual acepta un argumento: data

data

Los datos que son pasados a la función de llamada de retorno.

Valores devueltos

No devuelve ningún valor.

Ver también

  • newt_suspend() - Indica a newt que devuelva el terminal a su estado inicial
  • newt_resume() - Reanudación usando la interfaz de newt después de llamar a newt_suspend
