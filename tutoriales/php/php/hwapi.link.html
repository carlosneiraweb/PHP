hw_api::link

(PHP 4, PHP 5 < 5.2.0, PECL hwapi SVN)

hw_api::link — Crea un link a un objeto

Descripción

bool hw_api::link ( array $parameter )

Crea un link a un objeto. Acceder a este link es como acceder al objeto al que el link apunta.

Parámetros

parameter

El parámetro es un array que contiene los elementos obligatorios 'objectIdentifier' y 'destinationParentIdentifier'. 'destinationParentIdentifier' es la 'collection' de destino.

Valores devueltos

La función devuelve TRUE si ha acabado con éxito o un objeto de error en caso contrario.
