mb_get_info

(PHP 4 >= 4.2.0, PHP 5, PHP 7)

mb_get_info — Obtiene la configuración interna de mbstring

Descripción

mixed mb_get_info ([ string $type = "all" ] )

mb_get_info() devuelve los parámetros de configuración interna de mbstring.

Parámetros

type

Si no se estecifica type o si se especifica como "all", será devuelto un array con los elementos "internal_encoding", "http_output", "http_input", "func_overload", "mail_charset", "mail_header_encoding", "mail_body_encoding".

Si type se especifica como "http_output", "http_input", "internal_encoding", "func_overload", será devuelto el parámetro de configuración especificado.

Valores devueltos

Un array con el tipo de información si type no se especifica, o de otro modo, el type especificado.

Historial de cambios

Versión Descripción
5.1.3 Están disponibles las entradas "mail_charset", "mail_header_encoding", y "mail_body_encoding".
5.3.0 Está disponible la entrada "http_output_conv_mimetypes".
