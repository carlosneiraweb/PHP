ingres_pconnect

(PHP 4 >= 4.0.2, PHP 5 <= 5.0.5, PECL ingres >= 1.0.0)

ingres_pconnect — Abre una conexión persistente a una base de datos de Ingres

Descripción

resource ingres_pconnect ([ string $database [, string $username [, string $password [, array $options ]]]] )

Abre una conexión persistente a una base de datos de Ingres.

Existen solamente dos diferencias esta función y ingres_connect(): La primera es que, al conectarse, esta función inicialmente intentará encontrar un enlace (persistente) que ya esté abierto con los mismos parámetros. Si se encuentra uno, será devuelto un identificador para dicho enlace en lugar de abrir una nueva conexión. La segunda diferencia es que la conexión al servidor Ingres no se cerrará al finalizar la ejecución del script. En su lugar, el enlace permanecerá abierto para usos futuros (ingres_close() no cerrará los enlaces establecidos por ingres_pconnect()). Por esto, a este tipos de enlaces de les llama "persistentes".

Parámetros

database

El nombre de la base de datos. Debe serguir la sintaxis:

[vnode::]dbname[/svr_class]

username

El nombre de usuario de Ingres

password

La contraseña asociada con username

options

Véase ingres_connect() para una lista de las opciones que se pueden pasar

Valores devueltos

Devuelve un recurso de enlace a Ingres en caso de éxito o FALSE en caso de error

  • ingres_close() - Cierra una conexión a una base de datos de Ingres
