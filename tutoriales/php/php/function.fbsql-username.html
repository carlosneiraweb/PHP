fbsql_username

(PHP 4 >= 4.0.6, PHP 5, PHP 7)

fbsql_username — Obtiene o asigna el nombre de usuario de la conexión

Descripción

string fbsql_username ( resource $link_identifier [, string $username ] )

Obtiene o asigna el nombre de usuario usado en la conexión.

Parámetros

link_identifier

Un identificador de enlace FrontBase devuelto por fbsql_connect() o fbsql_pconnect().

Es opcional y si no se especifica, la función intentará encontrar un enlace abierto al servidor FrontBase y de no encontrarse ningún enlace abierto intentará crear uno como si fbsql_connect() fuese invocada sin parámetros.

username

Si se recibe, el nuevo nombre de usuario que se asignará.

Valores devueltos

Devuelve el nombre de usuario usado actualmente en la conexión, como cadena.

Ver también

  • fbsql_password() - Obtiene o asigna el password usado en una conexión
  • fbsql_hostname() - Obtiene o establece el nombre host de una conexión
