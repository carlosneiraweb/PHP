rmdir

(PHP 4, PHP 5, PHP 7)

rmdir — Elimina un directorio

Descripción

bool rmdir ( string $dirname [, resource $context ] )

Intenta eliminar el directorio nombrado por dirname. El directorio debe estar vacío, y los permisos relevantes deben permitirlo. Un error de nivel E_WARNING será generado si se produce un error.

Parámetros

dirname

Ruta del directorio

context

Nota: Soporte para context fue añadido en PHP 5.0.0. Para una descripción de contexts, refiérase a Flujos.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.

Historial de cambios

Versión Descripción
5.0.0 Desde PHP 5.0.0 rmdir() también se puede usar con algunas envolturas URL. Consulte Protocolos y Envolturas soportados para un listado de las envolturas que soporta rmdir().

Ejemplos

Ejemplo #1 Ejemplo de rmdir()

<?php
if (!is_dir('ejemplos')) {
    mkdir('ejemplos');
}

rmdir('ejemplos');
?>

Notas

Nota: Cuando el modo seguro está habilitado, PHP comprueba si el directorio en el cual el script está operando tiene el mismo UID (propietario) que el script que está siendo ejecutado.
