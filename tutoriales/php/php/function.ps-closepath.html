ps_closepath

(PECL ps >= 1.1.0)

ps_closepath — Cerrar un trazado

Descripción

bool ps_closepath ( resource $psdoc )

Conecta el último punto con el primer punto de un trazado. El trazado resultante puede usarse para contornear, rellenar, recortar, etc...

Parámetros

psdoc

El identificador de recursos del fichero postscript, como el devuelto por la función ps_new().

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.

  • ps_clip() - Realizar un recorte utilizando el trazado actual
  • ps_closepath_stroke() - Cerrar y contornear un trazado
