Configuración en tiempo de ejecución

El comportamiento de estas funciones se ve afectado por la configuración de php.ini.

Opciones de configuración de mongodb
Nombre Por defecto Cambiable Historial de cambios Comentarios
mongodb.debug "" PHP_INI_ALL Solo disponible para el controlador de PHP, no para el de HHVM

He aquí una breve explicación de las directivas de configuración.

mongodb.debug string

Almacenar el registro de depuración de actividad completo en este directorio o flujo. Se puede proporcionar el valor especial "stderr" para escribir en STDERR.

Nota:

No disponible para HHVM
