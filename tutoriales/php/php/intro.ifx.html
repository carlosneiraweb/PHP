Introducción

El controlador Informix para Informix (IDS) 7.x, SE 7.x, Universal Server (IUS) 9.x y para IDS 2000 está implementado en "ifx.ec" y en "php_informix.h" en el directorio de extensiones de informix. El soporte para IDS 7.x está prácticamente completo, con soporte completo para columnas BYTE y TEXT. El soporte para IUS 9.x está parcialmente terminado: los nuevos tipos de datos están ahí, pero el soporte para SLOB y CLOB aún está en construcción.

Nota:

Esta extensión se ha movido al repositorio de » PECL y ya no se distribuye con PHP a partir de PHP 5.2.1.
