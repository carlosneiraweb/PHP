ifx_byteasvarchar

(PHP 4, PHP <=5.2.0)

ifx_byteasvarchar — Establece el modo predeterminado byte

Descripción

bool ifx_byteasvarchar ( int $mode )

Establece el modo byte predeterminado para todas las consultas de selección.

Parámetros

mode

Modo "0" devolverá blob id, y modo "1" devolverá un varchar con el contenido del texto.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.
