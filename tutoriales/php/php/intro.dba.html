Introducción

Estas funciones construyen los cimientos para acceder a las bases de datos estilo Berkeley DB.

Es una capa de abstracción general para varias bases de datos basadas en ficheros. Como tal, la funcionalidad está limitada a un subconjunto común de características admitidas por las bases de datos modernas como » Oracle Berkeley DB.
