SphinxClient::getLastWarning

(PECL sphinx >= 0.1.0)

SphinxClient::getLastWarning — Obtener la última advertencia

Descripción

public string SphinxClient::getLastWarning ( void )

Devuelve el último mensaje de advertencia. Si no se generó ninguna advertencia durante la previa llamada API, es retornado un string vacío. Este método no restablece la advertencia, lo que con seguridad se puede llamar varias veces.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve el último mensaje de advertencia o un string vacío si no hay advertencias.
