bcompiler_write_footer

(PECL bcompiler >= 0.4)

bcompiler_write_footer — Escribe el carácter \x00 para indicar el Final de los datos compilados

Descripción

bool bcompiler_write_footer ( resource $filehandle )

Escribe el carácter \x00 para indicar el Final de los datos compilados.

Parámetros

filehandle

Un gestor de archivo como el devuelto por fopen().

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.

Ejemplos

Ejemplo #1 Ejemplo de bcompiler_write_footer()

<?php
$fh = fopen("/tmp/ejemplo","w");
bcompiler_write_header($fh);
bcompiler_write_class($fh,"DB");
bcompiler_write_class($fh,"DB_common");
bcompiler_write_footer($fh);
fclose($fh);

?>
