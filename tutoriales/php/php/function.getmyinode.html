getmyinode

(PHP 4, PHP 5, PHP 7)

getmyinode — Obtiene el inode del script actual

Descripción

int getmyinode ( void )

Obtiene el inode del script actual.

Valores devueltos

Devuelve el inodo del script actual como un entero, o FALSE en caso de error.
