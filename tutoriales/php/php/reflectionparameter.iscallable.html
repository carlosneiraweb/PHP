ReflectionParameter::isCallable

(PHP 5 >= 5.4.0, PHP 7)

ReflectionParameter::isCallable — Devuelve si el parámetro DEBE ser de tipo callable

Descripción

public bool ReflectionParameter::isCallable ( void )

Advertencia

Esta función no está documentada actualmente, solamente se encuentra disponible la lista de parámetros.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve TRUE si el parámetro es callable, FALSE si no, o NULL en caso de error.
