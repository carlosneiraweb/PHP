OCI-Lob::save

(PHP 5, PHP 7, PECL OCI8 >= 1.1.0)

OCI-Lob::save — Guarda datos en un objeto grande

Descripción

bool OCI-Lob::save ( string $data [, int $offset ] )

Guarda los datos dados por data en un objeto grande.

Parámetros

data

Los datos a guardar.

offset

Se puede usar para señalar el índice desde el comienzo del objeto grande.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.
