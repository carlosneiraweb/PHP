  • Funciones del SDO-DAS-Relacional
    • SDO_DAS_Relational::applyChanges — Aplicar los cambios hechos a un grafo de datos en la base de datos.
    • SDO_DAS_Relational::__construct — Crea una instancia de un Servicio de Acceso a Datos Relacional
    • SDO_DAS_Relational::createRootDataObject — Devuelve el objeto raíz especial en un grafo de datos en principio vacío. Usado para crear un grafo de datos desde cero.
    • SDO_DAS_Relational::executePreparedQuery — Ejecuta una consulta SQL pasada como una sentencia preparada, con una lista de valores para sustituir los parámetros de sustitución, y devuelve los resultados como un grafo de datos normalizado.
    • SDO_DAS_Relational::executeQuery — Ejecutar una consulta SQL dada en una base de datos relacional y devolver los resultados como un grafo de datos normalizado.
