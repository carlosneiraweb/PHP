SoapServer::addSoapHeader

(PHP 5 >= 5.0.1, PHP 7)

SoapServer::addSoapHeader — Añade un encabezado SOAP a la respuesta

Descripción

public void SoapServer::addSoapHeader ( SoapHeader $object )

Añade un encabezado SOAP a ser devuelto en la respuesta de la petición actual.

Parámetros

object

El encabezado a ser devuelto.

Valores devueltos

No devuelve ningún valor.
