QuickHashIntHash::get

(PECL quickhash >= Unknown)

QuickHashIntHash::get — Este método recupera un valor a partir del hash por su clave

Descripción

public int QuickHashIntHash::get ( int $key )

Este método recupera un valor a partir del hash por su clave.

Parámetros

key

La clave de la entrada para añadir.

Valores devueltos

El valor si la clave existe, o NULL si la clave no era parte del hash.

Ejemplos

Ejemplo #1 Ejemplo de QuickHashIntHash::get()

<?php
$hash = new QuickHashIntHash( 8 );
var_dump( $hash->get( 1 ) );

var_dump( $hash->add( 2 ) );
var_dump( $hash->get( 2 ) );

var_dump( $hash->add( 3, 5 ) );
var_dump( $hash->get( 3 ) );
?>

El resultado del ejemplo sería algo similar a:

bool(false)
bool(true)
int(1)
bool(true)
int(5)
