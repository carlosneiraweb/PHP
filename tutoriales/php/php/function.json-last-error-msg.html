json_last_error_msg

(PHP 5 >= 5.5.0, PHP 7)

json_last_error_msg — Devuelve el string con el error de la última llamada a json_encode() o a json_decode()

Descripción

string json_last_error_msg ( void )

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve el mensaje de error en caso de éxito, "No Error" si no ocurrió ningún error, o FALSE en caso de error.
