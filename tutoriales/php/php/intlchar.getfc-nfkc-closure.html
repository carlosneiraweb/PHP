IntlChar::getFC_NFKC_Closure

(PHP 7)

IntlChar::getFC_NFKC_Closure — Obtener la propiedad FC_NFKC_Closure de un punto de código

Descripción

public static string IntlChar::getFC_NFKC_Closure ( mixed $codepoint )

Obtiene la cadena de la propiedad FC_NFKC_Closure de un carácter.

Parámetros

codepoint

El valor de tipo integer del punto de código (p.ej. 0x2603 para U+2603 SNOWMAN), o el carácter codificado como un string UTF-8 (p.ej. "\u{2603}")

Valores devueltos

Devuelve la cadena de la propiedad FC_NFKC_Closure de codepoint, o una cadena vacía si no existe.

Ejemplos

Ejemplo #1 Probar diferentes puntos de código

<?php
var_dump(IntlChar::getFC_NFKC_Closure("\u{2121}"));
var_dump(IntlChar::getFC_NFKC_Closure("\u{1D2D}"));
?>

El resultado del ejemplo sería:

string(3) "tel"
string(2) "æ"
