ifx_update_blob

(PHP 4, PHP <=5.2.0)

ifx_update_blob — Actualizar el contenido de un objeto BLOB

Descripción

bool ifx_update_blob ( int $bid , string $content )

Actualiza el contenido de un objeto BLOB asociado con el identificador BLOB bid.

Parámetros

bid

Es el identificador del objeto BLOB.

content

La nueva información, como cadena.

Valores devueltos

Devuelve TRUE en caso de éxito, o FALSE en caso de error.
