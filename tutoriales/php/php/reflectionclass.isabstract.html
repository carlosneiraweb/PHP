ReflectionClass::isAbstract

(PHP 5, PHP 7)

ReflectionClass::isAbstract — Comprueba si la clase es abstracta

Descripción

public bool ReflectionClass::isAbstract ( void )

Comprueba si la clase es o no abstracta.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.

Ejemplos

Ejemplo #1 Ejemplo de ReflectionClass::isAbstract()

<?php
class          ClaseDePrueba { }
abstract class ClaseAbstractaDePrueba { }

$claseDePrueba  = new ReflectionClass('ClaseDePrueba');
$claseAbstracta = new ReflectionClass('ClaseAbstractaDePrueba');

var_dump($claseDePrueba->isAbstract());
var_dump($claseAbstracta->isAbstract());
?>

El resultado del ejemplo sería:

bool(false)
bool(true)
