bbcode_add_element

(PECL bbcode >= 0.9.0)

bbcode_add_element — Añade un elemento bbcode

Descripción

bool bbcode_add_element ( resource $bbcode_container , string $tag_name , array $tag_rules )

Añade una etiqueta al set de BBCode_Container usando tag_rules.

Parámetros

bbcode_container

Resource BBCode_Container, retornado por bbcode_create().

tag_name

La nueva etiqueta a añadir al set de BBCode_Container.

tag_rules

Un array asociativo conteniendo las reglas para el analizador; ver bbcode_create() para las claves disponibles.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.
