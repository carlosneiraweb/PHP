RecursiveIteratorIterator::getMaxDepth

(PHP 5 >= 5.1.0, PHP 7)

RecursiveIteratorIterator::getMaxDepth — Obtener el máximo de profundidad

Descripción

public mixed RecursiveIteratorIterator::getMaxDepth ( void )

Obtiene el máximo de profundidad permitido.

Advertencia

Esta función no está documentada actualmente, solamente se encuentra disponible la lista de parámetros.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

El máximo de profundidad aceptado, o FALSE si cualquier profundidad está permitida.
