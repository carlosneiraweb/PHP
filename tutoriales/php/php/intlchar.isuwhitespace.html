IntlChar::isUWhiteSpace

(PHP 7)

IntlChar::isUWhiteSpace — Comprobar si un punto de código tiene la propiedad de Unicode White_Space

Descripción

public static bool IntlChar::isUWhiteSpace ( mixed $codepoint )

Comprueba si un punto de código tiene la propiedad de Unicode White_Space.

Esto es lo mismo que IntlChar::hasBinaryProperty($codepoint, IntlChar::PROPERTY_WHITE_SPACE)

Nota:

Esto es diferente de IntlChar::isspace() y de IntlChar::isWhitespace().

Parámetros

codepoint

El valor de tipo integer del punto de código (p.ej. 0x2603 para U+2603 SNOWMAN), o el carácter codificado como un string UTF-8 (p.ej. "\u{2603}")

Valores devueltos

Devuelve TRUE si codepoint tiene la propiedad de Unicode White_Space, FALSE si no.

Ejemplos

Ejemplo #1 Probar diferentes puntos de código

<?php
var_dump(IntlChar::isUWhiteSpace("A"));
var_dump(IntlChar::isUWhiteSpace(" "));
var_dump(IntlChar::isUWhiteSpace("\n"));
var_dump(IntlChar::isUWhiteSpace("\t"));
var_dump(IntlChar::isUWhiteSpace("\u{00A0}"));
?>

El resultado del ejemplo sería:

bool(false)
bool(true)
bool(true)
bool(true)
bool(true)
