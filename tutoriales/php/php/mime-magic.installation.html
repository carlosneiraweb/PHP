Instalación

Advertencia

Las extensiones de Mimetype han sido eliminadas en PHP 5.3.0 a favor de Fileinfo.

Debe compilar PHP con el conmutador de configuración --with-mime-magic para obtener soporte de las funciones de mime-type. La extensión necesita una copia del fichero magic simplificado, distribuido con httpd de Apache.

Nota:

La opción de configuración ha sido cambiado de --enable-mime-magic a --with-mime-magic desde PHP 4.3.2

Nota:

Esta extensión no es capaz de manejar el fichero magic comletamente adornado que generalmente viene con las distribuciones estándar de Linux, y que se supone va a usarse con las versiones recientes del comando file.

Nota: Nota a los usuaricos de Win32

Para usar este módulo en un entorno Windows debe establecer la ruta al fichero incluido, magic.mime, en su php.ini.

Ejemplo #1 Establecer la ruta a magic.mime

mime_magic.magicfile = "$PHP_INSTALL_DIR\magic.mime"

Recuerde sustituir $PHP_INSTALL_DIR por la ruta real a PHP en el ejemplo de arriba. P.ej. c:\php
