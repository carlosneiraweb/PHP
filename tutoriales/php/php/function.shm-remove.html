shm_remove

(PHP 4, PHP 5, PHP 7)

shm_remove — Elimina la memoria compartida de sistemas Unix

Descripción

bool shm_remove ( resource $shm_identifier )

shm_remove() elimina el identificador de memoria compartida shm_identifier. Toda la información será destruida.

Parámetros

shm_identifier

El identificador de memoria compartida devuelto por shm_attach()

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.
