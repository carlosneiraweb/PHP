mcrypt_cfb

(PHP 4, PHP 5)

mcrypt_cfb — Encripta/decripta datos en modo CFB

Advertencia

Esta función está OBSOLETA en PHP 5.5.0, por lo tanto, será ELIMINADA en PHP 7.0.0.

Las alternativas a esta función son:

  • mcrypt_decrypt()
  • mcrypt_encrypt()

Descripción

string mcrypt_cfb ( int $cipher , string $key , string $data , int $mode , string $iv )
string mcrypt_cfb ( string $cipher , string $key , string $data , int $mode [, string $iv ] )

El primer prototipo es cuando se enlazó con libmcrypt 2.2.x, el segundo cuando se enlazó con libmcrypt 2.4.x o superiores. El parámetro mode debería ser o MCRYPT_ENCRYPT o MCRYPT_DECRYPT.
