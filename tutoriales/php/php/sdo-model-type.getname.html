SDO_Model_Type::getName — Obtener el nombre del tipo

Descripción

string SDO_Model_Type::getName ( void )
Advertencia

Esta función ha sido declarada EXPERIMENTAL. Su funcionamiento, nombre y la documentación que le acompaña puede cambiar sin previo aviso en futuras versiones de PHP. Utilícela bajo su propia responsabilidad.

Devuelve el nombre del tipo. La combinación del nombre del tipo y el URI del espacio de nombre se usa para identificar el tipo de manera única.

Parámetros

Ninguno.

Valores devueltos

Devuelve el nombre del tipo.
