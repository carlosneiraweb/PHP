RRDGraph::save

(PECL rrd >= 0.9.0)

RRDGraph::save — Guarda el resultado de la consulta en la imagen

Descripción

public array RRDGraph::save ( void )

Guarda el resultado de la consulta de base de datos RRD en la imagen definida por RRDGraph::__construct().

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Se devuelve un array con información acerca de la imagen generada, FALSE si se produce un error.
