svn_delete

(PECL svn >= 0.4.0)

svn_delete — Elimina ítems de una copia de trabajo o repositorio

Descripción

bool svn_delete ( string $path [, bool $force = false ] )

Elimina el fichero, directorio o enlace simbólico en el path del directorio de trabajo. El ítem será eliminado del repositorio la próxima vez que se use svn_commit() en la copia de trabajo.

Parámetros

path

Ruta o ítem a eliminar.

Nota: Las rutas relativas se resolverán como si el directorio de trabajo actual es el que contiene el binario de PHP. Para utilizar el directorio de trabajo que contiene el script a llamar, use realpath() o dirname(__FILE__).

force

En caso que sea TRUE, el fichero será eliminado incluso si se han hecho modificaciones en local. En caso contrario, el haber hecho modificaciones en local causará un error. El valor por omisión es FALSE

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.

Notas

Advertencia

Esta función ha sido declarada EXPERIMENTAL. Su funcionamiento, nombre y la documentación que le acompaña puede cambiar sin previo aviso en futuras versiones de PHP. Utilícela bajo su propia responsabilidad.
