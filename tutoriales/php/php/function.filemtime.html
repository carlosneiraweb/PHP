filemtime

(PHP 4, PHP 5, PHP 7)

filemtime — Obtiene el momento de la última modificación de un archivo

Descripción

int filemtime ( string $filename )

Esta función devuelve el momento de cuándo los bloques de información de un archivo fueron escritos, es decir, el momento de cuándo el contenido del archivo se modificó.

Parámetros

filename

Ruta del archivo.

Valores devueltos

Devuelve el momento de cuándo el archivo se modificó por última vez, o FALSE en caso de error. El momento es devuelto como tiempo Unix, lo cual es apropiado para la función date().

Ejemplos

Ejemplo #1 Ejemplo de filemtime()

<?php
// muestra, p.ej.  La última modificación de archivo.txt fue: December 29 2002 22:16:23.

$nombre_archivo = 'archivo.txt';
if (file_exists($nombre_archivo)) {
    echo "La última modificación de $nombre_archivo fue: " . date ("F d Y H:i:s.", filemtime($nombre_archivo));
}
?>

Errores/Excepciones

En caso de fallo, se emite un E_WARNING.

Notas

Nota:

Observe que el tiempo de resolución quizá varia según el sistema de ficheros.

Nota: Los resultados de esta función son guardados en caché. Para más información ver clearstatcache().

Sugerencia

Desde PHP 5.0.0, esta función también puede usarse con algunos URL wrappers. Consultar Protocolos y Envolturas soportados para deter4minar qué envolturas soportan la familia de funcionalidad de stat().
