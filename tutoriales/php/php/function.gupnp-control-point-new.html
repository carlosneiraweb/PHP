gupnp_control_point_new

(PECL gupnp >= 0.1.0)

gupnp_control_point_new — Crea un nuevo punto de control

Descripción

resource gupnp_control_point_new ( resource $context , string $target )

Crea un nuevo punto de control con un objetivo especificado.

Parámetros

context

Un identificador de contexto, devuelto por gupnp_context_new().

target

La búsqueda del objetivo. target debería ser un servicio o el nombre de un dispositivo, así como urn:schemas-upnp-org:service:WANIPConnection:1 o urn:schemas-upnp-org:device:MediaRenderer:1.

Valores devueltos

Un identificador de punto de control.
