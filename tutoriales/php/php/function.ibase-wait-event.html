ibase_wait_event

(PHP 5, PHP 7)

ibase_wait_event — Espera a que la base de datos emita un determinado evento

Descripción

string ibase_wait_event ( string $event_name1 [, string $event_name2 [, string $... ]] )
string ibase_wait_event ( resource $connection , string $event_name1 [, string $event_name2 [, string $... ]] )

Esta función suspende la ejecución del script hasta que la base de datos emita uno de los eventos especificados. Devuelve el nombre del evento emitido. Esta función acepta hasta 15 eventos como parámetros.

Parámetros

event_name1

Nombre de evento.

event_name2

...

Valores devueltos

Devuelve el nombre del evento que se emitió.

Ver también

  • ibase_set_event_handler() - Registra una función manejadora de un determinado evento
  • ibase_free_event_handler() - Cancela un manejador de eventos ya registrado
