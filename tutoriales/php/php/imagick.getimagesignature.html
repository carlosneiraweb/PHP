Imagick::getImageSignature

(PECL imagick 2.0.0)

Imagick::getImageSignature — Genera un resumen de un mensaje SHA-256

Descripción

string Imagick::getImageSignature ( void )

Genera un resumen de un mensaje SHA-256 para el flujo de píxeles de la imagen.

Valores devueltos

Devuelve un string que contiene el hash SHA-256 de el fichero.

Errores/Excepciones

Lanza ImagickException en caso de error.
