SDO_Model_Type::getBaseType

(^)

SDO_Model_Type::getBaseType — Obtener el tipo base de este tipo

Descripción

SDO_Model_Type SDO_Model_Type::getBaseType ( void )
Advertencia

Esta función ha sido declarada EXPERIMENTAL. Su funcionamiento, nombre y la documentación que le acompaña puede cambiar sin previo aviso en futuras versiones de PHP. Utilícela bajo su propia responsabilidad.

Obtiene el tipo base de este tipo. Devuelve el SDO_Model_Type del tipo base si este tipo hereda de otro, si no, devuelve NULL. Un ejemplo de cuándo se producen lo tipos base es cuando un tipo definido en el esquema XML hereda de otro tipo usando 

<extension base="...">
.

Parámetros

Ninguno.

Valores devueltos

Devuelve el SDO_Model_Type del tipo base si este tipo hereda de otro, si no, devuelve NULL.
