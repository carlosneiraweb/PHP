ResourceBundle::get

resourcebundle_get

(PHP >= 5.3.2, PECL intl >= 2.0.0)

ResourceBundle::get -- resourcebundle_get — Obtener datos de un paquete

Descripción

Estilo orientado a objetos

public mixed ResourceBundle::get ( string|int $index )

Estilo por procedimientos

mixed resourcebundle_get ( ResourceBundle $r , string|int $index )

Obtiene los datos de un paquete mediante claves de índice o cadena.

Parámetros

r

Un objeto de la clase ResourceBundle.

index

Índice del dato, debe ser de tipo string o integer.

Valores devueltos

Devuelve los datos ubicados en el índice o NULL en caso de error. Las cadenas, enteros y cadenas de datos binarios son devueltas como su tipo correspondiente en PHP, los array de enteros son devueltos como arrays de PHP. Los tipos complejos son devueltos como objetos de la clase ResourceBundle.

Ejemplos

Ejemplo #1 Ejemplo de resourcebundle_get()

<?php
$r = resourcebundle_create( 'es', "/usr/share/data/myapp");
echo resourcebundle_get($r, 'somestring');
?>

Ejemplo #2 Ejemplo orientado a objetos

<?php
$r = new ResourceBundle( 'es', "/usr/share/data/myapp");
echo $r->get('somestring');
?>

El resultado del ejemplo sería:

?Hola, mundo!
