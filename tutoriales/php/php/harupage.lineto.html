HaruPage::lineTo

(PECL haru >= 0.0.1)

HaruPage::lineTo — Dibuja una línea del punto actual al especificado

Descripción

bool HaruPage::lineTo ( float $x , float $y )

Dibuja una línea del punto actual al especificado.

Parámetros

x

y

Valores devueltos

Devuelve TRUE en éxito.

Errores/Excepciones

Lanza una HaruException cuando se produce un error.
