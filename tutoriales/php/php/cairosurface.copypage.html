CairoSurface::copyPage

cairo_copy_page

(PECL cairo >= 0.1.0)

CairoSurface::copyPage -- cairo_copy_page — El propósito de copyPage

Descripción

Estilo orientado a objetos (método):

public void CairoSurface::copyPage ( void )

Estilo por procedimientos:

void cairo_copy_page ( CairoContext $context )

Emite la página actual para entornos que soportan múltiples página, pero no la borra, así que el contenido de la página actual será mantenido para la siguiente página. Use CairoSurface::showPage() si quiere obtener una página vacía después de la emisión.

Parámetros

context
