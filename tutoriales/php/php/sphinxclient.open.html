SphinxClient::open

(PECL sphinx >= 1.0.3)

SphinxClient::open — Abre una conexión persistente con el servidor

Descripción

public bool SphinxClient::open ( void )

Abre una conexión persistente al servidor.

Parámetros

Esta función no tiene parámetros.

Historial de cambios

PECL/sphinx Versión Descripción
1.0.3 Añadido SphinxClient::open(), disponible sólo si se compila con libsphinxclient >= 0.9.9.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.
