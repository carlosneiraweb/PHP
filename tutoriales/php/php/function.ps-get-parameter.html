ps_get_parameter

(PECL ps >= 1.1.0)

ps_get_parameter — Obtener ciertos parámetros

Descripción

string ps_get_parameter ( resource $psdoc , string $name [, float $modifier ] )

Obtiene varios parámetros que se establecieron directamente con la función ps_set_parameter() o indirectamente por alguna de las otras funciones. Los parámetros son por definición valores de tipo string. Esta función no se puede usar para recuperar recursos que también fueron establecidos con la función ps_set_parameter().

El parámetro name puede tener los siguientes valores:

fontname

El nombre de la fuente activa o la fuente cuyo identificador es pasado al parámetro modifier.

fontencoding

La codificación de la fuente activa.

dottedversion

La versión de la biblioteca pslib subyacente con el formato <mayor>.<menor>.<submenor>

scope

El ámbito de dibujo actual. Puede ser un objeto, un documento, null, una página, un patrón, un trazado, una plantilla, un prólogo, una fuente, un glifo.

ligaturedisolvechar

El carácter que disuelve una ligadura. Si se está utilizando una fuente que contiene la ligadura 'ff' y '|' es el carácter para disolver la ligadura, 'f|f' resultará en dos 'f' en lugar de la ligadura 'ff'.

imageencoding

La codificación utilizada para codificar imágenes. Puede ser hex o 85. La codificación hex usa dos bytes en el fichero postscript por cada byte de la imagen. 85 representa la codificación Ascii85.

linenumbermode

Establecer a paragraph si las líneas están numeradas dentro de un párrafo, o box si están numeradas dentro de una caja circundante.

linebreak

Se usa solamente si el texto se imprime con la función ps_show_boxed(). Si se establece a TRUE un retorno de carro añadirá un salto de línea.

parbreak

Se usa solamente si el texto se imprime con la función ps_show_boxed(). Si se establece a TRUE un retorno de carro iniciará un nuevo párrafo.

hyphenation

Se usa solamente si el texto se imprime con la función ps_show_boxed(). Si se establece a TRUE el párrafo será dividido mediante guiones si está establecido y existe un diccionario de partición silábica.

hyphendict

Nombre de fichero del diccionario usado como patrón de partición silábica.

Parámetros

psdoc

El identificador de recursos del fichero postscript, como el devuelto por la función ps_new().

name

El nombre del parámetro.

modifier

Un identificador necesario si se solicita un parámetro de un recurso, p.ej. el tamaño de una imagen. En tal caso se pasa el ID del recurso.

Valores devueltos

Devuelve el valor del parámetro o FALSE en caso de error.
