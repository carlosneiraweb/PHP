Instalación

Para tener disponibilidad de estas funciones se debe compilar PHP con soporte para dbx usando la opción --enable-dbx y todas las opciones para las bases de datos que serán utilizadas, p.ej., para MySQL se debe especificar también --with-mysql=[DIR] . Para que otras bases de datos soportadas funcionen con el módulo, consulte sus documentaciones específicas.
