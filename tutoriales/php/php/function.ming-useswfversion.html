ming_useswfversion

(PHP 4 >= 4.2.0, PHP 5 <= 5.3.0, PECL ming SVN)

ming_useswfversion — Establece la versión de SWF

Descripción

void ming_useswfversion ( int $version )

Establece la versión de SWF a usar en la película. Afecta al comportamiento de Action Script.

Parámetros

version

Versión de SWF a usar.

Valores devueltos

No devuelve ningún valor.

Ejemplos

Ejemplo #1 Ejemplo de ming_useswfversion()

<?php

$movie = new SWFMovie();
ming_useswfversion(4); // Flash 4

?>
