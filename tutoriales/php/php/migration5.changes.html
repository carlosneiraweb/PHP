¿Qué ha cambiado en PHP 5.0.x?

Tanto PHP 5, como Zend Engine 2, han mejorado mucho el rendimiento y las capacidades de PHP, pero se ha tenido especial cuidado en hacer que el código existente fuera tan compatibles como fuera posible. De modo que migrar un código de PHP 4 a 5 debería de ser muy fácil. La mayoria de código PHP 4 existente debería estar listo para ejecutarse sin cambios, pero aún así deben conocerse las pequeñas diferencias y tomar las precauciones necesarias para comprobar el código antes de cambiar de versión en un entorno en producción.
