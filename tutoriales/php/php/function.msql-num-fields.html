msql_num_fields

(PHP 4, PHP 5, PHP 7)

msql_num_fields — Obtiene el número de campos en el resultado

Descripción

int msql_num_fields ( resource $result )

msql_num_fields() devuelve el número de campos en el conjunto de resultados.

Parámetros

result

El resultado resource que está siendo evaluado. Este resultado proviene de una llamada a msql_query().

Valores devueltos

Devuelve el número de campos en el conjunto de resultados.
