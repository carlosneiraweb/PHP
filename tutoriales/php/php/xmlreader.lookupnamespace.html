XMLReader::lookupNamespace

(PHP 5 >= 5.1.0, PHP 7)

XMLReader::lookupNamespace — Busca el namespace para un prefijo

Descripción

public bool XMLReader::lookupNamespace ( string $prefix )

Busca el ámbito del namespace para un prefijo dado.

Parámetros

prefix

La cadena conteniendo el prefijo.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.
