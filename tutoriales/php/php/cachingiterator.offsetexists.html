CachingIterator::offsetExists

(PHP 5 >= 5.2.0, PHP 7)

CachingIterator::offsetExists — Comprobar existencia de un índice

Descripción

public void CachingIterator::offsetExists ( string $index )
Advertencia

Esta función no está documentada actualmente, solamente se encuentra disponible la lista de parámetros.

Parámetros

index

El índice a ser comprobado.

Valores devueltos

Devuelve TRUE si la entrada a la que hace referencia el índice existe, en caso contrario FALSE.
