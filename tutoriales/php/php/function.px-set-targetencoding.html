px_set_targetencoding

(PECL paradox >= 1.0.0)

px_set_targetencoding — Establecer la codificación de los campos de caracteres (obsoleto)

Descripción

bool px_set_targetencoding ( resource $pxdoc , string $encoding )

Establece la codificación para los datos recuperados desde un campo de caracteres. Todos los campos de caracteres serán recodificados a la codificación establecida por esta función. Si la codificación no se ha establecido, los datos de carácter serán devueltos en el sistema de codificación DOS especificado en el archivo de base de datos. El parámetro encoding puede ser cualquier identificador de cadena conocido a iconv o recodificar. En los sistemas Unix, ejecutar iconv -l para obtener una lista de las codificaciones disponibles.

Esta función es obsoleta y debe sustituirse por llamadas a la función px_set_parameter().

Véase también la función px_get_info() para determinar el sistema de codificación DOS almacenado en el archivo de base de datos.

Parámetros

pxdoc

Identificador de recursos de la base de datos paradox devuelto por la función px_new().

encoding

La codificación de salida. Los datos que se están leyendo desde los campos de caracteres se recodifica en la codificación de destino.

Valores devueltos

Devuelve FALSE si la codificación no se puede establecer, por ejemplo, la codificación es desconocida, o pxlib no aprueba la recodificación de todos. En el segundo caso una advertencia será emitida.
