session_save_path

(PHP 4, PHP 5, PHP 7)

session_save_path — Obtener y/o establecer la ruta de almacenamiento de la sesión actual

Descripción

string session_save_path ([ string $path ] )

session_save_path() devuelve la ruta del directorio actual usado para guardar la información de sesión.

Parámetros

path

La ruta de la información de sesión. Si se especifica, la ruta de la información guardada se cambiará. session_save_path() necesita ser llamada antes de session_start() para ese propósito.

Nota:

En algunos sistemas operativos, se puede querer especificar una ruta en un sistema de archivos que gestione muchos archivos pequeños de manera eficaz. Por ejemplo, en Linux, reiserfs puede proporcionar un mejor rendimiento que ext2fs.

Valores devueltos

Devuelve la ruta del directorio actual usado para el almacenamiento de información.
