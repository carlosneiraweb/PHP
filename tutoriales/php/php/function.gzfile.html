gzfile

(PHP 4, PHP 5, PHP 7)

gzfile — Lee un archivo gz completo en una matriz

Descripción

array gzfile ( string $filename [, int $use_include_path = 0 ] )

Está función es identica a readgzfile(), excepto que retorna el archivo en una matriz.

Parámetros

filename

El nombre del archivo.

use_include_path

Se puede asignar este parámetro opcional en 1, si se desea buscar también el archivo en la ruta include_path.

Valores devueltos

Una matriz que contiene el archivo, una línea por celda.

Ejemplos

Ejemplo #1 Ejemplo de gzfile()

<?php
$lines = gzfile('somefile.gz');
foreach ($lines as $line) {
    echo $line;
}
?>
