SphinxClient::close

(PECL sphinx >= 1.0.3)

SphinxClient::close — Cierra conexión persistente previamente abierta

Descripción

public bool SphinxClient::close ( void )

Cierra conexión persistente previamente abierta.

Parámetros

Esta función no tiene parámetros.

Historial de cambios

PECL/sphinx Versión Descripción
1.0.3 Añadido SphinxClient::close(), disponible sólo si se compila con libsphinxclient >= 0.9.9.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.
