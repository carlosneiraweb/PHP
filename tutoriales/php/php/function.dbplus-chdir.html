dbplus_chdir

(PHP 4 <= 4.1.0, PECL dbplus >= 0.9)

dbplus_chdir — Obtener/Estabecer el directorio virtual actual de la base de datos

Descripción

string dbplus_chdir ([ string $newdir ] )

Cambia el directorio virtual actual donde se buscarán los archivos de relaciones mediante dbplus_open().

Parámetros

newdir

El nuevo directorio para los archivos de relaciones. Se puede omitir este parámetro para preguntar por el directorio de trabajo actual.

Valores devueltos

Devuelve la ruta absoluta del directorio actual.

Notas

Advertencia

Esta función ha sido declarada EXPERIMENTAL. Su funcionamiento, nombre y la documentación que le acompaña puede cambiar sin previo aviso en futuras versiones de PHP. Utilícela bajo su propia responsabilidad.
