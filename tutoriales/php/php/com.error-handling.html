Errores y manejo de errores

Excepciones (PHP 5)

Esta extensión lanzará instancias de la clase com_exception siempre que haya un error potencialmente fatal notificado por COM. Todas las excepciones de COM tienen una propiedad code bien definida que se corresponde con el valor devuelto HRESULT de diferentes operaciones COM. Puede usar este código para tomar decisiones programáticas de cómo manejar la excepción.
