SAMConnection::isConnected

(PECL sam >= 0.1.0)

SAMConnection::isConnected — Consulta si se ha establecido conexión a un Servidor de Mensajería

Descripción

bool SAMConnection::isConnected ( void )

La llamada al método "isConnected" en un objeto Connection verificará si el script PHP está conectado a un servidor de mensajería. No se podrá enviar ni recibir mensajes a menos que se haya establecido una conexión con un Servidor de Mensajería.

Valores devueltos

Este método devuelve TRUE si el objeto SAMConnection se ha conectado con éxito a un Servidor de Mensajería o FALSE en caso contrario.

Ejemplos

Ejemplo #1 Comprueba si se ha establecido una conexión a un servidor de mensajería

<?php

if ($conn->isConnected()) {
  echo 'Connection is active.';
} else {
  echo 'No active connection!';
}

?>
