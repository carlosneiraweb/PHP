fbsql_change_user

(PHP 4 >= 4.0.6, PHP 5, PHP 7)

fbsql_change_user — Cambia el usuario logueado en la conexión activa

Descripción

bool fbsql_change_user ( string $user , string $password [, string $database [, resource $link_identifier ]] )

fbsql_change_user() cambia el usuario logueado en la conexión especificada. Si falla la autorización del nuevo usuario/password, se mantiene activa la conexión con el usuario actual.

Parámetros

user

El nuevo nombre de usuario.

password

El nuevo password.

database

Si se especifica, se aplicará como base de datos por defecto una vez el usuario se haya cambiado.

link_identifier

Un identificador de enlace FrontBase devuelto por fbsql_connect() o fbsql_pconnect().

Es opcional y si no se especifica, la función intentará encontrar un enlace abierto al servidor FrontBase y de no encontrarse ningún enlace abierto intentará crear uno como si fbsql_connect() fuese invocada sin parámetros.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.
