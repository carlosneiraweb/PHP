sybase_data_seek

(PHP 4, PHP 5, PHP 7)

sybase_data_seek — Desplaza el apuntador de filas interno

Descripción

bool sybase_data_seek ( resource $result_identifier , int $row_number )

sybase_data_seek() desplaza el apuntador de filas interno del resultado Sybase asociado al identificador result_identifier hasta la línea row_number. La próxima llamada a la función sybase_fetch_row() devolverá esa fila.

Parámetros

result_identifier

row_number

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.

Ver también

  • sybase_fetch_row() - Devuelve una fila Sybase como un array indexado numéricamente
