md5_file

(PHP 4 >= 4.2.0, PHP 5, PHP 7)

md5_file — Calcula el resumen criptográfico md5 de un archivo dado

Descripción

string md5_file ( string $filename [, bool $raw_output = false ] )

Calcula el resumen criptográfico MD5 del archivo especificado por el parámetro filename usando el » Algoritmo de resumen de mensajes MD5 de RSA Data Security, Inc., y devuelve ese resumen. El resumen criptográfico es un número hexadecimal de 32 caracteres.

Parámetros

nombre_archivo

El nombre de archivo

raw_output

Cuando es TRUE, devuelve el resumen en un formato binario primitivo con una longitud de 16.

Valores devueltos

Devuelve una cadena en caso de éxito, FALSE de lo contrario.

Historial de cambios

Versión Descripción
5.1.0 Se modificó la función para usar la API de secuencias. Esto quiere decir que se puede usar con envolturas, como md5_file('http://example.com/..')

Ejemplos

Ejemplo #1 Ejemplo de uso de md5_file()

<?php
$file = 'php-5.3.0alpha2-Win32-VC9-x64.zip';

echo 'MD5 file hash of ' . $file . ': ' . md5_file($file);
?>
