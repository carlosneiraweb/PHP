dbplus_rquery

(PHP 4 <= 4.1.0, PECL dbplus >= 0.9)

dbplus_rquery — Realizar una consulta AQL local (en bruto)

Descripción

resource dbplus_rquery ( string $query [, string $dbpath ] )

dbplus_rquery() realiza una consulta AQL local (en bruto) usando un intérprete de AQL embebido dentro de la biblioteca cliente de db++. dbplus_rquery() es más rápitda que dbplus_aql() pero sólo trabajará sobre información local.

Parámetros

query

dbpath

Valores devueltos

Notas

Advertencia

Esta función ha sido declarada EXPERIMENTAL. Su funcionamiento, nombre y la documentación que le acompaña puede cambiar sin previo aviso en futuras versiones de PHP. Utilícela bajo su propia responsabilidad.
