Judy::next

(PECL judy >= 0.1.1)

Judy::next — Busca por el índice siguiente en el array Judy

Descripción

public mixed Judy::next ( mixed $index )

Búsqueda (global) para el índice siguiente presente que es mayor que el índice pasado.

Parámetros

index

El índice puede ser un integer o un string que corresponde al índice donde empieza la búsqueda.

Valores devueltos

Devuelve el índice que correspondiente al array.
