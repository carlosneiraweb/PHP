streamWrapper::mkdir

(PHP 5, PHP 7)

streamWrapper::mkdir — Crear un directorio

Descripción

public bool streamWrapper::mkdir ( string $path , int $mode , int $options )

Este método es llamado en respuesta a mkdir().

Nota:

Para que el mensaje de error apropiado sea devuelto, este método no debería ser definido si la envoltura no soporta la creación de directorios.

Parámetros

path

El directorio que debería ser creado.

mode

El valor pasdo a mkdir().

options

Una máscara a nivel de bits de valores, como STREAM_MKDIR_RECURSIVE.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.

Errores/Excepciones

Emite un E_WARNING si la llamada a este método falla (por ej., por no estar implementado).

Notas

Nota:

La propiedad streamWrapper::$context es actualizada si se pasa un contexto válido en la función invocante.
