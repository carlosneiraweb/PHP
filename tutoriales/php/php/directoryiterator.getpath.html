DirectoryIterator::getPath

(PHP 5, PHP 7)

DirectoryIterator::getPath — Obtiene la ruta del elemento iterador actual sin nombre de fichero

Descripción

public string DirectoryIterator::getPath ( void )

Obtiene la ruta del elemento actual DirectoryIterator.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve la ruta de el fichero, omitiendo el nombre del fichero y los slash.

Ejemplos

Ejemplo #1 Ejemplo de DirectoryIterator::getPath()

<?php
$iterator = new DirectoryIterator(dirname(__FILE__));
echo $iterator->getPath();
?>

El resultado del ejemplo sería algo similar a:

/home/examples/public_html
