udm_free_res

(PHP 4 >= 4.0.5, PHP 5 <= 5.0.5, PECL mnogosearch >= 1.0.0)

udm_free_res — Libera result mnoGoSearch

Descripción

bool udm_free_res ( resource $res )

Liberación de memoria asignada para obtener resultados.

Parámetros

res

Un enlace a un identificador de resultado, recibido después de llamar a udm_find().

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.
