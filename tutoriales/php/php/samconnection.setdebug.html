SAMConnection::setDebug

(PECL sam >= 1.1.0)

SAMConnection::setDebug — Activa o desactiva la información adicional de depuración.

Descripción

El método "setdebug" se utiliza para activar o desactivar la información adicional de depuración. La framework SAM framework proveerá las entradas de método/función y las salidas de datos de rastreo, más información adicional. La implementación específica del protocolo también proveerá información de salida adicional.

void SAMConnection::setDebug ( bool $switch )

Parámetros

switch

Si este parámetro se fija a TRUE, se generará información adicional de depuración. Si el valor se fija a FALSE, la salida de información adicional se interrumpirá.

Valores devueltos

No devuelve ningún valor.

Ejemplos

Ejemplo #1 Activar la salida de información de depuración

<?php
$conn->setdebug(TRUE);
?>

Ejemplo #2 Desactivar la salida de información de depuración

<?php
$conn->setdebug(FALSE);
?>
