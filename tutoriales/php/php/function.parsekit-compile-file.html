parsekit_compile_file

(PECL parsekit >= 0.2.0)

parsekit_compile_file — Compilar un fichero de PHP y devolver el array op resultante

Descripción

array parsekit_compile_file ( string $filename [, array &$errors [, int $options = PARSEKIT_QUIET ]] )
Advertencia

Esta función ha sido declarada EXPERIMENTAL. Su funcionamiento, nombre y la documentación que le acompaña puede cambiar sin previo aviso en futuras versiones de PHP. Utilícela bajo su propia responsabilidad.

Parámetros

filename

Una cadena que contiene el nombre del fichero a compilar. Similar al argumento pasado a include.

errors

Un hash 2D de errores (incluyendo errores fatales) encontrados durante la compilación. Devuelto por referencia.

options

O PARSEKIT_QUIET o PARSEKIT_SIMPLE. Para producir varios grados de verbosidad en la salida devuelta.

Valores devueltos

Devuelve una estructura de array multicapa compleja como está detallado abajo.

Ejemplos

Ejemplo #1 Ejemplo de parsekit_compile_file()

<?php
var_dump(parsekit_compile_file('hello_world.php', $errors, PARSEKIT_SIMPLE));
?>

El resultado del ejemplo sería:

array(5) {
  [0]=>
  string(37) "ZEND_ECHO UNUSED 'Hello World' UNUSED"
  [1]=>
  string(30) "ZEND_RETURN UNUSED NULL UNUSED"
  [2]=>
  string(42) "ZEND_HANDLE_EXCEPTION UNUSED UNUSED UNUSED"
  ["function_table"]=>
  NULL
  ["class_table"]=>
  NULL
}
