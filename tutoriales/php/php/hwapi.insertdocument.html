hw_api::insertdocument

(PHP 4, PHP 5 < 5.2.0, PECL hwapi SVN)

hw_api::insertdocument — Inserta un objeto nuevo de tipo 'document'

Descripción

hw_api_object hw_api::insertdocument ( array $parameter )

Esta función es una abreviatura para hwapi_insert(). Inserta un objeto con contenido y configura algunos de los atributos obligatorios para un objeto de este tipo.

Parámetros

parameter

El parámetro es un array que contiene los elementos obligatorios 'object', 'parentIdentifier' y 'content' y los elementos opcionales 'mode', 'parameter' y 'attributeSelector'.

Ver hwapi_insert() para conocer el significado de cada elemento.
