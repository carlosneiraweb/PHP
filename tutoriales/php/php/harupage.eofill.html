HaruPage::eofill

(PECL haru >= 0.0.1)

HaruPage::eofill — Relleno actual de la ruta usadando la regla even-odd

Descripción

bool HaruPage::eofill ( void )

Rellenar la ruta actual usadando la regla even-odd.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve TRUE en éxito.

Errores/Excepciones

Lanza una HaruException cuando se produce un error.
