IntlChar::isprint

(PHP 7)

IntlChar::isprint — Comprobar si un punto de código es un carácter imprimible

Descripción

public static bool IntlChar::isprint ( mixed $codepoint )

Determina si el punto de código especificado es un carácter imprimible.

TRUE para categorías generales distintas de "C" (controles).

Parámetros

codepoint

El valor de tipo integer del punto de código (p.ej. 0x2603 para U+2603 SNOWMAN), o el carácter codificado como un string UTF-8 (p.ej. "\u{2603}")

Valores devueltos

Devuelve TRUE si codepoint es un carácter imprimible, FALSE si no.

Ejemplos

Ejemplo #1 Probar diferentes puntos de código

<?php
var_dump(IntlChar::isprint("A"));
var_dump(IntlChar::isprint(" "));
var_dump(IntlChar::isprint("\n"));
var_dump(IntlChar::isprint("\u{200e}"));
?>

El resultado del ejemplo sería:

bool(true)
bool(true)
bool(false)
bool(false)
