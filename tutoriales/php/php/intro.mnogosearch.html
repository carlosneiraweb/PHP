Introducción

Estas funciones permiten acceder al motor de búsqueda libre de mnoGoSearch (ex UdmSearch). mnoGoSearch es un software de motor de búsqueda completa para servidores de intranet e internet, distribuido bajo la licencia GNU. mnoGoSearch tiene un número de características únicas, que son adecuadas para una amplia gama de aplicaciones de búsqueda dentro del sitio a un sistema de búsqueda especializados como recetas de cocina o búsqueda de periódico, búsqueda de archivo FTP, búsqueda de artículos de noticias. Ofrece indexación de texto completo y búsqueda de documentos de texto, HTML y PDF. mnoGoSearch consta de dos partes. El primero es un mecanismo de indexación (indizador). El propósito del indizador es caminar a través de HTTP, FTP, servidores de noticias o archivos locales, recursivamente agarrar todos los documentos y almacenar metadatos sobre documentos en base de datos SQL de manera inteligente y eficaz. Después de cada documento se hace referencia en su URL correspondiente, los metadatos son recogidos por el indizador para su uso posterior en un proceso de búsqueda. La búsqueda se realiza a través de interfaz Web. Búsquedas frontales C, CGI, PHP y Perl están incluidas.

Puede encontrar más información sobre mnoGoSearch en » http://www.mnogosearch.org/.

Nota:

Esta extensión se ha movido al repositorio de » PECL y ya no se distribuye con PHP a partir de PHP 5.1.0.

Nota: Esta extensión no está disponible en plataformas Windows.
