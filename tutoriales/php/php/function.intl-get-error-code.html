intl_get_error_code

(PHP 5 >= 5.3.0, PHP 7, PECL intl >= 1.0.0)

intl_get_error_code — Obtiene el último código de error

Descripción

int intl_get_error_code ( void )

Útil para controlar los errores acaecidos en los métodos estáticos cuando no hay un objeto para obtener el código de error de ellos.

Valores devueltos

Devolución del código de error por la última llamada a la función API.

Ejemplos

Ejemplo #1 intl_get_error_code() ejemplo

<?php
$coll = collator_create( '<bad_param>' );
if( !$coll ) {
    handle_error( intl_get_error_code() );
}
?>
