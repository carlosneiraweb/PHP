$php_errormsg

(PHP 4, PHP 5, PHP 7)

$php_errormsg — El mensaje de error anterior

Descripción

$php_errormsg es una variable que contiene el texto del último mensaje de error generado por PHP. Esta variable solo estará disponible dentro del ámbito donde ocurrió el error, y solamente funcionará si la opción de configuración track_errors está activada (por defecto es 'off').

Advertencia

Si está definido un gestor de errores (set_error_handler()), $php_errormsg estará establecido solamente si el gestor de errores devuelve FALSE

Ejemplos

Ejemplo #1 Ejemplo de $php_errormsg

<?php
@strpos();
echo $php_errormsg;
?>

El resultado del ejemplo sería algo similar a:

Wrong parameter count for strpos()
