CairoContext::textPath

cairo_text_path

(PECL cairo >= 0.1.0)

CairoContext::textPath -- cairo_text_path — El propósito de textPath

Descripción

Estilo orientado a objetos

public void CairoContext::textPath ( string $string )

Estilo por procedimientos

void cairo_text_path ( CairoContext $context , string $text )

Añade trazados cerrados para texto al trazado actual. El trazado generado, si se rellena, consigue un efecto similar al de CairoContext::showText().

La conversión y posicionamiento de texto se hace de manera similar a CairoContext::showText().

Al igual que en CairoContext::showText(), después de esta llamada el punto actual se mueve al origen de donde sería colocado el siguiente glifo en esta misma progresión. Es decir, el punto actual estará en el origen del índice de del glifo final mediante sus valores de avance. Esto permite cambiar múltiples llamadas a CairoContext::showText() sin tener que establecer el punto actual entre ellas.

Observación: La llamada a la función CairoContext::textPath() es parte de lo que los diseñadores de cairo llaman la API de texto "toy". Es conveniente para demos cortas y programas sencillos, pero no se espera que sea adecuada para aplicaciones serias que usan texto. Véase CairoContext::glyphPath() para la API "real" de trazado de texto de cairo.
