mcrypt_module_get_algo_block_size

(PHP 4 >= 4.0.2, PHP 5, PHP 7)

mcrypt_module_get_algo_block_size — Devuelve el tamaño de bloque del algoritmo especificado

Descripción

int mcrypt_module_get_algo_block_size ( string $algorithm [, string $lib_dir ] )

Toma el tamaño de bloque del algoritmo especificado.

Parámetros

algorithm

El nombre del algoritmo.

lib_dir

Este parámetro opcional puede contener la ubicación donde el módulo de modos se encuentra dentro del sistema.

Valores devueltos

Devuelve el tamaño de bloque del algoritmo especificado en bytes.
