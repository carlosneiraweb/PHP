HaruDoc::useCNSEncodings

(PECL haru >= 0.0.1)

HaruDoc::useCNSEncodings — Habilita las codificaciones Chinas simplificadas

Descripción

bool HaruDoc::useCNSEncodings ( void )

Habilita las codificaciones Chinas simplificadas.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve TRUE en caso de éxito.

Errores/Excepciones

Lanza una HaruException cuando se produce un error.
