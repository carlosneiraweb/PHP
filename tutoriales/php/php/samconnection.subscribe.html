SAMConnection::subscribe

(PECL sam >= 0.1.0)

SAMConnection::subscribe — Crea una suscripción a un tema específico.

Descripción

string SAMConnection::subscribe ( string $targetTopic )

El método "subscribe" se utiliza para crear una nueva suscripción a un tema específico.

Parámetros

targetTopic

La identidad del tema (topic://topicname) al que nos queremos suscribir.

Valores devueltos

Un identificador de suscripción que puede ser utilizado en una llamada de recepción posterior a modo de selector, para obtener cualquier información del tema o FALSE en caso de error. El identificador de suscripción debe ser utilizado en la llamada de recepción en vez del nombre del tema.

Ejemplos

Ejemplo #1 Suscripción a un tema

<?php
$subid = $conn->subscribe('topic://A');
if (!$subid) {
  // La suscripción falló!
  echo "Suscripción faliida ($conn->errno) $conn->error";
}
?>
