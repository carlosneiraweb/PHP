Memcached::increment

(PECL memcached >= 0.1.0)

Memcached::increment — Incrementar un valor numérico de un ítem

Descripción

public int Memcached::increment ( string $key [, int $offset = 1 [, int $initial_value = 0 [, int $expiry = 0 ]]] )

Memcached::increment() incrementa un valor numérico de un ítem por el offset especificado. Si el valor del ítem no es numérico, generará un error. Memcached::increment() establecerá el ítem al parámetro initial_value si la clave no existe.

Parámetros

key

La clave del ítem a incrementar.

offset

La cantidad por la que incrementar el valor del ítem.

initial_value

El valor para establecer el ítem si actualmente no existe.

expiry

El periodo de expiración a establecer para el ítem.

Valores devueltos

Devuelve el nuevo valor del ítem en caso de éxito o FALSE en caso de error.

Ejemplos

Ejemplo #1 Ejemplo de Memcached::increment()

<?php
$m = new Memcached();
$m->addServer('localhost', 11211);

$m->set('counter', 0);
$m->increment('counter');
$n = $m->increment('counter', 10);
var_dump($n);

$m->set('counter', 'abc');
$n = $m->increment('counter');
// ^ fallará debido a que el valor del ítem no es numérico
var_dump($n);
?>

El resultado del ejemplo sería:

int(11)
bool(false)
