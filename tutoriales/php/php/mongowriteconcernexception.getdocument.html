MongoWriteConcernException::getDocument

(PECL mongo >= 1.5.0)

MongoWriteConcernException::getDocument — Obtener el documento del error

Descripción

public array MongoWriteConcernException::getDocument ( void )

Devuelve la respuesta real del servidor que fue interpretada como un error.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Un documento de MongoDB, si está disponible, como un array.
