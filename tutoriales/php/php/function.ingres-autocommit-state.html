ingres_autocommit_state

(PECL ingres >= 2.0.0)

ingres_autocommit_state — Comprueba si la conexión usa la autoconsigna

Descripción

bool ingres_autocommit_state ( resource $link )

ingres_autocommit_state() se invoca para determinar si el enlace de conexión actual tiene habilitada la autoconsigna.

Parámetros

link

El identificador de enlace de la conexión

Valores devueltos

Devuelve TRUE si la autoconsigna está habilitada o FALSE cuando está deshabilitada
