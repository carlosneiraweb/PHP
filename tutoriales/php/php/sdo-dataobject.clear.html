SDO_DataObject::clear — Limpiar las propiedades de un SDO_DataObject

Descripción

void SDO_DataObject::clear ( void )
Advertencia

Esta función ha sido declarada EXPERIMENTAL. Su funcionamiento, nombre y la documentación que le acompaña puede cambiar sin previo aviso en futuras versiones de PHP. Utilícela bajo su propia responsabilidad.

Limpia las propiedades de un SDO_DataObject. Las propiedades de sólo lectura no son afectadas. Las subsiguientes llamadas a isset() para el objeto de datos devolverán FALSE.

Parámetros

Ninguno.

Valores devueltos

No devuelve ningún valor.
