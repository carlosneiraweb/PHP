Imagick::getImageProperty

(PECL imagick 2.0.0)

Imagick::getImageProperty — Devuelve la propiedad nominada de la imagen

Descripción

string Imagick::getImageProperty ( string $name )

Devuelve la propiedad nominada de la imagen. Este método está disponible si Imagick ha sido compilado con la versión 6.3.2 o superior de ImageMagick.

Parámetros

name

nombre de la propiedad (por ejemplo Exif:DateTime)

Ejemplos

Ejemplo #1 Usando Imagick::getImageProperty():

Establecer y obtener la propiedad de la imagen

<?php
$imagen = new Imagick();
$imagen->newImage(300, 200, "black");

$imagen->setImageProperty('Exif:Make', 'Imagick');
echo $imagen->getImageProperty('Exif:Make');
?>

Ver también

  • Imagick::setImageProperty() - Establece una propiedad de la imagen

Valores devueltos

Devuelve un string que contiene la propiedad de la imagen, false si no existe una propiedad con el nombre dado.
