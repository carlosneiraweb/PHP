sqlite_next

SQLiteResult::next

SQLiteUnbuffered::next

(PHP 5 < 5.4.0, PECL sqlite >= 1.0.0)

sqlite_next -- SQLiteResult::next -- SQLiteUnbuffered::next — Busca el número de fila siguiente

Descripción

bool sqlite_next ( resource $result )

Estilo orientado a objetos (método):

bool SQLiteResult::next ( void )
bool SQLiteUnbuffered::next ( void )

sqlite_next() avanza en el gestor de resultados result a la siguiente fila.

Parámetros

result

El recurso de resultados de SQLite. Este parámetro no es necesario cuando se usa el método orientado a objetos.

Valores devueltos

Devuelve TRUE en caso de éxito, o FALSE si no existen más filas.

Ver también

  • sqlite_seek() - Busca un número de fila en particular en un conjunto de resultados almacenado en buffer
  • sqlite_current() - Obtiene la fila actual como un array de un conjunto de resultado
  • sqlite_rewind() - Busca el primer número de fila
