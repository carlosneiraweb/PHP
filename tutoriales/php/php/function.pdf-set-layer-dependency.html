PDF_set_layer_dependency

(PECL pdflib >= 2.0.0)

PDF_set_layer_dependency — Definir la relación entre capas

Descripción

bool PDF_set_layer_dependency ( resource $pdfdoc , string $type , string $optlist )

Define la relación grupal y jerárquica entre capas. Devuelve TRUE en caso de éxito o FALSE en caso de error.

Esta función requiere PDF 1.5.
