fbsql_num_fields

(PHP 4 >= 4.0.6, PHP 5, PHP 7)

fbsql_num_fields — Obtener el número de campos del resultado

Descripción

int fbsql_num_fields ( resource $result )

Devuelve el número de campos del result set dado.

Parámetros

result

Un identificador de resultado devuelto por fbsql_query() o fbsql_db_query().

Valores devueltos

Devuelve el número de campos, como entero.

  • fbsql_query() - Realiza una consulta FrontBase

  • fbsql_num_rows() - Obtener el número de filas del resultado
