fam_pending

(PHP 5 <= 5.0.5)

fam_pending — Verifica si hay eventos FAM pendientes

Descripción

int fam_pending ( resource $fam )

Verifica si hay eventos FAM pendientes.

Parámetros

fam

Un recurso que representa una conexión al servicio FAM devuelto por fam_open()

Valores devueltos

Devuelve un valor distinto a cero si los eventos están disponibles para ser obtenidos por la función fam_next_event(), de lo contrario retornará cero.

  • fam_next_event() - Obtiene el próximo evento FAM en espera
