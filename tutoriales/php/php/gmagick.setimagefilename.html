Gmagick::setimagefilename

(PECL gmagick >= Unknown)

Gmagick::setimagefilename — Establece el nombre de archivo de una imagen en particular en una secuencia

Descripción

public Gmagick Gmagick::setimagefilename ( string $filename )

Establece el nombre de archivo de una imagen en particular en una secuencia.

Parámetros

filename

El nombre de archivo de la imagen.

Valores devueltos

El objeto Gmagick si se tuvo éxito.

Errores/Excepciones

Arroja un GmagickException en caso de error.
