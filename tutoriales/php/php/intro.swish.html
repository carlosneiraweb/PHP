Introducción

La extensión swish proporciona los enlaces para la API Swish-e. Swish-e proviene de "Simple Web Indexing System for Humans - Enhanced" y es un sistema de código abierto de indexación y búsqueda. Swish-e está disponible bajo licencia GPL, pero utiliza una cláusula que permite a las aplicaciones vincular nuevamente la biblioteca si todas las copias del conjunto de trabajo va acompañada de la URL en el código fuente Swish-e. Aquí lo tiene: » http://swish-e.org.

Advertencia

Esta extensión es EXPERIMENTAL. Esto significa que el comportamiento de esta extensión, los nombres de sus funciones y todo lo documentado sobre esta extensión, puede cambiar en una futura edición de PHP sin previo aviso. El uso de esta extensión queda bajo su propia responsabilidad.
