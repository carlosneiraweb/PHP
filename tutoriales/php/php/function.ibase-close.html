ibase_close

(PHP 5, PHP 7)

ibase_close — Cerrar una conexión con una base de datos InterBase

Descripción

bool ibase_close ([ resource $connection_id = NULL ] )

Cierra el enlace con una base de datos InterBase que esté asociado con un id de conexión devuelto por ibase_connect(). La transacción predeterminada en el enlace es aplicada, las demás transacciones son revertidas.

Parámetros

connection_id

Un identificador de enlace InterBase devuelto por ibase_connect(). Si se omite, se asume el último enlace abierto.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.

Ver también

  • ibase_connect() - Abrir una conexión a una base de datos
  • ibase_pconnect() - Abrir una conexión persistente con una base de datos InterBase
