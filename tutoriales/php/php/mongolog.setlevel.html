MongoLog::setLevel

(PECL mongo >= 1.2.3)

MongoLog::setLevel — Establece el nivel o niveles a registrar

Descripción

public static void MongoLog::setLevel ( int $level )

Esta función se puede usar para controlar la verbosidad del registro y los tipos de actividades que deberían registrarse. Las constantes de nivel de MongoLog se pueden emplear con operadores a nivel de bit para especificar múltiples niveles.

<?php

// primero, especificar un módulo de registro
MongoLog::setModule(MongoLog::CON);

// registrar mensajes de cada nivel
MongoLog::setLevel(MongoLog::ALL);

// registrar mensajes de advertencias e información solamente
MongoLog::setLevel(MongoLog::WARNING|MongoLog::INFO);

// registrar todo excepto actividades de grano fino
MongoLog::setLevel(MongoLog::ALL & (~MongoLog::FINE));

?>

Tenga presente que también puede llamar a MongoLog::setModule() para especificar qué módulo o módulos del controlador se deberían registrar.

Parámetros

level

Nivel o niveles que se desean registrar.
