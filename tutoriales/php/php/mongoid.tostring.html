MongoId::__toString

(PECL mongo >= 0.8.0)

MongoId::__toString — Devuelve la representación hexadecimal de este id

Esta extenisón que define este método está obsoleta. En su lugar debe usarse la extensión MongoDB. Las alternativas a este método son:

  • MongoDB\BSON\ObjectID::__toString()

Descripción

public string MongoId::__toString ( void )

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Este id.

Ejemplos

Ejemplo #1 Ejemplo de MongoId::__toString()

<?php

$m = new MongoClient();
$collection = $m->selectDB("foo")->selectCollection("bar");

$collection->insert(array( "x" => "y" ));
$collection->insert(array( "x" => "y" ));

$cursor = $collection->find();
$r1 = $cursor->next();
$r2 = $cursor->next();

echo $r1["_id"] . "\n";
echo $r2["_id"] . "\n";

?>

El resultado del ejemplo sería algo similar a:

49a7011a05c677b9a916612a
49a702d5450046d3d515d10d
