IntlDateFormatter::getCalendarObject

datefmt_get_calendar_object

(PHP 5 >= 5.5.0, PHP 7, PECL intl >= 3.0.0)

IntlDateFormatter::getCalendarObject -- datefmt_get_calendar_object — Obtener una copia del objeto calendario del formateador

Descripción

Estilo orientado a objetos

public IntlCalendar IntlDateFormatter::getCalendarObject ( void )

Estilo por procedimientos

IntlCalendar datefmt_get_calendar_object ( void )

Obtiene una copia del objeto calendario usado internamente por este formateador. Este calendario tendrá un tipo (como gregorian, japanese, buddhist, roc, persian, islamic, etc.) y una zona horaria que coincide con el tipo y la zona horaria usada por el formateador. La fecha/hora del objeto está sin especificar.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Una copia del calendario interno usado por este formateador.

Ejemplos

Ejemplo #1 Ejemplo IntlDateFormatter::getCalendarObject()

<?php
$formatter = IntlDateFormatter::create(
    "fr_FR@calendar=islamic", 
    NULL,
    NULL,
    "GMT-01:00",
    IntlDateFormatter::TRADITIONAL
);

$cal = $formatter->getCalendarObject();

var_dump(
    $cal->getType(),
    $cal->getTimeZone(),
    $cal->getLocale(Locale::VALID_LOCALE)
);

El resultado del ejemplo sería:

string(7) "islamic"
object(IntlTimeZone)#3 (4) {
  ["valid"]=>
  bool(true)
  ["id"]=>
  string(9) "GMT-01:00"
  ["rawOffset"]=>
  int(-3600000)
  ["currentOffset"]=>
  int(-3600000)
}
string(5) "fr_FR"
