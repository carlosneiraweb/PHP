imap_8bit

(PHP 4, PHP 5, PHP 7)

imap_8bit — Convertir una cadena 8bit a una cadena quoted-printable

Descripción

string imap_8bit ( string $string )

Convertir una cadena 8bit a una cadena quoted-printable (de acuerdo a su definición en » RFC2045, sección 6.7).

Parámetros

string

La cadena 8bit a convertir

Valores devueltos

Devuelve una cadena codificada con el método quoted-printable.
