ReflectionFunctionAbstract::isClosure

(PHP 5 >= 5.3.0, PHP 7)

ReflectionFunctionAbstract::isClosure — Comprobar si es una clausura

Descripción

public bool ReflectionFunctionAbstract::isClosure ( void )

Comprueba si la función reflejada es una Closure.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve TRUE si la función es una Closure, FALSE si no.

Ejemplos

Ejemplo #1 Ejemplo de ReflectionFunctionAbstract::isClosure()

<?php
// No clausura
$función1 = 'str_replace';
$$reflejo1 = new ReflectionFunction($función1);
var_dump($reflejo1->isClosure());

// Clausura
$función2 = function () {};
$reflejo2 = new ReflectionFunction($función2);
var_dump($reflejo2->isClosure());
?>

El resultado del ejemplo sería:

bool(false)
bool(true)
