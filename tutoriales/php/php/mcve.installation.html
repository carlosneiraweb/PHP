Instalación

Esta extensión » PECL no se distribuye con PHP.

Se puede encontrar información para la instalación de esta extensión PECL en el capítulo del manual titulado Instalación de extensiones PECL. Se puede encontrar información adicional, tal como nuevos lanzamientos, descargas, ficheros fuente, información de mantenimiento, y un CHANGELOG, aquí: » http://pecl.php.net/package/mcve.

Actualmente, no hay ninguna DLL disponible para esta extensión PECL. Véase también la sección Compilar en Windows .

Si se instala sin especificar la ruta de libmonetra, PHP intentará buscar en la ubicación de instalación predeterminada de LibMonetra (/usr/local). Si Monetra (MCVE) está en una ubicación no estándard, ejecute configure con: --with-mcve=$ruta_mcve , donde $ruta_mcve es la ruta de su instalación de MCVE/Monetra. Por favor, observe que el soporte MCVE/Monetra requiere que $ruta_mcve/lib y $ruta_mcve/include existe, e incluye mcve.h o monetra.h bajo el directorio de inclusión libmcve.so and/or libmcve.a y/o libmonetra.so y/o libmonetra.a bajo el directorio lib.

Ya que MCVE/Monetra tiene una verdadera separación servidor/cliente, no hay requerimientos adicionales para ejecutar PHP con soporte para MCVE. Para probar su extensión MCVE/Monetra en PHP, debe conectarse a testbox.monetra.com en el puerto 8333 para IP, o en el puerto 8444 para SSL utilizando la API MCVE/Monetra de PHP. Use 'vitale' como nombre de usuario, y 'test' como contraseña. Está disponible información adicional sobre facilidades de prueba en » http://www.mainstreetsoftworks.com/.
