PDO::pgsqlCopyToArray

(PHP 5 >= 5.3.3, PHP 7)

PDO::pgsqlCopyToArray — Copiar datos de una tabla de una base de datos a un array de PHP

Descripción

public array PDO::pgsqlCopyToArray ( string $table_name , array $rows [, string $delimiter = '\t' [, string $null_as = "\\\\N" [, string $fields ]]] )

Copia los datos de table a un array empleando delimiter como delimitador de campos y la lista de campos fields

Parámetros

table_name

Un string que contiene el nombre de la tabla

delimiter

El delimitador empleado en el array rows

null_as

Cómo interpretar valores nulos

fields

La lista de campos a exportar

Valores devueltos

Returns an array of rows, o FALSE en caso de error.
