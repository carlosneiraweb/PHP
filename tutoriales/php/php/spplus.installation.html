Instalación

Se puede encontrar información para la instalación de esta extensión PECL en el capítulo del manual titulado Instalación de extensiones PECL. Se puede encontrar información adicional, tal como nuevos lanzamientos, descargas, ficheros fuente, información de mantenimiento, y un CHANGELOG, aquí: » http://pecl.php.net/package/spplus.

Actualmente, no hay ninguna DLL disponible para esta extensión PECL. Véase también la sección Compilar en Windows .

Nota:

Se necesita PHP 4.3.0 o superior para que la compresión de datos funcione.
