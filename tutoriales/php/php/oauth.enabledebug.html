OAuth::enableDebug

(PECL OAuth >= 0.99.3)

OAuth::enableDebug — Activa la depuración detallada

Descripción

public bool OAuth::enableDebug ( void )

Activa la solititud de información detallada útil para depurar, la información de depuración es almacenada en el miembro debugInfo. Alternativamente, el miembro debug puede ser establecido a un valor no-FALSE para activar el depurador.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

TRUE

Historial de cambios

Versión Descripción
0.99.8 Los miembros debug y debugInfo fueron agregados.
