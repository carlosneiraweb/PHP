SphinxClient::escapeString

(PECL sphinx >= 0.1.0)

SphinxClient::escapeString — Escapa caracteres especiales

Descripción

public string SphinxClient::escapeString ( string $string )

Escapa caracteres que se tratan como operadores especiales por el analizador de lenguaje de consulta.

Parámetros

string

String a ser escapado.

Valores devueltos

Devuelve un string escapado.
