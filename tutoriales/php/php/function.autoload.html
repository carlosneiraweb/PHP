__autoload

(PHP 5, PHP 7)

__autoload — Intenta cargar una clase sin definir

Descripción

void __autoload ( string $class )

Puede definir esta función para habilitar lacarga de clases.

Parámetros

class

Nombre de la clase a cargar

Valores devueltos

No devuelve ningún valor.
