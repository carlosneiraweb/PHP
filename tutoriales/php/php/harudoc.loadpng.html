HaruDoc::loadPNG

(PECL haru >= 0.0.1)

HaruDoc::loadPNG — Carga y devuelve una imagen PNGHaruImage instance

Descripción

object HaruDoc::loadPNG ( string $filename [, bool $deferred = false ] )

Carga una imagen PNG.

La Libharu puede ser construida sin ayuda de libpng, en ste caso cada llamada a ésta función resultará en excepción.

Parámetros

filename

El nombre del archivo de la imagen PNG.

deferred

No carga los datos inmediatamente. Se debe establecer el parámetro deferred a TRUE para los datos de carga diferida, en éste caso solo el tamaño y el color son cargados inmediatamente.

Valores devueltos

Devuelve una instacia de HaruImage.

Errores/Excepciones

Lanza una HaruException cuando se produce un error.
