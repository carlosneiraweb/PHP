ReflectionClass::setStaticPropertyValue

(PHP 5 >= 5.1.0, PHP 7)

ReflectionClass::setStaticPropertyValue — Establece el valor de una propiedad estática

Descripción

public void ReflectionClass::setStaticPropertyValue ( string $name , string $value )

Establece el valor de una propiedad estática.

Advertencia

Esta función no está documentada actualmente, solamente se encuentra disponible la lista de parámetros.

Parámetros

name

Nombre de la propiedad.

value

Nuevo valor de la propiedad.

Valores devueltos

No devuelve ningún valor.
