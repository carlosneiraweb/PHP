fbsql_database

(PHP 4 >= 4.0.6, PHP 5, PHP 7)

fbsql_database — Obtiene o especifica la base de datos usada por la conexión

Descripción

string fbsql_database ( resource $link_identifier [, string $database ] )

Obtiene o especifica la base de datos que usa la conexión.

Parámetros

link_identifier

Un identificador de enlace FrontBase devuelto por fbsql_connect() o fbsql_pconnect().

Es opcional y si no se especifica, la función intentará encontrar un enlace abierto al servidor FrontBase y de no encontrarse ningún enlace abierto intentará crear uno como si fbsql_connect() fuese invocada sin parámetros.

database

El nombre de la base de datos. Si se indica, se cambiará la base de datos por defecto de la conexión por la database especificada.

Valores devueltos

Devuelve el nombre de la base de datos usada en esta conexión.
