DirectoryIterator::getType

(PHP 5, PHP 7)

DirectoryIterator::getType — Determina el tipo del elemento actual DirectoryIterator

Descripción

public string DirectoryIterator::getType ( void )

Determina el tipo del fichero al que pertenece el elemento actual DirectoryIterator. Puede ser file, link, o dir.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve un string que representa el tipo de el fichero. Puede ser file, link, or dir.

Ejemplos

Ejemplo #1 Ejemplo de DirectoryIterator::getType()

<?php
$iterator = new DirectoryIterator(dirname(__FILE__));
foreach ($iterator as $fileinfo) {
    echo $fileinfo->getFilename() . " " . $fileinfo->getType() . "\n";
}
?>

El resultado del ejemplo sería algo similar a:

. dir
.. dir
manzana.jpg file
banana.jpg file
ejemplo.php file
pera.jpg file
