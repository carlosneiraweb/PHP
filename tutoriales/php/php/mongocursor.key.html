MongoCursor::key

(PECL mongo >=0.9.0)

MongoCursor::key — Devuelve el _id del resultado actual, o su índice dentro del conjunto de resultados

Descripción

public string|int MongoCursor::key ( void )

Parámetros

Esta función no tiene parámetros.

Valores devueltos

El _id del resultado actual como string. Si el resultado no posee un _id, se devolverá como entero su índice numérico dentro del conjunto de resultados.
