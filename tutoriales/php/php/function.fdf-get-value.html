fdf_get_value

(PHP 4, PHP 5, PHP 7)

fdf_get_value — Obtener el valor de un campo

Descripción

mixed fdf_get_value ( resource $fdf_document , string $fieldname [, int $which = -1 ] )

Obtiene el valor del campo solicitado.

Parámetros

fdf_document

El gestor de documento FDF, devuelto por fdf_create(), fdf_open() o fdf_open_string().

fieldname

Nombre del campo FDF, como una cadena.

which

Los elementos de un campo tipo matriz pueden ser recuperados pasando este parámetro opcional, comenzando desde cero. Para campos que no son matrices, este parámetro será ignorado.

Valores devueltos

Devuelve el valor del campo.

Historial de cambios

Versión Descripción
4.3.0 El soporte para matrices y el parámetro cuales fue agregado.
