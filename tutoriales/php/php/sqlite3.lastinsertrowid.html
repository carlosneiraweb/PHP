SQLite3::lastInsertRowID

(PHP 5 >= 5.3.0, PHP 7)

SQLite3::lastInsertRowID — Devolver el ID de la fila de la sentencia INSERT más reciente realizada en la base de datos

Descripción

public int SQLite3::lastInsertRowID ( void )

Devuelve el ID de la fila de la sentencia INSERT más reciente realizada en la base de datos.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve el ID de la fila de la sentencia INSERT más reciente realizada en la base de datos
