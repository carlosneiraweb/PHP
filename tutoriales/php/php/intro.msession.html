Introducción

msession es un interface a un demonio de alta velocidad que puede ejecutarse local o remotamente. Está diseñado para proporcionar control de sesiones para una granja de webs en PHP. Más información acerca de msession y del software servidor puede encontrase en » http://www.mohawksoft.org/?q=node/8.

Nota: Esta extensión no está disponible en plataformas Windows.

Nota:

Esta extensión se ha movido al repositorio de » PECL y ya no se distribuye con PHP a partir de PHP 5.1.3.
