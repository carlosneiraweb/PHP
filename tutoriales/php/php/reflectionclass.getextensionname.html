ReflectionClass::getExtensionName

(PHP 5, PHP 7)

ReflectionClass::getExtensionName — Obtiene el nombre de la extensión que definió una clase

Descripción

public string ReflectionClass::getExtensionName ( void )

Obtiene el nombre de la extensión que definió una clase.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

El nombre de la extensión que definió la clase, o FALSE para clases definidas por el usuario.

Ejemplos

Ejemplo #1 Uso básico de ReflectionClass::getExtensionName()

<?php
$clase = new ReflectionClass('ReflectionClass');
$extensión = $clase->getExtensionName();
var_dump($extensión);
?>

El resultado del ejemplo sería:

string(10) "Reflection"
