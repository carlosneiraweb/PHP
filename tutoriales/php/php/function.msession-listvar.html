msession_listvar

(PHP 4 >= 4.2.0, PHP 5 <= 5.1.2)

msession_listvar — Lista sesiones con variable

Descripción

array msession_listvar ( string $name )

Usado para buscar sesiones con atributos comunes.

Parámetros

name

El nombre a buscar.

Valores devueltos

Devuelve un array asociativo de valor/sesión para todas la sesiones con una variable nombrada name.
