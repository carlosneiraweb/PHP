msql_pconnect

(PHP 4, PHP 5, PHP 7)

msql_pconnect — Abrir una conexión mSQL persistente

Descripción

resource msql_pconnect ([ string $hostname ] )

msql_pconnect() funciona de forma similar a msql_connect() con dos claras diferencias.

En primer lugar, al conectarse, la primera función sería tratar de encontrar un enlace (persistente) que esté ya abierto con el mismo host. Si se encuentra alguno, un identificador de éste será devuelto en vez de abrir una nueva conexión.

En segundo lugar, la conexión al servidor SQL no será cerrada cuando la ejecución del script finalice. En su lugar, el enlace permanecerá abierto para su futuro uso (msql_close() no cerrará los enlaces establecidos con esta función).

Parámetros

hostname

El hostname también puede incluir un número de puerto, por ejemplo: hostname,port.

Si no se especifica, la conexión se establece por medio de un dominio socket de Unix, siendo más eficaz que una conexión TCP al servidor local.

Nota: Mientras esta función acepte los dos puntos (:) como un separador de host/puerto, una coma (,) será el método preferido.

Valores devueltos

Devuelve un identificador positivo de enlace mSQL en caso de éxito, o FALSE en caso de error.
