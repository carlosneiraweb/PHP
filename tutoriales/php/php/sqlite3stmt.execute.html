SQLite3Stmt::execute

(PHP 5 >= 5.3.0, PHP 7)

SQLite3Stmt::execute — Ejecutar una sentencia preparada y devuolver un objeto de conjunto de resultados

Descripción

public SQLite3Result SQLite3Stmt::execute ( void )

Ejecuta una sentencia preparada y devuelve un objeto de conjunto de resultados.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve un objeto de la clase SQLite3Result si la ejecución de la sentencia preparada tuvo éxito, FALSE en caso de fallo.
