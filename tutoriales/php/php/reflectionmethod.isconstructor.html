ReflectionMethod::isConstructor

(PHP 5, PHP 7)

ReflectionMethod::isConstructor — Comprueba si el método es un constructor

Descripción

public bool ReflectionMethod::isConstructor ( void )

Comprueba si el método es un constructor.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

TRUE si el método es un constructor, o si no FALSE
