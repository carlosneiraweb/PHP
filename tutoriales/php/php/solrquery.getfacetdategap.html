SolrQuery::getFacetDateGap

(PECL solr >= 0.9.2)

SolrQuery::getFacetDateGap — Devuelve el valor del parámetro facet.date.gap

Descripción

public string SolrQuery::getFacetDateGap ([ string $field_override ] )

Devuelve el valor del parámetro facet.date.gap. Acepta una sobrescritura opcional de campos

Parámetros

field_override

El nombre del campo

Valores devueltos

Devuelve una cadena en caso de éxito y NULL si no se estabelció
