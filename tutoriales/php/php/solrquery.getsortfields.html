SolrQuery::getSortFields

(PECL solr >= 0.9.2)

SolrQuery::getSortFields — Devuelve todos los campos de ordenación

Descripción

public array SolrQuery::getSortFields ( void )

Devuelve todos los campos de ordenación

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve una matriz en caso de éxito y NULL si no se estableció ningún parámetro.
