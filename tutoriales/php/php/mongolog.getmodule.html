MongoLog::getModule

(PECL mongo >= 1.2.3)

MongoLog::getModule — Devuelve el módulo o módulos que se están registrando actualmente

Descripción

public static int MongoLog::getModule ( void )

Esta función se puede usar para ver qué módulos del controlador están siendo registrados actualmente. El valor de tipo integer devuelto puede ser comparado con las constantes de módulos de MongoLog empleando operadores a nivel de bit para comprobar qué módulos están siendo registrados.

<?php

if (MongoLog::getModule() & MongoLog::RS) {
    echo "registrando conjuntos de réplicas\n";
}

if (MongoLog::getModule() ^ MongoLog::NONE) {
    echo "registrando algo\n";
}

if ((MongoLog::getModule() & MongoLog::IO) == 0) {
    echo "no se están registrando entradas/salidas\n";
}

?>

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve el módulo o los módulos que se están registrando actualmente.
