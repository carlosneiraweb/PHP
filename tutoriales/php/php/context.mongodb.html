Opciones de contexto de MongoDB

Opciones de contexto de MongoDB — Listado de opciones de contexto de MongoDB

Descripción

Opciones de contexto para transportes mongodb://.

Opciones

log_cmd_insert callable

Una función de retrollamada invocada al insertar un documento, véase log_cmd_insert().

log_cmd_delete callable

Una función de retrollamada invocada al borrar un documento, véase log_cmd_delete().

log_cmd_update callable

Una función de retrollamada invocada al actualizar un documento, véase log_cmd_update().

log_write_batch callable

Una función de retrollamada invocada al ejecutar un Lote de Escrituras, véase log_write_batch().

log_reply callable

Una función de retrollamada invocada al leer una réplica de MongoDB, véase log_reply().

log_getmore callable

Una función de retrollamada invocada al recuperar más resultados de un cursor MongoDB, véase log_getmore().

log_killcursor callable

Una función de retrollamada invocada al ejecutar un código de operación 'killcursor', véase log_killcursor().
