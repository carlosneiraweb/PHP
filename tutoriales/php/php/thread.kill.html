Thread::kill

(PECL pthreads >= 2.0.0)

Thread::kill — Ejecución

Descripción

public void Thread::kill ( void )

Fuerza a terminar al hilo referenciado.

Advertencia

El programador no debería terminar hilos por la fuerza normalmente.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Una indicación booleana indicando el éxito.

Ejemplos

Ejemplo #1 Termina el hilo referenciado

<?php
class T extends Thread {
    public function run() {
        $stdin = fopen("php://stdin", "r");
        while(($line = fgets($stdin))) {
            echo $line;
        }
    }
}

$t = new T();
$t->start();

var_dump($t->kill());
?>

El resultado del ejemplo sería:

bool(true)
