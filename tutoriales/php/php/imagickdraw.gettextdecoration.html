ImagickDraw::getTextDecoration

(PECL imagick 2.0.0)

ImagickDraw::getTextDecoration — Devuelve la decoración del texto

Descripción

int ImagickDraw::getTextDecoration ( void )
Advertencia

Esta función no está documentada actualmente, solamente se encuentra disponible la lista de parámetros.

Devuelve la decoración aplicada cuando se escribe texto.

Valores devueltos

Devuelve una de las constantes DECORATION_ y 0 si no está establecida la decoración.
