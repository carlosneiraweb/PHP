fileatime

(PHP 4, PHP 5, PHP 7)

fileatime — Obtiene el momento del último acceso a un archivo

Descripción

int fileatime ( string $filename )

Obtiene el momento del último acceso a un archivo dado.

Parámetros

filename

Ruta del archivo.

Valores devueltos

Devuelve el momento en el que se acceció al archivo por última vez, o FALSE en caso de error. El momento es devuelto como tiempo Unix.

Ejemplos

Ejemplo #1 Ejemplo de fileatime()

<?php

// muestra, p.ej.  El último acceso a archivo.txt fue: December 29 2002 22:16:23.

$nombre_archivo = 'archivo.txt';
if (file_exists($nombre_archivo)) {
    echo "El último acceso a $nombre_archivo fue: " . date("F d Y H:i:s.", fileatime($nombre_archivo));
}

?>

Errores/Excepciones

En caso de fallo, se emite un E_WARNING.

Notas

Nota:

El atime de un archivo se supone que cambia siempre que los bloques de información de un archivo están siendo leídos. Esto puede costar rendimiento a la larga cuando una aplicación accede regularmente a un número muy grande de archivos o directorios.

Algunos sistemas de archivos Unix se pueden montar con las actualizaciones de atime deshabitiladas para incrementar el rendimiento de tales aplicaciones; las colas de noticias de USENET son un ejemplo común. En tales sistemas de archivos esta función será inservible.

Nota:

Observe que el tiempo de resolución quizá varia según el sistema de ficheros.

Nota: Los resultados de esta función son guardados en caché. Para más información ver clearstatcache().

Sugerencia

Desde PHP 5.0.0, esta función también puede usarse con algunos URL wrappers. Consultar Protocolos y Envolturas soportados para deter4minar qué envolturas soportan la familia de funcionalidad de stat().
