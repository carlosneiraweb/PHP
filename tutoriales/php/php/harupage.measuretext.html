HaruPage::measureText

(PECL haru >= 0.0.1)

HaruPage::measureText — Calcula el número de caracteres los cuales pueden ser incluidos dentro del ancho especificado

Descripción

int HaruPage::measureText ( string $text , float $width [, bool $wordwrap = false ] )

Calcula el número de caracteres los cuales pueden ser incluidos dentro del ancho especificado.

Parámetros

text

El texto a medir.

width

El ancho del área a colocar el texto.

wordwrap

Cuando éste parámetro se establece a TRUE la función "emula" la palabra envolver y no incluye la parte de la palabra actual si se ha llegado al final del área.

Valores devueltos

Devuelve el número de caracteres Returnslos cuales puede ser incluidos dentro del ancho especificado.

Errores/Excepciones

Lanza una HaruException cuando se produce un error.
