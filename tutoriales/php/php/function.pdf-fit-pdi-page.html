PDF_fit_pdi_page

(PECL pdflib >= 2.0.0)

PDF_fit_pdi_page — Colocar una página PDF importada

Descripción

bool PDF_fit_pdi_page ( resource $pdfdoc , int $page , float $x , float $y , string $optlist )

Coloca una página PDF importada en la página, sujeta a varias opciones. Devuelve TRUE en caso de éxito o FALSE en caso de error.
