posix_getpgrp

(PHP 4, PHP 5, PHP 7)

posix_getpgrp — Devolver el identificador de grupo de proceso actual

Descripción

int posix_getpgrp ( void )

Devuelve el identificador de grupo de proceso del proceso actual.

Valores devueltos

Devuelve el identificador, como valor de tipo integer.

Ver también

  • POSIX.1 y la página del manual getpgrp(2) en los sistemas POSIX para más información sobre grupos de procesos.
