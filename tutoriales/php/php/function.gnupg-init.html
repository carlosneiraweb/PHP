gnupg_init

(PECL gnupg >= 0.4)

gnupg_init — Inicializa una conexión

Descripción

resource gnupg_init ( void )

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Una conexión de tipo GnuPG resource usada por otras funciones GnuPG.

Ejemplos

Ejemplo #1 Ejemplo de gnupg_init() mediante funciones

<?php
$res = gnupg_init();
?>

Ejemplo #2 Ejemplo del inicializador gnupg mediante OO

<?php
$gpg = new gnupg();
?>
