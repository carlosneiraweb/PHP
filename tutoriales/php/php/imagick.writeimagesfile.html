Imagick::writeImagesFile — Escribe fotogramas al gestor de archivo

Descripción

bool Imagick::writeImagesFile ( resource $filehandle )

Escribe todos los fotogramas en un gestor de archivo abierto. Este método se puede usar para escribir gifs u otras imágenes multi-fotograma en un gestor de archivo abierto. Este método está disponible si Imagick ha sido compilado con la versión 6.3.6 o superior de ImageMagick.

Parámetros

filehandle

El gestor de archivo donde se van a escribir las imágenes

Valores devueltos

Devuelve TRUE en caso de éxito.
