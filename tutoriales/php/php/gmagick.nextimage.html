Gmagick::nextimage

(PECL gmagick >= Unknown)

Gmagick::nextimage — Se mueve a la siguiente imagen

Descripción

public bool Gmagick::nextimage ( void )

Asocia la siguiente imagen de la lista de ímágenes con un objeto Gmagick.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

El objeto Gmagick si se tuvo éxito

Errores/Excepciones

Arroja un GmagickException en caso de error.
