mcrypt_get_cipher_name

(PHP 4, PHP 5, PHP 7)

mcrypt_get_cipher_name — Toma el nombre del cifrador especificado

Descripción

string mcrypt_get_cipher_name ( int $cipher )
string mcrypt_get_cipher_name ( string $cipher )

mcrypt_get_cipher_name() es utilizada para tomar el nombre del cifrador especificado.

mcrypt_get_cipher_name() toma el número del cifrador como un argumento (libmcrypt 2.2.x) o toma el nombre del cifrador como un argumento (libmcrypt 2.4.x o superiores) y devuelve el nombre del cifrador o FALSE si el cifrador no existe.

Parámetros

cipher

Una de las constantes MCRYPT_nombredelcifrado, o el nombre del algoritmo como string.

Valores devueltos

Esta función devuelve el nombre del cifrador o FALSE si el cifrador no existe.

Ejemplos

Ejemplo #1 Ejemplo de mcrypt_get_cipher_name()

<?php
   $cipher = MCRYPT_TripleDES;

   echo mcrypt_get_cipher_name($cipher);
?>

El resultado del ejemplo sería:

3DES
