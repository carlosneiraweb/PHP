DirectoryIterator::getInode

(PHP 5, PHP 7)

DirectoryIterator::getInode — Obtiene el inodo para el elemento actual DirectoryIterator

Descripción

public int DirectoryIterator::getInode ( void )

Obtiene el número de inodo para el elemento actual DirectoryIterator.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve el número de inodo para el fichero.

Ejemplos

Ejemplo #1 Ejemplo de DirectoryIterator::getInode()

Este ejemplo muestra el número de inodo para el directorio que contiene al script.

<?php
$iterator = new DirectoryIterator(dirname(__FILE__));
echo $iterator->getInode();
?>
