fbsql_insert_id

(PHP 4 >= 4.0.6, PHP 5, PHP 7)

fbsql_insert_id — Obtiene el valor id generado por la anterior operación INSERT

Descripción

int fbsql_insert_id ([ resource $link_identifier ] )

Obtiene el id generado por la anterior operación INSERT que haya creado un valor DEFAULT UNIQUE.

Nota:

El valor de la función SQL FrontBase fbsql_insert_id() siempre contiene el valor DEFAULT UNIQUE generado más recientemente, y no se reinicia entre consultas.

Parámetros

link_identifier

Un identificador de enlace FrontBase devuelto por fbsql_connect() o fbsql_pconnect().

Es opcional y si no se especifica, la función intentará encontrar un enlace abierto al servidor FrontBase y de no encontrarse ningún enlace abierto intentará crear uno como si fbsql_connect() fuese invocada sin parámetros.

Valores devueltos

Devuelve el ID generado por la anterior consulta INSERT, o 0 si la consulta anterior no generó un valor DEFAULT UNIQUE.

Si se necesita guardar el valor para después, hay que asegurarse de llamar a esta función inmediatamente después de que la consulta que genera el valor.
