fam_cancel_monitor

(PHP 5 <= 5.0.5)

fam_cancel_monitor — Detiene el monitor

Descripción

bool fam_cancel_monitor ( resource $fam , resource $fam_monitor )

Termina de monitorizar un recurso.

Además se produce un evento FAMAcknowledge.

Parámetros

fam

Un recuro representa una conexión al servicio FAM devuelto por fam_open()

fam_monitor

Un recurso devuelto por uno de las funciones fam_monitor_XXX

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.

Ver también

  • fam_monitor_file() - Vigila cambios en un archivo regular
  • fam_monitor_directory() - Vigila cambios en un directorio
  • fam_monitor_collection() - Vigila los cambios en una colección de archivos de un directorio
  • fam_suspend_monitor() - Suspende temporalmente un monitoreo
