Nuevas Extensiones

Los siguientes son nuevas extensiones agregadas (por defecto) a partir de PHP 5.2.0:

  • Filter - valida y filtra datos, y está diseñado para ser usado con datos inseguros como ser los ingresados por el usuario. Esta extensión está activada de forma predeterminada; el modo predeterminado RAW no afecta los datos de entrada de ninguna manera.
  • JSON - implementa el formato de intercambio de datos 'JavaScript Object Notation' (JSON). Esta extensión está activada por defecto.
  • Zip - permite leer y escribir de forma transparente archivos comprimidos con ZIP y los ficheros dentro de ellos.
