SwishResults::nextResult

(PECL swish >= 0.1.0)

SwishResults::nextResult — Obtener el siguiente resultado de la búsqueda

Descripción

object SwishResults::nextResult ( void )
Advertencia

Esta función ha sido declarada EXPERIMENTAL. Su funcionamiento, nombre y la documentación que le acompaña puede cambiar sin previo aviso en futuras versiones de PHP. Utilícela bajo su propia responsabilidad.

Valores devueltos

Devuelve el siguiente objeto SwishResult en el conjunto de resultados o FALSE si no hay más resultados disponibles.

Ejemplos

Ejemplo #1 Un ejemplo básico de SwishResults::nextResult()

<?php

try {

    $swish = new Swish("index.swish-e");
    $search = $swish->prepare();

    $results = $search->execute("lost");
    while($result = $results->nextResult()) {
        /* hacer algo con el objeto resultado */
    }

} catch (SwishException $e) {
    echo $e->getMessage(), "\n";
}

?>
