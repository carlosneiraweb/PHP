HaruDoc::setPassword

(PECL haru >= 0.0.1)

HaruDoc::setPassword — Establecer propietario y contraseñas de usuarios para el documento

Descripción

bool HaruDoc::setPassword ( string $owner_password , string $user_password )

Define propietario y contraseñas de usuario para el documento. Asignar contraseñas hace que se encripte el contenido del documento.

Parámetros

owner_password

La contraseña de el propietario, que puede cambiar los permisos del documento. La contraseña vacía no es aceptada. El propietario de la contraseña no puede ser el mismo que el de la contraseña de usuario.

user_password

La contraseña de el usuario. Puede ser vacía.

Valores devueltos

devuelve TRUE en caso de éxito.

Errores/Excepciones

Lanza una HaruException cuando se produce un error.

  • HaruDoc::setPermission() - Establece los permisos para el documento
