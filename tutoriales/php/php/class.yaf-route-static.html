La clase Yaf_Route_Static

(Yaf >=1.0.0)

Introducción

Por defecto, Yaf_Router sólo tiene un Yaf_Route_Static como su ruta predeterminada.

Y Yaf_Route_Static está diseñada para manejar el 80% de los requisitos.

Por favor, *OBSERVE* que no es necesario instanciar un objeto de la clase Yaf_Route_Static, y que también es inncesesario añadirlo a la pila de enrutamientos de Yaf_Router, ya que siempre hay una en la pila de enrutamiento de Yaf_Router, y que siempre será llamada en el último momento.

Sinopsis de la Clase

Yaf_Route_Static implements Yaf_Router {
/* Métodos */
public string assemble ( array $info [, array $query ] )
public void match ( string $uri )
public bool route ( Yaf_Request_Abstract $request )
}
