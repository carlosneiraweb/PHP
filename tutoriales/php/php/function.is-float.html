is_float

(PHP 4, PHP 5, PHP 7)

is_float — Comprueba si el tipo de una variable es float

Descripción

bool is_float ( mixed $var )

Comprueba si el tipo de variable dado es float.

Nota:

Para comprobar si una variable es un número o un string numérico (por ejemplo, un formulario de entrada, que es siempre un string), debe utilizar is_numeric().

Parámetros

var

La variable a ser evaluada.

Valores devueltos

Devuelve TRUE si var es de tipo float, o FALSE de lo contrario.

Ejemplos

Ejemplo #1 Ejemplo de is_float()

<?php
if(is_float(27.25)) {
    echo "es float\n";
}else {
    echo "no es float\n";
}
var_dump(is_float('abc'));
var_dump(is_float(23));
var_dump(is_float(23.5));
var_dump(is_float(1e7));  //Notación Científica
var_dump(is_float(true));
?>

El resultado del ejemplo sería:

es float
bool(false)
bool(false)
bool(true)
bool(true)
bool(false)
