RecursiveFilterIterator::hasChildren

(PHP 5 >= 5.1.0, PHP 7)

RecursiveFilterIterator::hasChildren — Comprueba si el elemento actual del iterador interno tiene hijos

Descripción

public bool RecursiveFilterIterator::hasChildren ( void )

Comprueba si el elemento actual del iterador interno tiene hijos.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

TRUE si el iterador interno tiene hijos, en caso contrario FALSE
