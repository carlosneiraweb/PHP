SolrDocument::__set

(PECL solr >= 0.9.2)

SolrDocument::__set — Añade otro campo al documento

Descripción

public bool SolrDocument::__set ( string $fieldName , string $fieldValue )

Añade otro campo al documento. Se usa para establecer el campo como propiedades nuevas.

Parámetros

fieldName

Nombre del campo.

fieldValue

Valor del campo.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.
