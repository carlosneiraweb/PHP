Introducción

Gmagick es una extensión de php para crear, modificar y obtener meta-información de imágenes usando la API GraphicsMagick.

GraphicsMagick se enorgullece de ser la navaja suiza del procesamiento de imágenes. Trabaja con más de 88 formatos importantes incluyendo formatos como DPX, GIF, JPEG, JPEG-2000, PNG, PDF, PNM, y TIFF.

Gmagick consiste en una clase Gmagick principal, una clase GmagickDraw que de hecho es una varita mágica de dibujo y una clase GmagickPixel cuyas instancias representan un sólo píxel de una imagen (color, opacidad).
