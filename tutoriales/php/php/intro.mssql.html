Advertencia

Esta característica está OBSOLETA en PHP 7.0.0.

Las alternativas a esta característica son:

  • PDO_SQLSRV (solo Windows)
  • PDO_ODBC
  • SQLSRV (solo Windows)
  • API de ODBC unificada

Estas funciones permiten el acceso a bases de datos de MS SQL Server.

Esta extensión ya no está disponible en Windows con PHP 5.3 o posterior.

SQLSRV, una extensión alternativa para conectividad con MS SQL, está disponible desde Microsoft: » http://msdn.microsoft.com/en-us/sqlserver/ff657782.aspx.
