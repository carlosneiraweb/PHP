ReflectionExtension::getName

(PHP 5, PHP 7)

ReflectionExtension::getName — Obtiene el nombre de la extensión

Descripción

public string ReflectionExtension::getName ( void )

Devuelve el nombre de la extensión.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Nombre de la extensión.

Ejemplos

Ejemplo #1 Ejemplo de ReflectionExtension::getName()

<?php
$ext = new ReflectionExtension('mysqli');
var_dump($ext->getName());
?>

El resultado del ejemplo sería algo similar a:

string(6) "mysqli"
