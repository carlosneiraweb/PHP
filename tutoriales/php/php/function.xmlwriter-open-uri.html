XMLWriter::openURI

xmlwriter_open_uri

(PHP 5 >= 5.1.2, PHP 7, PECL xmlwriter >= 0.1.0)

XMLWriter::openURI -- xmlwriter_open_uri — Crea un nuevo xmlwriter usando una fuente uri para la salida

Descripción

Estilo orientado a objetos

bool XMLWriter::openURI ( string $uri )

Estilo por procedimientos

resource xmlwriter_open_uri ( string $uri )

Crea un nuevo XMLWriter usando uri para la salida.

Parámetros

uri

El recurso de URI para la salida.

Valores devueltos

Estilo orientado a objetos: Devuelve TRUE en caso de éxito o FALSE en caso de error.

Estilo por procedimientos: Devuelve un nuevo xmlwriter recurso para usarlo después con las funciones xmlwriter en caso de éxito, FALSE en caso de error.
