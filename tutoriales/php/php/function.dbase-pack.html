dbase_pack

(PHP 4, PHP 5, PHP 7)

dbase_pack — Empaquetar una base de datos

Descripción

bool dbase_pack ( int $dbase_identifier )

Empaqueta una base de datos borrando permanentemente todos los registros marcados para el borrado usando dbase_delete_record().

Parámetros

dbase_identifier

El identificador de enlace de la base de datos, devuelto por dbase_open() o dbase_create()..

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.

Ejemplos

Ejemplo #1 Vaciar una base de datos dBase

<?php

// abrir en modo lectura-escritura
$db = dbase_open('/tmp/test.dbf', 2);

if ($db) {
  $número_registros = dbase_numrecords($db);
  for ($i = 1; $i <= $número_registros; $i++) {
      dbase_delete_record($db, $i);
  }
  // expurgar la base de datos
  dbase_pack($db);
}

?>
