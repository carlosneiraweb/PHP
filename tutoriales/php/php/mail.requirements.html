Requerimientos

Para que las funciones de Mail estén disponibles, PHP debe tener acceso al binario de sendmail en el sistema durante el momento de compilación. Si se usa otro sistema de correo como qmail o postfix, es necesario que el wrapper apropiado venga con él. PHP buscará primero sendmail en PATH y entonces en: /usr/bin:/usr/sbin:/usr/etc:/etc:/usr/ucblib:/usr/lib. Es altamente recomendable que sendmail se encuentre en PATH. Además, el usuario con el que está compilado PHP debe tener permiso para acceder al binario de sendmail.
