cubrid_send_glo

(PECL CUBRID >= 8.3.0)

cubrid_send_glo — Leer información de un glo y enviarla a la salida estándar

Descripción

int cubrid_send_glo ( resource $conn_identifier , string $oid )

La función cubrid_send_glo() se usa para leer información de una instancia glo y enviarla a la salida estándar de PHP.

Parámetros

conn_identifier

Identificador de conexión.

oid

OID de la instancia glo de la que se quiere leer información.

Valores devueltos

TRUE, cuando el proceso tiene éxito.

FALSE, cuando el proceso no tiene éxito.

Ejemplos

Ejemplo #1 Ejemplo de cubrid_send_glo()

<?php
$req = cubrid_execute ($con, "select image from person where id =1");
if ($req) {
  list ($oid) = cubrid_fetch($req);
  cubrid_close_request($req);
  Header ("Content-type: image/jpeg");
  cubrid_send_glo ($con, $oid);
}
?>

Notas

Nota:

Por razones de compatibilidad con versiones anteriores, los siguientes alias obsoletos podrían usarse: cubrid_send_glo()

Nota:

Esta función está eliminada desde CUBRID 3.1.
