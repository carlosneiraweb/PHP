DOMNode::isSameNode

(PHP 5, PHP 7)

DOMNode::isSameNode — Indica si dos nodos son el mismo nodo

Descripción

public bool DOMNode::isSameNode ( DOMNode $node )

Esta función indica si dos nodos son el mismo nodo. La comparación no está basada en el contenido

Parámetros

node

El nodo comparado.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.
