ibase_blob_create

(PHP 5, PHP 7)

ibase_blob_create — Crea un nuevo blob al que añadir datos

Descripción

resource ibase_blob_create ([ resource $link_identifier = NULL ] )

ibase_blob_create() crea un nuevo BLOB al que añadir datos.

Parámetros

link_identifier

Identificador de enlace a InterBase. Si se omite, se utilizará el último enlace que se haya abierto.

Valores devueltos

Devuelve un manejador BLOB para usar posteriormente con ibase_blob_add() o FALSE en caso de error.

Ver también

  • ibase_blob_add() - Añade datos a un nuevo blob
  • ibase_blob_cancel() - Cancela la creación de un blob
  • ibase_blob_close() - Cierra un blob
  • ibase_blob_import() - Crea un blob, copia un fichero en él, y lo cierra
