geoip_isp_by_name

(PECL geoip >= 1.0.2)

geoip_isp_by_name — Obtiene el nombre del proveedor de servicios de Internet (ISP)

Descripción

string geoip_isp_by_name ( string $hostname )

La función geoip_isp_by_name() devolverá el nombre del proveedor de servicios de Internet (ISP) que tenga asignado una dirección IP.

Esta función sólo se encuentra disponible para los usuarios que han comprado un GeoIP ISP Edición Comercial. Se devolverá una advertencia si no puede ser localizado en la base de datos.

Parámetros

hostname

El nombre de equipo o la dirección IP.

Valores devueltos

Devuelve el nombre del ISP si es correcto, o FALSE si la dirección IP no puede ser localizada en la base de datos.

Ejemplos

Ejemplo #1 Ejemplo de geoip_isp_by_name()

Esto mostrará el ISP de ejemplo.com.

<?php
$isp = geoip_isp_by_name('www.example.com');
if ($isp) {
    echo 'This host IP is from ISP: ' . $isp;
}
?>

El resultado del ejemplo sería:

This host IP is from ISP: ICANN c/o Internet Assigned Numbers Authority
