MysqlndUhConnection::getHostInformation

(PECL mysqlnd-uh >= 1.0.0-alpha)

MysqlndUhConnection::getHostInformation — Devuelve una representación de tipo string del tipo de conexión usada

Descripción

public string MysqlndUhConnection::getHostInformation ( mysqlnd_connection $connection )

Devuelve una representación de tipo string del tipo de conexión usada.

Parámetros

connection

Gestor de conexión de mysqlnd. ¡No modificar!

Valores devueltos

La descripción de la conexión.

Ejemplos

Ejemplo #1 Ejemplo de MysqlndUhConnection::getHostInformation()

<?php
class proxy extends MysqlndUhConnection {
 public function getHostInformation($res) {
   printf("%s(%s)\n", __METHOD__, var_export(func_get_args(), true));
   $ret = parent::getHostInformation($res);
   printf("%s returns %s\n", __METHOD__, var_export($ret, true));
   return $ret;
 }
}
mysqlnd_uh_set_connection_proxy(new proxy());

$mysqli = new mysqli("localhost", "root", "", "test");
var_dump($mysqli->host_info);
?>

El resultado del ejemplo sería:

proxy::getHostInformation(array (
  0 => NULL,
))
proxy::getHostInformation returns 'Localhost via UNIX socket'
string(25) "Localhost via UNIX socket"
