gupnp_context_get_subscription_timeout

(PECL gupnp >= 0.1.0)

gupnp_context_get_subscription_timeout — Obtiene el tiempo de espera de la suscripción de eventos

Descripción

int gupnp_context_get_subscription_timeout ( resource $context )

Obtiene el tiempo de espera de la suscripción de eventos (en segundos), o 0 refiriénsose a que no hay tiempo de espera.

Parámetros

context

Un identificador de contexto, devuelto por gupnp_context_new().

Valores devueltos

El tiempo de espera del suscriptor de eventos en segundos.

Ver también

  • gupnp_context_new() - Crea un nuevo contexto
  • gupnp_context_set_subscription_timeout() - Establece el timpo de espera del suscriptor de eventos
