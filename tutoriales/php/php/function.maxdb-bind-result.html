maxdb_bind_result

(PECL maxdb 1.0)

maxdb_bind_result — Alias de maxdb_stmt_bind_result()

Descripción

Esta función es un alias de: maxdb_stmt_bind_result().

Esta función alias está obsoleta y sólo existe por razones de compatibilidad con versiones anteriores. El uso de esta función no es recomendado, y puede ser eliminado de PHP en futuras versiones.
