GearmanClient::addServers

(PECL gearman >= 0.5.0)

GearmanClient::addServers — Añade una lista de servidores de trabajos al cliente

Descripción

public bool GearmanClient::addServers ([ string $servers = 127.0.0.1:4730 ] )

Añade una lista de servidores de trabajos que puede usarse para ejecutar una tarea. No se realiza ninguna actividad de entrada y salida sobre el socket; los servidores únicamente se añaden a la lista completa de servidores.

Parámetros

servers

Una lista separadas por comas de servidores, cada servidor especificado en el formato 'host:port'.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.

Ejemplos

Ejemplo #1 Añade dos servidores de trabajos

<?php

# Creamos el objeto con el cliente
$gmclient= new GearmanClient();

# Añadimos múltiples servidores, el primero en el puerto por defecto: 4730
$gmclient->addServers("10.0.0.1,10.0.0.2:7003");

?>
