OAuthProvider::callTimestampNonceHandler

(PECL OAuth >= 1.0.0)

OAuthProvider::callTimestampNonceHandler — Llama al callback timestampNonceHandler

Descripción

public void OAuthProvider::callTimestampNonceHandler ( void )

Llama a la función callback del timestamp registrado, que fue establecido con OAuthProvider::timestampNonceHandler().

Advertencia

Esta función no está documentada actualmente, solamente se encuentra disponible la lista de parámetros.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

No devuelve ningún valor.

Errores/Excepciones

Muestra un error E_ERROR level si la llamada de retorno no puede ser llamada, o no se especificó.
