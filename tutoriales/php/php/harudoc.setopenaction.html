HaruDoc::setOpenAction

(PECL haru >= 0.0.1)

HaruDoc::setOpenAction — Definir en qué página se muestra cuando se abre el documento

Descripción

bool HaruDoc::setOpenAction ( object $destination )

Define en qué página se debe mostrar cuando se abre el documento.

Parámetros

destination

Una válido HaruDestination instancia.

Valores devueltos

Devuelve TRUE en el éxito.

Errores/Excepciones

Lanza una HaruException cuando se produce un error.
