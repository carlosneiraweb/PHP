Cómo encontrar más información sobre PHP

Este manual no pretende dar prácticas de programación en general. Si es su primera vez o es un principiante, seguramente encontrará difícil aprender como programar con PHP usando solamente este manual. Busque mejor un libro orientado a principiantes.

Hay un gran número de listas de correo activas para discutir cualquier aspecto relacionado con la programación con PHP. Si se queda bloqueado con un problema, puede usar estas listas para obtener ayuda. Para ver las opciones de soporte, incluidas listas de correo, ver » página de soporte de PHP.net.
