win32_delete_service

(PECL win32service SVN)

win32_delete_service — Borra una entrada de servicio de la base de datos del ACS

Descripción

mixed win32_delete_service ( string $servicename [, string $machine ] )

Intenta borrar un servicio de la base de datos del ACS. Se necesitan privilegios adminstrativos para que esto suceda.

Esta función en realidad marca el servicio para su eliminación. Si otro proceso (tal como la Aplicación de Servicios) está abierto, el borrado será aplazado hasta que tales aplicaciones se cierren. Si un servicio es marcado para su eliminación, más intentos de borrarlo fallarán, y los intentos de crear un nuevo servicio con ese nombre también fallarán.

Parámetros

servicename

El nombre corto del servicio.

machine

El nombre opcional de la máquina. Si se omite, será usada la máquina local.

Valores devueltos

Devuelve WIN32_NO_ERROR en caso de éxito, FALSE si hay un problema con los parámetros o Código de error Win32 en caso de error.

Ejemplos

Ejemplo #1 Un ejemplo de win32_delete_service()

Borra el servicio dummyphp.

<?php
win32_delete_service('dummyphp');
?>
