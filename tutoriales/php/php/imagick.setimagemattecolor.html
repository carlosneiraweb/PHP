Imagick::setImageMatteColor

(PECL imagick 2.0.0)

Imagick::setImageMatteColor — Establece el color mate de la imagen

Descripción

bool Imagick::setImageMatteColor ( mixed $matte )

Establece el color mate de la imagen.

Parámetros

matte

Valores devueltos

Devuelve TRUE en caso de éxito.

Errores/Excepciones

Lanza ImagickException en caso de error.

Historial de cambios

Versión Descripción
2.1.0 Ahora se permite que un string represente el color como parámetro. Versiones anteriores sólo permitían un objeto ImagickPixel.
