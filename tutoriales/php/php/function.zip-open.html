zip_open

(PHP 4 >= 4.1.0, PHP 5 >= 5.2.0, PHP 7, PECL zip >= 1.0.0)

zip_open — Abre un fichero ZIP

Descripción

resource zip_open ( string $filename )

Abre un nuevo fichero ZIP para su lectura.

Parámetros

filename

El nombre del fichero ZIP a abrir.

Valores devueltos

Devuelve un manejador del recurso para usarlo después con zip_read() y zip_close() o devolver el número de error si filename no existe o en caso de otro error.

  • zip_read() - Leer la siguiente entrada en el fichero ZIP
  • zip_close() - Cierra un fichero ZIP
