fann_set_train_stop_function

(PECL fann >= 1.0.0)

fann_set_train_stop_function — Establece la función de parada empleada durante el entrenamiento

Descripción

bool fann_set_train_stop_function ( resource $ann , int $stop_function )

Establece la función de parada empleada durante el entrenamiento.

Las funciones de parada están descritas en las constantes de funciones de parada.

Parámetros

ann

Un resource de red neuronal.

stop_function

La constante de función de parada.

Valores devueltos

Devuelve TRUE en caso de éxito, o FALSE de lo contrario.
