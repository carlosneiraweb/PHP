tidy_load_config

(PECL tidy >= 0.5.2)

tidy_load_config — Carga un archivo de configuración ASCII Tidy con la codificación especificada

Descripción

void tidy_load_config ( string $filename , string $encoding )

Carga un archivo de configuración Tidy, con la codificación encoding especificada.

Parámetros

filename

encoding

Valores devueltos

No devuelve ningún valor.

Notas

Nota: Esta función está disponible sólo en Tidy 1.0. Quedó obsoleta en Tidy 2.0, y por tanto ha sido eliminada.
