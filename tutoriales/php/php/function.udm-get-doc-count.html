udm_get_doc_count

(PHP 4 >= 4.0.5, PHP 5 <= 5.0.5, PECL mnogosearch >= 1.0.0)

udm_get_doc_count — Obtener el número total de documentos en la base de datos

Descripción

int udm_get_doc_count ( resource $agent )

udm_get_doc_count() devuelve el número de documentos en la base de datos.

Parámetros

agent

Un enlace a Agent, recibido después de llamar a udm_alloc_agent().

Valores devueltos

Devuelve el número de documentos.

Notas

Nota:

Esta función es soportada sólo en mnoGoSearch 3.1.11 o posterior.
