HaruPage::setTextLeading

(PECL haru >= 0.0.1)

HaruPage::setTextLeading — Establece el texto guía (línea de espacio) para la página

Descripción

bool HaruPage::setTextLeading ( float $text_leading )

Establece el texto guía (línea de espacio) para la página.

Parámetros

text_leading

Define la línea de espacio para la página.

Valores devueltos

Devuelve TRUE en éxito.

Errores/Excepciones

Lanza una HaruException cuando se produce un error.
