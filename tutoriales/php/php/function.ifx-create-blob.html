ifx_create_blob

(PHP 4, PHP <=5.2.0)

ifx_create_blob — Crea un objeto Blob

Descripción

int ifx_create_blob ( int $type , int $mode , string $param )

Crea un objeto blob.

Parámetros

type

1 = TEXT, 0 = BYTE

mode

0 = blob-object contiene el contenido en la memoria, 1 = blob-object contiene el contenido en un archivo.

param

if mode = 0: puntero al contenido, if mode = 1: puntero al string del archivo.

Valores devueltos

Devuelve el id del objeto BLOB, o FALSE en caso de error.

Ver también

  • ifx_copy_blob() - Duplica el objeto blob
  • ifx_free_blob() - Elimina el objeto blob
