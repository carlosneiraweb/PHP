mysqli::thread_safe

mysqli_thread_safe

(PHP 5, PHP 7)

mysqli::thread_safe -- mysqli_thread_safe — Devuelve si la seguridad a nivel de hilos está dada o no

Descripción

Estilo por procedimientos

bool mysqli_thread_safe ( void )

Indica si la biblioteca cliente está compilada con segurdid a nivel de hilos.

Valores devueltos

TRUE si la biblioteca cliente es segura a nivel de hilos, FALSE si no.
