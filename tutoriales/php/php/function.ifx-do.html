ifx_do

(PHP 4, PHP <=5.2.0)

ifx_do — Ejecutar una sentencia SQL previamente preparada

Descripción

bool ifx_do ( resource $result_id )

Ejecuta una sentencia previamente preparada o abre un puntero hacia una.

No libera al identificador result_id en caso de error.

Además establece el número real que devolverá ifx_affected_rows() para las sentencias "NO-SELECT".

Parámetros

result_id

result_id es el identificador válido devuelto por ifx_query() o ifx_prepare() (sólo para consultas "SELECT").

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.

Ver también

Ejemplo #1 Ejemplo de ifx_do()

<?php
$conn = fx_connect( "db", "user", "password" );
$result = ifx_prepare("SELECT customer_num, company FROM customer", $conn);
ifx_do($result);
?>
