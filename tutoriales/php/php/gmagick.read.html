Gmagick::read

(PECL gmagick >= Unknown)

Gmagick::read — Lee una imagen desde un nombre de archivo

Descripción

public Gmagick Gmagick::read ( string $filename )

Lee una imagen desde un nombre de archivo.

Parámetros

filename

El nombre de archivo de la imagen.

Valores devueltos

El objeto Gmagick si se tuvo éxito

Errores/Excepciones

Arroja un GmagickException en caso de error.
