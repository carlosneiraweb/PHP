PDF_close_pdi

(PHP 4 >= 4.0.5, PECL pdflib >= 1.0.0)

PDF_close_pdi — Cerrar el documento PDF de entrada [obsoleta]

Descripción

bool PDF_close_pdi ( resource $p , int $doc )

Cierra todos los gestores de página abiertos y cierra el documento PDF de entrada. Devuelve TRUE en caso de éxito o FALSE en caso de error.

Esta función está obsoleta desde la versión 7 de PDFlib, use PDF_close_pdi_document() en su lugar.
