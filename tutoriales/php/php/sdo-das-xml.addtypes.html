SDO_DAS_XML::addTypes

(^)

SDO_DAS_XML::addTypes — Para cargar un segundo o subsiguiente fichero de esquema en un objteo SDO_DAS_XML

Descripción

void SDO_DAS_XML::addTypes ( string $xsd_file )

Carga un segundo o subsiguiente fichero de esquema en un DAS XML que ya ha sido creado con el método estático create(). Aunque el fichero puede ser cualquier fichero de esquema válido, una posible razón para usar este método es para añadir un fichero de esquema que contenga definiciones de tipos complejos extra, de ahí su nombre. Véase el Ejemplo 4 de documento padre para un ejemplo.

Parámetros

xsd_file

Ruta al fichero de esquema XSD.

Valores devueltos

Ninguno si tiene éxito, de otro modo lanza una excepción tal como se describe abajo.

Errores/Excepciones

SDO_TypeNotFoundException

Lanzada si un tipo no está definido en el modelo subyacente.

SDO_DAS_XML_ParserException

Lanzada por cualqier problema al analizar el fichero XSD dado.

SDO_DAS_XML_FileException

Lanzada si el fichero especificado no se puede encontrar.
