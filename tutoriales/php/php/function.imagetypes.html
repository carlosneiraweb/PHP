imagetypes

(PHP 4 >= 4.0.2, PHP 5, PHP 7)

imagetypes — Devolver los tipos de imagen soportados por la versión actual de PHP

Descripción

int imagetypes ( void )

Devuelve los tipos de imagen soportados por la instalación actual de PHP.

Valores devueltos

Devuelve un campo de bit correspondiente a los formatos de imagen soportados por la versión de GD vinculada a PHP. Los siguietes bits son devueltos, IMG_GIF | IMG_JPG | IMG_PNG | IMG_WBMP | IMG_XPM.

Ejemplos

Ejemplo #1 Comprobar el soporte para PNG

<?php
if (imagetypes() & IMG_PNG) {
    echo "El soporte para PNG está habilitado";
}
?>
