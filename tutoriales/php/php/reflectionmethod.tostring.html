ReflectionMethod::__toString

(PHP 5, PHP 7)

ReflectionMethod::__toString — Devuelve la representación en forma de texto de este objeto

Descripción

public string ReflectionMethod::__toString ( void )

Devuelve la representación en forma de texto de este objeto.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Representación en forma de texto de esta instancia de ReflectionMethod.

Ejemplos

Ejemplo #1 Ejemplo de ReflectionMethod::__toString()

<?php
class HolaMundo {

    public function saludarA($nombre) {
        return 'Hola ' . $nombre;
    }

}

$metodoReflexionado = new ReflectionMethod('HolaMundo', 'saludarA');
echo $metodoReflexionado;
?>
