SDO_Model_Property::isContainment — Comprobar si la propiedad define una relación de contención

Descripción

bool SDO_Model_Property::isContainment ( void )
Advertencia

Esta función ha sido declarada EXPERIMENTAL. Su funcionamiento, nombre y la documentación que le acompaña puede cambiar sin previo aviso en futuras versiones de PHP. Utilícela bajo su propia responsabilidad.

Comprueba si la propiedad se corresponde con una relación de contención. Devuelve TRUE si la propiedad define una relación de contención, o FALSE si es referencia.

Parámetros

Ninguno.

Valores devueltos

Devuelve TRUE si la propiedad define una relación de contención, o FALSE si es referencia.
