Cairo::version

cairo_version

(PECL cairo >= 0.1.0)

Cairo::version -- cairo_version — Recupera la versión de la biblioteca cairo

Descripción

Estilo orientado a objetos (método):

public static int Cairo::version ( void )

Estilo por procedimientos:

int cairo_version ( void )

Recupera la versión actual de la biblioteca cairo como un valor entero

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Valor entero de la versión de la biblioteca Cairo

Ejemplos

Ejemplo #1 Estilo orientado a objetos

<?php
var_dump(Cairo::version());
?>

El resultado del ejemplo sería algo similar a:

int(10808)

Ejemplo #2 Estilo por procedimientos

<?php
var_dump(cairo_version());
?>

El resultado del ejemplo sería algo similar a:

int(10808)
