px_delete_record

(PECL paradox >= 1.4.0)

px_delete_record — Eliminar los registros de base de datos paradox

Descripción

bool px_delete_record ( resource $pxdoc , int $num )

Esta función elimina un registro de la base de datos. Esto no libera el espacio en el archivo de base de datos, solamente lo marca como eliminado. Insertando a continuación un nuevo registro se volverá a utilizar el espacio.

Nota:

Esta función sólo está disponible si se utiliza pxlib >= 0.6.0.

Parámetros

pxdoc

Identificador de recursos de la base de datos paradox devuelto por la función px_new().

num

El registro numérico es un número artificial que cuenta los registros a medida en que se almacenan en la base de datos. El primer registro tiene el número 0.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.
