SolrQuery::getFacetDateFields

(PECL solr >= 0.9.2)

SolrQuery::getFacetDateFields — Devuelve todos los campos de facet.date

Descripción

public array SolrQuery::getFacetDateFields ( void )

Devuelve todos los campos de facet.date

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve todos los campos de facet.date como matriz o NULL si no se estableció nada
