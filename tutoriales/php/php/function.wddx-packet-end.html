wddx_packet_end

(PHP 4, PHP 5, PHP 7)

wddx_packet_end — Finaliza un paquete WDDX con un ID específico

Descripción

string wddx_packet_end ( resource $packet_id )

Finaliza y devuelve el paquete WDDX.

Parámetros

packet_id

Un paquete WDDX, devuelto por wddx_packet_start().

Valores devueltos

Devuelve la cadena que contiene el paquete WDDX.
