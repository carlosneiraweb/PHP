HaruDoc::loadRaw

(PECL haru >= 0.0.1)

HaruDoc::loadRaw — Carga una imagen RAW

Descripción

object HaruDoc::loadRaw ( string $filename , int $width , int $height , int $color_space )

Carga una imagen RAW.

Parámetros

filename

El nombre de una archivo de imagen RAW.

width

The width of the image.

height

El ancho de la imagen.

color_space

El espacio de color de la imagen. Puede ser uno de los siguientes valores:

  • HaruDoc::CS_DEVICE_GRAY
  • HaruDoc::CS_DEVICE_RGB
  • HaruDoc::CS_DEVICE_CMYK

Valores devueltos

Devuelve una instancia de HaruImage.

Errores/Excepciones

Lanza una HaruException cuando se produce un error.
