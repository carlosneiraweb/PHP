ReflectionFunctionAbstract::getShortName

(PHP 5 >= 5.3.0, PHP 7)

ReflectionFunctionAbstract::getShortName — Obtiene el nombre abreviado de la función

Descripción

public string ReflectionFunctionAbstract::getShortName ( void )

Devuelve el nombre abreviado de la función (sin la parte del espacio de nombres).

Advertencia

Esta función no está documentada actualmente, solamente se encuentra disponible la lista de parámetros.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Nombre abreviado de la función.
