bcscale

(PHP 4, PHP 5, PHP 7)

bcscale — Establece los parametros de scale por defecto para todas las funciones matemáticas de bc

Descripción

bool bcscale ( int $scale )

Establece los parametros de scale para todas las subsiguientes llamadas a funciones matemáticas bc, donde no se especifica el parametro de scale.

Parámetros

scale

El factor scale.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.

Ejemplos

Ejemplo #1 Ejemplo bcscale()

<?php

// default scale : 3
bcscale(3);
echo bcdiv('105', '6.55957'); // 16.007

// this is the same without bcscale()
echo bcdiv('105', '6.55957', 3); // 16.007

?>
