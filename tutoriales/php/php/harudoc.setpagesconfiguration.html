HaruDoc::setPagesConfiguration

(PECL haru >= 0.0.1)

HaruDoc::setPagesConfiguration — Establece el número de páginas que contiene cada conjunto de páginas

Descripción

bool HaruDoc::setPagesConfiguration ( int $page_per_pages )

De manera predeterminada, cada documento tiene un objeto de páginas raíz para todas las páginas. Todos los objetos de páginas se crean a partir de este objeto. Un objeto de páginas sólo puede albergar 8191, por lo que el número máximo de páginas por documento es 1891. Pero esto se puede modificar mediante el parámetro page_per_pages, de manera que el objeto de páginas raíz contenga 8191 objetos de páginas más, que a su vez puedan albergar cada uno de ellos hasta 8191 páginas. De esta forma, el número máximo de páginas en un documento sería 8191*page_per_pages.

Parámetros

page_per_pages

Número de páginas que puede albergar un objeto de páginas.

Valores devueltos

En caso de éxito devuelve TRUE.

Errores/Excepciones

Lanza una HaruException cuando se produce un error.
