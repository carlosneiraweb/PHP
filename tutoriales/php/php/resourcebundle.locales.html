ResourceBundle::getLocales

resourcebundle_locales

(PHP >= 5.3.2, PECL intl >= 2.0.0)

ResourceBundle::getLocales -- resourcebundle_locales — Obtener las configuraciones regionales soportadas

Descripción

Estilo orientado a objetos

public array ResourceBundle::getLocales ( string $bundlename )

Estilo por procedimientos

array resourcebundle_locales ( string $bundlename )

Obtiene las configuraciones regionales disponibles desde el nombre de ResourceBundle.

Parámetros

bundlename

La ruta del ResourceBundle del que obtener las configuraciones regionales disponibles, o una cadena vacía para la lista de configuraciones regionales predeterminadas.

Valores devueltos

Devuelve la lista de las configuraciones regionales soportadas por el paquete.

Ejemplos

Ejemplo #1 Ejemplo de resourcebundle_locales()

<?php
$bundle = "/user/share/data/myapp";
echo join(PHP_EOL, resourcebundle_locales($bundle));
?>

El resultado del ejemplo sería algo similar a:

es
root

Ejemplo #2 Ejemplo orientado a objetos

<?php
$bundle = "/usr/share/data/myapp";
$r = new ResourceBundle( 'es', $bundle);
echo join("\n", $r->getLocales($bundle));
?>

El resultado del ejemplo sería algo similar a:

es
root
