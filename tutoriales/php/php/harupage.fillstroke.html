HaruPage::fillStroke

(PECL haru >= 0.0.1)

HaruPage::fillStroke — Rellenar la ruta actual utilizando la regla de zigzagueo de número distinto de cero, luego colorea la ruta

Descripción

bool HaruPage::fillStroke ([ bool $close_path = false ] )

Rellena la ruta actual utilizando la regla de zigzagueo de número distinto de cero, luego colorea la ruta.

Parámetros

close_path

Parámetro opcional. Cuando se pone en TRUE, la función cierra la ruta actual. De forma predeterminada está en FALSE.

Valores devueltos

Devuelve TRUE al tener éxito.

Errores/Excepciones

Lanza una HaruException cuando se produce un error.
