Instalación

Se puede encontrar información para la instalación de esta extensión PECL en el capítulo del manual titulado Instalación de extensiones PECL. Se puede encontrar información adicional, tal como nuevos lanzamientos, descargas, ficheros fuente, información de mantenimiento, y un CHANGELOG, aquí: » http://pecl.php.net/package/mnogosearch.

Esta extensión se ha movido al repositorio de » PECL y ya no se distribuye con PHP a partir de PHP 5.1.0

Actualmente, no hay ninguna DLL disponible para esta extensión PECL. Véase también la sección Compilar en Windows .

Nota:

PHP contiene biblioteca incorporada de acceso de MySQL, que puede utilizarse para acceder a MySQL. Se sabe que mnoGoSearch no es compatible con esta biblioteca incorporada y puede trabajar con bibliotecas de MySQL genéricas. Por lo tanto, si utilizas mnoGoSearch con MySQL, durante la configuración de PHP, se tiene que indicar el directorio de la instalación de MySQL, que se utilizó durante la configuración de mnoGoSearch, por ejemplo: --with-mnogosearch --with-mysql=/usr .
