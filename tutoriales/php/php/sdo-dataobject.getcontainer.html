(^)

SDO_DataObject::getContainer — Obtener el contenedor de un objeto de datos

Descripción

SDO_DataObject SDO_DataObject::getContainer ( void )
Advertencia

Esta función ha sido declarada EXPERIMENTAL. Su funcionamiento, nombre y la documentación que le acompaña puede cambiar sin previo aviso en futuras versiones de PHP. Utilícela bajo su propia responsabilidad.

Obtiene el objeto de datos que contiene este objeto de datos.

Parámetros

Ninguno.

Valores devueltos

Devuelve el SDO_DataObject que contiene este SDO_DataObject, o NULL si éste es un SDO_DataObject raíz (es decir, no tiene contenedor).
