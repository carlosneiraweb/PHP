SplFileObject::getCsvControl

(PHP 5 >= 5.2.0, PHP 7)

SplFileObject::getCsvControl — Obtiene el carácter delimitador y el circundante para campos CSV

Descripción

public array SplFileObject::getCsvControl ( void )

Obtiene el carácter delimitador y el circundante usados para analizar campos CSV.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve un array indexado que contiene el carácter delimitador y el circundante.

Ejemplos

Ejemplo #1 Ejemplo de SplFileObject::getCsvControl()

<?php
$fichero = new SplFileObject("datos.txt");
print_r($fichero->getCsvControl());
?>

El resultado del ejemplo sería algo similar a:

Array
(
    [0] => ,
    [1] => "
)
