(^)

SDO_Model_ReflectionDataObject::getContainmentProperty — Obtener la propiedad que define la relación de contención del objeto de datos

Descripción

SDO_Model_Property SDO_Model_ReflectionDataObject::getContainmentProperty ( void )
Advertencia

Esta función ha sido declarada EXPERIMENTAL. Su funcionamiento, nombre y la documentación que le acompaña puede cambiar sin previo aviso en futuras versiones de PHP. Utilícela bajo su propia responsabilidad.

Obtiene el SDO_Model_Property que contiene el SDO_DataObject. Este método se usa para navegar hasta la propiedad padre que contiene el objeto de datos que ha sido reflejado.

Parámetros

Ninguno.

Valores devueltos

Devuelve el SDO_Model_Property conteneder que referencia al SDO_DataObject, o NULL si es un SDO_DataObject raíz.
