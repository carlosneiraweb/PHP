fdf_set_ap

(PHP 4, PHP 5, PHP 7)

fdf_set_ap — Establecer la apariencia de un campo

Descripción

bool fdf_set_ap ( resource $fdf_document , string $field_name , int $face , string $filename , int $page_number )

Establece la apariencia de un campo (es decir, el valor de la clave /AP).

Parámetros

fdf_document

El gestor de documento FDF, devuelto por fdf_create(), fdf_open() o fdf_open_string().

field_name

face

Los valores posibles son FDFNormalAP, FDFRolloverAP y FDFDownAP.

filename

page_number

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.
