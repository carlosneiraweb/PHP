ifx_nullformat

(PHP 4, PHP <=5.2.0)

ifx_nullformat — Establecer el valor devuelto de manera predeterminada al extraer una fila

Descripción

bool ifx_nullformat ( int $mode )

Establece el valor que será devuelto predeterminadamente para las columnas con valor "NULL" al extraer una fila.

Parámetros

mode

Si el modo es "0" devuelve "" y si el modo es "1" devuelve "NULL".

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.
