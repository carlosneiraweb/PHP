DSN de PDO_4D — Conectarse a un servidor de 4D SQL

Descripción

El DSN de PDO_4D consiste en:

prefijo DSN

El prefijo DSN es 4D:.

host

La máquina anfitriona en la que está el servidor 4D SQL.

port

El número de puerto del servidor. Es opcinal.

user

El nombre de identificación al conectarse a la base de datos.

password

La contraseña para la identificación anterior.

dbname

El nombre de la base de datos. Este parámetro es opcional, y no se utiliza.

charset

El conjunto de caractares de 4D.

Ejemplos

Ejemplo #1 Ejemplo de DSN para PDO_4D

Los siguientes ejemplos tienen dos DSN para PDO_4D, que conectan a una base de datos de 4D: 

4D:host=localhost;charset=UTF-8
Otros valores posibles : 
4D:host=localhost
