DirectoryIterator::getGroup

(PHP 5, PHP 7)

DirectoryIterator::getGroup — Obtiene el grupo para el elemento actual DirectoryIterator

Descripción

public int DirectoryIterator::getGroup ( void )

Obtiene el id del grupo de el fichero.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve el id del grupo de el elemento actual DirectoryIterator en formato numérico.

Ejemplos

Ejemplo #1 Ejemplo de DirectoryIterator::getGroup()

<?php
$iterator = new DirectoryIterator(dirname(__FILE__));
$groupid  = $iterator->getGroup();
echo 'Directorio pertenece al grupo con id ' . $groupid . "\n";
print_r(posix_getgrgid($groupid));
?>

El resultado del ejemplo sería algo similar a:

Directorio pertenece al grupo con id 42
Array
(
    [name]    => toons
    [passwd]  => x
    [members] => Array
        (
            [0] => tom
            [1] => jerry
        )
    [gid]     => 42
)
