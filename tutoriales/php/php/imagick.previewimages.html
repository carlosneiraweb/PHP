Imagick::previewImages

(PECL imagick 2.0.0)

Imagick::previewImages — Precisa rápidamente los parámetros apropiados para el procesamiento de la imagen

Descripción

bool Imagick::previewImages ( int $preview )

Crea un mosaico de 9 miniaturas de la imagen especificada con la operación de proceso de imágenes aplicada en varias intensidades. Esto es de útil para precisar rápidamente un parámetro apropiado para una operación de proceso de imágenes.

Parámetros

preview

Tipo de previsualización. Véase constantes de tipos de previsualización

Valores devueltos

Devuelve TRUE en caso de éxito.

Errores/Excepciones

Lanza ImagickException en caso de error.
