PDO::pgsqlGetNotify

(PHP 5 >= 5.6.0, PHP 7)

PDO::pgsqlGetNotify — Obtener una notificación asíncrona

Descripción

public array PDO::pgsqlGetNotify ([ int $result_type = PDO::FETCH_USE_DEFAULT [, int $ms_timeout = 0 ]] )

Devuelve un conjunto de resultados que representa una notificación asíncrona pendiente.

Parámetros

result_type

El formato del conjunto de resultados debería ser devueto como, representado como una constante PDO::FETCH_*.

ms_timeout

La duración del tiempo de espera de una respuesta, en milisegundos.

Valores devueltos

Si hay una o más notificaciones pendientes, devuelve una única fila con los campos message y pid, o de lo contrario devuelve FALSE.
