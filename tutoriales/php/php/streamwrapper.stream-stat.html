streamWrapper::stream_stat

(PHP 4 >= 4.3.2, PHP 5, PHP 7)

streamWrapper::stream_stat — Recuperar información sobre un recurso de archivo

Descripción

public array streamWrapper::stream_stat ( void )

Este método es llamado en respuesta a fstat().

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Véase stat().

Errores/Excepciones

Emite un E_WARNING si la llamada a este método falla (por ej., por no estar implementado).
