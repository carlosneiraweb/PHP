dbx_escape_string

(PHP 4 >= 4.3.0, PHP 5 <= 5.0.5, PECL dbx >= 1.1.0)

dbx_escape_string — Escapar una cadena para que pueda ser usada de forma segura en una declaración sql

Descripción

string dbx_escape_string ( object $link_identifier , string $text )

Escapa la cadena dada para que pueda ser usada de forma segura en una declaración sql

Parámetros

link_identifier

El objeto de enlace de DBX devuelto por dbx_connect()

text

La cadena a escapar.

Valores devueltos

Devuelve el texto escapado cuando sea necesario (tal como comillas, barras invertidas, etc). Se devuelve NULL en caso de error.

Ejemplos

Ejemplo #1 Ejemplo de dbx_escape_string()

<?php
$enlace   = dbx_connect(DBX_MYSQL, "localhost", "db", "username", "password")
    or die("No se pudo conectar");

$texto = dbx_escape_string($enlace, "It\'s quoted and backslashed (\\).");
$result = dbx_query($enlace, "insert into tbl (txt) values ('" . $texto . "')");
if ($result == 0) {
    echo dbx_error($enlace);
}
dbx_close($enlace);
?>
