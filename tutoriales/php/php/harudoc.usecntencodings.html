HaruDoc::useCNTEncodings

(PECL haru >= 0.0.1)

HaruDoc::useCNTEncodings — Habilita las codificaciones Chinas tradicionales

Descripción

bool HaruDoc::useCNTEncodings ( void )

Habilita las codificaciones Chinas tradicionales.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve TRUE en caso de éxito.

Errores/Excepciones

Lanza una HaruException cuando se produce un error.
