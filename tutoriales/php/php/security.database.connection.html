Conexión a una base de datos

Se pueden establecer las conexiones sobre SSL para encriptar las comunicaciones cliente/servidor y aumentar la seguridad, o también emplear ssh para encriptar la conexión de red entre los clientes y el servidor de bases de datos. Si se utiliza algunas de estas opciones, será difícil para un posible atacante la monitorización del tráfico y la obtención de información de la base de datos.
