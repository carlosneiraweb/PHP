IntlChar::isWhitespace

(PHP 7)

IntlChar::isWhitespace — Comprobar si un punto de código es un carácter de espacio en blanco según ICU

Descripción

public static bool IntlChar::isWhitespace ( mixed $codepoint )

Determina si el punto de código es un carácter de espacio en blanco según ICU.

Un carácter se considera un carácter de espacio en blanco de Java si satisface uno de los siguientes criterios:

  • Es un carácter Separador de Unicode (categorías "Z" = "Zs" o "Zl" o "Zp"), pero no es también un espacio de no ruptura (U+00A0 NBSP o U+2007 Figure Space o U+202F Narrow NBSP).
  • Es un U+0009 HORIZONTAL TABULATION.
  • Es un U+000A LINE FEED.
  • Es un U+000B VERTICAL TABULATION.
  • Es un U+000C FORM FEED.
  • Es un U+000D CARRIAGE RETURN.
  • Es un U+001C FILE SEPARATOR.
  • Es un U+001D GROUP SEPARATOR.
  • Es un U+001E RECORD SEPARATOR.
  • Es un U+001F UNIT SEPARATOR.

Parámetros

codepoint

El valor de tipo integer del punto de código (p.ej. 0x2603 para U+2603 SNOWMAN), o el carácter codificado como un string UTF-8 (p.ej. "\u{2603}")

Valores devueltos

Devuelve TRUE si codepoint es un carácter de espacio en blanco según ICU, FALSE si no.

Ejemplos

Ejemplo #1 Probar diferentes puntos de código

<?php
var_dump(IntlChar::iswhitespace("A"));
var_dump(IntlChar::iswhitespace(" "));
var_dump(IntlChar::iswhitespace("\n"));
var_dump(IntlChar::iswhitespace("\t"));
var_dump(IntlChar::iswhitespace("\u{00A0}"));
?>

El resultado del ejemplo sería:

bool(false)
bool(true)
bool(true)
bool(true)
bool(false)
