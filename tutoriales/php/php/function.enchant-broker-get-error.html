enchant_broker_get_error

(PHP 5 >= 5.3.0, PHP 7, PECL enchant >= 0.1.0 )

enchant_broker_get_error — Devuelve el último error del agente

Descripción

string enchant_broker_get_error ( resource $broker )

Devuelve el último error que ocurrión en este agente.

Parámetros

broker

Recurso de agente.

Valores devueltos

Devuelve la cadena de mensaje si se encontró un error o FALSE
