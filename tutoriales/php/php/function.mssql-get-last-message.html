mssql_get_last_message

(PHP 4, PHP 5, PECL odbtp >= 1.1.1)

mssql_get_last_message — Devuelve el último mensaje de error generado por el servidor MSSQL

Advertencia

Esta función está OBSOLETA en PHP 7.0.0.

Las alternativas a esta función son:

  • PDOStatement::errorInfo()
  • sqlsrv_errors()
  • odbc_errormsg()

Descripción

string mssql_get_last_message ( void )

Devuelve el último mensaje de error del servidor MS-SQL

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve el último mensaje de error del servidor, o un string vacío si ningún mensaje de error ha sido devuelto por MSSQL.

Ejemplos

Ejemplo #1 mssql_get_last_message() example

<?php
// Conectar a MSSQL y seleccionar la base de datos
mssql_connect('KALLESPC\SQLEXPRESS', 'sa', 'phpfi');
mssql_select_db('php');

// Una consulta que fallará...
$query = @mssql_query('SELECT * FROM [php].[dbo].[not-found]');

if (!$query) {
    // La consulta ha fallado, muestra un mensaje de error
    // utilizando mssql_get_last_message()
    die('MSSQL error: ' . mssql_get_last_message());
}
?>

El resultado del ejemplo sería algo similar a:

MSSQL error: Invalid object name 'php.dbo.not-found'.
