GearmanTask::isRunning

(PECL gearman >= 0.5.0)

GearmanTask::isRunning — Comprueba si la tarea está siendo ejecutada

Descripción

public bool GearmanTask::isRunning ( void )

Indica si la tarea está actualmente siendo ejecutada.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

TRUE si la tarea está siendo ejecutada, FALSE en otro caso.
