OCI-Lob::write

(PHP 5, PHP 7, PECL OCI8 >= 1.1.0)

OCI-Lob::write — Escribe datos en un objeto grande

Descripción

int OCI-Lob::write ( string $data [, int $length ] )

Escribe los datos del parámetro data en la posición actual del puntero interno del LOB.

Parámetros

data

Los datos a escribir en el LOB.

length

Si se proporciona este parámetro, la escritura se dentendrá después de que se hayan escrito length bytes o se alcance el final de data, lo que suceda primero.

Valores devueltos

Devuelve el número de bytes escritos o FALSE en caso de error.
