sqlite_close

(PHP 5 < 5.4.0, PECL sqlite >= 1.0.0)

sqlite_close — Cierra un base de datos SQLite abierta

Descripción

void sqlite_close ( resource $dbhandle )

Cierra el manejador de db_handle dado. Si la base de datos fue persistente, será cerrada y eliminada de la lista de persistentes.

Parámetros

dbhandle

El recurso de base de datos SQLite; devuelto desde sqlite_open().

Valores devueltos

No devuelve ningún valor.

Ejemplos

Ejemplo #1 Ejemplo de sqlite_close()

<?php
$dbhandle = sqlite_open('sqlitedb');
sqlite_close($dbhandle);
?>

  • sqlite_popen() - Abrir un gestor persistente a una base de datos SQLite y crearla si no existe
