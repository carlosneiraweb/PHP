unlink

(PHP 4, PHP 5, PHP 7)

unlink — Borra un fichero

Descripción

bool unlink ( string $filename [, resource $context ] )

Elimina filename. Es similar a la función unlink() de C en Unix. Se generará un error de nivel E_WARNING si se produce un error.

Parámetros

filename

Ruta del fichero.

context

Nota: Soporte para context fue añadido en PHP 5.0.0. Para una descripción de contexts, refiérase a Flujos.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.

Historial de cambios

Versión Descripción
5.0.0 A partir de PHP 5.0.0 unlink() también se puede utilizar con algunas envolturas de URL. Consulte Protocolos y Envolturas soportados para un listado de la envolturas admitidas por unlink().

Ejemplos

Ejemplo #1 Uso básico de unlink()

<?php
$fh = fopen('prueba.html', 'a');
fwrite($fh, '<h1>¡Hola mundo!</h1>');
fclose($fh);

unlink('prueba.html');
?>
