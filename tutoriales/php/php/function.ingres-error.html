ingres_error

(PECL ingres >= 1.1.0)

ingres_error — Obtener un mensaje de error coherente del último error generado

Descripción

string ingres_error ([ resource $link ] )

Devuelve un valor de tipo string que contiene el último error, o NULL si ocurrió ninguno.

Si se pasa un recurso link a ingres_error(), devolverá el último error registrado para el enlace. Si no se pasa ningún enlace, ingres_error() devolverá el último error notificado usando el enlace predeterminado.

La función ingres_error() debería llamarse siempre después de ejecutar una consulta a la base de datos. La llamada a otra función antes de llamar a ingres_error() reiniciará o cambiará cualquier código de error de la última función de Ingres.

Parámetros

link

El identificador de enlace de la conexión

Valores devueltos

Devuelve un valor de tipo string que contiene el último error, o NULL si ocurrió ninguno.

Ejemplos

Ejemplo #1 Obtener un mensaje del último error generado

<?php
$enlace = ingres_connect($base_datos, $usuario, $contraseña);

$resultado = ingres_query($enlace, "select * from table");

$texto_error = ingres_error();

if (!is_null($texto_error)) {
   echo "Ocurrió un error - " . $texto_error;
}
?>
