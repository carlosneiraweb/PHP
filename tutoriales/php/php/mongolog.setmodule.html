MongoLog::setModule

(PECL mongo >= 1.2.3)

MongoLog::setModule — Establece el módulo o módulos a registrar

Descripción

public static void MongoLog::setModule ( int $module )

Esta función se puede usar para establecer qué módulos deberían registrarse. Las constantes de módulos de MongoLog se pueden emplear con operadores a nivel de bit para especificar múltiples módulos.

<?php

// primero, especificar un nivel de registro
MongoLog::setLevel(MongoLog::ALL);

// registrar las actividades de conjuntos de réplicas
MongoLog::setModule(MongoLog::RS);

// registrar las actividades de conjuntos de réplicas y conexiones
MongoLog::setModule(MongoLog::RS|MongoLog::CON);

// registramos todo excepto actividades de E/S
MongoLog::setModule(MongoLog::ALL & (~MongoLog::IO));

?>

Tenga presente que también se puede invocar a MongoLog::setLevel() para habilitar el registro.

Parámetros

module

El/los módulo/s que desea registrar.
