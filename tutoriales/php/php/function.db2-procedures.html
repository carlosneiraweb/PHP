db2_procedures

(PECL ibm_db2 >= 1.0.0)

db2_procedures — Devolver un resultado con los procedimientos almacenados registrados en la base de datos

Descripción

resource db2_procedures ( resource $connection , string $qualifier , string $schema , string $procedure )

Devuelve un resultado con los procedimientos almacenados registrados en una base de datos.

Parámetros

connection

Una conexión válida a una base de datos IBM DB2, Cloudscape, o Apache Derby.

qualifier

Un calificador para bases de datos DB2 ejecutándose sobre servidores OS/390 o z/OS. Para otras bases de datos se puede pasar NULL o una cadena vacia.

schema

El esquema que contiene los procedimientos. Este parámetro acepta las etiquetas _ y % para realizar búsquedas de cadena.

procedure

El nombre del procedimiento. Este parámetro acepta las etiquetas _ y % para realizar búsquedas de cadena.

Valores devueltos

Devuelve un recurso de sentencia con un resultado que contiene las filas que representan a los procedimientos almacenados que coinciden con el criterio de búsqueda. Las filas se componen de las siguientes columnas:

Columna Descripción
PROCEDURE_CAT El catálogo que contiene el procedimiento. El valor es NULL si la tabla no tiene catálogos.
PROCEDURE_SCHEM Nombre del esquema que contiene el procediento almacenado.
PROCEDURE_NAME Nombre del procedimiento.
NUM_INPUT_PARAMS Número de parámetros de entrada (IN) para el procedimiento almacenado.
NUM_OUTPUT_PARAMS Número de parámetros de salida (OUT) para el procedimiento almacenado.
NUM_RESULT_SETS Número de resultados devueltos por el procedimiento almacenado.
REMARKS Descripción del procedimiento almacenido.
PROCEDURE_TYPE Siempre devuelve 1, indicando que el procedimiento almacenado no devuelve un valor.
