Introducción

Enlace de PHP para la biblioteca FANN (Fast Artificial Neural Network - Red Neuronal Artificial Rápida), la cual implementa redes neuronales artificiales multicapa con soporte para redes completa y escasamente conectadas. Incluye un marco de trabajo para el manejo sencillo de conjuntos de datos de entrenamiento. Es sencilla de utilizar, es versátil, está bien documentada, y es rápida.
