px_get_value

(PECL paradox >= 1.1.0)

px_get_value — Obtener un valor

Descripción

float px_get_value ( resource $pxdoc , string $name )

Obtiene varios valores.

Parámetros

pxdoc

Identificador de recursos de la base de datos paradox devuelto por la función px_new().

name

El parámetro name puede ser uno de los siguientes.

numprimkeys

El número de índices primarios. Las bases de datos paradox utiliza siempre los primeros campos numprimkeys para el índice primario.

Valores devueltos

Devuelve el valor del parámetro o FALSE en caso de error.
