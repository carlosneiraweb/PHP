ReflectionFunctionAbstract::isUserDefined

(PHP 5, PHP 7)

ReflectionFunctionAbstract::isUserDefined — Comprueba si es una función definida por un usuario

Descripción

public bool ReflectionFunctionAbstract::isUserDefined ( void )

Comprueba si una función está definida por un usuario, en contraposición a las funciones internas.

Advertencia

Esta función no está documentada actualmente, solamente se encuentra disponible la lista de parámetros.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

TRUE si está definida por el usuario, y en cualquier otro caso FALSE.
