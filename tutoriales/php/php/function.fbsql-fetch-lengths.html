fbsql_fetch_lengths

(PHP 4 >= 4.0.6, PHP 5, PHP 7)

fbsql_fetch_lengths — Obtiene la longitud de cada salida en un resultado

Descripción

array fbsql_fetch_lengths ( resource $result )

Almacena en un array las longitudes de cada columna de resultado, de la última fila devuelta por fbsql_fetch_row(), fbsql_fetch_array() y fbsql_fetch_object().

Parámetros

result

Un identificador de resultado devuelto por fbsql_query() o fbsql_db_query().

Valores devueltos

Devuelve un array (desde offset 0) que corresponde a las longitudes de cada campo en la última fila obtenida por fbsql_fetch_row(), o FALSE en caso de error.
