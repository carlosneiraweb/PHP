spl_autoload_call

(PHP 5 >= 5.1.2, PHP 7)

spl_autoload_call — Probar todos las funciones registradas __autoload() para cargar la clase solicitada

Descripción

void spl_autoload_call ( string $class_name )

Esta función puede ser usada manualmente para buscar una clase o interface usando las funciones registradas __autoload.

Parámetros

class_name

El nombre de la clase a ser buscada.

Valores devueltos

No devuelve ningún valor.
