streamWrapper::dir_opendir

(PHP 4 >= 4.3.2, PHP 5, PHP 7)

streamWrapper::dir_opendir — Abrir un gestor de directorio

Descripción

public bool streamWrapper::dir_opendir ( string $path , int $options )

Este método es llamado en respuesta a opendir().

Parámetros

path

Especifica la URL que fue pasada a opendir().

Nota:

La URL se puede desmontar con parse_url().

options

Si forzar o no safe_mode (0x04).

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.

Ver también

  • opendir() - Abre un gestor de directorio
  • streamWrapper::dir_closedir() - Cerrar un gestor de directorio
  • parse_url() - Analiza un URL y devuelve sus componentes
