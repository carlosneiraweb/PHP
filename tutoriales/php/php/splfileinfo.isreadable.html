SplFileInfo::isReadable

(PHP 5 >= 5.1.2, PHP 7)

SplFileInfo::isReadable — Comprueba si el fichero se puede leer

Descripción

public bool SplFileInfo::isReadable ( void )

Comprueba si el fichero se puede leer.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve TRUE si se puede leer, en caso contrario FALSE.

Ejemplos

Ejemplo #1 Ejemplo de SplFileInfo::isReadable()

<?php
$info = new SplFileInfo(__FILE__);
var_dump($info->isReadable());

$info = new SplFileInfo('foo');
var_dump($info->isReadable());
?>

El resultado del ejemplo sería algo similar a:

bool(true)
bool(false)
