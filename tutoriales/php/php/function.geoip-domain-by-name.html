geoip_domain_by_name

(PECL geoip >= 1.1.0)

geoip_domain_by_name — Obtener el nombre de dominio de segundo nivel

Descripción

string geoip_domain_by_name ( string $hostname )

La función geoip_domain_by_name() devolverá los nombres de dominios de segundo nivel asociados con un nombre de host o una dirección IP.

Esta función sólo está disponible actualmente a usuarios que hayan comprado una edición comercial de GeoIP Domain Edition. Se emitirá una advertencia si no se puede localizar la propia base de datos.

Parámetros

hostname

El nombre de host o dirección IP.

Valores devueltos

Devuelve el nombre de dominio en caso de éxito, o FALSE si no se puede encontrar la dirección en la base de datos.

Ejemplos

Ejemplo #1 Un ejemplo de geoip_domain_by_name()

Mostrar el dominio asociado a la IP 61.106.139.1.

<?php
$domain = geoip_domain_by_name('61.106.139.1');

if ($domain) {
    echo 'El dominio es: '. $domain;
}

?>

El resultado del ejemplo sería:

El dominio es: von.co.kr
