newt_run_form

(PECL newt >= 0.1)

newt_run_form — Ejecuta un formulario

Descripción

resource newt_run_form ( resource $form )

Esta función ejecuta el formulario proporcionado.

Parámetros

form

El componente formulario.

Valores devueltos

El componente que ocasionó que se dejara de ejecutar el formulario.

Nota:

Nótese que esta función no encaja con la convención de nombre normal de newt. Es una interfaz antigua que no funcionará para todos los formularios. Se ha dejado en newt únicamente para aplicaciones antiguas. Aunque es una interfaz más sencilla que la nueva newt_form_run(), y aún se usa muy a menudo como resultado. Cuando una aplicación se termina con un formulario, destruye el formulario y todos los componentes que contiene el formulario.

Ver también

  • newt_form_run() - Ejecuta un formulario
  • newt_form_destroy() - Destruye un formulario
