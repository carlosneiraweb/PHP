oci_free_statement

(PHP 5, PHP 7, PECL OCI8 >= 1.1.0)

oci_free_statement — Libera todos los recursos asociados con una sentencia o cursor

Descripción

bool oci_free_statement ( resource $statement )

Libera los recursos asociados con un cursor o sentencia de Oracle, el cual fue recibido desde una resultado de oci_parse() u obtenido desde Oracle.

Parámetros

statement

Un identificador de sentecia válido de OCI.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.
