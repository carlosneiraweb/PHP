Collator::sortWithSortKeys

collator_sort_with_sort_keys

(PHP 5 >= 5.3.0, PHP 7, PECL intl >= 1.0.0)

Collator::sortWithSortKeys -- collator_sort_with_sort_keys — Ordenar un array usando el cotejador y las claves de ordenación especificados

Descripción

Estilo orientado a objetos

public bool Collator::sortWithSortKeys ( array &$arr )

Estilo por procedimientos

bool collator_sort_with_sort_keys ( Collator $coll , array &$arr )

Esta función es similar collator_sort() excepto que utiliza las claves de ordenación de ICU producidas por ucol_getSortKey() para conseguir más velocidad en arrays grandes.

Parámetros

coll

Un objeto Collator.

arr

El array de cadenas a ordenar.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.

Ejemplos

Ejemplo #1 Ejemplo de collator_sort_with_sort_keys()

<?php
$arr  = array( 'Köpfe', 'Kypper', 'Kopfe' );
$coll = collator_create( 'sv' );

collator_sort_with_sort_keys( $coll, $arr );
var_export( $arr );
?>

El resultado del ejemplo sería:

array (
  0 => 'Kopfe',
  1 => 'Kypper',
  2 => 'Köpfe',
)
