PDF_get_minorversion

(PHP 4 >= 4.2.0, PECL pdflib >= 1.0.0)

PDF_get_minorversion — Obtener el número de versión menor [obsoleta]

Descripción

int PDF_get_minorversion ( void )

Devuelve el número de versión menor de la versión de PDFlib.

Esta función está obsoleta desde la versión 5 de PDFlib, use PDF_get_value() con el parámetro minor en su lugar.
