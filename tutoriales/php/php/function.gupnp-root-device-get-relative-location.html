gupnp_root_device_get_relative_location

(PECL gupnp >= 0.1.0)

gupnp_root_device_get_relative_location — Obtiene la ubicación relativa del dispositivo raíz

Descripción

string gupnp_root_device_get_relative_location ( resource $root_device )

Obtiene la ubicación relativa del dispositivo raíz.

Parámetros

root_device

Un identificador de dispositivo raíz, devuelto por gupnp_root_device_new().

Valores devueltos

La ubicación relativa del dispositivo raíz
