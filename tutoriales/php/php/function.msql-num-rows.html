msql_num_rows

(PHP 4, PHP 5, PHP 7)

msql_num_rows — Obtiene el número de filas en el resultado

Descripción

int msql_num_rows ( resource $query_identifier )

msql_num_rows() devuelve el número de filas en el conjunto de resultados.

Parámetros

result

El resultado resource que está siendo evaluado. Este resultado proviene de una llamada a msql_query().

Valores devueltos

Devuelve el número de filas en el conjunto de resultados.
