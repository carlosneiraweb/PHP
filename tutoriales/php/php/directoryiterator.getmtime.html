DirectoryIterator::getMTime

(PHP 5, PHP 7)

DirectoryIterator::getMTime — Obtiene la última modificación para el elemento actual DirectoryIterator

Descripción

public int DirectoryIterator::getMTime ( void )

Obtiene la última modificación para el elemento actual DirectoryIterator, en formato Unix timestamp.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

la última modificación del fichero, en formato Unix timestamp.

Ejemplos

Ejemplo #1 Ejemplo de DirectoryIterator::getMTime()

Mostrará una lista de ficheros en el directorio donde se encuentra el script y su última fecha de modificación.

<?php
$iterator = new DirectoryIterator(dirname(__FILE__));
foreach ($iterator as $fileinfo) {
    if ($fileinfo->isFile()) {
        echo $fileinfo->getFilename() . " " . $fileinfo->getMTime() . "\n";
    }
}
?>

El resultado del ejemplo sería algo similar a:

manzana.jpg 1240047118
banana.jpg 1240065176
index.php 1240047208
pera.jpg 12240047979
