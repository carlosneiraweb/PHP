Mongo::getPoolSize

(PECL mongo >=1.2.0)

Mongo::getPoolSize — Obtener el tamaño de las agrupaciones de conexiones

Descripción

public static int Mongo::getPoolSize ( void )
Advertencia

Esta característica ha sido declarada OBSOLETA a partir de la versión 1.2.3. Se recomienda encarecidamente no confiar en ella. Por favor, use MongoPool::getSize() en su lugar.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve el tamaño de la agrupación actual.

Historial de cambios

Versión Descripción
1.2.11 Emite un error de nivel E_DEPRECATED al usarla.

Ejemplos

Ejemplo #1 Cambiar el tamaño de la agrupación

Devuelve el tamaño de la agrupación predeterminada, establece un nuevo tamaño, imprime ese nuevo tamaño y la información de depuración de la agrupación. Observe que cambiar el tamaño de la agrupación únicamente afecta a agrupaciones conexiones nuevas, no cambia las antiguas.

<?php

$conexión = new Mongo("host1");

// el tamaño de la agrupación es -1
echo "el tamaño de la agrupación es: ".Mongo::getPoolSize()."\n";

echo "establecer el tamaño de la agrupación a 200\n";

Mongo::setPoolSize(200);

// el tamaño de la agrupación es 200
echo "el tamaño de la agrupación es: ".Mongo::getPoolSize()."\n";

$conexión2 = new Mongo("host2");

// lo que queda para host1 es -2
// lo que queda para host2 es 199
var_dump(Mongo::poolDebug());

?>
