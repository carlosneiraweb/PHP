dbplus_rcrtlike

(PHP 4 <= 4.1.0, PECL dbplus >= 0.9)

dbplus_rcrtlike — Crea una copia vacía de una relación con los índeces predeterminados

Descripción

mixed dbplus_rcrtlike ( string $name , resource $relation [, int $overwrite ] )

dbplus_rcrtexact() creará una copia vacía de la relación dada por relation bajo un nuevo nombre dado por name, pero con los índices predeterminados.

Parámetros

name

relation

La relación copiada, abierta mediante dbplus_open().

overwrite

Una relación existente con el mismo nombre dado por name sólo será sobrescirta si este parámetro está establecido a TRUE y ningún otro proceso está actualmente usando la relación.

Valores devueltos

Devuelve un recurso si se tuvo éxito o DBPLUS_ERR_UNKNOWN si falló.

Notas

Advertencia

Esta función ha sido declarada EXPERIMENTAL. Su funcionamiento, nombre y la documentación que le acompaña puede cambiar sin previo aviso en futuras versiones de PHP. Utilícela bajo su propia responsabilidad.
