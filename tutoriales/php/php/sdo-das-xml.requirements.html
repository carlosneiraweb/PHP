Requerimientos

El Servicio de Acceso a Datos XML de SDO requiere PHP 5.1.0 o superior. Está empaquetado con la extensión SDO y requiere SDO para que sea instalado. Véase las instrucciones de instalación de SDO para los detalles de cómo realizarla.
