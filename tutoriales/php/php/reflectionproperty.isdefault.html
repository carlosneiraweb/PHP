ReflectionProperty::isDefault

(PHP 5, PHP 7)

ReflectionProperty::isDefault — Comprueba si una propiedad es predeterminada

Descripción

public bool ReflectionProperty::isDefault ( void )

Comprueba si la propiedad se declaró durante la compilación o si se declaró dinámicamente durante la ejecución.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

TRUE si la propiedad se declaró en tiempo de compilación, o FALSE si se creó en tiempo de ejecución.

Ejemplos

Ejemplo #1 Ejemplo de ReflectionClass::isDefault()

<?php
class Foo {
    public $bar;
}

$o = new Foo();
$o->bar = 42;
$o->baz = 42;

$ro = new ReflectionObject($o);
var_dump($ro->getProperty('bar')->isDefault());
var_dump($ro->getProperty('baz')->isDefault());
?>

El resultado del ejemplo sería:

bool(true)
bool(false)
