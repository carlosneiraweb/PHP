Locale::getDisplayName

locale_get_display_name

(PHP 5 >= 5.3.0, PHP 7, PECL intl >= 1.0.0)

Locale::getDisplayName -- locale_get_display_name — Devolver un nombre manifiesto debidamente localizado de la configuración regional de entrada

Descripción

Estilo orientado a objetos

public static string Locale::getDisplayName ( string $locale [, string $in_locale ] )

Estilo por procedimientos

string locale_get_display_name ( string $locale [, string $in_locale ] )

Devuelve un nombre manifiesto debidamente localizado de la configuración regional de entrada. Si es NULL se utiliza la configuración regional predeterminada.

Parámetros

locale

La configuración regional de la que devolver un nombre manifiesto.

in_locale

Formato regional opcional

Valores devueltos

El nombre manifiesto de la configuración regional con el formato apropiado dado por $in_locale.

Ejemplos

Ejemplo #1 Ejemplo de locale_get_display_name()

<?php
echo locale_get_display_name('sl-Latn-IT-nedis', 'en');
echo ";\n";
echo locale_get_display_name('sl-Latn-IT-nedis', 'fr');
echo ";\n";
echo locale_get_display_name('sl-Latn-IT-nedis', 'de');
?>

Ejemplo #2 Ejemplo orientado a objetos

<?php
echo Locale::getDisplayName('sl-Latn-IT-nedis', 'en');
echo ";\n";
echo Locale::getDisplayName('sl-Latn-IT-nedis', 'fr');
echo ";\n";
echo Locale::getDisplayName('sl-Latn-IT-nedis', 'de');
?>

El resultado del ejemplo sería:

Slovenian (Latin, Italy, Natisone dialect);
slov\xc3\xa8ne (latin, Italie, dialecte de Natisone;
Slowenisch (Lateinisch, Italien, NEDIS)
