fann_set_rprop_delta_zero

(PECL fann >= 1.0.0)

fann_set_rprop_delta_zero — Establece el tamaño de paso inicial

Descripción

bool fann_set_rprop_delta_zero ( resource $ann , float $rprop_delta_zero )

El tamaño de paso inicial es un número positivo que determina el tamaño de paso inicial.

Parámetros

ann

Un resource de red neuronal.

rprop_delta_zero

El tamaño de paso inicial.

Valores devueltos

Devuelve TRUE en caso de éxito, o FALSE de lo contrario.
