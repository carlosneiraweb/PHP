Transliterator::createFromRules

transliterator_create_from_rules

(PHP >= 5.4.0, PECL intl >= 2.0.0)

Transliterator::createFromRules -- transliterator_create_from_rules — Crear un transliterador desdes normas

Descripción

Estilo orientado a objetos

public static Transliterator Transliterator::createFromRules ( string $rules [, string $direction ] )

Estilo por procedimientos

Transliterator transliterator_create_from_rules ( string $id [, int $direction ] )

Crea un transliterador desdes normas.

Advertencia

Esta función no está documentada actualmente, solamente se encuentra disponible la lista de parámetros.

Parámetros

rules

Las normas.

direction

La dirección, por defecto es >Transliterator::FORWARD. También se puede establecer a Transliterator::REVERSE.

Valores devueltos

Devuelve un objeto de la clase Transliterator en caso de éxito, o NULL en caso de error.
