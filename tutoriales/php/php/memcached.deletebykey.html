Memcached::deleteByKey

(PECL memcached >= 0.1.0)

Memcached::deleteByKey — Eliminar un ítem de un servidor específico

Descripción

public bool Memcached::deleteByKey ( string $server_key , string $key [, int $time = 0 ] )

La funcionalidad de Memcached::deleteByKey() es equivalente a la de Memcached::delete(), con la excepción de que server_key se puede utilizar para que la clave dada por key corresponda a un servidor específico.

Parámetros

server_key

La clave que identifica al servidor donde guardar o recuperar el valor. En lugar de realizar un hash en la clave real para el ítem, se realiza en la clave del servidor cuando se decide con qué servidor de memcached conversar. Esto permite que los ítems relacionados sean agrupados en un único servidor para una mayor eficacia con operaciones múltiples.

key

La clave a eliminar.

time

La cantidad de tiempo que el servidor esperará para eliminar el ítem.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error. Memcached::getResultCode() devolverá Memcached::RES_NOTFOUND si la clave no existe.
