judy_type

(PECL judy >= 0.1.1)

judy_type — Devuelve el tipo de un array de Judy

Descripción

int judy_type ( Judy $array )

judy_type() devuelve un tipo de dato entero correspondiente al tipo de dato de Judy de su array específico.

Parámetros

array

El array de Judy a probar.

Valores devueltos

Devuelve un tipo de dato entero correspondiente a un tipo de dato de Judy.
