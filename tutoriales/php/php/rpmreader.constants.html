Constantes predefinidas

Estas constantes están definidas por esta extensión y estarán disponibles sólo cuando la extensión haya sido compilada con PHP, o bien sea cargada dinámicamente en ejecución.

La siguiente lista de constantes son usadas para obtener información al usar la función rpm_get_tag(). Estas constantes son representadas por el número de etiqueta a ser obtenida de la sección de encabezado del fichero RPM. A continuación se muestran las descripciones y a que número de constantes corresponden.

RPMREADER_MINIMUM (integer)
El valor mínimo de cualquier valor de cualquier número de etiqueta RPM.
RPMREADER_NAME (integer)
El nombre del paquete RPM.
RPMREADER_VERSION (integer)
La versión del paquete RPM.
RPMREADER_RELEASE (integer)
La release del paquete RPM.
RPMREADER_EPOCH (integer)
RPMREADER_SERIAL (integer)
RPMREADER_SUMMARY (integer)
El texto sumario del paquete RPM.
RPMREADER_DESCRIPTION (integer)
El texto de la descripción completa del paquete RPM.
RPMREADER_BUILDTIME (integer)
La fecha y hora de cuando se compiló el paquete RPM.
RPMREADER_BUILDHOST (integer)
El nombre del host en donde se compiló el paquete RPM.
RPMREADER_INSTALLTIME (integer)
RPMREADER_SIZE (integer)
El tamaño del paquete RPM.

RPMREADER_OLDFILENAMES (integer)
La lista de ficheros de un paquete RPM (formato absoleto). La forma correcta de hacerlo es usando la combinación de estas 3 etiquetas (RPMREADER_BASENAMES, RPMREADER_DIRINDEXES, RPMREADER_DIRNAMES) en la que RPM ahora se llama "CompressedFileNames". Esta etiqueta todavía se sigue usando en ficheros RPM antiguos que no usaron el método "CompressedFileNames" y se mantiene para compatibilidad con versiones anteriores.
