Memcached::setSaslAuthData

(PECL memcached >= 2.0.0)

Memcached::setSaslAuthData — Establecer las credenciales para la autenticación

Descripción

public void Memcached::setSaslAuthData ( string $username , string $password )

Memcached::setSaslAuthData() establece el nombre de usuario y la contraseña que deberían utilizarse para la autenticación SASL con los servidores de memcache.

Este método solamente está disponible cuando la extensión memcached está construida con soporte para SASL. Consulte la Configuración de memcached para saber cómo se hace esto.

Parámetros

username

El nombre de usuario para la autenticación.

password

La contraseña para la autenticación.

Valores devueltos

No devuelve ningún valor.
