SolrQuery::getTermsLowerBound

(PECL solr >= 0.9.2)

SolrQuery::getTermsLowerBound — Devuelve el término en el que comenzar

Descripción

public string SolrQuery::getTermsLowerBound ( void )

Devuelve el término en el que comenzar

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve una cadena en caso de éxito y NULL si no se estableció.
