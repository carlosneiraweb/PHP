La clase CURLFile

(PHP 5 >= 5.5.0, PHP 7)

Introducción

CURLFile debería emplearse para subir un fichero con CURLOPT_POSTFIELDS.

Sinopsis de la Clase

CURLFile {
/* Propiedades */
public $name ;
public $mime ;
public $postname ;
/* Métodos */
public __construct ( string $filename [, string $mimetype [, string $postname ]] )
public string getFilename ( void )
public string getMimeType ( void )
public string getPostFilename ( void )
public void setMimeType ( string $mime )
public void setPostFilename ( string $postname )
public void __wakeup ( void )
}

Propiedades

name

Nombrte del fichero a subir.

mime

Tipo MIME del fichero (el predeterminado es application/octet-stream).

postname

El nombre del fichero en los datos de subida (lo predeterminado es la propiedad name).
