XMLWriter::startDTDEntity

xmlwriter_start_dtd_entity

(PHP 5 >= 5.1.2, PHP 7, PECL xmlwriter >= 0.1.0)

XMLWriter::startDTDEntity -- xmlwriter_start_dtd_entity — Crea un ente DTD inicial

Descripción

Estilo orientado a objetos

bool XMLWriter::startDTDEntity ( string $name , bool $isparam )

Estilo por procedimientos

bool xmlwriter_start_dtd_entity ( resource $xmlwriter , string $name , bool $isparam )

Inicia un ente DTD.

Parámetros

xmlwriter

Sólo para llamadas por procedimientos. El resource XMLWriter que está siendo modificado. Este recurso proviene de una llamada a xmlwriter_open_uri() o xmlwriter_open_memory().

name

El nombre de la entidad.

isparam

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.

  • XMLWriter::endDTDEntity() - Finaliza el actual ente DTD
  • XMLWriter::writeDTDEntity() - Escribe una etiqueta completa de un ente DTD
