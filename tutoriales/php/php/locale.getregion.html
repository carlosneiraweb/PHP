Locale::getRegion

locale_get_region

(PHP 5 >= 5.3.0, PHP 7, PECL intl >= 1.0.0)

Locale::getRegion -- locale_get_region — Obtener la región de la configuración local de entrada

Descripción

Estilo orientado a objetos

public static string Locale::getRegion ( string $locale )

Estilo por procedimientos

string locale_get_region ( string $locale )

Obtiene la región de la configuración local de entrada.

Parámetros

locale

La configuración regional desde la que extraer el código de región

Valores devueltos

La subetiqueta de región de la configuración regional o NULL si no está presente

Ejemplos

Ejemplo #1 Ejemplo de locale_get_region()

<?php
echo locale_get_region('de-CH-1901');
?>

Ejemplo #2 Ejemplo orientado a objetos

<?php
echo Locale::getRegion('de-CH-1901');
?>

El resultado del ejemplo sería:

CH
