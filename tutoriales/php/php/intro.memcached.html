Introducción

» memcached es un sistema de alto rendimiento para el almacenamiento de objetos en caché de memoria distribuida, genérico por naturaleza, aunque pensado principalmente para acelerar aplicaciones web dinámicas aliviando la carga de bases de datos.

Esta extensión utiliza la biblioteca libmemcached para proveer una API de comunicación con servidores de memcached. También proporciona un gestor de sesiones (memcached).

Puede encontrarse más información acerca de libmemcached en » http://libmemcached.org/libMemcached.html.
