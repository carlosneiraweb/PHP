newt_wait_for_key

(PECL newt >= 0.1)

newt_wait_for_key — No devolver hasta que se pulse una tecla

Descripción

void newt_wait_for_key ( void )

Esta función no devuelve hasata que se pulse una tecla. La pulsación de la tecla es ignorada. Si una tecla ya está en el búfer del terminal, esta función desecha una pulsación de una tecla y devuelve inmediatamente.

Valores devueltos

No devuelve ningún valor.

Ver también

  • newt_clear_key_buffer() - Desecha el contenido del búfer de entrada del terminal sin esperar a entradas adicionales
