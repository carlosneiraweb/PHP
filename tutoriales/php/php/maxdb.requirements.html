Requerimientos

Con el fin de tener estas funciones disponibles, se debe compilar PHP con soporte para MaxDB. Además, debe tener disponible la librería MaxDB SQLDBC para acceder al servidor MaxDB.

Documentación para MaxDB SQLDBC se puede encontrar en » http://maxdb.sap.com/documentation/.

Se puede descargar el paquete MaxDB SQLDBC de » http://www.sdn.sap.com/irj/sdn/maxdb-downloads.
