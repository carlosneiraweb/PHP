XMLWriter::writeComment

xmlwriter_write_comment

(PHP 5 >= 5.1.2, PHP 7, PECL xmlwriter >= 0.1.0)

XMLWriter::writeComment -- xmlwriter_write_comment — EScribe la etiqueta del comentario completa

Descripción

Estilo orientado a objetos

bool XMLWriter::writeComment ( string $content )

Estilo por procedimientos

bool xmlwriter_write_comment ( resource $xmlwriter , string $content )

Escribe un comentario completo.

Parámetros

xmlwriter

Sólo para llamadas por procedimientos. El resource XMLWriter que está siendo modificado. Este recurso proviene de una llamada a xmlwriter_open_uri() o xmlwriter_open_memory().

content

El contenido del comentario.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.

Ver también

  • XMLWriter::startComment() - Crea un comentario inicial
  • XMLWriter::endComment() - Crea un comentario final
