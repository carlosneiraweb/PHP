variant_get_type

(PHP 5, PHP 7)

variant_get_type — Devuelve el tipo de un objeto variante

Descripción

int variant_get_type ( variant $variant )

Devuelve el tipo de un objeto variante.

Parámetros

variant

El objeto variante.

Valores devueltos

Esta función devuelve un valor de tipo integer que indica el tipo de variant, que puede ser una instancia de las clases COM, DOTNET o VARIANT. El valor devuelto puede ser comparado con una de las constantes VT_XXX.

El valor devuelto para objetos COM y DOTNET será normalmente VT_DISPATCH; la única razón de que esta función funcione con estas clases es que COM y DOTNET son descendientes de VARIANT.

En versiones de PHP anteriores a 5 se podría obtener esta información ÚNICAMENTE desde instancias de la clase VARIANT, leyendo una propiedad type falsa. Véase la clase VARIANT para más información sobre esto.
