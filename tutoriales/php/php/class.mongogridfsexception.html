Clase MongoGridFSException

(PECL mongo >= 1.0.0)

Introducción

Lanzada cuando hay errores al leer o escribir ficheros hacia o desde la base de datos.

Sinopsis de la Clase

MongoGridFSException extends MongoException {
}
