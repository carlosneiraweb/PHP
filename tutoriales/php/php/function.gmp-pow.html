gmp_pow

(PHP 4 >= 4.0.4, PHP 5, PHP 7)

gmp_pow — Aumenta el número a la potencia

Descripción

GMP gmp_pow ( GMP $base , int $exp )

Aumenta la base a la potencia exp.

Parámetros

base

La base del número.

resource de número de GMP en PHP 5.5 y anterior, o un objeto de GMP en PHP 5.6 y posterior, o un string numérico siempre que sea posible convertirlo a un número.

exp

La potencia positiva a elevar la base.

Valores devueltos

El nuevo (elevado) número, como un número GMP. El caso de 0^0 produce 1.

Ejemplos

Ejemplo #1 Ejemplo de gmp_pow()

<?php
$pow1 = gmp_pow("2", 31);
echo gmp_strval($pow1) . "\n";
$pow2 = gmp_pow("0", 0);
echo gmp_strval($pow2) . "\n";
$pow3 = gmp_pow("2", -1); // exponente negativo, genera peligro
echo gmp_strval($pow3) . "\n";
?>

El resultado del ejemplo sería:

2147483648
1
