La clase SoapClient

(PHP 5 >= 5.0.1, PHP 7)

Introducción

La clase SoapClient proporciona un cliente para servidores » SOAP 1.1, » SOAP 1.2. Puede ser usada en modo WSDL o modo no-WSDL.
