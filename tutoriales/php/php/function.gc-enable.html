gc_enable

(PHP 5 >= 5.3.0, PHP 7)

gc_enable — Activa el colector de referencia circular

Descripción

void gc_enable ( void )

Activa el colector de referencia circular, estableciendo zend.enable_gc a 1.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

No devuelve ningún valor.
