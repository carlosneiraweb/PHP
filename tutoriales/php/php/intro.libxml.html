Introducción

Estas funciones/constantes están disponibles a partir de PHP 5.1.0. Las siguientes extensiones del núcleo dependen de esta extensión libxml: DOM, libxml, SimpleXML, SOAP, WDDX, XSL, XML, XMLReader, XMLRPC y XMLWriter.
