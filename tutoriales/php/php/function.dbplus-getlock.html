dbplus_getlock

(PHP 4 <= 4.1.0, PECL dbplus >= 0.9)

dbplus_getlock — Obtener un bloqueo de escritura en una tupla

Descripción

int dbplus_getlock ( resource $relation , string $tuple )

Solicita un bloqueo de escritura en la tupla tuple especificada.

Parámetros

relation

Una relación abierta con dbplus_open().

tuple

Valores devueltos

Devuelve cero si se tuvo éxito o un código de error distinto de cero, especialmente DBPLUS_ERR_WLOCKED, si falló.

Notas

Advertencia

Esta función ha sido declarada EXPERIMENTAL. Su funcionamiento, nombre y la documentación que le acompaña puede cambiar sin previo aviso en futuras versiones de PHP. Utilícela bajo su propia responsabilidad.
