GearmanWorker::returnCode

(PECL gearman >= 0.5.0)

GearmanWorker::returnCode — Obtiene el último código de retorno de Gearman

Descripción

public int GearmanWorker::returnCode ( void )

Retorna el último código de retorno de Gearman.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Un código de retorno válido de Gearman.
