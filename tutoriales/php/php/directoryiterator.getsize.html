DirectoryIterator::getSize

(PHP 5, PHP 7)

DirectoryIterator::getSize — obtiene el tamaño de el elemento actual DirectoryIterator

Descripción

public int DirectoryIterator::getSize ( void )

Obtiene el tamaño del fichero de el elemento actual DirectoryIterator.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve el tamaño del fichero en bytes.

Ejemplos

Ejemplo #1 Ejemplo de DirectoryIterator::getSize()

<?php
$iterator = new DirectoryIterator(dirname(__FILE__));
foreach ($iterator as $fileinfo) {
    if ($fileinfo->isFile()) {
        echo $fileinfo->getFilename() . " " . $fileinfo->getSize() . "\n";
    }
}
?>

El resultado del ejemplo sería algo similar a:

manzana.jpg 15385
banana.jpg 15190
ejemplo.php 170
pera.jpg 34406
