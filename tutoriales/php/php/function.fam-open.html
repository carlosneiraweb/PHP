fam_open

(PHP 5 <= 5.0.5)

fam_open — Abre una conexión al demonio FAM

Descripción

resource fam_open ([ string $appname ] )

Abre una conexión al demonio del servicio FAM.

Parámetros

appname

Una cadena que identifica la aplicación por razones de registro

Valores devueltos

Devuelve un recurso que representa una conexión al servicio FAM al tener éxito o FALSE en caso de error.

  • fam_close() - Cierra la conexión FAM
