Imagick::getImageOrientation

(PECL imagick 2.0.0)

Imagick::getImageOrientation — Obtiene la orientación de la imagen

Descripción

int Imagick::getImageOrientation ( void )

Obtiene la orientación de la imagen. El valor de retorno es una de las constantes de orientación.

Valores devueltos

Devuelve un integer si se tuvo éxito.

Errores/Excepciones

Lanza ImagickException en caso de error.
