HaruPage::setGrayFill

(PECL haru >= 0.0.1)

HaruPage::setGrayFill — Establece el color de relleno para la página

Descripción

bool HaruPage::setGrayFill ( float $value )

Define el color de relleno para la página.

Parámetros

value

El valor de gris entre 0 y 1.

Valores devueltos

Devuelve TRUE en éxito.

Errores/Excepciones

Lanza una HaruException cuando se produce un error.
