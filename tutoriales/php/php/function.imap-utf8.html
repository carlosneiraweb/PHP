imap_utf8

(PHP 4, PHP 5, PHP 7)

imap_utf8 — Convierte texto codificado MIME en UTF-8

Descripción

string imap_utf8 ( string $mime_encoded_text )

Convierte el parámetro dado por mime_encoded_text en UTF-8.

Parámetros

mime_encoded_text

Una cadena codificada MIME. El método de codificación MIME y la especificación UTF-8 están descritas en » RFC2047 y » RFC2044 respectivamente.

Valores devueltos

Devuelve una cadena codificada en UTF-8.
