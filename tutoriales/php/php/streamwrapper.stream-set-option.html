streamWrapper::stream_set_option

(PHP 5 >= 5.3.0, PHP 7)

streamWrapper::stream_set_option — Cambiar las opciones del flujo

Descripción

public bool streamWrapper::stream_set_option ( int $option , int $arg1 , int $arg2 )

Este método es llamado para establecer las opciones del flujo.

Parámetros

option

Una de las siguientes opciones:

  • STREAM_OPTION_BLOCKING (El método fue llamado en respuesta a stream_set_blocking())
  • STREAM_OPTION_READ_TIMEOUT (El método fue llamado en respuesta a stream_set_timeout())
  • STREAM_OPTION_WRITE_BUFFER (El método fue llamado en respuesta a stream_set_write_buffer())

arg1

Si option es

  • STREAM_OPTION_BLOCKING: modo de bloqueo solicitado (1 significa bloqueo, 0 sin bloqueo).
  • STREAM_OPTION_READ_TIMEOUT: tiempo de espera en segundos.
  • STREAM_OPTION_WRITE_BUFFER: modo del buffer (STREAM_BUFFER_NONE o STREAM_BUFFER_FULL).

arg2

Si option es

  • STREAM_OPTION_BLOCKING: Esta opción no está establecida.
  • STREAM_OPTION_READ_TIMEOUT: tiempo de espera en microsegundos.
  • STREAM_OPTION_WRITE_BUFFER: el tamaño del buffer solicitado.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error. Si option no está implementado, debería devolver FALSE..

Ver también

  • stream_set_blocking() - Establecer el modo bloqueo/no-bloqueo en un flujo
  • stream_set_timeout() - Establecer un perido de tiempo de espera en un flujo
  • stream_set_write_buffer() - Establece el búfer para escritura de ficheros en el flujo dado
