HaruDoc::getCurrentEncoder

(PECL haru >= 0.0.1)

HaruDoc::getCurrentEncoder — Obteniene la HaruEncoder actualmente usada en el documento

Descripción

object HaruDoc::getCurrentEncoder ( void )

Obteniene la HaruEncoder actualmente usada en el documento.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve la HaruEncoder actualmente usada en el documento o FALSE si el codificador no está establecido.

Errores/Excepciones

Lanza una HaruException cuando se produce un error.
