MongoCursor::current

(PECL mongo >=0.9.0)

MongoCursor::current — Devuelve el elemento actual

Descripción

public array MongoCursor::current ( void )

Devuelve NULL hasta que se llame a MongoCursor::next().

Parámetros

Esta función no tiene parámetros.

Valores devueltos

El documento de resultados actual como array asociativo. Se devolverá NULL si no hubiera resultados.
