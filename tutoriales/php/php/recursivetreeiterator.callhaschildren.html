RecursiveTreeIterator::callHasChildren

(PHP 5 >= 5.3.0, PHP 7)

RecursiveTreeIterator::callHasChildren — Comprobar si tiene hijos

Descripción

public bool RecursiveTreeIterator::callHasChildren ( void )

Se llama para cada elemento para comprobar si tiene hijos o no.

Advertencia

Esta función no está documentada actualmente, solamente se encuentra disponible la lista de parámetros.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

TRUE si tiene hijos, en caso contrario FALSE.
