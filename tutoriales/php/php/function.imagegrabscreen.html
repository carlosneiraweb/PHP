imagegrabscreen

(PHP 5 >= 5.2.2, PHP 7)

imagegrabscreen — Capturar la pantalla completa

Descripción

resource imagegrabscreen ( void )

Toma una captura de pantalla de la pantalla completa.

Valores devueltos

Devuelve un identificador de recurso de imagen si tiene éxito, FALSE si falló.

Ejemplos

Ejemplo #1 Ejemplo de imagegrabscreen()

Este ejemplo demuestra cómo tomar una captura de pantalla de la pantalla actual y guardarla como una imagen png.

<?php
$im = imagegrabscreen();
imagepng($im, "mi_captura_de_pantalla.png");
imagedestroy($im);
?>

Notas

Nota:

Esta función sólo está disponible en Windows.
