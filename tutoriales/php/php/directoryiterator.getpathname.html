DirectoryIterator::getPathname

(PHP 5, PHP 7)

DirectoryIterator::getPathname — Devuelve la ruta y nombre de fichnero del elemento actual DirectoryIterator

Descripción

public string DirectoryIterator::getPathname ( void )

Obtiene la ruta y nombre de el fichero actual.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve la ruta y el nombre del fichero actual. Directorios no tienen slash.

Ejemplos

Ejemplo #1 Ejemplo de DirectoryIterator::getPathname()

<?php
$iterator = new DirectoryIterator(dirname(__FILE__));
foreach ($iterator as $fileinfo) {
    echo $fileinfo->getPathname() . "\n";
}
?>

El resultado del ejemplo sería algo similar a:

/home/examples/.
/home/examples/..
/home/examples/manzana.jpg
/home/examples/banana.jpg
/home/examples/getpathname.php
/home/examples/pera.jpg
