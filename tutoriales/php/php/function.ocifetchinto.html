ocifetchinto

(PHP 4, PHP 5, PHP 7, PECL OCI8 >= 1.0.0)

ocifetchinto — Variante obsoleta de oci_fetch_array(), oci_fetch_object(), oci_fetch_assoc() y oci_fetch_row()

Descripción

Variante absoleta de oci_fetch_array(), oci_fetch_object(), oci_fetch_assoc() y oci_fetch_row()

Advertencia

Este alias ha sido declarado OBSOLETO desde PHP 5.4.0. Su uso está totalmente desaconsejado.
