filepro_fieldtype

(PHP 4, PHP 5 <= 5.1.6, PHP 7)

filepro_fieldtype — Obtiene el tipo de campo

Descripción

string filepro_fieldtype ( int $field_number )

Devuelve el tipo de edición del campo correspondiente a field_number.

Parámetros

field_number

El campo numérico.

Valores devueltos

Devuelve el tipo de campo de edición como cadena, o FALSE en caso de error.
