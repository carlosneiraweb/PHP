IntlChar::isIDStart

(PHP 7)

IntlChar::isIDStart — Comprobar si un punto de código es permisible como el primer carácter de un identificador

Descripción

public static bool IntlChar::isIDStart ( mixed $codepoint )

Determna si el carácter especficado es permisible como el primer carácter de un identificador según Unicode (El Estándar de Unicode, Versión 3.0, capítulo 5.16 Identificadores).

TRUE para caracteres con categorías generales "L" (letras) y "Nl" (números de letra).

Parámetros

codepoint

El valor de tipo integer del punto de código (p.ej. 0x2603 para U+2603 SNOWMAN), o el carácter codificado como un string UTF-8 (p.ej. "\u{2603}")

Valores devueltos

Devuelve TRUE si codepoint podría iniciar un identificador, FALSE si no.

Ejemplos

Ejemplo #1 Probar diferentes puntos de código

<?php
var_dump(IntlChar::isIDStart("A"));
var_dump(IntlChar::isIDStart("$"));
var_dump(IntlChar::isIDStart("\n"));
var_dump(IntlChar::isIDStart("\u{2603}"));
?>

El resultado del ejemplo sería:

bool(true)
bool(false)
bool(false)
bool(false)
