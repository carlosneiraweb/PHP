Gmagick::normalizeimage

(PECL gmagick >= Unknown)

Gmagick::normalizeimage — Mejora el contraste de una imagen a color

Descripción

public Gmagick Gmagick::normalizeimage ([ int $channel ] )

Mejora el contraste de una imagen a color ajustando los colores de los píxeles para abarcar el rango completo de colores disponibles.

Parámetros

channel

Identifica el canal a normalizar

Valores devueltos

El objeto Gmagick si se tuvo éxito

Errores/Excepciones

Arroja un GmagickException en caso de error.
