HaruDoc::loadTTC

(PECL haru >= 0.0.1)

HaruDoc::loadTTC — Carga la fuente con el indice especificado del archivo TTC

Descripción

string HaruDoc::loadTTC ( string $fontfile , int $index [, bool $embed = false ] )

Carga la fuente TruType con el indice especificado de un archivo de colección TrueType.

Parámetros

fontfile

El archivo de colección TrueType.

index

El indice de la fuente en el archivo de colección.

embed

Cuando se establece a TRUE, los datos del glifo de la fuente son encrustados dentro del archivo PDF, sin embargo solo la matriz de los datos es incluida.

Valores devueltos

Deveuleve el nombre de la fuente cargada en éxito.

Errores/Excepciones

Lanza una HaruException cuando se produce un error.
