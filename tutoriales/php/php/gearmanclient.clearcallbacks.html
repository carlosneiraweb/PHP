GearmanClient::clearCallbacks

(PECL gearman >= 0.5.0)

GearmanClient::clearCallbacks — Elimina todas las funciones de retorno de llamada

Descripción

public bool GearmanClient::clearCallbacks ( void )

Elimina todas las funciones de retorno de llamada establecidas anteriormente.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Retorna siempre TRUE.
