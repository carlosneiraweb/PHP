La clase MongoTimestamp

(PECL mongo >=1.0.1)

Advertencia

Esta extensión que define esta clase está obsoleta. En su lugar debe usarse la extensión MongoDB. Las alternativas a esta clase son:

  • MongoDB\BSON\Timestamp

Introducción

MongoTimestamp es un tipo interno empleado por MongoDB para la replicación y fragmentación. Consiste en una marca temporal (es decir, segundos desde la época) de 4 bytes y un incremento de 4 bytes. Este tipo no está pensado para almacenar valores de horas o fechas (p.ej., un campo "creadoEn" de un documento).

Nota: A menos que se esté escribiendo una aplicación que interactue con el registro de operaciones de replicación o con los componentes de fragmentación de MongoDB: pare, vaya directamente a MongoDate, sin pasar por la casilla de salida y sin cobrar 200 dólares. Esta no es la clase que está buscando.
