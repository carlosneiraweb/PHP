Nuevos métodos

Se han introducido varios métodos nuevos en la versión 5.3.0:

Fecha/Hora:

  • DateTime::add() - Añade una cantidad de días, meses, años, horas, minutos y segundos al objeto DateTime.
  • DateTime::createFromFormat() - Devuelve un nuevo objeto DateTime formateado según el formato especificado.
  • DateTime::diff() - Devuelve la diferencia entres dos objetos DateTime.
  • DateTime::getLastErrors() - Devuelve las advertencias y los errores de la última operación de fecha/hora.
  • DateTime::sub() - Sustrae una cantidad de días, meses, años, horas, minutos y segundos de un objeto DateTime.

Excepciones:

  • Exception::getPrevious() - Obtiene la excepción anterior.

DOM:

  • DOMNode::getLineNo() - Obtiene el número de línea del nodo analizado.

PDO_FIREBIRD:

  • PDO::setAttribute() - Establece un atributo.

Reflection:

  • ReflectionClass::getNamespaceName() - Obtiene el nombre espacio de nombres en el que esta clase está definida.
  • ReflectionClass::getShortName() - Devuelve el nombre reducido de esta clase (sin el espacio de nombres).
  • ReflectionClass::inNamespace() - Comprueba si esta clase está definida dentro de un espacio de nombres.
  • ReflectionFunction::getNamespaceName() - Obtiene el nombre del espacio de nombres en el que esta función está definida.
  • ReflectionFunction::getShortName() - Devuelve el nombre reducido de esta función (sin el espacio de nombres).
  • ReflectionFunction::inNamespace() - Comprueba si esta función está definida dentro de un espacio de nombres.
  • ReflectionProperty::setAccessible() - Establece si se pueden requerir propiedades no públicas.

SPL:

  • SplObjectStorage::addAll() - Añade todos los elementos de otro objeto SplObjectStorage.
  • SplObjectStorage::removeAll() - Eliminar todos los elementos de otro objeto SplObjectStorage.

XSL:

  • XSLTProcessor::setProfiling() - Asigna el fichero de salida de información.
