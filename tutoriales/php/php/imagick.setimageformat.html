Imagick::setImageFormat

(PECL imagick 2.0.0)

Imagick::setImageFormat — Establece el formato de una imagen en particular

Descripción

bool Imagick::setImageFormat ( string $format )

Establece el formato de una imagen en particular en una secuencia.

Parámetros

format

La representación string de un formato de imagen. Los formatos soportados dependen de la instalación de ImageMagick.

Valores devueltos

Devuelve TRUE en caso de éxito.
