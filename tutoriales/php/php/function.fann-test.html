fann_test

(PECL fann >= 1.0.0)

fann_test — Probar con un conjunto de entradas, y un conjunto de salidas deseadas

Descripción

bool fann_test ( resource $ann , array $input , array $desired_output )

Probar con un conjunto de entradas, y un conjunto de salidas deseadas. Esta operación actualiza el error cuadrático medio, aunque no cambia la red de ninguna manera.

Parámetros

ann

Un resource de red neuronal.

input

Un array de entradas. Este array debe ser exactamente de fann_get_num_input() de longitud.

desired_output

Un array de salidas deseadas. Este array debe ser exactamente de fann_get_num_output() de longitud.

Valores devueltos

Devuelve TRUE en caso de éxito, o FALSE de lo contrario.

Ver también

  • fann_test_data() - Prueba un conjunto de datos de entrenamiento y calcula el ECM de dichos datos
  • fann_train() - Entrenar una iteración con un conjunto de entradas y un conjunto de salidas deseadas
  • fann_get_num_input() - Obtener el número de neuronas de entrada
  • fann_get_num_output() - Obtener el número de neuronas de salida
