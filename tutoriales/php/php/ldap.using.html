Utilizando las llamadas PHP LDAPs

Antes de que pueda utilizar las llamadas LDAP necesitará saber ...

  • El nombre o dirección del servidor de directorio que utilizará

  • El "dn base" del servidor (la parte del directorio del mundo que es mantenida en este servidor, la que podría ser "o=Mi Compañía,c=US")

  • Puede ser que necesite una contraseña para acceder al servidor (muchos servidores le proveerán acceso de lectura para una "vinculación anónima" pero requieren una contraseña para cualquier otra cosa)

La secuencia típica de llamadas LDAP que hará en una aplicación va a seguir este patrón: 

  ldap_connect()    // Establecer una conexión al servidor
     |
  ldap_bind()       // Una "entrada" anónima o autenticada
     |
  Realizar algo como una búsqueda o actualizar el directorio
  y mostrar los resultados
     |
  ldap_close()      // "Cerrar la sesión"
