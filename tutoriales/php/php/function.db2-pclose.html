db2_pclose

(PECL ibm_db2 >= 1.8.0)

db2_pclose — Cerrar una conexión persistente a la base de datos

Descripción

bool db2_pclose ( resource $resource )

Cierra una conexión cliente de DB2 creada con db2_pconnect() y devuelve los recursos correspondientes al servidor de la base de datos.

Nota:

Esta función solo está disponible en i5/OS es respuesta al sistema de administración de i5/OS.

Si se tiene una conexión persistente a DB2 creda con db2_pconnect(), se puede usar esta función para cerrar la conexión. Para evitar reducciones en el rendimiento de las conexiones, esta función solo debería usarse en casos especiales donde la conexión persistente no responda o la conexión no se necesite por mucho tiempo.

Parámetros

connection

Especifica una conexión cliente activa de DB2.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.

Ejemplos

Ejemplo #1 Cerrando una conexión persistente

El siguiente ejemplo demuestra como cerrar una conexión a una base de datos en IBM DB2 i5/OS.

<?php
$conn = db2_pconnect('', '', '');
$rc = db2_pclose($conn);
if ($rc) {
    echo "Connection was successfully closed.";
}
?>

El resultado del ejemplo sería:

Connection was successfully closed.
