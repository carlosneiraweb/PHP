iis_get_server_by_path

(PECL iisfunc SVN)

iis_get_server_by_path — Devuelve el número de instancia asociado con Path

Descripción

int iis_get_server_by_path ( string $path )

Cada servidor virtual de IIS está asociado con número de instancia. iis_get_server_by_path() busca el número de instancia desde la ruta actual hacia el directorio raíz.

Parámetros

path

La ruta al directorio raíz

Valores devueltos

Devuelve el número de instancia del servidor.
