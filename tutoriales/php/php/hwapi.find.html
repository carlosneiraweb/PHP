hw_api::find

(PHP 4, PHP 5 < 5.2.0, PECL hwapi SVN)

hw_api::find — Busca objetos

Descripción

array hw_api::find ( array $parameter )

Esta función busca objetos ejecutando una clave o/y una query de texto completo. Los objetos encontrados pueden ser filtrados posteriormente por un objeto query opcional. Están ordenados según su importancia. La segunda operación de búsqueda es relativamente lenta y su resultado puede ser limitado a un número determinado de valores. Esto permite realizar una búsqueda incremental, devolviendo únicamente un subconjunto de todos los documentos encontrados, empezando desde un índice dado.

Parámetros

parameter

El parámetro es un array que contiene el 'keyquery' o/y 'fulltextquery' dependiendo de lo que se deseee buscar. Los parámetros opcionales son: 'objectquery', 'scope', 'languages' y 'attributeselector'. En el caso de una búsqueda incremental, los parámetros opcionales 'startIndex', 'numberOfObjectsToGet' y 'exactMatchUnit' pueden ser pasados.
