rpm_open

(PECL rpmreader >= 0.1.0)

rpm_open — Abre un fichero RPM

Descripción

resource rpm_open ( string $filename )

rpm_open() abrirá un fichero RPM y comprobará si es un fichero RPM válido.

Parámetros

filename

El nombre del fichero RPM que se desea abrir.

Valores devueltos

Si el fichero se puede abrir, la función rpm_open() devolverá un puntero al fichero que se acaba de abrir. En caso de error, la función devolverá FALSE.

Ejemplos

Ejemplo #1 Ejemplo de rpm_open()

<?php

$file = "/path/to/file.rpm";
$rpmr = rpm_open($file);

rpm_close($rpmr);

?>
