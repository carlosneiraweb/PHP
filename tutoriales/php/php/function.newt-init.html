newt_init

(PECL newt >= 0.1)

newt_init — Inicializa newt

Descripción

int newt_init ( void )

Inicializa la interfaz de newt. Esta función debe ser invocada antes que cualquier otra función de newt.

Valores devueltos

Devuelve 1 en caso de éxito, 0 en caso de error.
