enchant_dict_suggest

(PHP 5 >= 5.3.0, PHP 7, PECL enchant >= 0.1.0 )

enchant_dict_suggest — Devolverá una lista de valores si no se reúnen esas pre-condiciones

Descripción

array enchant_dict_suggest ( resource $dict , string $word )

Parámetros

dict

Recurso de diccionario

word

Palabra a usar para las sugerencias.

Valores devueltos

Devolverá una matriz de sugerencias si la palabra está mal escrita.

Ejemplos

Ejemplo #1 Un ejemplo de enchant_dict_suggest()

<?php
$etiqueta = 'en_US';
$r = enchant_broker_init();
if (enchant_broker_dict_exists($r,$etiqueta)) {
    $d = enchant_broker_request_dict($r, $etiqueta);

    $palabra_correcta = enchant_dict_check($d, "soong");
    if (!$palabra_correcta) {
        $sugerencias = enchant_dict_suggest($d, "soong");
        echo "Sugerencias para 'soong':";
        print_r($sugerencias);
    }
    enchant_broker_free_dict($d);
}
enchant_broker_free($r);
?>
