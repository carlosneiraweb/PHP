Nuevos Parámetros

Algunas funciones recibieron nuevos parámetros opcionales en PHP 5.2.x:

Núcleo de PHP:

  • htmlentities() - añadido double_encode en PHP 5.2.3.
  • htmlspecialchars() - añadido double_encode en PHP 5.2.3.
  • base64_decode() - añadido strict
  • setcookie() - añadido httponly
  • setrawcookie() - añadido httponly
  • session_set_cookie_params() - añadido httponly
  • memory_get_usage() - añadido real_usage
  • get_loaded_extensions() - añadido zend_extensions en PHP 5.2.4

curl:

  • curl_multi_info_read() - añadido msgs_in_queue

datetime

  • date() - añadido carácter de formato "u" (milisegundos) en PHP 5.2.2

imap:

  • imap_open() - añadido n_retries
  • imap_reopen() - añadido n_retries

mbstring:

  • mb_strrpos() - añadido offset
    Advertencia

    El parámetro offset se puso en la posición en la que se encontraba el parámetro encoding. Se ha proporcionado compatibilidad con versiones anteriores al permitir que se especifique la codificación (encoding) como tercer parámetro. No se recomienda el uso de este modo de compatibilidad debido a que será removido en futuros lanzamientos de PHP.

ming:

  • SWFMovie::streamMP3() - añadido skip en PHP 5.2.1

openssl:

  • openssl_verify() - añadido signature_algo

pgsql:

  • pg_escape_bytea() - añadido connection
  • pg_escape_string() - añadido connection

simplexml:

  • SimpleXMLElement::__construct() - añadido is_prefix
  • SimpleXMLElement::attributes() - añadido is_prefix
  • SimpleXMLElement::children() - añadido is_prefix
  • simplexml_load_file() - añadido is_prefix
  • simplexml_load_string() - añadido is_prefix

spl:

  • array iterator_to_array(Traversable iterator [, bool use_keys = true]) - fue añadido use_keys en PHP 5.2.1

xmlreader:

  • XMLReader::open() - añadidos encoding y options
  • XMLReader::XML() - añadidos encoding y options

XMLWriter:

  • xmlwriter_write_element() - el parámetro content se hizo opcional en PHP 5.2.3
  • xmlwriter_write_element_ns() - el parámetro content se hizo opcional en PHP 5.2.3
