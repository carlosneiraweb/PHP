ob_implicit_flush

(PHP 4, PHP 5, PHP 7)

ob_implicit_flush — Habilitar/deshabilitar el volcado implícito

Descripción

void ob_implicit_flush ([ int $flag = true ] )

ob_implicit_flush() habilitará o deshabilitará el volcado implícito. El volcado implícito resultará en una operación de volcado después de cada llamada que produzca salida, de modo que no son necesarias las llamadas explícitas a flush().

Parámetros

flag

TRUE para habilitar el volcado implícito, FALSE para deshabilitarlo.

Valores devueltos

No devuelve ningún valor.

Ver también

  • flush() - Vaciar el búfer de salida del sistema
  • ob_start() - Activa el almacenamiento en búfer de la salida
  • ob_end_flush() - Volcar (enviar) el búfer de salida y deshabilitar el almacenamiento en el mismo
