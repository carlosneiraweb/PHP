HaruDoc::getCurrentPage

(PECL haru >= 0.0.1)

HaruDoc::getCurrentPage — Devuelve la página corriente del documento

Descripción

object HaruDoc::getCurrentPage ( void )

Obtiene la página corriente del documento.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve una instancia de HaruPage en éxito o FALSE si no hay página corriente en el momento.

Errores/Excepciones

Lanza una HaruException cuando se produce un error.
