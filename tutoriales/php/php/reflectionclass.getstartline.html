ReflectionClass::getStartLine

(PHP 5, PHP 7)

ReflectionClass::getStartLine — Obtiene el número de linea de inicio

Descripción

public int ReflectionClass::getStartLine ( void )

Devuelve el número de la línea de inicio.

Advertencia

Esta función no está documentada actualmente, solamente se encuentra disponible la lista de parámetros.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

El número de la línea de inicio, en forma de integer.
