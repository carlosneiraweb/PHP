ReflectionFunctionAbstract::inNamespace

(PHP 5 >= 5.3.0, PHP 7)

ReflectionFunctionAbstract::inNamespace — Comprueba si pertenece a un espacio de nombres

Descripción

public bool ReflectionFunctionAbstract::inNamespace ( void )

Comprueba si la función está definida o no en un espacio de nombres.

Advertencia

Esta función no está documentada actualmente, solamente se encuentra disponible la lista de parámetros.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

TRUE si está en un espacio de nombres, o si no FALSE
