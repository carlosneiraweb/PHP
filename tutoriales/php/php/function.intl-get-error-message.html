intl_get_error_message

(PHP 5 >= 5.3.0, PHP 7, PECL intl >= 1.0.0)

intl_get_error_message — Obtener una descripción del último error

Descripción

string intl_get_error_message ( void )

Obtener el mensaje de error de la última llamada a la función internacionalizada.

Valores devueltos

Descripción de un error sucedido en la última llamada a la función API.

Ejemplos

Ejemplo #1 intl_get_error_message() ejemplo

<?php
if( Collator::getAvailableLocales() === false ) {
    show_error( intl_get_error_message() );
}
?>
