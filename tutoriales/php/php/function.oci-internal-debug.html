oci_internal_debug

(PHP 5, PHP 7, PECL OCI8 >= 1.1.0)

oci_internal_debug — Habilita o deshabilita la salida de la depuración interna

Descripción

void oci_internal_debug ( bool $onoff )

Habilita o deshabilita la salida de la depuración interna.

Parámetros

onoff

Establecer a FALSE para activar la salida de la depuración, o a TRUE para desactivarla.

Valores devueltos

No devuelve ningún valor.

Notas

Nota:

En versiones de PHP anteriores a la 5.0.0 se debe usar ociinternaldebug() en su lugar. Este nombre aún se puede usar; se dejó como un alias de oci_internal_debug() por razones de retrocompatibilidad. Sin embargo, este nombre es obsoleto y no se recomienda.
