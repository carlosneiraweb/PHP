Yar_Client::__call

(PECL yar >= 1.0.0)

Yar_Client::__call — Llamar a un servicio

Descripción

public void Yar_Client::__call ( string $method , array $parameters )

Emite una llamada a un método RPC remoto.

Parámetros

method

Nombre del método RPC remono.

parameters

Parámetros.

Valores devueltos

Ejemplos

<?php

$cliente = new Yar_Client("http://host/api/");

/* llamar al servicio remoto */
