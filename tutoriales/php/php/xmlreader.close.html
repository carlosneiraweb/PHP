XMLReader::close

(PHP 5 >= 5.1.0, PHP 7)

XMLReader::close — Cierra la entrada del XMLReader

Descripción

public bool XMLReader::close ( void )

Cierra la entrada del objeto XMLReader que es actualmente análizado.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.

Ver también

  • XMLReader::open() - Establece el URI conteniendo el XML a analizar
  • XMLReader::xml() - Establece el contenido del XML a analizar
