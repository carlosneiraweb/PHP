Gmagick::resizeimage

(PECL gmagick >= Unknown)

Gmagick::resizeimage — Escala una imagen

Descripción

public Gmagick Gmagick::resizeimage ( int $width , int $height , int $filter , float $blur [, bool $fit = false ] )

Escala una imagen a las dimensiones deseadas con un filtro.

Parámetros

width

El número de columnas en la imagen escalada.

height

El número de filas en la imagen escalada.

filter

El filtro de imagen a usar.

blur

El factor de borrosidad dondo mayor que 1 es borroso, menor que 1 es nítido.

Valores devueltos

El objeto Gmagick si se tuvo éxito

Errores/Excepciones

Arroja un GmagickException en caso de error.
