GearmanTask::uuid

(PECL gearman <= 0.5.0)

GearmanTask::uuid — Obtiene el identificador único de una tarea (obsoleto)

Descripción

public string GearmanTask::uuid ( void )

Retorna el identificador único de una tarea. Este es asignado por GearmanClient, a diferencia del manejador de trabajo que lo define el servidor de trabajos Gearman.

Nota:

Este método ha sido reemplazado por GearmanTask::unique() en la versión 0.6.0 de la extensión Gearman.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

El identificador único o FALSE si no tiene identificador asignado.

Ver también

  • GearmanClient::do() - Ejecuta una sola tarea y retorna el resultado [obsoleto]
  • GearmanClient::addTask() - Añade una tarea para ser ejecutada en paralelo
