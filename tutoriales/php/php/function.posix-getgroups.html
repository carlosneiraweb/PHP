posix_getgroups

(PHP 4, PHP 5, PHP 7)

posix_getgroups — Devolver el conjunto de grupos del proceso actual

Descripción

array posix_getgroups ( void )

Obtiene el conjunto de grupos del proceso actual.

Valores devueltos

Devuelve un array de enteros que contiene los ids numéricos del conjunto de grupos del proceso actual.

Ejemplos

Ejemplo #1 Ejemplo de uso de posix_getgroups()

<?php

$groups = posix_getgroups();

print_r($groups);
?>

El resultado del ejemplo sería algo similar a:

Array
(
    [0] => 4
    [1] => 20
    [2] => 24
    [3] => 25
    [4] => 29
    [5] => 30
    [6] => 33
    [7] => 44
    [8] => 46
    [9] => 104
    [10] => 109
    [11] => 110
    [12] => 1000
)
