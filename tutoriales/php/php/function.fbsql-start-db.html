fbsql_start_db

(PHP 4 >= 4.0.6, PHP 5, PHP 7)

fbsql_start_db — Inicia una base de datos en un servidor local o remoto

Descripción

bool fbsql_start_db ( string $database_name [, resource $link_identifier [, string $database_options ]] )

Inicia la base de datos en un servidor local o remoto.

Parámetros

database_name

El nombre de la base de datos.

link_identifier

Un identificador de enlace FrontBase devuelto por fbsql_connect() o fbsql_pconnect().

Es opcional y si no se especifica, la función intentará encontrar un enlace abierto al servidor FrontBase y de no encontrarse ningún enlace abierto intentará crear uno como si fbsql_connect() fuese invocada sin parámetros.

database_options

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.

Ver también

  • fbsql_db_status() - Obtiene el estado de una base de datos dada
  • fbsql_stop_db() - Detiene una base de datos en un servidor local o remoto
