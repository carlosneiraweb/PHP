newt_form

(PECL newt >= 0.1)

newt_form — Crea un formulario

Descripción

resource newt_form ([ resource $vert_bar [, string $help [, int $flags ]]] )

Crea un nuevo formulario.

Parámetros

vert_bar

Barra de desplazamiento vertical que debería estar asociada al formulario

help

Cadena de texto de ayuda

flags

Banderas varias

Valores devueltos

Devuelve un enlace a un recurso al componente formulario creado, o FALSE en caso de error.

Ejemplos

Ejemplo #1 Un ejemplo de newt_form()

Muestra un simple botoón "Salir", que cierra la aplicación una vez pulsado.

<?php
newt_init();
newt_cls();

$myform = newt_form();
$button = newt_button (5, 12, "Salir");

newt_form_add_component ($myform, $button);
newt_refresh ();
newt_run_form ($myform);

newt_finished ();
newt_form_destroy ($myform);
?>
