mcrypt_generic_end

(PHP 4 >= 4.0.2, PHP 5)

mcrypt_generic_end — Esta función termina el cifrado

Advertencia

Esta función está OBSOLETA en PHP 5.3.0, por lo tanto, será ELIMINADA en PHP 7.0.0.

Las alternativas a esta función son:

  • mcrypt_generic_deinit()

Descripción

bool mcrypt_generic_end ( resource $td )

Advertencia

Debería utilizarse mcrypt_generic_deinit() en lugar de esta función, ya que puede ocasionar colisiones al utilizarla con mcrypt_module_close() debido a múltiples libreaciones de búfer.

Esta función termina con el cifrado esecificado por el descriptor de cifrado (el parámetro td). En realidad, limpia todos los buffers y cierra los módulos utilizados. Devuelve FALSE en caso de error, o TRUE en caso de éxito.
