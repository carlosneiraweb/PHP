GearmanJob::data

(PECL gearman <= 0.5.0)

GearmanJob::data — Envía datos para un trabajo en ejecución (obsoleto)

Descripción

public bool GearmanJob::data ( string $data )

Envía datos al servidor de trabajos (y a cualquier cliente a la escucha) para este trabajo.

Nota:

Este método ha sido reemplazado por GearmanJob::sendData() en la versión 0.6.0 de la extensión Gearman.

Parámetros

data

Datos arbitrarios serializados.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.

Ver también

  • GearmanJob::workload() - Obtiene la carga de trabajo
  • GearmanTask::data() - Obtiene los datos retornados para una tarea
