SolrQuery::__construct

(PECL solr >= 0.9.2)

SolrQuery::__construct — Constructor

Descripción

public SolrQuery::__construct ([ string $q ] )

Constructor.

Parámetros

q

Consulta de búsqueda opcional

Valores devueltos

Ninguno.

Errores/Excepciones

Emite una SolrIllegalArgumentException en caso de proporcionar un parámetro no válido.

Historial de cambios

Versión Descripción
2.0.0 Si q fuera inválido, se lanza una SolrIllegalArgumentException. Anteriormente se emitía un error.
