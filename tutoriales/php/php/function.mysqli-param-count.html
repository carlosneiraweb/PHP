mysqli_param_count

(PHP 5 < 5.4.0)

mysqli_param_count — Alias de mysqli_stmt_param_count()

Descripción

Esta función es un alias de mysqli_stmt_param_count().

Advertencia

Esta función ha sido declarada OBSOLETA desde PHP 5.3.0 y ELIMINADA a partir de PHP 5.4.0.
