CairoMatrix::initIdentity

cairo_matrix_init_identity

(PECL cairo >= 0.1.0)

CairoMatrix::initIdentity -- cairo_matrix_init_identity — Crea una nueva matriz de identidad

Descripción

Estilo orientado a objetos (método):

public static void CairoMatrix::initIdentity ( void )

Estilo por procedimientos:

object cairo_matrix_init_identity ( void )

Crea una nueva matriz que es una transformación de identidad. Una transformación de identidad significa que la información de origen es copiada a la información destino sin cambios

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve un nuevo objeto CairoMatrix que se puede usar con superficies, contextos y patrones.

Ejemplos

Ejemplo #1 Estilo orientado a objetos

<?php
/* Crear una nueva Matriz */
$matriz = CairoMatrix::initIdentity();
?>

Ejemplo #2 Estilo por procedimientos

<?php
/* Crear una nueva Matriz */
$matriz = cairo_matrix_init_identity();
?>
