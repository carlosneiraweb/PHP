Cambios en las características

  • Actualización del gestor de almacenamiento APC

    • Construcción corregida para APC 3.1.9+
    • Nota: El uso del gestor de almacenamiento APC actualmente no está recomendado debido a problemas de estabilidad del mismo APC.

  • Nuevas directivas de configuración de PHP

    • mysqlnd_qc.collect_statistics_log_file. Se agregó el fichero de registro de estadísticas de la caché escrito después de cada 10 peticiones servidas por el proceso de PHP.
    • mysqlnd_qc.ignore_sql_comments. Controlar si los comentarios SQL son ignorados para la generación hash de claves de la caché.

  • Nuevas constantes y sugerencias SQL

    • MYSQLND_QC_SERVER_ID_SWITCH permite el agrupamiento de entradas de la caché desde diferentes conexiones físicas. Esto es necesario para PECL/mysqlnd_ms.
    • MYSQLND_QC_CONDITION_META_SCHEMA_PATTERN para usarla con mysqlnd_qc_set_cache_condition().

  • La nueva función mysqlnd_qc_set_cache_condition() para el almacenamiento en caché basado en patrones de esquemas internos. Probablemente para dar soporte a un rango más amplio de condiciones en el futuro.

  • Informar de la marca de tiempo valid_until para entradas de la caché del gestor predeterminado a través de mysqlnd_qc_get_cache_info().

  • Incluir el número de conjuntos de caracteres para el uso de hash con las entradas de la caché. Esto debería evitar el servicio de conjuntos de resultados que tienen un conjunto de caracters erróneo.

    Cambio en la API: get_hash_key espera el nuevo parámetro "charsetnr" (int) después de "port".

  • Cambio en la API: cambiar la firma de is_select() de bool is_select() a mixed is_select(). Mixed puede ser o bien un valor booleano o un array(long ttl, string server_id). Esto es necesario para PECL/mysqlnd_ms.

Otros

  • Soporte para actuar como un sistema de caché para PECL/mysqlnd_ms 1.3.0-beta o superior para reemplazar de forma transparente las lecturas de esclavos de la Replicación MySQL con accesos a la caché, si el usuario lo permite explícitamente.

Errores corregidos

  • Error #59959 corregido (config.m4, biblioteca errónea - construcción del gestor de memcache de 64 bit) (Créditos: Remi Collet)
