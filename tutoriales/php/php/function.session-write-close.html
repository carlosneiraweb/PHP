session_write_close

(PHP 4 >= 4.0.4, PHP 5, PHP 7)

session_write_close — Escribir información de sesión y finalizar la sesión

Descripción

void session_write_close ( void )

Finaliza la sesión actual y almacena la información de la sesión.

La información de la sesión normalmente se almacena después de que el script termine sin la necesidad de llamar a session_write_close(), pero ya que la información de la sesión está bloqueada para prevenir escrituras concurrentes, sólo el script puede operar sobre una sesión cada vez. Cuando se usan sub-ventanas (frames) junto con sesiones, experimentará que las sub-ventanas se cargan de una en una debido a este bloqueo. Puede reducir el tiempo necesario para cargar todas las sub-ventanas finalizando la sesión tan pronto como todos los cambios a las variables de sesión estén hechos.

Valores devueltos

No devuelve ningún valor.
