QuickHashIntHash::update

(PECL quickhash >= Unknown)

QuickHashIntHash::update — Este método actualiza una entrada del hash con un nuevo valor

Descripción

public bool QuickHashIntHash::update ( int $key , int $value )

Este método actualiza una entrada con un nuevo valor, y devuelve si la entrada fue actualizada. Si update. Si existen claves duplicadas, únicamente se actualizará el valor del primer elemento encontrado. Use QuickHashIntHash::CHECK_FOR_DUPES durante la creación de un hash para evitar que las claves duplicadas sean parte de un hash.

Parámetros

key

La clave de la entrada a añadir.

value

El nuevo valor con el que actualizar la entrada.

Valores devueltos

TRUE cuando la entrada se encontró y se actualizó, y FALSE si la entrada ya no es parte del hash.

Ejemplos

Ejemplo #1 Ejemplo de QuickHashIntHash::update()

<?php
$hash = new QuickHashIntHash( 1024 );

var_dump( $hash->add( 141421, 173205 ) );
var_dump( $hash->update( 141421, 223606 ) );
var_dump( $hash->get( 141421 ) );
?>

El resultado del ejemplo sería algo similar a:

bool(true)
bool(true)
int(223606)
