sqlite_field_name

SQLiteResult::fieldName

SQLiteUnbuffered::fieldName

(PHP 5 < 5.4.0, PECL sqlite >= 1.0.0)

sqlite_field_name -- SQLiteResult::fieldName -- SQLiteUnbuffered::fieldName — Devuelve el nombre de un campo en particular

Descripción

string sqlite_field_name ( resource $result , int $field_index )

Estilo orientado a objetos (método):

string SQLiteResult::fieldName ( int $field_index )
string SQLiteUnbuffered::fieldName ( int $field_index )

Dado el número ordinal de columna, field_index, sqlite_field_name() devuelve el nombre de ese campo en el conjunto de resultados dado por result.

Parámetros

result

El recurso de resultados de SQLite. Este parámetro no es necesario cuando se usa el método orientado a objetos.

field_index

El número ordinal de la columna del conjunto de resultados.

Valores devueltos

Devuelve el nombre de un campo de un conjunto de resultados de SQLite, dado el número ordinal de columna; FALSE en caso de error.

Los nombre de las columnas devueltos por SQLITE_ASSOC y SQLITE_BOTH serán mayúsculas o no de acuerdo al valor de la opción de configuración sqlite.assoc_case.
