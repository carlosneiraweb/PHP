PDF_set_parameter

(PHP 4, PECL pdflib >= 1.0.0)

PDF_set_parameter — Establecer un parámetro de cadena

Descripción

bool PDF_set_parameter ( resource $p , string $key , string $value )

Establece algún parámetro PDFlib con el tipo string. Devuelve TRUE en caso de éxito o FALSE en caso de error.
