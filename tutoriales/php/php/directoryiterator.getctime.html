DirectoryIterator::getCTime

(PHP 5, PHP 7)

DirectoryIterator::getCTime — Obtiene el cambio de tiempo del inodo para el elemento actual DirectoryIterator

Descripción

public int DirectoryIterator::getCTime ( void )

Obtiene el cambio de tiempo del inodo para el elemento actual DirectoryIterator.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve el último cambio de tiempo de el fichero, en formato Unix timestamp.

Ejemplos

Ejemplo #1 DirectoryIterator::getCTime() example

Este ejemplo mostrará el nombre del fichero y el último cambio de tiempo de los ficheros de el directorio que contiene el script.

<?php
$iterator = new DirectoryIterator(dirname(__FILE__));
foreach ($iterator as $fileinfo) {
    if ($fileinfo->isFile()) {
        echo $fileinfo->getFilename() . " cambiado en " . $fileinfo->getCTime() . "\n";
    }
}
?>

El resultado del ejemplo sería algo similar a:

manzana.jpg cambiado en 1240398312
banana.jpg cambiado en 1238605440
index.php cambiado en 1240398935
pera.jpg cambiado en 1237423740
