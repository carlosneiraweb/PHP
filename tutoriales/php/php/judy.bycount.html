Judy::byCount

(PECL judy >= 0.1.1)

Judy::byCount — Localiza el enésimo indice presente en el array Judy

Descripción

public int Judy::byCount ( int $nth_index )

Lozaliza el enésimo indice presente en el array Judy.

Parámetros

nth_index

El enésimo indice a ser vuelto. Si nth_index es igual a 1, devuelve el índice en la posición enésima dada.

Valores devueltos

Devuelve el índice en la posición enésima dada.
