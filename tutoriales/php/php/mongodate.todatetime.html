MongoDate::toDateTime

(PECL mongo >= 1.6.0; PHP > 5.3.4)

MongoDate::toDateTime — Devuelve un objeto DateTime que representa esta fecha

Esta extenisón que define este método está obsoleta. En su lugar debe usarse la extensión MongoDB. Las alternativas a este método son:

  • MongoDB\BSON\UTCDateTime::toDateTime()

Descripción

public DateTime MongoDate::toDateTime ( void )

Devuelve una representación de un objeto DateTime de esta fecha.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Un objeto DateTime con esta fecha.

Ejemplos

Ejemplo #1 Ejemplo de MongoDate::toDateTime()

Este ejemplo demuestra la creación de un objeto DateTime desde un objeto MongoDate.

<?php
$d = new MongoDate(strtotime("2014-11-18 11:01:25"));
var_dump( $d->toDateTime() );
?>

El resultado del ejemplo sería algo similar a:

class DateTime#2 (3) {
  public $date =>
  string(26) "2014-11-18 11:01:25.000000"
  public $timezone_type =>
  int(1)
  public $timezone =>
  string(6) "+00:00"
}
