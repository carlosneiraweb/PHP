fbsql_close

(PHP 4 >= 4.0.6, PHP 5, PHP 7)

fbsql_close — Cierra la conexión FrontBase

Descripción

bool fbsql_close ([ resource $link_identifier ] )

Cierra la conexión con el servidor FrontBase asociado al link_identifier especificado.

No suele ser necesario usar fbsql_close(), ya que los enlaces no-persistentes abiertos se cierran de forma automática al final de la ejecución del script.

Parámetros

link_identifier

Un identificador de enlace FrontBase devuelto por fbsql_connect() o fbsql_pconnect().

Es opcional y si no se especifica, la función intentará encontrar un enlace abierto al servidor FrontBase y de no encontrarse ningún enlace abierto intentará crear uno como si fbsql_connect() fuese invocada sin parámetros.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.

Ejemplos

Ejemplo #1 Ejemplo de fbsql_close()

<?php
$link = fbsql_connect("localhost", "_SYSTEM", "secret")
    or die("No se puede conectar");
echo "Conectado correctamente";
fbsql_close($link);
?>
