Introducción

La extensión win32service es una extensión específica de Windows que permite a PHP comunicarse con el Administrador de Control de Servicios para iniciar, detener, registrar y dejar de registrar servicios, e incluso permite a los scripts de PHP ejecutarse como un servicio.
