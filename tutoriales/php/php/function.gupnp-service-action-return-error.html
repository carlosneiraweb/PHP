gupnp_service_action_return_error

(PECL gupnp >= 0.1.0)

gupnp_service_action_return_error — Devuelve el código de error

Descripción

bool gupnp_service_action_return_error ( resource $action , int $error_code [, string $error_description ] )

Devuelve el código de error.

Parámetros

action

Un identificador de acción de servicio.

error_code

El código de error. La señal puede ser uno de las siguientes valores o ser definido por el usuario:

GUPNP_CONTROL_ERROR_INVALID_ACTION
El nombre de acción es inválido.
GUPNP_CONTROL_ERROR_INVALID_ARGS
Los argumentos de acción fueron inválidos.
GUPNP_CONTROL_ERROR_OUT_OF_SYNC
Fuera de sincronización (obsoleto).
GUPNP_CONTROL_ERROR_ACTION_FAILED
La acción ha fallado.

error_description

La descripción del error.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.
