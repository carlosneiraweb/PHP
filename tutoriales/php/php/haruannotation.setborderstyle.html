HaruAnnotation::setBorderStyle

(PECL haru >= 0.0.1)

HaruAnnotation::setBorderStyle — Define el estilo de borde de la anotación

Descripción

bool HaruAnnotation::setBorderStyle ( float $width , int $dash_on , int $dash_off )

Define el estilo de borde de la anotación. Ésta función puede ser usada con enlaces de anotación solamente.

Parámetros

width

El ancho de la linea del borde.

dash_on

El guión de estilo.

dash_off

El guión de estilo.

Valores devueltos

Devuelve TRUE en éxito.

Errores/Excepciones

Lanza una HaruException cuando se produce un error.
