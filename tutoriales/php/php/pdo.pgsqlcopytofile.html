PDO::pgsqlCopyToFile

(PHP 5 >= 5.3.3, PHP 7)

PDO::pgsqlCopyToFile — Copiar datos desde una tabla a un fichero

Descripción

public bool PDO::pgsqlCopyToFile ( string $table_name , string $filename [, string $delimiter = '\t' [, string $null_as = "\\\\N" [, string $fields ]]] )

Copia los datos desde la tabla al fichero especificado por filename empleanto delimiter como delimitador de campos y la lista de campos fields

Parámetros

table_name

Un string que contiene el nombre de la tabla

filename

El nombre del fichero a donde exportar los datos

delimiter

El delimitador empleado en el fichero especificado por filename

null_as

Cómo interpretar valores nulos

fields

La lista de campos a insertar

Valores devueltos

Returns TRUE on success, o FALSE en caso de error.
