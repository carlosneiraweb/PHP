DOMNode::isDefaultNamespace

(PHP 5, PHP 7)

DOMNode::isDefaultNamespace — Comprueba si la URI del espacio de nombres especificada es el espacio de nombres predeterminado

Descripción

public bool DOMNode::isDefaultNamespace ( string $namespaceURI )

Indica si namespaceURI es el espacio de nombres especificado.

Parámetros

namespaceURI

La URI del espacio de nombres a buscar.

Valores devueltos

Devuelve TRUE si namespaceURI es el espacio de nombres predeterminado, FALSE si no.
