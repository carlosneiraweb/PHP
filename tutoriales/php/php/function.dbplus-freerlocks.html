dbplus_freerlocks

(PHP 4 <= 4.1.0, PECL dbplus >= 0.9)

dbplus_freerlocks — Liberar todos los bloqueos de una relación dada

Descripción

int dbplus_freerlocks ( resource $relation )

Libera todos los bloqueos de tuplas mantenidos por relation.

Parámetros

relation

Una relación abierta con dbplus_open().

Advertencia
