gupnp_root_device_stop

(PECL gupnp >= 0.1.0)

gupnp_root_device_stop — Detiene el bucle principal

Descripción

bool gupnp_root_device_stop ( resource $root_device )

Detiene el bucle principal en el servidor raíz.

Parámetros

root_device

Un identificadorde servicio raíz, devuelto por gupnp_root_device_new().

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.
