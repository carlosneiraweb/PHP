spl_autoload_functions

(PHP 5 >= 5.1.2, PHP 7)

spl_autoload_functions — Devolver todas las funciones __autoload() registradas

Descripción

array spl_autoload_functions ( void )

Obtiene todas las funciones __autoload() registradas.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Un array con todas las funciones __autoload registradas. Si la cola de autocarga no está activada, el valor devuelto es FALSE. Si no hay ninguna función registrada, el valor devuelto será un array vacío.
