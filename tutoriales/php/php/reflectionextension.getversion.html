ReflectionExtension::getVersion

(PHP 5, PHP 7)

ReflectionExtension::getVersion — Obtiene la versión de una extensión

Descripción

public string ReflectionExtension::getVersion ( void )

Devuelve la versión de la extensión.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Versión de la extensión.

Ejemplos

Ejemplo #1 Ejemplo de ReflectionExtension::getVersion()

<?php
$ext = new ReflectionExtension('mysqli');
var_dump($ext->getVersion());
?>

El resultado del ejemplo sería algo similar a:

string(3) "0.1"
