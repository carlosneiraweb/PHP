sybase_fetch_row

(PHP 4, PHP 5, PHP 7)

sybase_fetch_row — Devuelve una fila Sybase como un array indexado numéricamente

Descripción

array sybase_fetch_row ( resource $result )

sybase_fetch_row() devuelve una fila de datos del resultado asociado al identificador de resultado result.

Las siguientes llamadas a sybase_fetch_row() devolverán la siguiente fila del conjunto de resultado, o FALSE si ya no hay más filas.

Parámetros

result

Valores devueltos

Devuelve un array que corresponde a la fila retornada, o FALSE si no hay más filas. Cada columna del resultado se almacena en un array indexado nuéricamente, empezando desde el índice 0.

Tipos de datos
PHP Sybase
string VARCHAR, TEXT, CHAR, IMAGE, BINARY, VARBINARY, DATETIME
int NUMERIC (sin precisión), DECIMAL (sin precisión), INT, BIT, TINYINT, SMALLINT
float NUMERIC (con precisión), DECIMAL (con precisión), REAL, FLOAT, MONEY
NULL NULL
