SAMConnection::error

(PECL sam >= 0.1.0)

SAMConnection::error — Contiene el texto con la descripción de la última operación SAM fallida

Descripción

string $SAMConnection->error;

Contiene el texto con la descripción de la última operación SAM fallida en esa conexión. Si la última operación se completó con éxito esta propiedad contiene una cadena vacía.

Valores devueltos

Una cadena que contiene la descripción del último error encontrado en la conexión. Una cadena vacía indica que la última operación en esa conexión se completó con éxito.

Ejemplos

Ejemplo #1 Uso de las propiedades de número y descripción de error

<?php
$conn = new SAMConnection();
$conn->connect(SAM_WMQ, array(SAM_HOST => 'localhost', SAM_PORT => 1506));
$msg = new SAMMessage('Este es un elemento simple de texto');
if (!$conn->send('topic://test', $msg)) {
    // El Send falló!
    echo "Send fallido ($conn->errno) $conn->error";
}
?>
