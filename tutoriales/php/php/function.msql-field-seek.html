msql_field_seek

(PHP 4, PHP 5, PHP 7)

msql_field_seek — Determina el desplazamiento de campo

Descripción

bool msql_field_seek ( resource $result , int $field_offset )

Busca el desplazamiento de campo especificado. Si la siguiente llamada a msql_fetch_field() no inluye un desplazamiento de campo, el campo será devuelto.

Parámetros

result

El resultado resource que está siendo evaluado. Este resultado proviene de una llamada a msql_query().

field_offset

El número del campo a buscar. El valor de field_offset comienza en 1.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.
