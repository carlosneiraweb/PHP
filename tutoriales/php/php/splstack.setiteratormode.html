SplStack::setIteratorMode

(PHP 5 >= 5.3.0, PHP 7)

SplStack::setIteratorMode — Establece el modo de iteración

Descripción

void SplStack::setIteratorMode ( int $mode )

Parámetros

mode

Este es el único parámetro de iteración que se puede modificar.

  • El comportamiento del iterador (ya sea uno o el otro):
    • SplDoublyLinkedList::IT_MODE_DELETE (Elements are deleted by the iterator)
    • SplDoublyLinkedList::IT_MODE_KEEP (Elements are traversed by the iterator)

El modo por omisión es 0x2 : SplDoublyLinkedList::IT_MODE_LIFO | SplDoublyLinkedList::IT_MODE_KEEP

Advertencia

La dirección de iteración puede La dirección de la iteración ya no puede ser cambiado para SplStacks. Si se intenta hacer esto lanzará una RuntimeException.

Valores devueltos

No devuelve ningún valor.
