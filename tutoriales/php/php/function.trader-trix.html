trader_trix

(PECL trader >= 0.2.0)

trader_trix — Ritmo de cambio de 1 día (RDC) de una MME suave triple

Descripción

array trader_trix ( array $real [, integer $timePeriod ] )

Parámetros

real

Array de valores reales.

timePeriod

Número del periodo. Rango válido desde 2 hasta 100000.

Valores devueltos

Devuelve un array con los datos calculados o false en caso de error.
