MongoCursorInterface::info

(PECL mongo >=1.5.0)

MongoCursorInterface::info — Obtiene información sobre la creación e iteración del cursor

Descripción

abstract public array MongoCursorInterface::info ( void )

Devuelve información sobre la creación e iteración del cursor. Esta función se puede llamar antes o después de que el cursor haya empezado a iterar.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve el espacio de nombres, tamaño del lote, límite, salto, banderas, consulta, y campos protegidos de este cursor. Si el cursor ha empezado a iterar, se incluirá información adicional sobre la iteración y la conexión.
