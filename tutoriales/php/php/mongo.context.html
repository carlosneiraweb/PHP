Opciones de Contexto de flugo

La extensión de MongoDB para PHP proporciona Soporte para contextos de flujo usando el contexto de mongodb.

Un contexto de flujo se debe crear con stream_context_create() y pasarlo a MongoClient::__construct() antes de que se realice la conexión real a MongoDB. No es posible aplicar un contexto de flujo a flujos ya creados.

También hay soporte para parámetros y opciones de contexto adicionales, como ssl y notification parameters.

Las opciones de contexto de MongoDB proporcionan una inteerfaz rica para registrar tráfico de red entre el controlador y servidores de MongoDB. Esta interfaz se puede usar para proporcionar el registro de consultas, perfilamiento, depuradores, o cualquier cosa que fuera necesaria para inspeccionar las opciones de protocolos y comandos subyacentes.
