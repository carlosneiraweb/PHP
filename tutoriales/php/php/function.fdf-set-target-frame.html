fdf_set_target_frame

(PHP 4 >= 4.3.0, PHP 5, PHP 7)

fdf_set_target_frame — Definir contenedor en el que mostrar el formulario

Descripción

bool fdf_set_target_frame ( resource $fdf_document , string $frame_name )

Define el contenedor en el que mostrar el PDF resultante definido en fdf_save_file().

Parámetros

fdf_document

Manejador de documentos FDF, devuelto por fdf_create(), fdf_open() o por fdf_open_string().

frame_name

Nombre del contenedor, en forma de string.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.
