fbsql_next_result

(PHP 4 >= 4.0.6, PHP 5, PHP 7)

fbsql_next_result — Mover el puntero del resultado interno por el resultado de al lado

Descripción

bool fbsql_next_result ( resource $result )

Al enviar más de una instrucción SQL a el servidor o ejecutar un procedimiento almacenado con varios resultados hará que el servidor retorne varios conjuntos de resultados. Esta función pondrá a prueba para obtener resultados adicionales disponible en el servidor. Si un conjunto de resultados adicionales que existe el resultado conjunto existente y prepararse para ir a buscar las palabras del nuevo conjunto de resultados.

Parámetros

result

Un identificador de resultado devuelto por fbsql_query() o fbsql_db_query().

Valores devueltos

Retorna TRUE si un conjunto de resultados esta disponible o FALSE de otro modo.

Ejemplos

Ejemplo #1 fbsql_next_result() ejemplo

<?php
$link = fbsql_connect("localhost", "_SYSTEM", "secret");
fbsql_select_db("MyDB", $link);
$SQL = "Select * from table1; select * from table2;";
$rs = fbsql_query($SQL, $link);
do {
    while ($row = fbsql_fetch_row($rs)) {
    }
} while (fbsql_next_result($rs));
fbsql_free_result($rs);
fbsql_close($link);
?>
