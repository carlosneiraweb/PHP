recode_file

(PHP 4, PHP 5, PHP 7)

recode_file — Recodificar de fichero a fichero según la petición de recoficación

Descripción

bool recode_file ( string $request , resource $input , resource $output )

Recodifica el fichero señalado por el gestor de ficheros input al fichero señalado por el gestor de ficheros output según la petición de recodificación dada por request.

Parámetros

request

El tipo de solicitud de recodificación deseada

input

Un resource gestor de ficheros local para input

output

Un resource gestor de ficheros local para output

Valores devueltos

Devuelve FALSE, si no se puede cumplir, o de lo contrario TRUE.

Ejemplos

Ejemplo #1 Ejemplo básico de recode_file()

<?php
$input = fopen('input.txt', 'r');
$output = fopen('output.txt', 'w');
recode_file("us..flat", $input, $output);
?>

Notas

Esta función actualmente no procesa gestores de ficheros que señalan a ficheros remotos (URLs). Ambos gestores de ficheros deben señalar a ficheros locales.
