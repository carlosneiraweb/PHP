Imagick::profileImage

(PECL imagick 2.0.0)

Imagick::profileImage — Añade o elimina un perfil de una imagen

Descripción

bool Imagick::profileImage ( string $name , string $profile )

Añade o elimina un perfil ICC, IPTC, o genérico de una imagen. Si el perfil es NULL, se elimina de la imagen, si no, se añade. Use como nombre '*' y un perfil NULL para eliminar todos los perfiles de la imagen.

Parámetros

name

profile

Valores devueltos

Devuelve TRUE en caso de éxito.

Errores/Excepciones

Lanza ImagickException en caso de error.
