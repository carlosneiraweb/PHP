mssql_field_name

(PHP 4, PHP 5, PECL odbtp >= 1.1.1)

mssql_field_name — Devuelve el nombre de un campo

Advertencia

Esta función está OBSOLETA en PHP 7.0.0.

Las alternativas a esta función son:

  • PDOStatement::getColumnMeta()
  • sqlsrv_field_metadata()
  • odbc_field_name()

Descripción

string mssql_field_name ( resource $result [, int $offset = -1 ] )

Devuelve el nombre del campo número offset en result.

Parámetros

result

El resultado que está siendo evaluado. Este resultado proviene de una llamada a mssql_query().

offset

La posición del campo, empezando por 0. Si se omite, se utilizará el campo actual.

Valores devueltos

El nombre del campo en la posicion especificada en caso de éxito o FALSE en caso de error.

Ejemplos

Ejemplo #1 mssql_field_name() example

<?php
// Enviar una consulta a MSSQL
$query = mssql_query('SELECT [username], [name], [email] FROM [php].[dbo].[userlist]');

echo 'El resultado contiene los siguientes campos:', PHP_EOL;

// Volcar todos los nombres de campo del resultado
for ($i = 0; $i < mssql_num_fields($query); ++$i) {
    echo ' - ' . mssql_field_name($query, $i), PHP_EOL;
}

// Liberar el recurso
mssql_free_result($query);
?>

El resultado del ejemplo sería algo similar a:

El resultado contiene los siguientes campos:
 - nombre de usuario
 - nombre
 - email
