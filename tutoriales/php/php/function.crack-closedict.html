crack_closedict

(PECL crack >= 0.1)

crack_closedict — Cierra un diccionario abierto de CrackLib

Descripción

bool crack_closedict ([ resource $dictionary ] )

crack_closedict() cierra el diccionario indicado mediante el identificador dictionary.

Advertencia

Esta función ha sido declarada EXPERIMENTAL. Su funcionamiento, nombre y la documentación que le acompaña puede cambiar sin previo aviso en futuras versiones de PHP. Utilícela bajo su propia responsabilidad.

Parámetros

dictionary

El diccionario a cerrarse. Si no se especifica, el diccionario actualmente abierto será cerrado.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.
