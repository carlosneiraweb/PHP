gc_disable

(PHP 5 >= 5.3.0, PHP 7)

gc_disable — Desactiva el recolector de referencia circular

Descripción

void gc_disable ( void )

Desactiva el recolector de referencia circular. zend.enable_gc a 0.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

No devuelve ningún valor.
