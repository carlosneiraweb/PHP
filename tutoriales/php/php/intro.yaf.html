Introducción

La extensión Yet Another Framework (Yaf) -Otro Framework Más, en español- es un framework de PHP usado para desarrollar aplicaciones web.

Yaf requiere una versión de PHP superior a la 5.2.1. Las versiones anteriores podrían no funcionar.

Se puede encontrar una comparativa sencilla en » Rendimiento de Yaf.

Para una guía rápida, consulte la sección Turoriales.
