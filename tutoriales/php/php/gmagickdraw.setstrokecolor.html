GmagickDraw::setstrokecolor

(PECL gmagick >= Unknown)

GmagickDraw::setstrokecolor — Establece el color usado para contornear los perfiles del objeto.

Descripción

public GmagickDraw GmagickDraw::setstrokecolor ( GmagickPixel $color )

Establece el color usado para contornear los perfiles del objeto.

Parámetros

color

Objeto GmagickPixel que representa el color del contorno.

Valores devueltos

El objeto GmagickDraw si se tuvo éxito
