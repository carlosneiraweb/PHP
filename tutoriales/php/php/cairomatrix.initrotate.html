CairoMatrix::initRotate

cairo_matrix_init_rotate

(PECL cairo >= 0.1.0)

CairoMatrix::initRotate -- cairo_matrix_init_rotate — Crea una nueva matriz rotada

Descripción

Estilo orientado a objetos (método):

public static void CairoMatrix::initRotate ( float $radians )

Estilo por procedimientos:

object cairo_matrix_init_rotate ( float $radians )

Crea una nueva matriz para una transformación que rota por los radianes proporcionados

Parámetros

radians

Ángulo de rotación, es radianes. La dirección de rotación está definida tal que los ángulos positivos rotan en la dirección desde el eje X positivo hacia el eje Y positivo. Con orientación de ejes predeterminada de cairo, los ángulos positivos rotan en dirección de las agujas del reloj.

Valores devueltos

Devuelve un nuevo objeto CairoMatrix que se puede usar con superficies, contexto y patrones.

Ejemplos

Ejemplo #1 Estilo orientado a objetos

<?php
/* Crear una nueva Matriz */
$matriz = CairoMatrix::initRotate(0.3);
?>

Ejemplo #2 Estilo por procedimientos

<?php
/* Crear una nueva Matriz */
$matriz = cairo_matrix_init_rotate(0.3);
?>
