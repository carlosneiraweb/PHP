hw_api::children

(PHP 4, PHP 5 < 5.2.0, PECL hwapi SVN)

hw_api::children — Devuelve los hijos de un objeto

Descripción

array hw_api::children ( array $parameter )

Devuelve los hijos de una colección o los atributos de un documento. Los hijos pueden ser filtrados posteriormente especificando un objeto de tipo query.

Parámetros

parameter

El parámetro es un array que contiene los elementos obligatorios 'objectIdentifier' y los elementos opcionales 'attributeSelector' y 'objectQuery'.

Valores devueltos

El valor devuelto es un array de objetos del tipo HW_API_Object o HW_API_Error.
