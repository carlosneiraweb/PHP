SplFixedArray::offsetSet

(PHP 5 >= 5.3.0, PHP 7)

SplFixedArray::offsetSet — Establece un nuevo valor para el índice específicado

Descripción

public void SplFixedArray::offsetSet ( int $index , mixed $newval )

Establece el valor en el index específicado en newval.

Parámetros

index

El índice a ser establecido.

newval

El nuevo valor para index.

Valores devueltos

No devuelve ningún valor.

Errores/Excepciones

Lanza una excepción de tipo RuntimeException cuando index está fuera del tamaño definido del array o cuando index no se puede procesar como un entero.
