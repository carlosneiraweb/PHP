SplFileObject::hasChildren

(PHP 5 >= 5.1.2, PHP 7)

SplFileObject::hasChildren — SplFileObject no tiene hijos

Descripción

public bool SplFileObject::hasChildren ( void )

Una clase SplFileObject no tiene hijos. Este método siempre devuelve FALSE.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve FALSE
