Imagick::getImageChannelMean

(PECL imagick 2.0.0)

Imagick::getImageChannelMean — Obtiene la media y la desviación estándar

Descripción

array Imagick::getImageChannelMean ( int $channel )

Obtiene la media y la desviación estándar de uno o más canales de imagen. El valor devuelto es una matriz asociativa con las claves "mean" (media) y "standardDeviation" (desviación estándar).

Parámetros

channel

Proporcione cualquier constante de canal que sea válida para su modo de canal. Para aplicar más de un canal, combine las constantes channeltype usando operadores a nivel de bits. Consulte esta lista de constantes de canal.

Valores devueltos

Devuelve TRUE en caso de éxito.

Errores/Excepciones

Lanza ImagickException en caso de error.
