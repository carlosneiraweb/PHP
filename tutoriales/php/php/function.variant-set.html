variant_set

(PHP 5, PHP 7)

variant_set — Asigna un nuevo valor para un objeto variante

Descripción

void variant_set ( variant $variant , mixed $value )

Convierte value a una variante y lo asigna al objeto variant; no se crean un nuevo objeto variante, y el valor antiguo de variant es liberado.

Parámetros

variant

La variante.

value

Valores devueltos

No devuelve ningún valor.
