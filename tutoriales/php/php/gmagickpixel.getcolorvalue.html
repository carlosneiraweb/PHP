GmagickPixel::getcolorvalue

(PECL gmagick >= Unknown)

GmagickPixel::getcolorvalue — Obtiene el valor normalizado del canal de color proporcionado

Descripción

public float GmagickPixel::getcolorvalue ( int $color )

Recupera el canal de color especificado, como un número de coma flotante entre 0 y 1.

Parámetros

color

El canal a comprobar, especificado como una de las constantes de canal de Gmagick.

Valores devueltos

El valor del canal, como un número de coma flotante normalizado, lanzando una excepción de tipo GmagickPixelException si se produjo un error.
