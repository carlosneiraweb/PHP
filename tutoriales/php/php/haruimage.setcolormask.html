HaruImage::setColorMask

(PECL haru >= 0.0.1)

HaruImage::setColorMask — Establece el color de mascara de la imagen

Descripción

bool HaruImage::setColorMask ( int $rmin , int $rmax , int $gmin , int $gmax , int $bmin , int $bmax )

Define el color trasnparente de la imagen usando los rangos de valores RGB. El colo dentro del rango is mostrado como un color transparente. El espacio de color de la imagen debe ser RGB.

Parámetros

rmin

El mas bajo limite de rojo. Debe estar entre 0 y 255.

rmax

El mas alto limite de rojo. Debe estar entre 0 y 255.

gmin

El mas bajo limite de verde. Debe estar entre 0 y 255.

gmax

El mas alto limite de verde. Debe estar entre 0 y 255.

bmin

El mas bajo limite de azul. Debe estar entre 0 y 255.

bmax

El mas alto limite de azul. Debe estar entre 0 y 255.

Valores devueltos

Devuelve TRUE en éxito.

Errores/Excepciones

Lanza una HaruException cuando se produce un error.
