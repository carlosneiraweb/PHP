Memcached::getVersion

(PECL memcached >= 0.1.5)

Memcached::getVersion — Obtener la información de la versión de la agrupación de servidores

Descripción

public array Memcached::getVersion ( void )

Memcached::getVersion() devuelve un array que contine la información de la versión de todos los servidores de memcache disponibles.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Array de las versiones de los servidores, una entrada por servidor.

Ejemplos

Ejemplo #1 Ejemplo de Memcached::getVersion()

<?php
$m = new Memcached();
$m->addServer('localhost', 11211);

print_r($m->getVersion());
?>

El resultado del ejemplo sería algo similar a:

Array
(
    [localhost:11211] => 1.2.6
)
