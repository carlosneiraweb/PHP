gmp_abs

(PHP 4 >= 4.0.4, PHP 5, PHP 7)

gmp_abs — Valor absoluto

Descripción

GMP gmp_abs ( GMP $a )

Obtiene el valor absoluto de un número.

Parámetros

a

resource de número de GMP en PHP 5.5 y anterior, o un objeto de GMP en PHP 5.6 y posterior, o un string numérico siempre que sea posible convertirlo a un número.

Valores devueltos

Devuelve el valor absoluto a, como un número GMP.

Ejemplos

Ejemplo #1 Ejemplo de gmp_abs()

<?php
$abs1 = gmp_abs("274982683358");
$abs2 = gmp_abs("-274982683358");

echo gmp_strval($abs1) . "\n";
echo gmp_strval($abs2) . "\n";
?>

El resultado del ejemplo sería:

274982683358
274982683358
