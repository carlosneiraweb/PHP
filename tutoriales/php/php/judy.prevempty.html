Judy::prevEmpty

(PECL judy >= 0.1.1)

Judy::prevEmpty — Busca el índice ausente anterior en el array Judy

Descripción

public int Judy::prevEmpty ( mixed $index )

Busca el índice ausente anterior (no incluido) que sea menor que el índice pasado.

Parámetros

index

El índice puede ser un integer o un string correspondiente al índice donde empezar la búsqueda.

Valores devueltos

Devuelve el correspondiente índice del array.
