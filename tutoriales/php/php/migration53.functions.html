Nuevas Funciones

PHP 5.3 añade algunas nuevas funciones:

Nuevo de PHP:

  • array_replace() - Remplaza los elementos de los arrays pasados en el primer array.
  • array_replace_recursive() - Reemplaza los elementos de los arrays pasados al primer array de forma recursiva.
  • class_alias() - Crea un alias para una clase.
  • forward_static_call() - Llamar a un método estático.
  • forward_static_call_array() - Llamar a un método estático y pasar los argumentos como matriz.
  • gc_collect_cycles() - Fuerza la recolección de un ciclo de basura.
  • gc_disable() - Desactiva la recolección de referencias circulares.
  • gc_enable() - Activa la recolección de referencias circulares.
  • gc_enabled() - Devuelve el estado de la recolección de referencias circulares.
  • get_called_class() - Obtiene el nombre de la clase desde la que se llama al método estático.
  • gethostname() - Obtiene el nombre de host de la máquina local.
  • header_remove() - Elimina las cabeceras HTTP asignadas anteriormente usando la función header().
  • lcfirst() - Convierte a minúscula el primer caracter de un string.
  • parse_ini_string() - Analiza un string de configuración.
  • quoted_printable_encode() - Convierte un string de 8 bits a un string a codificación QP.
  • str_getcsv() - Convierte una cadena CSV en un array.
  • stream_context_set_default() - Establece el flujo de contexto por omisión.
  • stream_supports_lock() - Indica si el flujo soporta bloqueo.
  • stream_context_get_params() - Recuperar los parámetros de un contexto.
  • streamWrapper::stream_cast() - Recuperar el recurso subyacente.
  • streamWrapper::stream_set_option() - Cambiar las opciones de flujo.

Fecha/Hora:

  • date_add() - Añade una cantidad de días, meses, años, horas, minutos y segundos a un objeto DateTime.
  • date_create_from_format() - Devuelve un nuevo objeto DateTime formateado según el formato especificado.
  • date_diff() - Devuelve la diferencia entre dos objetos DateTime.
  • date_get_last_errors() - Devuelve las advertencias y errores de la última operación de fecha/hora.
  • date_parse_from_format() - Devuelve información de una fecha.
  • date_sub() - Sustrae una cantidad de días, meses, años, horas, minutos y segundos de un objeto DateTime.
  • timezone_version_get() - Obtiene la versión de la base de datos timezonedb.

GMP:

  • gmp_testbit() - Comprueba si un bit está establecido.

Hash:

  • hash_copy() - Copia el contexto de Hash.

IMAP:

  • imap_gc() - Limpia la caché de IMAP.
  • imap_utf8_to_mutf7() - Codifica una cadena UTF-8 a UTF-7 modificado.
  • imap_mutf7_to_utf8() - Decodifica de UTF-7 modificado a una cadena UTF-8.

JSON:

  • json_last_error() - Obtiene el último error de JSON que ocurrió.

Extensión mejorada de MySQL:

  • mysqli_fetch_all() - Extrae todos los resultados en un array asociativo, en un array numérico, o en ambos.
  • mysqli_get_connection_stats() - Obtiene estadísticas sobre la conexión cliente.
  • mysqli_poll() - Apila conexiones.
  • mysqli_reap_async_query() - Obtiene resultados de una consulta asíncrona.

OpenSSL:

  • openssl_random_pseudo_bytes() - Devuelve una cadena de la longitud especificada, rellenada con bytes pseudo-aleatorios.

PCNTL:

  • pcntl_signal_dispatch() - Realiza una llamada a los manejadores de señales para las señales pendientes.
  • pcntl_sigprocmask() - Establece y obtiene las señales bloqueadas.
  • pcntl_sigtimedwait() - Realiza una espera a señales con un tiempo de expiración.
  • pcntl_sigwaitinfo() - Espera a una señal.

PCRE:

  • preg_filter() - Realiza búsquedas y reemplazos por expresiones regulares, devolviendo solo los resultados que coincidan con el patrón.

Semáforos:

  • msg_queue_exists() - Comprueba si existe una cola de mensajes o no.
  • shm_has_var() - Comprueba si existe o no una clave específica en un segmento de memoria compartida.

Las siguientes funciones están ahora disponibles de forma nativa, de modo que están disponibles en todos los sistemas operativos que puedan ejecutar PHP:

  • acosh()
  • asinh()
  • atanh()
  • expm1()
  • log1p()
