fbsql_connect

(PHP 4 >= 4.0.6, PHP 5, PHP 7)

fbsql_connect — Abre una conexión a un servidor FrontBase

Descripción

resource fbsql_connect ([ string $hostname = ini_get("fbsql.default_host") [, string $username = ini_get("fbsql.default_user") [, string $password = ini_get("fbsql.default_password") ]]] )

fbsql_connect() establece una conexión a un servidor FrontBase.

Si se hace una segunda llamada a fbsql_connect() con los mismos argumentos, no se establecerá un nuevo enlace, sino que se devolverá el identificador del enlace ya abierto en su lugar.

El enlace al servidor se cerrará tan pronto como acabe la ejecución del script, a no ser que se cierre antes explicitamente llamando fbsql_close().

Parámetros

hostname

El nombre de host del servidor.

username

El nombre de usuario para la conexión.

password

El password para la conexión.

Valores devueltos

Devuelve un identificador de enlace FrontBase positivo, o FALSE en caso de error.

Ejemplos

Ejemplo #1 Ejemplo de fbsql_connect()

<?php

$link = fbsql_connect("localhost", "_SYSTEM", "secret")
    or die("No se puede conectar");
echo "Conectado correctamente";
fbsql_close($link);

?>
