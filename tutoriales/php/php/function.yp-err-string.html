yp_err_string

(PHP 4 >= 4.0.6, PHP 5 <= 5.0.5)

yp_err_string — Devuelve la cadena de error asociada con el código de error dado

Descripción

string yp_err_string ( int $errorcode )

Devuelve el mensaje de error relacionado con el código de error determinado. Útil para indicar exactamente lo que salió mal.

Parámetros

errorcode

El código de error.

Valores devueltos

Devuelve el mensaje de error, como una cadena.

Ejemplos

Ejemplo #1 Ejemplo con yp_err_string()

<?php
echo "Error: " . yp_err_string(yp_errno());
?>
