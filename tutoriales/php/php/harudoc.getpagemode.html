HaruDoc::getPageMode

(PECL haru >= 0.0.1)

HaruDoc::getPageMode — Coloca al corriente el modo de la página

Descripción

int HaruDoc::getPageMode ( void )

Coloca al corriente el modo de la página. Ver HaruDoc::setPageMode() para la lista de los posibles valores.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve el modo de la página actualmente colocado en el documento. Ver HaruDoc::setPageMode() para la lista de los posibles valores.

Errores/Excepciones

Lanza una HaruException cuando se produce un error.
