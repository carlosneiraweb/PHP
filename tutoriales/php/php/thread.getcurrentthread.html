Thread::getCurrentThread

(PECL pthreads >= 2.0.0)

Thread::getCurrentThread — Identificación

Descripción

public static Thread Thread::getCurrentThread ( void )

Devuelve una referencia al hilo que está siendo ejecutado.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Un objeto representando la ejecución del hilo actual.

Ejemplos

Ejemplo #1 Devuelve el hilo que se está ejecutando

<?php
class My extends Thread {
    public function run() {
        var_dump(Thread::getCurrentThread());
    }
}
$my = new My();
$my->start();
?>

El resultado del ejemplo sería:

object(My)#2 (0) {
}
