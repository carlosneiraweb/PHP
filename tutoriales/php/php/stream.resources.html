Clases de Flujos

Las envolturas diseñadas por el usuario se pueden registrar mediante stream_wrapper_register(), usando la definición de la clase mostrada en esa página del manual.

La clase php_user_filter está predefinida y es una clase base abstracta para usarla con filtros definidos por el usuario. Véase la página del manual de stream_filter_register() para detalles sobre la implementación de filtros definidos por el usuario.
