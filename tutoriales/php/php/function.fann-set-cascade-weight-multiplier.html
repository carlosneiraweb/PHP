fann_set_cascade_weight_multiplier

(PECL fann >= 1.0.0)

fann_set_cascade_weight_multiplier — Establece el multiplicador de peso

Descripción

bool fann_set_cascade_weight_multiplier ( resource $ann , float $cascade_weight_multiplier )

Establece el multiplicador de peso.

Parámetros

ann

Un resource de red neuronal.

cascade_weight_multiplier

El multiplicador de peso.

Valores devueltos

Devuelve TRUE en caso de éxito, o FALSE de lo contrario.
