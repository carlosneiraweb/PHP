HaruDoc::setInfoDateAttr

(PECL haru >= 0.0.1)

HaruDoc::setInfoDateAttr — Establecer los atributos datetime info del documento

Descripción

bool HaruDoc::setInfoDateAttr ( int $type , int $year , int $month , int $day , int $hour , int $min , int $sec , string $ind , int $off_hour , int $off_min )

Establece atributos datetime info del documento.

Parámetros

type

El tipo de atributo. Puede ser uno de los siguientes:

  • HaruDoc::INFO_CREATION_DATE
  • HaruDoc::INFO_MOD_DATE

year

month

Entre 1 y 12.

day

Entre 1 y 31, 30, 29 o 28 (diferente para cada mes).

hour

Entre 0 y 23.

min

Entre 0 y 59.

sec

Entre 0 y 59.

ind

La zona horaria respecto a UTC, puede ser "", " ", "+", "-" y "Z".

off_hour

Si ind no es " " o "", los valores entre 0 y 23 son válidos. De lo contrario, este parámetro se ignora.

off_min

Si ind no es " " o "", los valores entre 0 y 59 son válidos. De lo contrario, este parámetro se ignora.

Valores devueltos

Devuelve TRUE si tiene éxito.

Errores/Excepciones

Lanza una HaruException cuando se produce un error.
