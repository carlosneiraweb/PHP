fdf_get_ap

(PHP 4 >= 4.3.0, PHP 5, PHP 7)

fdf_get_ap — Consultar la publicación de un campo

Descripción

bool fdf_get_ap ( resource $fdf_document , string $field , int $face , string $filename )

Consulta la publicación de un field (es decir, el valor de la clave /AP) y lo almacena en un fichero.

Parámetros

fdf_document

Manejador de documentos FDF, devuelto por fdf_create(), fdf_open() o por fdf_open_string().

field

face

Los valores posibles son FDFNormalAP, FDFRolloverAP o FDFDownAP.

filename

Nombre del fichero en el que se guardará la publicación.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.
