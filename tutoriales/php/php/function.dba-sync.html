dba_sync

(PHP 4, PHP 5, PHP 7)

dba_sync — Sincronizar base de datos

Descripción

bool dba_sync ( resource $handle )

dba_sync() sincroniza la base de datos. Esto probablemente desencadenará una escritura en disco, si es soportado.

Parámetros

handle

El gestor de base de datos, devuelto por dba_open() o dba_popen().

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.
