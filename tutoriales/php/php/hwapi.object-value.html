hw_api_object::value

(PHP 4, PHP 5 < 5.2.0, PECL hwapi SVN)

hw_api_object::value — Devuelve el valor de un atributo

Descripción

string hw_api_object::value ( string $name )

Devuelve el valor de un atributo.

Parámetros

name

El nombre del atributo.

Valores devueltos

Devuelve el calor de el atributo con el nombre dado o FALSE Si ocurre un error.
