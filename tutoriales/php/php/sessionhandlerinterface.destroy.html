SessionHandlerInterface::destroy

(PHP 5 >= 5.4.0, PHP 7)

SessionHandlerInterface::destroy — Destruir una sesión

Descripción

abstract public bool SessionHandlerInterface::destroy ( string $session_id )

Destruye una sesión. Llamado por session_regenerate_id() (con $destroy = TRUE), session_destroy() y cuando session_decode() falla.

Parámetros

session_id

El ID de sesión a ser destruido.

Valores devueltos

El valor devuelto (normalmente TRUE en caso de éxito, FALSE en caso de error). Observe que este valor es devuelto internamente a PHP para su procesamiento.
