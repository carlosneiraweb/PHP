Puesta en marcha

El complemento está imlementado como una extensión de PHP. Véanse también las instrucciones de instalación para instalar la extensión » PECL/mysqlnd_qc.

Se ha de compilar o configurar la extensión (mysqli, PDO_MYSQL, mysql) que se planee utilizar con soporte para la biblioteca mysqlnd. PECL/mysqlnd_qc es un complemento para la biblioteca mysqlnd. Para usar el complemento con cualquier extensión de MySQL para PHP existente (APIs), la extensión ha de utilizar la biblioteca mysqlnd.

Luego, se ha de cargar la extensión en PHP y activar el complemento en el fichero de configuración de PHP usando la directiva de configuración de PHP llamada mysqlnd_qc.enable_qc.

Ejemplo #1 Habilitar el complemento (php.ini)

mysqlnd_qc.enable_qc=1
