SplObserver::update

(PHP 5 >= 5.1.0, PHP 7)

SplObserver::update — Recibe actualizaciones de sujeto

Descripción

abstract public void SplObserver::update ( SplSubject $subject )

Este método es llamado cuando cualquier SplSubject se use para llamar llamadas adjuntas SplSubject::notify().

Advertencia

Esta función no está documentada actualmente, solamente se encuentra disponible la lista de parámetros.

Parámetros

subject

El objeto SplSubject notificando al observador de una actualización.

Valores devueltos

No devuelve ningún valor.
