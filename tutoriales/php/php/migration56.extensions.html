Otros cambios en extensiones

cURL

Se han eliminado varias constantes marcadas como obsoletas en la biblioteca cURL:

  • CURLOPT_CLOSEPOLICY
  • CURLCLOSEPOLICY_CALLBACK
  • CURLCLOSEPOLICY_LEAST_RECENTLY_USED
  • CURLCLOSEPOLICY_LEAST_TRAFFIC
  • CURLCLOSEPOLICY_OLDEST
  • CURLCLOSEPOLICY_SLOWEST

OCI8

  • Se ha añadido soporte para conjuntos de resultados implícitos para Oracle Database 12c mediante la nueva función oci_get_implicit_resultset().
  • El empleo de oci_execute($s, OCI_NO_AUTO_COMMIT) para una SELECT ya no inicia innecesariamente un ROLLBACK interno durante el cierre de la conexión.
  • Se han añadido las sondas DTrace controladas por la opción de configuración --enable-dtrace.
  • oci_internal_debug() ahora es una instrucción no-op.
  • Ha cambiado el formato de salida de phpinfo() para OCI8.

Zip

Se ha añadido una opción de configuración, --with-libzip, para emplear un sistema de instalación de libzip. Se requiere la versión 0.11 de libzip, y se recomienda la 0.11.2 o posterior.

MySQLi

Se ha añadido una nueva opción, mysqli.rollback_on_cached_plink, para revertir conexiones que han sido puestas de nuevo en la provisión de conexiones persistentes.
