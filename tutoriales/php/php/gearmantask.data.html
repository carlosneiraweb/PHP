GearmanTask::data

(PECL gearman >= 0.5.0)

GearmanTask::data — Obtiene los datos retornados para una tarea

Descripción

public string GearmanTask::data ( void )

Obtiene los datos retornados para una tarea por un trabajador.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Los datos serialziados, o FALSE en caso de no haberlos.
