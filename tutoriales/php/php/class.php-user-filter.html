La clase php_user_filter

(PHP 5, PHP 7)

Introducción

Los hijos de esta clase son pasados a stream_filter_register().

Sinopsis de la Clase

php_user_filter {
/* Propiedades */
public $filtername ;
public $params ;
/* Métodos */
public int filter ( resource $in , resource $out , int &$consumed , bool $closing )
public void onClose ( void )
public bool onCreate ( void )
}

Propiedades

filtername

El nombre del filtro registrado por stream_filter_append().

params

Tabla de contenidos

  • php_user_filter::filter — Llamado cuando se aplica un filtro
  • php_user_filter::onClose — Llamado cuando se cierra el filtro
  • php_user_filter::onCreate — Llamado cuando se crea el filtro
