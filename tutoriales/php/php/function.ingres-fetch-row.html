ingres_fetch_row

(PHP 4 >= 4.0.2, PHP 5 <= 5.0.5, PECL ingres >= 1.0.0)

ingres_fetch_row — Obtiene una fila de resultados en un array enumerado

Descripción

array ingres_fetch_row ( resource $result )

ingres_fetch_row() devuelve un array que se corresponde con la fila obtenida, o FALSE si no existen más filas. Cada columna del resultado se almacena en un índice del array, empezando en el índice 1.

Las llamadas subsiguientes a ingres_fetch_row() devolverán la siguientes fila del conjunto de resultados, o FALSE si no existieran más filas.

De manera predeterminada, los arrays creados por ingres_fetch_row() empiezan en la posición 1, y no en la 0 como en otras extensiones de SGBDs. La posición de inicio se puede ajustar a 0 usando el parámetro de configuración ingres.array_index_start.

Nota: Configuraciones relacionadas

Véanse también las directivas ingres.array_index_start, ingres.fetch_buffer_size y ingres.utf8 de la sección Configuración en tiempo de ejecución.

Parámetros

result

El identificador de resultados de la consulta

Valores devueltos

Devuelve un array que se corresponde con la fila obtenida, o FALSE si no existieran más filas

Ejemplos

Ejemplo #1 Obttener una fila de resultados en un array enumerado

<?php
$enlace = ingres_connect($base_datos, $usuario, $contraseña);

$resultado = ingres_query($enlace, "select * from table");
while ($fila = ingres_fetch_row($resultado)) {
    echo $fila[1];
    echo $fila[2];
}
?>
