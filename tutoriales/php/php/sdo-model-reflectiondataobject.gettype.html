SDO_Model_ReflectionDataObject::getType — Obtener el SDO_Model_Type del SDO_DataObject

Descripción

SDO_Model_Type SDO_Model_ReflectionDataObject::getType ( void )
Advertencia

Esta función ha sido declarada EXPERIMENTAL. Su funcionamiento, nombre y la documentación que le acompaña puede cambiar sin previo aviso en futuras versiones de PHP. Utilícela bajo su propia responsabilidad.

Devuelve el SDO_Model_Type del SDO_DataObject. El SDO_Model_Type contiene toda la información sobre el tipo del objeto de datos, como el URI del espacio de nombres, nombre de tipo, si es un tipo de datos primitivo, etc.

Parámetros

Ninguno.

Valores devueltos

Devuelve el SDO_Model_Type del SDO_DataObject.
