HaruPage::setHeight

(PECL haru >= 0.0.1)

HaruPage::setHeight — Establece el alto de la página

Descripción

bool HaruPage::setHeight ( float $height )

Define el alto de la página.

Parámetros

height

El alto definido para la página.

Valores devueltos

Devuelve TRUE en éxito.

Errores/Excepciones

Lanza una HaruException cuando se produce un error.
