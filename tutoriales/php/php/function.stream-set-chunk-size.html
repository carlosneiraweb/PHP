stream_set_chunk_size

(PHP 5 >= 5.4.0, PHP 7)

stream_set_chunk_size — Establecer el tamaño de trozo de flujo

Descripción

int stream_set_chunk_size ( resource $fp , int $chunk_size )

Estblece el tamaño de trozo de flujo.

Parámetros

fp

El flujo objetivo.

chunk_size

El nuevo tamaño de trozo deseado.

Valores devueltos

Devuelve el tamaño de trozo anterior en caso de éxito.

Devolverá FALSE si chunk_size es menor que 1 o mayor que PHP_INT_MAX.

Errores/Excepciones

Emitirá un error de nivel E_WARNING si chunk_size es menor que 1 o mayor que PHP_INT_MAX.
