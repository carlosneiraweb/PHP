filepro_fieldcount

(PHP 4, PHP 5 <= 5.1.6, PHP 7)

filepro_fieldcount — Encuentra cuantos campos hay en una base de datos filePro

Descripción

int filepro_fieldcount ( void )

Devuelve el número de campos (columnas) de la base de datos abierta filePro.

Valores devueltos

Devuelve el número de campos en la base de datos abierta filePro, o FALSE si ocurre un error.
