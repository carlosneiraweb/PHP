ifx_num_fields

(PHP 4, PHP <=5.2.0)

ifx_num_fields — Devolver el número de columnas de una consulta

Descripción

int ifx_num_fields ( resource $result_id )

Devuelve el número de columnas de una consulta ya preparada o ejecutada.

Parámetros

result_id

result_id es el identificador válido devuelto por ifx_query() o ifx_prepare() (sólo para consultas "SELECT").

Valores devueltos

Devuelve el número de columnas de una consulta asociada a result_id, o FALSE en caso de error.

Ejemplos

Ejemplo #1 Ejemplo de ifx_num_fields()

<?php
$conn_id = ifx_connect("db", "user", "password");
$res_id = ifx_query("select * from systables", $conn_id);
echo ifx_num_fields($res_id);
?>
