Introducción

Esta extensión proporciona acceso a la funcionalidad de mensajería y sistemas de encolado de mensajes, como la familia de productos IBM WebSphere MQSeries , de los scripts PHP. El interface está diseñado para que sea extremadamente simple realizar las tareas más comunes, como distribuir mensajes simples de texto a las colas, pero al mismo tiempo permite a los usuarios expertos realizar operaciones de mensajería más complejas. Para muchos usuarios las complejidades de configurar numerosas opciones pueden ser simplemente ignoradas.

La extensión SAM es una framework que proporciona un API muy simple que puede ser utilizado para acceder a muchos sistemas middleware de mensajería . Actualmente el paquete incluye soporte interno para el protocolo de mensajería MQTT (MQ Telemetry Transport) y soporte para los productos middleware de Mensajería y Encolado de mensajes de IBM. SAM ha sido diseñado para ser fácilmente extensible para soportar otros sistemas de mensajería y módulos de extensión, que pueden ser escritos en C o PHP.
