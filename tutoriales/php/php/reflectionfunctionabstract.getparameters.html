ReflectionFunctionAbstract::getParameters

(PHP 5, PHP 7)

ReflectionFunctionAbstract::getParameters — Obtiene los parámetros

Descripción

public array ReflectionFunctionAbstract::getParameters ( void )

Devuelve los parámetros en forma de array de ReflectionParameter.

Advertencia

Esta función no está documentada actualmente, solamente se encuentra disponible la lista de parámetros.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Parámetros, en forma de objetos ReflectionParameter.

Ver también

  • ReflectionFunctionAbstract::getNumberOfParameters() - Obtiene el número total de parámetros
  • func_get_args() - Devuelve un array que se compone de una lista de argumentos de función
