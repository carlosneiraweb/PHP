event_base_loopbreak

(PECL libevent >= 0.0.1)

event_base_loopbreak — Aborta un bucle de eventos

Descripción

bool event_base_loopbreak ( resource $event_base )

Aborta inmediatamente el bucle de eventos activo. El comportamiento es similar a la sentencia break .

Parámetros

event_base

Base de eventos válida.

Valores devueltos

event_base_loopbreak() devuelve TRUE en caso de éxito o FALSE en caso de error.
