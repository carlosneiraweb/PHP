trader_stoch

(PECL trader >= 0.2.0)

trader_stoch — Estocástico

Descripción

array trader_stoch ( array $high , array $low , array $close [, integer $fastK_Period [, integer $slowK_Period [, integer $slowK_MAType [, integer $slowD_Period [, integer $slowD_MAType ]]]]] )

Parámetros

high

Precio alto, array de valores reales.

low

Precio bajo, array de valores reales.

close

Precio de cierre, array de valores reales.

fastK_Period

Periodo de tiempo para la construcción de la línea K-Rápida. Rango válido desde 1 hasta 100000.

slowK_Period

Suavizado para la realización de la línea K-Lenta. Rango válido desde 1 hasta 100000, normalmente establecido a 3.

slowK_MAType

Tipo de media móvil para K-lento. Se debería usar la serie de constantes TRADER_MA_TYPE_*.

slowD_Period

Suavizado para la realización de la línea D-Lenta. Rango válido desde 1 hasta 100000.

slowD_MAType

Tipo de media móvil para D-lento. Se debería usar la serie de constantes TRADER_MA_TYPE_*.

Valores devueltos

Devuelve un array con los datos calculados o false en caso de error.
