floatval

(PHP 4 >= 4.2.0, PHP 5, PHP 7)

floatval — Obtener el valor flotante de una variable

Descripción

float floatval ( mixed $var )

Obtiene el valor valor float de var.

Parámetros

var

Puede ser cualquier tipo escalar. No se puede utilizar floatval() en objetos, si es usado emitirá un error de nivel E_NOTICE y devolverá 1.

Valores devueltos

El valor flotante de la variable dada. Arrays vacios devolverán 0, arrays no vacios devolverán 1.

Las cadenas generalmente devolverán 0, aunque esto depende de los primeros caracteres de la cadena. Las reglas comunes de conversión a flotante se aplican en cualquier caso.

Ejemplos

Ejemplo #1 Ejemplo de floatval()

<?php
 $var = '122.34343El';
 $valor_float_de_var = floatval($var);
 echo $valor_float_de_var; // 122.34343
 ?>

Ejemplo #2 Ejemplo de floatval() con caracteres a la izquierda no numéricos

<?php
$var = 'El122.34343';
$float_value_of_var = floatval($var);
echo $float_value_of_var; // 0
?>
