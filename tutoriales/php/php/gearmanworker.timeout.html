GearmanWorker::timeout

(PECL gearman >= 0.6.0)

GearmanWorker::timeout — Retorna el tiempo máximo de espera para actividad de entrada/salida del socket

Descripción

public int GearmanWorker::timeout ( void )

Retorna el tiempo actual a esperar, en milisegundos, para actividad de entrada/salida del socket.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Un período de tiempo en milisegundos. Un valor negativo indica un tiempo de espera infinito.
