IntlChar::getIntPropertyMinValue

(PHP 7)

IntlChar::getIntPropertyMinValue — Obtener el valor mínimo de una propiedad de Unicode

Descripción

public static int IntlChar::getIntPropertyMinValue ( int $property )

Obtiene el valor mínimo de una propiedad de Unicode enumerada/entera/binaria.

Parámetros

property

La propiedad de Unicode a consultar (véanse las constantes IntlChar::PROPERTY_*).

Valores devueltos

El valor máximo devuelto por IntlChar::getIntPropertyValue() para una propiedad de Unicode. 0 si el selector de la propiedad está fuera de rango.

Ejemplos

Ejemplo #1 Probar diferentes propiedades

<?php
var_dump(IntlChar::getIntPropertyMinValue(IntlChar::PROPERTY_BIDI_CLASS));
var_dump(IntlChar::getIntPropertyMinValue(IntlChar::PROPERTY_SCRIPT));
var_dump(IntlChar::getIntPropertyMinValue(IntlChar::PROPERTY_IDEOGRAPHIC));
var_dump(IntlChar::getIntPropertyMinValue(999999999)); // Un valor inventado
?>

El resultado del ejemplo sería:

int(0)
int(0)
int(0)
int(0)
