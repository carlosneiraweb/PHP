HaruDoc::output

(PECL haru >= 0.0.1)

HaruDoc::output — Escribe los datos del documento a la salida del buffer

Descripción

bool HaruDoc::output ( void )

Escribe los datos del documento dentro de una salida estándar.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve TRUE en caso de éxito.

Errores/Excepciones

Lanza una HaruException cuando se produce un error.
