ifx_error

(PHP 4, PHP <=5.2.0)

ifx_error — Devolver el código del error de la última llamada a Informix

Descripción

string ifx_error ([ resource $link_identifier ] )

Devuelve como cadena un caracter que describe de manera general el resultado de una sentencia así como los códigos SQLSTATE y SQLCODE asociados con la última sentencia SQL ejecutada.

Parámetros

link_identifier

Identificador del enlace.

Valores devueltos

Los códigos de error de Informix (SQLSTATE & SQLCODE) en un formato x [SQLSTATE = aa bbb SQLCODE=cccc].

donde x = espacio : no hay error

E : error

N : no hay más datos

W : advertencia

? : no definido

Si el caracter "x" no es un espacio, los códigos SQLSTATE y SQLCODE describen el error con mayor detalle.

Consulte el manual de Informix para ver las descripciones de los códigos SQLSTATE y SQLCODE
