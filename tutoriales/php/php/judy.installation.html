Instalación

Esta extensión » PECL no se distribuye con PHP.

Se puede encontrar información para la instalación de esta extensión PECL en el capítulo del manual titulado Instalación de extensiones PECL. Se puede encontrar información adicional, tal como nuevos lanzamientos, descargas, ficheros fuente, información de mantenimiento, y un CHANGELOG, aquí: » http://pecl.php.net/package/judy

La última PECL/judy Win32 DLL está disponible aquí: » php_judy.dll

Instalación en Sistemas Linux

Desde los fuentes

Descargar e instalar » libJudy, entonces: 

     phpize
     ./configure --with-judy[=DIR]
     make
     make test
     make install

Ubuntu/Debian

libJudy puede ser instalado con apt-get : 

     apt-get install libjudydebian1 libjudy-dev
Luego la instalación de la extensión de Judy PECL o de las fuentes.

Instalación en Sistemas Windows

Descargar » libJudy, luego extraer las fuentes y abrir el símbolo de comandos de Visual Studio y navegue hasta el directorio fuente. Luego ejecute : 

build
Esto creará "Judy.lib", copie esto en la carpeta de librerias de php-sdk y nómbrela como "libJudy.lib" Luego copie el archivo incluido "judy.h" dentro de la carpeta includes del php-sdk.

A continuación, la extensión Judy PHP puede ser instalada desde PECL o desde los fuentes extrayendo el pecl/judy dentro de la carpeta de contrucción donde los scripts estarán disponibles para ser tomados, e.g.: 

    C:\php\pecl\judy\
Si su código fuente de PHP está ubicado en: 
    C:\php\src\
El resto de los pasos son bastante simples, como cualquier otra extensión: 
    buildconf
    configure --with-judy=shared
    nmake

Instalación en Mac OS X

Descargar e instalar » libJudy. Luego instale la extensión Judy desde PECL o desde sus fuentes.
