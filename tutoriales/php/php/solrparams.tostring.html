SolrParams::toString

(PECL solr >= 0.9.2)

SolrParams::toString — Devuelve todos los parámetros de los pares nombre-valor del objeto

Descripción

final public string SolrParams::toString ([ bool $url_encode = false ] )

Devuelve todos los parámetros de los pares nombre-valor del objeto.

Parámetros

url_encode

Si devolver o no valores de URL codificados

Valores devueltos

Devuelve una cadena en caso de éxito y FALSE en caso de fallo.
