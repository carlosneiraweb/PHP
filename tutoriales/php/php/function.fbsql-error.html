fbsql_error

(PHP 4 >= 4.0.6, PHP 5, PHP 7)

fbsql_error — Devuelve el mensaje de error de la última operación

Descripción

string fbsql_error ([ resource $link_identifier ] )

Devuelve el mensaje de error de la última operación FrontBase.

Los errores procedentes el gestor de base de datos fbsql no emiten warnings. En su lugar, use fbsql_error() para obtener el mensaje de error. Recuerde que esta función solo devuelve el mensaje de error generado por la función fbsql ejecutada más recientemente (sin incluir fbsql_error() y fbsql_errno()), así que para usarla hay que asegurarse de comprobar el valor antes de ejecutar otra función de fbsql.

Parámetros

link_identifier

Un identificador de enlace FrontBase devuelto por fbsql_connect() o fbsql_pconnect().

Es opcional y si no se especifica, la función intentará encontrar un enlace abierto al servidor FrontBase y de no encontrarse ningún enlace abierto intentará crear uno como si fbsql_connect() fuese invocada sin parámetros.

Valores devueltos

Returns the error text from the last fbsql function, or '' (the empty string) if no error occurred.

Ejemplos

Ejemplo #1 Ejemplo de fbsql_error()

<?php
fbsql_connect("marliesle");
echo fbsql_errno() . ": " . fbsql_error() . "<br />";
fbsql_select_db("nonexistentdb");
echo fbsql_errno() . ": " . fbsql_error() . "<br />";
$conn = fbsql_query("SELECT * FROM nonexistenttable;");
echo fbsql_errno() . ": " . fbsql_error() . "<br />";
?>
