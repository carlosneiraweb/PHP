Cambios en el módulo de SAPI

  • Ahora está disponible un nuevo módulo de SAPI llamado cli-server.
  • Se ha añadido la opción CLI --rz que muestra información de la llamada extensión Zend.
  • Se ha añadido el atajo #ajusteini=valor para cambiar configuraciones de php.ini en tiempo de ejecución para readline de CLI interactivo.
  • Se han añadido funciones compatibles con apache: apache_child_terminate(), getallheaders(), apache_request_headers() y apache_response_headers() para la SAPI FastCGI.
  • PHP-FPM: Se ha añadido el ajuste process.max para controlar el número de procesos que FPM puede bifurcar.
