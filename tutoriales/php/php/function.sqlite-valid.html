sqlite_valid

SQLiteResult::valid

SQLiteUnbuffered::valid

(PHP 5 < 5.4.0)

sqlite_valid -- SQLiteResult::valid -- SQLiteUnbuffered::valid — Devuelve si están disponibles más filas

Descripción

bool sqlite_valid ( resource $result )

Estilo orientado a objetos (método):

bool SQLiteResult::valid ( void )
bool SQLiteUnbuffered::valid ( void )

Busaca si hay más filas disponibles en el gestor de resultados dado.

Parámetros

result

El recurso de resultados de SQLite. Este parámetro no es necesario al usar el método orientado a objetos.

Nota:

Esta función no puede ser utilizada con manejos de resultados no pasados por buffer.

Valores devueltos

Devuelve TRUE si hay más filas disponibles en el gestor de resultados dado por result, o FALSE si no.

Ver también

  • sqlite_num_rows() - Devolver el número de filas de un conjunto de resultados almacenado en buffer
  • sqlite_changes() - Devuelve el número de filas que han sido cambiadas mediante la sentencia SQL más reciente.
