yp_errno

(PHP 4 >= 4.0.6, PHP 5 <= 5.0.5)

yp_errno — Devuelve el código de error de la operación más reciente

Descripción

int yp_errno ( void )

Devuelve el código de error de la operación más reciente.

Valores devueltos

Devuelve una de las constantes de error YPERR_XXX.
