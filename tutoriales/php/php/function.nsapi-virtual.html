nsapi_virtual

(PHP 4 >= 4.3.3, PHP 5, PHP 7)

nsapi_virtual — Realiza una sub-petición NSAPI

Descripción

bool nsapi_virtual ( string $uri )

nsapi_virtual() es una función específica NSAPI la cual es equivalete a <!--#include virtual...--> en SSI (archivos .shtml). Esto realiza una sub-petición NSAPI. Es útil para incluir en scripts CGI o archivos .shtml, o cualquier otro que desee analizar a través del servidor web.

Para ejecutar una sub-petición, todos los buffers son finalizados y arrojados al bavegador, las cabeceras pendientes son enviadas también.

No se puede hacer peticiones recursivas con esta función a otro script PHP. Si se desea incluir a un script PHP, use include o require.

Nota:

Esta función depende de una característica no documentada de los servidores Netscape/iPlanet/Sun. Use phpinfo() para determinar si está disponible. En un entorno Unix debería siempre funcionar, en Windows depende del nombre de un archivo ns-httpdXX.dll.

Lea la nota sobre las sub-peticiones en la sección NSAPI (UNIX, Windows) si experimenta este problema.

Parámetros

uri

La URI del script.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.
