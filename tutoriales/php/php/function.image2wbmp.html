image2wbmp

(PHP 4 >= 4.0.5, PHP 5, PHP 7)

image2wbmp — Exportar la imagen al navegador o a un fichero

Descripción

bool image2wbmp ( resource $image [, string $filename [, int $threshold ]] )

image2wbmp() imprime o guarda una versión WBMP de image.

Parámetros

image

Un recurso image, es devuelto por una de las funciones de creación de imágenes, como imagecreatetruecolor().

filename

Ruta del fichero guardado. Si no se porporciona, el flujo de imagen sin tratar será impreso directamente.

threshold

El valor del umbral, entre 0 y 255 (inclusive).

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.

Ejemplos

Ejemplo #1 Ejemplo de image2wbmp()

<?php
$fichero = 'php.png';
$imagen = imagecreatefrompng($fichero);

header('Content-Type: ' . image_type_to_mime_type(IMAGETYPE_WBMP));
image2wbmp($imagen); // imprime el flujo directamente
imagedestroy($imagen);
?>

Notas

Nota:

El soporte para WBMP está disponible sólamente si PHP se compiló con GD-1.8 o posterior.
