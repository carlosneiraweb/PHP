fann_set_cascade_max_out_epochs

(PECL fann >= 1.0.0)

fann_set_cascade_max_out_epochs — Establece el máximo de épocas de salida

Descripción

bool fann_set_cascade_max_out_epochs ( resource $ann , int $cascade_max_out_epochs )

Establece el máximo de épocas de salida.

Parámetros

ann

Un resource de red neuronal.

cascade_max_out_epochs

El máximo de épocas de salida.

Valores devueltos

Devuelve TRUE en caso de éxito, o FALSE de lo contrario.
