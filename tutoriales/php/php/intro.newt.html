Introducción

Esta es una extensión del lenguaje PHP para la biblioteca RedHat Newt, una biblioteca de ventana basada en terminal y en widget para escribir aplicaciones con interfaz de usuario amigable. Una vez que esta extensión está activada en PHP, ofrecerá el uso de widgets Newt, tales como ventanas, botones, casillas de verificación, botones de radio, etiquetas, cajas de texto, barras de desplazamiento, áreas de texto, escalas, etc. El uso de esta extensión es muy similar a la API de Newt original del lenguaje de programación C.
