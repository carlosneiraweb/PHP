Imagick::__toString — Devuelve la imagen como un string

Descripción

string Imagick::__toString ( void )

Devuelve la imagen actual como un string. Solamente devolverá una única imagen; no debería usarse con objetos Imagick que contengan varias imágenges, p.ej., un GIF animado o un PDF con varias páginas.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve el contenido del string en caso de éxito o un string vacío en caso de fallo.

Ver también

  • Imagick::getImageBlob() - Devuelve la secuencia de imágenes como un valor de tipo blob
  • Imagick::getImagesBlob() - Devuelve todas las secuencias de imágenes como un blob
