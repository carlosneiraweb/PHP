zip_entry_filesize

(PHP 4 >= 4.1.0, PHP 5 >= 5.2.0, PHP 7, PECL zip >= 1.0.0)

zip_entry_filesize — Devuelve el tamaño del fichero actual de una entrada de directorio

Descripción

int zip_entry_filesize ( resource $zip_entry )

Devuelve el tamaño actual de una entrada de directorio específica.

Parámetros

zip_entry

Una entrada de directorio devuelva por zip_read().

Valores devueltos

El tamaño de una entrada de directorio.

Ver también

  • zip_open() - Abre un fichero ZIP
  • zip_read() - Leer la siguiente entrada en el fichero ZIP
