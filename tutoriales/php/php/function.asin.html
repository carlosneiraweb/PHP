asin

(PHP 4, PHP 5, PHP 7)

asin — Arco seno

Descripción

float asin ( float $arg )

Devuelve el arco seno de arg en radianes. asin() es la función complementaria de sin(), lo que quiere decir que a==sin(asin(a)) para cada valor de a en el rango de asin().

Parámetros

arg

El argumento a procesar

Valores devueltos

El arco seno de arg en radianes
