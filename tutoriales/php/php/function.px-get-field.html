px_get_field

(PECL paradox >= 1.0.0)

px_get_field — Devuelve la especificación de un solo campo

Descripción

array px_get_field ( resource $pxdoc , int $fieldno )

Parámetros

pxdoc

Identificador de recursos de la base de datos paradox devuelto por la función px_new().

fieldno

Número del campo. El primer campo tiene el número 0. Especificando un número de campo menor que 0 y mayor o igual al número de campos provocará un error trigger.

Valores devueltos

Devuelve la específicación del fieldno ésimo campo de la base de datos como un array asociado. La matriz contiene tres campos denominados name, type, y size.
