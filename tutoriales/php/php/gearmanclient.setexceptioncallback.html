GearmanClient::setExceptionCallback

(PECL gearman >= 0.5.0)

GearmanClient::setExceptionCallback — Establece una llamada de retorno para excepciones del trabajador

Descripción

public bool GearmanClient::setExceptionCallback ( callable $callback )

Especifica una función a llamar cuando un trabajador envía una excepción.

Parámetros

callback

Función a llamar cuando un trabajador lanza una excepción

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.
