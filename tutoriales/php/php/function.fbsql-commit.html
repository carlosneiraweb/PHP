fbsql_commit

(PHP 4 >= 4.0.6, PHP 5, PHP 7)

fbsql_commit — Ejecuta un commit de la transacción sobre la base de datos

Descripción

bool fbsql_commit ([ resource $link_identifier ] )

Finaliza la transaccion actual escribiendo todos los INSERTs, UPDATEs y DELETEs en el disco y desbloqueando todos los bloqueos de fila y tabla mantenidos por la transacción. Este comando sólo se necesita si el autocommit está asignado a false.

Parámetros

link_identifier

Un identificador de enlace FrontBase devuelto por fbsql_connect() o fbsql_pconnect().

Es opcional y si no se especifica, la función intentará encontrar un enlace abierto al servidor FrontBase y de no encontrarse ningún enlace abierto intentará crear uno como si fbsql_connect() fuese invocada sin parámetros.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.

Ver también

  • fbsql_autocommit() - Activa o desactiva el autocommit
  • fbsql_rollback() - Retrocede una transacción a la base de datos
