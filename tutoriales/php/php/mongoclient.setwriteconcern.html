MongoClient::setWriteConcern

(PECL mongo >=1.5.0)

MongoClient::setWriteConcern — Establecer el asunto de escritura para esta conexión

Esta extenisón que define este método está obsoleta. En su lugar debe usarse la extensión MongoDB. No hay equivalente para este método en la nueva extensión.

Descripción

public bool MongoClient::setWriteConcern ( mixed $w [, int $wtimeout ] )

Parámetros

w

El asunto de escritura. Podría ser un valor de tipo integer denotando el número de servidores requeridos para reconocer la escritura, o un string con el modo (p.ej. "majority").

wtimeout

El número máximo de milisegundos que tienen que esperar es servidor para satisfacer el asunto de escritura.

Valores devueltos

Devuelve TRUE en caso de éxito, o FALSE en caso contrario.

Errores/Excepciones

Emite un E_WARNING si el parámetro w no es un valor de tipo integer o string.

Ejemplos

Ejemplo #1 Ejemplo de MongoClient::setWriteConcern()

<?php

$mc = new MongoClient('mongodb://rs1.example.com,rs2.example.com');

// Requiere que la mayoría de los servidores en el conjunto de réplica reconozcan escrituras
// en un intervalo de tiempo de tres segundos.
$mc->setWriteConcern('majority', 3000);
?>
