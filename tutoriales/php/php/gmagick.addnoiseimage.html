Gmagick::addnoiseimage

(PECL gmagick >= Unknown)

Gmagick::addnoiseimage — Añade ruido aleatorio a la imagen

Descripción

public Gmagick Gmagick::addnoiseimage ( int $NOISE )

Añade ruido aleatorio a la imagen.

Parámetros

noise_type

El tipo de ruido. Consulte esta lista de constantes de ruido.

Valores devueltos

El objeto Gmagick con el ruido añadido.

Errores/Excepciones

Arroja un GmagickException en caso de error.
