EventBase::reInit

(PECL event >= 1.2.6-beta)

EventBase::reInit — Reinicializa el evento base (después de un 'fork')

Descripción

public bool EventBase::reInit ( void )

Reinicializa el evento base. Debe ser llamado después de un 'fork'.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve TRUE en caso de éxito. FALSE en cualquier otro caso.
