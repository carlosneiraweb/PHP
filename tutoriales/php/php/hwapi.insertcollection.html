hw_api::insertcollection

(PHP 4, PHP 5 < 5.2.0, PECL hwapi SVN)

hw_api::insertcollection — Inserta un objeto nuevo de tipo 'collection'

Descripción

hw_api_object hw_api::insertcollection ( array $parameter )

Esta función es una abreviatura de hwapi_insert(). Inserta un objeto de tipo 'collection' y configura algunos de los atributos obligatorios para un objeto de este tipo.

Parámetros

parameter

El parámetro es un array que contiene los elementos obligatorios 'object' y 'parentIdentifier' y los elementos opcionales 'parameter' y 'attributeSelector'. Ver hwapi_insert() para conocer el significado de cada elemento.
