SphinxClient::runQueries

(PECL sphinx >= 0.1.0)

SphinxClient::runQueries — Ejecutar un lote de consultas de búsqueda

Descripción

public array SphinxClient::runQueries ( void )

Se conecta a searchd, ejecuta un lote de todas las consultas se agregan utilizando SphinxClient::addQuery, obtiene y devuelve el conjunto de resultados.

Valores devueltos

Devuelve FALSE en caso de error y array de conjuntos de resultados en caso de éxito.
