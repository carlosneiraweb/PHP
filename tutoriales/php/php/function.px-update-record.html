px_update_record

(PECL paradox >= 1.4.0)

px_update_record — Actualizar el registro en la base de datos paradox

Descripción

bool px_update_record ( resource $pxdoc , array $data , int $num )

Actualiza un registro existente en la base de datos. El registro comienza en 0.

Los datos del registro se pasan como una matriz de valores de campo. Los elementos de la matriz deben corresponder a los campos de la base de datos. Si la matriz tiene menos elementos que los campos en la base de datos, los campos restantes se establecerán a null.

Nota:

Esta función sólo está disponible si se utiliza pxlib >= 0.6.0.

Parámetros

pxdoc

Resource identifier of the paradox database as returned by px_new().

data

Identificador de recursos de la base de datos paradox devuelto por la función px_new().

num

El registro numérico es un número artificial que cuenta los registros a medida en que se almacenan en la base de datos. El primer registro tiene el número 0.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.
