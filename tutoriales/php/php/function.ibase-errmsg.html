ibase_errmsg

(PHP 5, PHP 7)

ibase_errmsg — Devuelve un mensaje de error

Descripción

string ibase_errmsg ( void )

Devuelve el mensaje de error que hubo en la última llamada a una función de InterBase.

Valores devueltos

Devuelve el mensaje de error como string, o FALSE si no hubiera ningún error.
