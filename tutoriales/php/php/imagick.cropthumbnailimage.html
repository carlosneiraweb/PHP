Imagick::cropThumbnailImage

(PECL imagick 2.0.0)

Imagick::cropThumbnailImage — Crea una miniatura recortada

Descripción

bool Imagick::cropThumbnailImage ( int $width , int $height )

Crea una miniatura de tamaño fijo ampliando o reduciendo de escala la imagen y recortando un área específica desde el centro.

Parámetros

width

El ancho de la miniatura

height

El alto de la miniatura

Valores devueltos

Devuelve TRUE en caso de éxito.

Errores/Excepciones

Lanza ImagickException en caso de error.
