ReflectionMethod::isAbstract

(PHP 5, PHP 7)

ReflectionMethod::isAbstract — Comprueba si el método es abstracto

Descripción

public bool ReflectionMethod::isAbstract ( void )

Comprueba si el método es abstracto.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

TRUE si el método es abstracto, o si no FALSE
