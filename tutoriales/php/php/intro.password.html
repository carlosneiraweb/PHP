Introducción

La API de hash de contraseñas proporciona una envoltura fácil de usar sobre crypt() para hacerala sencilla para crear y administrar contraseñas de una forma segura.

Esta extensión está disponible desde PHP 5.5.0, aunque también existe una » implementación del espacio de usuario para PHP >= 5.3.7.
