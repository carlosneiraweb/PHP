XMLWriter::startCData

xmlwriter_start_cdata

(PHP 5 >= 5.1.2, PHP 7, PECL xmlwriter >= 0.1.0)

XMLWriter::startCData -- xmlwriter_start_cdata — Crea la etiqueta de inicio de CDATA

Descripción

Estilo orientado a objetos

bool XMLWriter::startCData ( void )

Estilo por procedimientos

bool xmlwriter_start_cdata ( resource $xmlwriter )

Inicia un CDATA.

Parámetros

xmlwriter

Sólo para llamadas por procedimientos. El resource XMLWriter que está siendo modificado. Este recurso proviene de una llamada a xmlwriter_open_uri() o xmlwriter_open_memory().

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.

Ver también

  • XMLWriter::endCData() - Fin del actual CDATA
  • XMLWriter::writeCData() - Escribe una etiqueta completa del CDATA
