Introducción

El monitor de archivos y directorio FAM, notifica a las aplicaciones interesadas de cambios. Más información sobre FAM está disponible en » http://oss.sgi.com/projects/fam/.

En un script PHP se puede especificar una lista de archivos para ser monitorizados por FAM usando las funciones provistas por esta extensión.

El proceso FAM se inicia cuando la primera conexión de cualquier aplicación se establece. Termina cuando la conexión también finaliza.

Nota:

Esta extensión se ha movido al repositorio de » PECL y ya no se distribuye con PHP a partir de PHP 5.1.0.

Nota: Esta extensión no está disponible en plataformas Windows.
