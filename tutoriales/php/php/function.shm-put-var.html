shm_put_var

(PHP 4, PHP 5, PHP 7)

shm_put_var — Inserta o actualiza una variable en la memoria compartida

Descripción

bool shm_put_var ( resource $shm_identifier , int $variable_key , mixed $variable )

shm_put_var() inserta o actualiza variable con la clave cada por variable_key.

Se emitirán advertencias (nivel E_WARNING) si shm_identifier no es un índice de memoria compartida SysV válido o si no hay suficiente memoria compartida restante para completar la petición.

Parámetros

shm_identifier

Un gestor de recurso de memoria compartida devuelto por shm_attach()

variable_key

La clave de la variable.

variable

La variable. Todos los tipos de variables que soporta serialize() se pueden usar: generalmente esto significa todos los tipos excepto los recursos y algunos objetos internos que no pueden ser serialzados.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.
