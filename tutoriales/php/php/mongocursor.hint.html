MongoCursor::hint

(PECL mongo >=0.9.0)

MongoCursor::hint — Da a la base de datos un indicio sobre la consulta

Descripción

public MongoCursor MongoCursor::hint ( mixed $index )

Parámetros

index

El índice a usar para la consulta. Si se proporciona un string, debería corresponderse con un nombre de índice. Si se proporciona un array u objeto, debería corresponderse con la especificación usada para crear el índice (esto es, el primer argumento de MongoCollection::ensureIndex()).

Valores devueltos

Devuelve este cursor.

Errores/Excepciones

Lanza una MongoCursorException si este cursor ha empezado a iterar.

Historial de cambios

Versión Descripción
1.4.0

El argumento index ahora admite nombres de índices como valores de tipo string. En versiones anteriores a la 1.4.0, solamente eran aceptados valores de array u objeto.
