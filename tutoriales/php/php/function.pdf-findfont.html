PDF_findfont

(PHP 4 >= 4.0.5, PECL pdflib >= 1.0.0)

PDF_findfont — Preparar una fuente para un uso posterior [obsoleta]

Descripción

int PDF_findfont ( resource $p , string $fontname , string $encoding , int $embed )

Busca una fuente y la prepara para un uso posterior con PDF_setfont(). La métrica será cargada y si el párametro embed no es cero, el archivo de fuente será comprobado, pero no usado todavía. encoding es builtin, macroman, winansi, host, un nombre de codificación definido por el usuario o el nombre de un CMap. El parámetro embed es opcional antes de PHP 4.3.5 o con PDFlib anterior a 5.

Esta función está obsoleta desde la versión 5 de PDFlib, use PDF_load_font() en su lugar.
