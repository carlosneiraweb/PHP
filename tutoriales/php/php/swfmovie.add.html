SWFMovie::add

(PHP 5 <= 5.3.0, PECL ming SVN)

SWFMovie::add — Añade cualquier tipo de información a un película

Descripción

mixed SWFMovie::add ( object $instance )
Advertencia

Esta función ha sido declarada EXPERIMENTAL. Su funcionamiento, nombre y la documentación que le acompaña puede cambiar sin previo aviso en futuras versiones de PHP. Utilícela bajo su propia responsabilidad.

Añade una instancia de un objeto SWF dada por instance a la película actual.

Parámetros

instance

Cualquier tipo de instancia de objeto, como SWFShape, SWFText, SWFFont.

Valores devueltos

Para tipos mostrables (forma, texto, botón, sprite), devuelve una instancia de SWFDisplayItem, un gestor a un objeto en una lista de muestras. Por lo tanto, se puede añadir a una película la misma forma múltiples veces y obtener gestores por serpado para cada instancia.

Ver también

  • SWFMovie::remove() - Elimina la instancia del objeto de la lista de muestras
