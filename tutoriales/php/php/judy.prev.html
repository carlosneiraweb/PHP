Judy::prev

(PECL judy >= 0.1.1)

Judy::prev — Busca el índice anterior en el array Judy

Descripción

public mixed Judy::prev ( mixed $index )

Búsqueda (global) del índice anterior presente que es menor que el índice pasado.

Parámetros

index

El índice puede ser un integer o un string que corresponde al indicé donde empieza la búsqueda.

Valores devueltos

Devielve el índice correspondiente en el array
