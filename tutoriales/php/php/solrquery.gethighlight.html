SolrQuery::getHighlight

(PECL solr >= 0.9.2)

SolrQuery::getHighlight — Devuelve el estado del parámtero hl

Descripción

public bool SolrQuery::getHighlight ( void )

Devuelve un booleano que indica si habilitar o no la generación de trozos remarcados en la respuesta de la consulta.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve un booleano en caso de éxito y NULL si no se estableció
