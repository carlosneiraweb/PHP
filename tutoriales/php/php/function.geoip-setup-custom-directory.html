geoip_setup_custom_directory

(PECL geoip >= 1.1.0)

geoip_setup_custom_directory — Establecer un directorio personalizado para la base de datos de GeoIP

Descripción

void geoip_setup_custom_directory ( string $path )

La función geoip_setup_custom_directory() cambiará el directorio personalizado de la base de datos de GeoIP. Esto equivale a cambiar geoip.custom_directory.

Parámetros

path

La ruta completa del disco donde se encuentra la base de datos de GeoIP.

Valores devueltos

No devuelve ningún valor.

Ejemplos

Ejemplo #1 Un ejemplo de geoip_setup_custom_directory()

Esto cambiará la ruta de la base de datos predeterminada de GeoIP.

<?php

geoip_setup_custom_directory('/alguna/otra/ruta');

print geoip_db_filename(GEOIP_COUNTRY_EDITION);

?>

El resultado del ejemplo sería:

/some/other/path/GeoIP.dat
