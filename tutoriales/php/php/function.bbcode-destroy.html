bbcode_destroy

(PECL bbcode >= 0.9.0)

bbcode_destroy — Cierra un recurso BBCode_container

Descripción

bool bbcode_destroy ( resource $bbcode_container )

Esta función cierra el recurso abierto con bbcode_create().

Parámetros

bbcode_container

Recurso BBCode_Container devuelto por bbcode_create().

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.
