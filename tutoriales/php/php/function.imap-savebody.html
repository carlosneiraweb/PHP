imap_savebody

(PHP 5 >= 5.1.3, PHP 7)

imap_savebody — Guardar una sección del cuerpo específica en un fichero

Descripción

bool imap_savebody ( resource $imap_stream , mixed $file , int $msg_number [, string $part_number = "" [, int $options = 0 ]] )

Guarda una parte del cuerpo completo de un mensaje especificado.

Parámetros

imap_stream

IMAP stream devuelto por imap_open().

file

La ruta al fichero guardado como una cadena, o un descriptor de fichero válido devuelto por fopen().

msg_number

El número de mensaje

part_number

El número de parte. Es una cadena de enteros delimitada por puntos que indexa a una lista de partes de cuerpos como por la especificación IMAP4

options

Una máscara de bits de uno o más de lo siguiente:

  • FT_UID - msg_number es un UID
  • FT_PEEK - No establecer la bandera \Seen si no lo está ya
  • FT_INTERNAL - La cadena devuelta está en formato intermo, no canonizará a CRLF.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.
