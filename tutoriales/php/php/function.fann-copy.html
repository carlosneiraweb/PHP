fann_copy

(PECL fann >= 1.0.0)

fann_copy — Crea una copia de una estructura fann

Descripción

resource fann_copy ( resource $ann )

Crea una copia de una estructura fann.

Parámetros

ann

Un resource de red neuronal.

Valores devueltos

Devuelve una copia de un recurso de red neuronal en caso de éxito, o FALSE en caso de error.
