XSLTProcessor::importStylesheet

(PHP 5, PHP 7)

XSLTProcessor::importStylesheet — Importa una hoja de estilos

Descripción

void XSLTProcessor::importStylesheet ( object $stylesheet )

Este método importa la hoja de estilos a XSLTProcessor para realizar transformaciones.

Parámetros

stylesheet

La hoja de estilos importada como objeto DOMDocument o SimpleXMLElement.

Valores devueltos

No devuelve ningún valor.

Historial de cambios

Versión Descripción
5.2.8 Acepta SimpleXMLElement de nuevo; no funcionaba desde PHP 5.2.6.
