SplFileInfo::getPathInfo

(PHP 5 >= 5.1.2, PHP 7)

SplFileInfo::getPathInfo — Obtiene un objeto SplFileInfo para la ruta

Descripción

public SplFileInfo SplFileInfo::getPathInfo ([ string $class_name ] )

Obtiene un objeto SplFileInfo para el padre de el fichero actual.

Parámetros

class_name

Nombre de una clase SplFileInfo derivada a usar.

Valores devueltos

Devuelve un objeto SplFileInfo para la ruta padre de el fichero.

Ejemplos

Ejemplo #1 Ejemplo de SplFileInfo::getPathInfo()

<?php
$info = new SplFileInfo('/usr/bin/php');
$parent_info = $info->getPathInfo();
var_dump($parent_info->getRealPath());
?>

El resultado del ejemplo sería algo similar a:

string(8) "/usr/bin"
