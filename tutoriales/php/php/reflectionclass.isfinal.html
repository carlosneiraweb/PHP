ReflectionClass::isFinal

(PHP 5, PHP 7)

ReflectionClass::isFinal — Comprueba si la clase inal

Descripción

public bool ReflectionClass::isFinal ( void )

Comprueba si la clase es final.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.

Ejemplos

Ejemplo #1 Ejemplo de ReflectionClass::isFinal()

<?php
class       ClaseDePrueba { }
final class ClaseFinalDePrueba { }

$claseNormal = new ReflectionClass('ClaseDePrueba');
$claseFinal  = new ReflectionClass('ClaseFinalDePrueba');

var_dump($claseNormal->isFinal());
var_dump($claseFinal->isFinal());

?>

El resultado del ejemplo sería:

bool(false)
bool(true)
