HaruDoc::setPageLayout

(PECL haru >= 0.0.1)

HaruDoc::setPageLayout — Indica cómo deben mostrarse las páginas

Descripción

bool HaruDoc::setPageLayout ( int $layout )

Indica cómo se deben mostrar las páginas.

Parámetros

layout

Lista de valores aceptados:

  • HaruDoc::PAGE_LAYOUT_SINGLE - sólo se muestra una página.
  • HaruDoc::PAGE_LAYOUT_ONE_COLUMN - muestra las páginas en una columna.
  • HaruDoc::PAGE_LAYOUT_TWO_COLUMN_LEFT - muestra las páginas en dos columnas, la primera en la columna izquierda.
  • HaruDoc::PAGE_LAYOUT_TWO_COLUMN_RIGHT - muestra las páginas en dos columnas, la primera en la columna derecha.

Valores devueltos

En caso de éxito devuelve TRUE.

Errores/Excepciones

Lanza una HaruException cuando se produce un error.
