PDF_get_pdi_value

(PHP 4 >= 4.0.5, PECL pdflib >= 1.0.0)

PDF_get_pdi_value — Obtener un parámetro númerico de un PDI [obsoleta]

Descripción

float PDF_get_pdi_value ( resource $p , string $key , int $doc , int $page , int $reserved )

Obtiene el contenido de un parámetro de un documento PDI de tipo numérico.

Esta función está obsoleta desde la versión 7 de PDFlib, use PDF_pcos_get_number() en su lugar.
