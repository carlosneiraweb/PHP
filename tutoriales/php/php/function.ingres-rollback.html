ingres_rollback

(PHP 4 >= 4.0.2, PHP 5 <= 5.0.5, PECL ingres >= 1.0.0)

ingres_rollback — Revierte una transacción

Descripción

bool ingres_rollback ( resource $link )

ingres_rollback() revierte la transacción abierta actualmente, realmente cancelando todos los cambios hechos a la base de datos durante dicha transacción.

Esta función cierra la transacción. Se puede abrir una nueva transacción enviando una consulta con ingres_query().

Parámetros

link

El identificador de enlace de la conexión

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.

Ver también

  • ingres_query() - Envía una consulta SQL a Ingres
  • ingres_commit() - Consigna una trasacción
  • ingres_autocommit() - Activa o desactiva la autoconsigna
