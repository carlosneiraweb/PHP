OAuthProvider::callconsumerHandler — Llama al callback consumerNonceHandler

Descripción

public void OAuthProvider::callconsumerHandler ( void )

Llama al callback del manejador del consumidor registrado, la cual es establecida con OAuthProvider::consumerHandler().

Advertencia

Esta función no está documentada actualmente, solamente se encuentra disponible la lista de parámetros.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

No devuelve ningún valor.

Errores/Excepciones

Muestra un error E_ERROR level si la llamada de retorno no puede ser llamada, o no se especificó.
