yp_get_default_domain

(PHP 4, PHP 5 <= 5.0.5)

yp_get_default_domain — Obtiene el dominio NIS por defecto

Descripción

string yp_get_default_domain ( void )

Devuelve el dominio predeterminado del nodo. Puede ser usado como parámetro de dominio en posteriores llamados a NIS.

Un dominio NIS puede ser descrito como un grupo de mapas NIS. Cada cliente que necesita buscar información se enlaza a un dominio determinado. Refiérase a los documentos mencionados al principio para obtener más información.

Valores devueltos

Devuelve el dominio predeterminado del nodo o FALSE. Puede ser usado como parámetro de dominio para posteriores llamados a NIS.

Ejemplos

Ejemplo #1 Ejemplo para un dominio predeterminado

<?php
$domain = yp_get_default_domain();
echo "Dominio NIS predeterminado es: " . $domain;
?>
