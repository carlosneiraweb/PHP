La clase Directorio

(No hay información de versión disponible, podría estar únicamente en SVN)

Introducción

Las instancias de Directory se crean mediante una llamada a la funcion dir() , no por el operador new.

Sinopsis de la Clase

Directory {
/* Propiedades */
public string $path ;
public resource $handle ;
/* Métodos */
public void close ([ resource $dir_handle ] )
public string read ([ resource $dir_handle ] )
public void rewind ([ resource $dir_handle ] )
}

Propiedades

path

El directorio que se abrió.

handle

Se puede usar junto con otras funciones de directorio, como readdir(), rewinddir() y closedir().

Tabla de contenidos

  • Directory::close — Cierra un gestor de directorio
  • Directory::read — Lee una entrada desde el controlador de directorio
  • Directory::rewind — Rebobina el manejador de directorio
