ibase_blob_cancel

(PHP 5, PHP 7)

ibase_blob_cancel — Cancela la creación de un blob

Descripción

bool ibase_blob_cancel ( resource $blob_handle )

Esta función descarta un BLOB, siempre y cuando no se haya cerrado ya mediante ibase_blob_close().

Parámetros

blob_handle

Manejador de BLOB, abierto con ibase_blob_create().

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error.

Ver también

  • ibase_blob_close() - Cierra un blob
  • ibase_blob_create() - Crea un nuevo blob al que añadir datos
  • ibase_blob_import() - Crea un blob, copia un fichero en él, y lo cierra
