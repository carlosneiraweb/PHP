imap_fetchmime

(PHP 5 >= 5.3.6, PHP 7)

imap_fetchmime — Obtener las cabeceras MIME para una sección en particular del mensaje

Descripción

string imap_fetchmime ( resource $imap_stream , int $msg_number , string $section [, int $options = 0 ] )

Obtener las cabeceras MIME de una sección en particular del cuerpo de los mensajes especificados.

Parámetros

imap_stream

IMAP stream devuelto por imap_open().

msg_number

El número de mensaje

section

El número de parte. Es una cadnea de enteros delimitados por un punto que indexa dentro de una lista de partes del cuerpo como en la especificación IMAP4

options

Una máscara de bits con uno o más de lo siguiente:

  • FT_UID - msg_number es un UID
  • FT_PEEK - No establecer la bandera \Seen si no está ya establecida
  • FT_INTERNAL - La cadena devuelta está en formato interno, no se canonizará a CRLF.

Valores devueltos

Devuelve las cabeceras MIME de una sección en particular del cuerpo de los mensajes especificados como una cadena de texto.
