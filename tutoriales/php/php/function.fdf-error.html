fdf_error

(PHP 4 >= 4.3.0, PHP 5, PHP 7)

fdf_error — Consultar la descripción de un código de error FDF

Descripción

string fdf_error ([ int $error_code = -1 ] )

Devuelve una descripción textual del código de error FDF de error_code.

Parámetros

error_code

Código de error obtenido con fdf_errno(). Si se omite, se utilizará el código de error interno que estableció la última operación.

Valores devueltos

Devuelve el mensaje de error en forma de string, o el texto no error si no hubo ningún error.
