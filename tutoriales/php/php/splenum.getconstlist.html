SplEnum::getConstList

(PECL spl_types >= 0.1.0)

SplEnum::getConstList — Devuelve todas las constantes (o valores posibles) como un array

Descripción

public array SplEnum::getConstList ([ bool $include_default = false ] )

Advertencia

Esta función no está documentada actualmente, solamente se encuentra disponible la lista de parámetros.

Parámetros

include_default

Ya sea para incluir una propiedad __default.

Valores devueltos

Ejemplos

Ejemplo #1 Ejemplo de SplEnum::getConstList()

<?php
$bool = new SplBool;
var_dump($bool->getConstList(true));
?>

El resultado del ejemplo sería:

array(3) {
  ["__default"]=>
  bool(false)
  ["false"]=>
  bool(false)
  ["true"]=>
  bool(true)
}
