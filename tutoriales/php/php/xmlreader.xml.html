XMLReader::XML

(PHP 5 >= 5.1.0, PHP 7)

XMLReader::XML — Establece el contenido del XML a analizar

Descripción

public bool XMLReader::xml ( string $source [, string $encoding [, int $options = 0 ]] )

Establece el contenido del XML a analizar.

Parámetros

source

La cadena que contiene el XML a ser analizado.

encoding

La codificación del documento o NULL.

options

Una mascara bit de la constante LIBXML_*.

Valores devueltos

Devuelve TRUE en caso de éxito o FALSE en caso de error. Si es llamado de forma estática, devuelve un XMLReader o FALSE en caso de error.

Errores/Excepciones

Este método puede ser llamado de forma estática, pero lanzará un error E_STRICT.

Historial de cambios

5.2.0 encoding ay options fueron agregados.
